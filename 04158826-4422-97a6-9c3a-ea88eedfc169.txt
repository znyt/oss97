　　可他却不清楚，刘哔比他更不适合在镇上露面。
<br/>　　“走快点，我赶时间。”看到任山有些磨磨蹭蹭的样子，刘哔皱了皱眉，开口督促道。
<br/>　　现在已经是中午了，他问过任山那个村子的具体位置，如果靠现在的脚程，估计天黑之前也赶不回去了。
<br/>　　毕竟是去鬼修的老巢，难免会有一些古怪的东西，白天总要比夜晚要安全一些。
<br/>　　听到刘哔的督促，任山也不敢怠慢，当下提升了些许速度，两人疾驰在土黄色的道路上。
<br/>　　紧赶慢赶之下，就在太阳即将落山的时候，两人终于来到了罗家村的附近。
<br/>　　夕阳很快沉入了地平线，四周的环境也开始变得昏暗起来，刘哔观察了一下四周，发现周围的环境很是荒凉。
<br/>　　或许是因为太长时间没有人经过，这条本来就接近于荒废的道路上长满了杂草，随着两人的经过，惊起了偏偏藏身于草中的秋虫。
<br/>　　不知为何，刘哔突然觉得气氛有些诡异，内心略微升起了一丝不安。
<br/>　　走了大致十分钟的时间，任山突然停下，指着前方道：“我们到了。”
<br/>　　顺着任山手指的方向看去，荒地中出现了一个村落，在昏暗的天色衬托下，略显破败。
<br/>　　这个村落并不大，在夜色中只能看见一点轮廓，里面也没有灯光亮起，如果不认识路的话，很难让人发现它的存在。
<br/>　　看着眼前并不明显的村庄，刘哔皱起了眉头，脸上的神情反而变得更加警惕。
<br/>　　他闻到了一股熟悉的味道，和尸体腐烂掉的臭味相同，这绝对不是一个普通的村子应该有的味道。
<br/>　　将手指放在嘴边，示意任山不要发出声音，刘哔抓着任山的脖领，慢慢来到了罗家村的入口。
<br/>　　村子里空无一人，仅有的二十几户房屋都是紧闭着大门，几条残破的白布散落在地上，在夜色的衬托下有些阴森。
<br/>　　四周都是十分安静，连虫鸣的声音都没有，整个村子，如同一个巨大的坟墓一般。
<br/>　　在这种氛围之下，刘哔连自己的呼吸声都清晰可闻，随着靠近，他感觉到空气中的腐臭味更加浓重了。
<br/>　　“那个鬼修的藏身地点，在哪一间？”压低了声音，刘哔伏在任山的耳旁，悄悄的问道。
<br/>　　似乎是想到了什么，任山脸色有些惨白，伸手指了指中央位置的一间宅院，嘴唇不住的哆嗦着。
<br/>　　看到任山的这幅模样，刘哔更加确认这个村庄有古怪了。
<br/>　　之前他询问过这里的情况，但任山的回答却是他也不知道，只是听说路过这里的人从来就没有活着出去过的。
<br/>　　但他现在已经发现问题所在了，任山绝对是骗他的，否则又怎么会因为一个传言就吓成这样。
<br/>　　更不会知道鬼修老头的具体居住位置。
<br/>　　注意到任山踌躇不前的动作，刘哔心中提高了警惕，伸手抓过任山的脖颈，小心翼翼的向村内走去。
<br/>　　村头距离鬼修老头的宅院并不远，很快两人就深入到了这里，指了指眼前有些老旧的木门，刘哔示意任山将门打开。
<br/>　　吞了一口唾液，任山不敢反抗，将手扣在木门上，轻轻的推开。
<br/>　　院子里有些简陋，只有寥寥几件物事，房屋也不是很多，看上去反倒简洁。
<br/>　　“任山，你怎么来了，这个人是谁？”就在刘哔仔细观察四周情况的时候，一个面相清秀的少年从屋中走了出来，惊讶开口。
<br/>　　与此同时，连环任务也提示更新了，刘哔进入系统查看，发现接下来的任务，是获得鬼修积攒下来的修炼资源。
<br/>　　“哦，我来找鹰先生，之前货物的运送出了问题。”感受到脖颈后面的锋芒，任山担心自身的安危，只能被迫给刘哔打起了掩护。
<br/>　　轻笑了一下，刘哔用一缕灵气缠绕住任山的脖子，随后放下左手，走上了前去。
<br/>　　“我是任家的护卫队长，初次见面，请多指教。”抱了抱拳，刘哔笑盈盈的看着少年。
<br/>　　他在这个少年身上，感受到了一股压力，对方应该也是一个不弱于他的修炼者。
<br/>　　现在局势有些不太明朗，不知道少年的身份，刘哔暂时不想发起冲突。
<br/>　　“恩，任家却是该雇个护卫队了，免得每次运货都成问题。”嘀咕了一声，少年还了一个礼：“我是鬼鹰先生的徒弟‘豺’，师父出门去了，现在不在家里，要不你们在这里住一晚，等师父回来了再说。”
<br/>　　昨天的时候师父就和他交代过，要出去一趟，如果不出什么意外，现在这个时间也快回来了。
<br/>　　至于要出去做什么，师父没和他说，他也没问，他从来不参与这些事情。
<br/>　　听到要在此地留宿，任山想要拒绝，但脖子上的丝线猛然一紧，无奈改口道：“好，那我们就恭敬不如从命了。”
<br/>　　点了点头，豺伸手指了指一旁的偏房，道：“你们就住那间屋子吧，有什么事情再叫我、”
<br/>　　说罢，便再也没有理会两人，少年转身走进了屋子，随后关闭了房间大门。
<br/>　　饱含深意的看了一眼紧闭的屋门，刘哔带着任山，往少年指派的偏房走去。
<br/>　　这间屋子并不大，但是里面的家具都挺齐全，炕上的被褥都是崭新的，看样子是专门用来招待客人的客房。
<br/>　　房间的中央，还有一个老式的火炉，旁边还放着不少柴火，炉子上面放着一个水壶，可以用来烧水。
<br/>　　虽然还是有些简陋，但是这里的条件，比起义庄来说可就好太多了，刘哔心中倒也还算满意。
<br/>　　随手将外套脱在一边，刘哔支使任山去烧一壶热水，虽然变成丧尸后不会口渴，但他还是保留下来了喝水的习惯。
<br/>　　说起来奇怪，他试过咖啡，茶叶，糖水等许多饮品，但都让他产生排斥，目前为止，也只有白水才能入口。
<br/>　　遗憾的是，白水并没有什么味道，之所以喝，也是他想要尽量保持曾经属于人类的一些习惯罢了。
<br/>　　虽然他心中清楚，很难再重新成为人类了，但他始终觉得，总要该去做点什么。
