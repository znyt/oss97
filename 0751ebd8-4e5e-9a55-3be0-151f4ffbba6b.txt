　　血海深处，冥河道人紧紧地看着眼前不断挣扎的冤魂。
<br/>　　自近日以来，血海之中便会经常出现一些冤魂，这些冤魂之中，有人族的，有妖族的，有散仙……各种种族都有。他们也不这是为何，仿佛是一股神秘的力量将他们牵引至此。
<br/>　　冤魂厉鬼们见到血海不由得大喜，因为血海中散发出来的气息让他们觉得十分亲切，这激发了他们内心的渴望，这是一种潜意识中对母亲怀抱的渴望。
<br/>　　于是，这些漂泊许久，意识已经渐渐模糊的魂魄们，懵懵懂懂地来到血海之地，见到犹如亲人一般的血海，毫不犹豫地便投了进去。
<br/>　　若是平时冥河见到这些魂魄，会觉得十分不耐烦，因为这些冤魂在血海之中太吵了，而今日见到他们，冥河只觉得自己好似找到了一个新的方向。
<br/>　　他心内深处有一种感觉，凭借这些冤魂，他能够创出一个新的种族。
<br/>　　想到就做，就这样，冥河道人投入到对冤魂的研究之中。
<br/>　　不知过了多久，冥河眼中的明悟之色渐渐增多了。他盘坐在十二品业火红莲之上，手中拿着一团精血，这是血海之中淬炼出来的精血。
<br/>　　他手中不知从何处抓来一道魂魄，他手捏法诀，一指点在魂魄之上，一道精血精气被他注入到那魂魄之中，之后再以他对造化之道的领悟创新创造新魂，道韵环绕之中，在冥河造化道的影响下，那魂魄宛若从虚幻之中，缓缓化为真实，凝聚出了肉身。
<br/>　　同时，冥河凭借着自己对血杀之道的领悟，他还在这新魂之内灌注了许多杀性，让他一旦出生，便能够为他不停的杀戮！
<br/>　　用了九九八十一日，冥河才创造出第一个生灵。在那生灵被创出来的一刻，这幽冥血海被一股无边无尽的道韵笼罩，天道之音，扶摇万里长空！
<br/>　　血海翻滚，波浪咆哮，造化之光通天彻地，那被冥河创造出来的第一个生灵，出现在洪荒天地间之时，天道有感，雷霆轰鸣，于那九天之上，风云倒卷，无边功德在血海之上凝聚。
<br/>　　“轰隆隆……”
<br/>　　血海深处，如此大的动静，有如此大的功德凝聚，顿时惊动了洪荒，被无数大能察觉，大量高手神念横空，破开大地，来到血海深处，想要探查究竟。
<br/>　　“冥河到底是在干什？莫非是在创造什么大神通？不对！嘶！那是……那不是妖族，也不是人族、巫族，更不是洪荒之中的任何生灵！这……难道说这冥河道人，在效仿女娲，要创造出一个新的种族，因此才有了如此多功德？”
<br/>　　“天降功德，说明此事乃顺天而行，不可阻拦，否则定遭天谴……冥河难道是想要效仿女娲，创造新的一族，借此成圣？！嘶！这……这不可能！”
<br/>　　无数大神通者们，待查清了冥河的目的之后，脸色纷纷狂变。若是真让冥河借此成圣了，那后果简直不堪设想。
<br/>　　“拜见老祖！叩谢老祖造化之恩！”
<br/>　　那第一个被冥河创造出来的生灵，眼中闪过惊天杀意，仿佛可以杀尽世界一切敌人。唯独面对冥河时，露出敬畏之情。
<br/>　　冥河淡淡的看了他一眼，这生灵并不是先天道体，实力也只有人仙左右，但是他们可以修炼。
<br/>　　不再理会其他，冥河仿佛找到了方向，他继续创造生灵。
<br/>　　慢慢地，岁月流转，数百年时光就此过去了。幽冥血海之上，道韵缭绕，造化玄光闪耀，通天彻地，越发的宏大缥缈，笼罩了整个血海。
<br/>　　随着一个又一个的生灵被创造出来，无尽的功德在血海上空汇聚，凝聚成磅礴之势，让越来越多的洪荒大能，嫉妒不已！
<br/>　　这些被冥河创造出来的生灵形态各不相同，男的面目极为狰狞丑陋，有的手生八臂，有的肋骨生翅，有的身穿皮甲，有的口露獠牙……女的却又相貌美丽，婀娜多姿！
<br/>　　越到最后，这些生灵的修为便越高，到后面甚是可以堪比天仙了。这些修为在洪荒之中可能不算什么，无比渺小，但别忘了，这是冥河刚刚创造出来的种族，潜力非凡，以后修炼，修为可以不断提高。最重要的是，他们灵魂深处有一种嗜杀的本性，他们极好杀戮，仿佛就是为了杀戮而生，与万劫魔尊的劫魔有得一拼。
<br/>　　“这……这怎么可能？冥河所创造出来的生灵，越来越强了，洪荒以后又要再多出一个顶尖势力了。”
<br/>　　大神通者们震惊了，以前冥河只是孤家寡人，虽然修为强横，有无数血神子，倒也不算什么，但现在他的背后却站着一个种族，若是他一旦建立势力，便会是洪荒顶尖的势力。
<br/>　　“拜见老祖！”
<br/>　　无数生灵密密麻麻地跪拜在冥河面前，在他们的记忆深处，是冥河创造了他们，他们的血脉源自冥河。
<br/>　　冥河闭目不语，好似在参悟什么，周身道韵流转，血海沸腾，良久，他才睁开眼睛，看着被自己创造出来的种族，对着修为最高的那一个说道大：
<br/>　　“从今以后，你为阿修罗王，吾赐汝名为……帝释天！”
<br/>　　“多谢老祖赐名，我为帝释天！吼！”帝释天向着冥河恭敬一拜，之后跪地仰天大吼，身上战意冲宵，杀气惊世！
<br/>　　“天道在上，贫道冥河，今日创造一族，此族好战嗜杀，名为……阿修罗！”冥河不理会其他，而是抬头仰天，大声开口道。
<br/>　　“吼！阿修罗！阿修罗！……”
<br/>　　无数阿修罗仰天大吼，吼声连片，让苍穹色变，风云倒卷！
<br/>　　九天之上，汇聚了无数的功德，在此刻一瞬间降落下来，直至落入冥河身上。冥河脚踏十二品红莲，抬头仰天，血发飞舞，衣衫飘扬，一股无形中的威势凌天而起，扩散到整片血海。
<br/>　　他全身的气势在这一刻暴涨而起，在这血海深处扩散开来，覆灭苍穹，他的境界在无限拔高，瞬间便突破准圣中期，斩去自己的执尸，成为准圣后期，最后缓缓停在后期巅峰。
<br/>　　冥河不停，继续仰天大声喊道：
<br/>　　“吾，血海冥河，今日以血海为域，以阿修罗为众，以十二品红莲镇压气运，立幽冥教！阿修罗，战天、战地、战人、战仙、战佛、战神、战魔、杀天、杀地、杀人、杀神、杀仙、杀佛、杀魔、杀圣！天地间无物不杀！杀！杀！杀！杀……”
