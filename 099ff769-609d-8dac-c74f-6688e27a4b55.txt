　　“还真是够雷厉风行的，当兵的都这样吧，说一就一说二就二，不过到底他们调查到了什么重要的线索，这么急。”
<br/>　　林峰摇了摇头目光落在了那些死掉的毒蜂尸体上，这群毒蜂尸体都被烧没了，只剩下一颗颗大概指甲盖大小的晶石。
<br/>　　林峰自然不会错过这些晶石，正要去收集的时候。
<br/>　　刷。
<br/>　　一根弩箭落在了林峰身边。
<br/>　　“小子，这里没你事了，这些晶石归我们了。”
<br/>　　一道嚣张的声音在林峰背后响起。
<br/>　　林峰转过身就看到是一伙人。
<br/>　　这伙人来人不少，足足二十多人，看样子是在外面猎杀丧尸的。

<br/>　　“这是基地内的哪个大势力。”
<br/>　　林峰扫了对方一眼，能够拿得出来这种实力的，肯定不简单。
<br/>　　战士会是肯定不可能的，看来应该是地下组织的势力。
<br/>　　“这些怪物都是我杀的，为什么晶石要归你们？”
<br/>　　林峰说道。
<br/>　　“你杀的又如何？你能证明这些是你杀的吗？”
<br/>　　“哈哈，小子，识相点的速度滚蛋，不然别怪我们骷髅会手下无情！！”
<br/>　　“没错，我们骷髅会要的东西，你还敢不给？？”
<br/>　　几个汉子戏谑的看着林峰。
<br/>　　说着其中一个人居然还掏出了一把手枪在那里转来转去，一副看上去十分牛逼的样子。
<br/>　　“看到没小子，枪！人类的大杀器！你小子识相的，还不麻利点滚蛋？”
<br/>　　其中领头的六阶高手笑呵呵道。
<br/>　　“副会长，这种垃圾，还需要用枪？”
<br/>　　一人呵呵一笑。
<br/>　　“也是哦，这种垃圾，估计顶多三四阶，用刀就可以砍死的玩意。”
<br/>　　几个汉子压根没有把林峰放在眼里。
<br/>　　“这么嚣张？”
<br/>　　林峰皱起了眉头，骷髅会他是知道的，地下势力之中第二的存在，很厉害，会长是个八阶的。
<br/>　　“就是这么嚣张！”
<br/>　　骷髅会的人哈哈大笑。
<br/>　　“不服就来砍我啊，小子。”
<br/>　　“爸爸站在这里让你砍，你要是砍的过爸爸，这些晶石爸爸让给你。”
<br/>　　“是吗？”
<br/>　　林峰冷笑了一下然后就放出了自己的气势。
<br/>　　顿时那群人都傻了眼了。
<br/>　　“七阶的？”
<br/>　　“握草，这家伙是七阶高手？”
<br/>　　“怎么可能。”
<br/>　　一群人顿时震惊了。
<br/>　　“七阶又如何，我们手里有家伙，他就算是七阶的，人没我们多，武器没我们好，他能把我们怎么办？”
<br/>　　骷髅会的副会长回过神来就是掏出了手枪对准了林峰。
<br/>　　“小子，这件事情算我们看走眼了，这些晶石全归你，和我们没有关系，走！“
<br/>　　副会长大手一挥就是要撤走。
<br/>　　“慢着，你们这话就很搞笑了，什么叫晶石全归我，这本来就是我的，你们之前不是很牛逼的吗？在我面前逞英雄，还自称爸爸，你是谁的爸爸？”
<br/>　　林峰冷笑道。
<br/>　　顿时一群骷髅会的人都不敢说话了，之前无比嚣张，现在一个个支支吾吾话都不敢说一句。
<br/>　　“这件事情是我们错了。我们骷髅会是地下第二大组织，得饶人处且饶人，大家各退一步。如何？”
<br/>　　骷髅会副会长皱着眉头道。
<br/>　　“地下第二大组织又如何？今天若不是我实力不怎么样的话，这群晶石岂不是要被你们给抢走了？所以，既然敢站出来打这个劫，那么就做好死的准备。”
<br/>　　林峰冷笑道。
<br/>　　林峰的为人处世，就是人不犯我，我不犯人，人若犯我，斩草除根。
<br/>　　“你真的要鱼死网破？就算你是七阶高手，我们这么多人，还有这么多武器，你一个人，可不见得能够讨得好处？”
<br/>　　骷髅会副会长威胁道。

<br/>　　林峰不屑道。
<br/>　　“什么意思？”
<br/>　　骷髅会副会长一脸不解。
<br/>　　不过下一刻，他就什么都明白了。
<br/>　　林峰把那个火箭筒扛在了肩上瞄准了他们。
<br/>　　“这才叫武器，知道没有！你们那玩意，是给小孩子玩的！”

<br/>　　“日！”
<br/>　　骷髅会众人都是下意识的后腿了好几部满脸的骇然之色。
<br/>　　现阶段，别说七阶了就是八阶九阶的人类，被这玩意来一下，那都得粉身碎骨，毕竟人是血肉之躯，火箭筒多大的威力。根本没有什么可比性。

<br/>　　“你，你到底想怎么样。”
<br/>　　顿时他认怂了。

<br/>　　林峰冷笑道。

<br/>　　骷髅会副会长顿时面色难看，他堂堂副会长，六阶的高手，居然要跪下给一个学生喊爸爸。

<br/>　　“爸爸！”
<br/>　　噗咚噗咚。
