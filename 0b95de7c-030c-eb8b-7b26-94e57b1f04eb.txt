<br/>				吴老回家以后，就去了医院做了详细的检查。虽然他感觉不错，身体已经没有以前那种疲乏甚至有些喘不上气不上起来的感觉，而是浑身的舒畅和有力，可以说好像年轻了十岁一般。<br/>　　<br/>　　走路也好，说话也好，都非常的中气十足不说，另外就是吃的好睡的香，基本上在陈默哪里住的时间段中，是最近十几年来最舒服的日子了。<br/>　　<br/>　　但是家人不知道也不敢肯定啊！虽然看着不错，而且很有精神，但是也不能放心！所以就到医院做了最彻底的检查。<br/>　　<br/>　　这一检查就让医院的医生都感觉很惊讶，包括李院长在内。因为从各种迹象表明，吴老的心脏已经好的多了。最后不相信的情况下，做了造影检测后，发现血管堵塞已经降到了百分之50左右。<br/>　　<br/>　　这还是陈默不敢用药的后果，否则绝对是变成疏通百分百。就这都让李院长惊讶了半天，而且吴老的家人也就放心了，没有想到不用动手苏，就是采用包养和中药的方法，真的能将身体给治疗好。<br/>　　<br/>　　等回家以后，对于吴老提出的和老伴一起去居住，当然就非常的同意。至于说居住的房费，呵呵，还真的不算什么。<br/>　　<br/>　　陈默不知道吴老回去后发生的事情，他还是按部就班的生活着。现在很多事情已经走上正轨了，所以他也就能够稍稍闲一些。<br/>　　<br/>　　这几天都是早上起来后，在周围转转，屁股后面这个大黄和大灰。偶尔回家里一趟，老爸老妈都惊讶大黄大灰的个头，怎么长这么大。<br/>　　<br/>　　然后逛逛菜地什么的，在看看自己的别墅的修建进度。只要框架修好以后，盖楼层是相当的快，在有几天就要封顶了。<br/>　　<br/>　　鱼塘有马有为和另外一个人照顾，到是没有事情发生。不过明显有些人手不足的问题，但是这个事情现在还不突出，他准备等大海回来后，再招几个人。<br/>　　<br/>　　大海已经给他打过电话了，牛种和猪种都已经购买好，不过都非常的贵，好在自己也没有准备养多少，倒是花费不了多少钱，也就是百万元左右。他们买好种苗以后，就跟车回来，现在可能还在路上，在于一天的时间就差不多到了。<br/>　　<br/>　　陈默今天已经安排陈金贵他们，将牛棚和猪舍全部都打扫了一遍，然后等着投放种苗。<br/>　　<br/>　　前面买回来的上万只小鸡苗，由于陈默在偷偷喂食了些溪水，所有的鸡苗都活蹦乱跳的，没有一个死亡的。这在其他人看来都是不可能的事情，没有想到这里就是发生了。<br/>　　<br/>　　每隔两天的拉菜，就有一大堆的菜叶等等，现在光是羊和鸡子吃，却是吃不完的，鸡和羊都还小，消耗的食物也就比较小，等猪和牛苗拉过来后，可能就不够吃了。<br/>　　<br/>　　等到晚上的时候，陈默还是雷打不动的制作符箓和阵基，然后就是修炼。经过几天的修炼，练气三层算是巩固了下来，并有一定程度的精深。<br/>　　<br/>　　当然，还有个好宝贝，现在可以用了。那就是青玉剑，现在他可以驱动了。不过也就在九米的范围内驱动，超出就无能为力。<br/>　　<br/>　　还有就是由于他自身的原因，只能驱动飞剑一次飞出后收回，至于第二次，呵呵！别指望了，全身的真元都耗费完了。<br/>　　<br/>　　主要是飞剑虽然在他真元内养着，但是现在青玉剑就是个剑胚不说，还缺少最后一次的炼制。至于说飞剑用还是可以用的，但是要消耗更多的真元不说，还在操控上来说不是那么随心所欲的。<br/>　　<br/>　　但是飞剑所需的金属，他还从来没有听说过，也许在这星球上也找不到吧，只能先就这样凑活着用了。<br/>　　<br/>　　陈默已经自己一个人去山谷中，试了一次青玉剑的威力，很不错!在他操控中直接将足足一尺多厚的岩石穿孔不说，还犹如穿过豆腐般轻松。但是也就这样了！<br/>　　<br/>　　他都有些丧气的说，飞剑确实好用，但是飞出去后再收回来，就让他丹田内空空如也，精神力也耗费的差不多见底，也让他瘫软在地上。要不是有灵液的话，想要恢复也要行功一周天才行。<br/>　　<br/>　　其实青玉剑要是祭炼好以后，将最后一道工序补足了话，那么自己绝对能操控青玉剑飞起来攻击三次以上没有问题，而且会更加的精细化操控，哪里想是现在一样，总是感觉在操控飞剑的时候，有种别扭感。神识传来的感觉，就是让飞剑转弯的时候总有种心有余而力不足之感觉。<br/>　　<br/>　　夜殇师父以前得到这个青玉剑的时候，为什么不将它弄好呢？想起那束头发，也想到那个乾坤袋，还有那本丹书，可能就是夜殇师父慢慢的回忆吧。<br/>　　<br/>　　这剑也可能是他的一个怀念，可能被人送给他以后，就是这个样子，也就没有想着去补足。<br/>　　<br/>　　陈默有时候也在想，要是夜殇师父的乾坤戒没有被毁坏就好了，那么自己至少有很多东西能用，也许里面有其他飞剑也说不定。但是现在说这些也没有什么用了，一切的一切都是过去式了。<br/>　　<br/>　　就在陈默的思绪飘飞的时候，感应法阵却感应到一个物体进入了自己的菜地。<br/>　　<br/>　　陈默不得不缓缓收功，然后去看看是什么。等到他来到菜地种植区域的时候，已经没有任何的发现了。走到了感应法阵的大概区域后，却没有什么发现。<br/>　　<br/>　　神识一扫之后，除了几根毛发和脚印以外，再也没有其他发现，看来闯进来的是个动物，看看脚印，也看不出是什么动物，只能以后再说了。<br/>　　<br/>　　这个时候，远远一束手电转了过来，陈默也就回应了一下。来的是陈金贵，他看到有人在菜地里打着手电，所以就过来看看。<br/>　　<br/>　　“金贵叔，回去吧，没有什么，我也就是随便看看！”陈默笑着说道。他其实在夜间是不用打手电的，但是那样做就有问题了，所以也是随大流，弄个手电装装样子。<br/>　　<br/>　　“呵呵！是二娃啊！那就好，那我就回去了！”<br/>　　<br/>　　陈金贵其实每天夜里都有巡逻的，很是认真负责。<br/>　　<br/>　　陈默没有发现什么，所以也就走了回去。感应阵虽然能提醒陈默，但是其也是有些限制的，只能让他感觉到有东西侵入，但是具体是什么也就不知道了。<br/>　　<br/>　　当然在感应阵的激发过程着，也是去除一些小型动物的，所以像是老鼠什么的小动物，感应阵是不会发出提醒的。<br/>　　<br/>　　等天亮的时候，陈默就去葫芦谷内修炼王家长拳，拳法已经被他修炼的有些像模像样了。拳法的修炼，只不过陈默的兴趣，在以后可能也是个遮掩，还是对他自己有好处的。<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
