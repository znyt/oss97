　　炼器师公会之中，一楼是一个极大的大殿，占地方圆千丈，其内摆放着一个个如同水晶般的柜台，有序的排列成数百排。
<br/>　　每一个水晶柜台之内，都有一件兵器，或是宝剑、或是长刀、亦或是长枪、大斧、长矛等等。
<br/>　　这些兵器没有一件是凡品，最低级的都是皇级兵器，其中最多的是帝级兵器。
<br/>　　甚至，还有一排水晶柜台之内，摆放的兵器极为不凡，光芒流转，威势隐隐能透过柜台传递而出。
<br/>　　很显然，这些是尊级兵器，武尊级别强者使用的兵器。
<br/>　　大殿中有零零散散的数十名武者，在各个水晶柜台观看，显然在选购兵器。
<br/>　　苏莫和秦不死，在大殿中漫步而行，进入炼器师公会，两人就将巫娴和另外两名巫族青年，全部收进了空间宝器之中。
<br/>　　毕竟，带着三名巫族，不甚方便。
<br/>　　望着这些神兵利器，苏莫不禁心中感慨，这炼器师公会果然不简单，居然还有尊级神兵售卖！
<br/>　　不过，他不禁有些犯愁了，他本以为炼器师公会，是一个专门培养炼器师的势力。
<br/>　　然后他加入炼器师公会，就能得到炼器师公会的庇护。
<br/>　　毕竟，看这炼器师公会的规模和排场，应该背后也有武尊级别的强者。
<br/>　　但现在，发现所谓的炼器师公会，居然是一个专门售卖兵器的商铺，他不禁有些失望。
<br/>　　不过，既然来了，他自然不能就这么离开，最起码待在这炼器师公会中，应该很安全。
<br/>　　见大殿中有不少穿着统一服饰的年轻炼器师，在售卖兵器，苏莫便大步走了过去。
<br/>　　“这位大师，敢问贵公会是否招收炼器师？”苏莫走到一名高瘦的炼器师身前，抱拳问道。
<br/>　　这名高瘦炼器师，年约二十余岁，鼻梁高挺、面容儒雅，虽然年轻但气质却是不俗，想必炼器之术也不低。
<br/>　　“我炼器师公会常年招收炼器师，难道你不知道？”高瘦炼器师瞥了苏莫和秦不死一眼，反问道。
<br/>　　“是吗？”苏莫闻言大喜，如此的话他就能加入炼器师公会，有了庇护之所。
<br/>　　没想到自己运气居然如此之好，真是犹如神助啊！
<br/>　　“敢问大师，炼器师公会有没有武尊境的强者守护？”苏莫再次问道。
<br/>　　他也是要确定炼器师公会的实力，不然的话，若是太弱，根本护不住他。
<br/>　　呃~~
<br/>　　听闻苏莫的话，高瘦炼器师顿时满脸愕然，仿佛看傻子一般的看着苏莫。
<br/>　　怎么了？
<br/>　　苏莫见此心中有些疑惑，对方这是什么表情？难道炼器师公会没有武尊级别的强者？
<br/>　　不会吧！
<br/>　　看此城中武者的修为水平，再加上炼器师公会的规模，应该有武尊境强者吧！
<br/>　　少倾，高瘦炼器师的脸色沉了下来，冷冷的道：“你是来捣乱的吧？”
<br/>　　对方居然问炼器师公会有没有武尊强者守护，这在高瘦炼器师看来，分明就是来捣乱的。
<br/>　　“不敢！在下来自小地方，初来乍到，的确不甚了解！”苏莫立刻解释道，同时他亦是心中疑惑，对方为何会认为自己是来捣乱的呢！
<br/>　　“哦？”高瘦炼器师闻言打量了苏莫一番，又看了看旁边的秦不死，脸色稍缓。
<br/>　　看苏莫两人一副乡巴佬的样子，也不像是来闹事的人，而且也没有人敢来炼器师公会闹事。
<br/>　　“我炼器师公会，乃是天荒星域一等一的大势力，你说有没有武尊级别的武者？”高瘦炼器师傲然一笑，眸中蕴含戏虐之色。
<br/>　　苏莫闻言心中一震，面现震惊之色，天荒星域一等一的大势力？这么强大？
<br/>　　若真是如此的话，看来此地的炼器师公会，只是一处分会了！
<br/>　　秦不死同样面带震惊之色，没这么夸张吧？他们随便找一个地方，就是天荒星域一等一的大势力？
<br/>　　高瘦炼器师见苏莫和秦不死面带震惊的模样，面上带着一丝笑意，一股自豪感油然而生。
<br/>　　“好了，你们两人都是想加入炼器师公会？”高瘦炼器师问道。
<br/>　　“只有我一人。”苏莫道，秦不死并不懂炼器，显然不可能加入炼器师公会。
<br/>　　不过，只要他加入了炼器师公会就够了。
<br/>　　“你？”高瘦炼器师闻言，面色顿时不再客气，不屑的摇头道：“你最多成为炼器学徒，不可能成为公会中的炼器师。”
<br/>　　他本以为秦不死是帝级炼器师，所以客气有佳，没想到原来并不是。
<br/>　　他看苏莫不过是武皇境修为，又如此年轻，最多也就是皇级炼器师而已，想成为公会的炼器师根本不可能。
<br/>　　“为何？”苏莫问道。
<br/>　　“只有帝级炼器师，才能成为本公会的炼器师，以下级别的炼器师，只能成为公会中的学徒！”
<br/>　　高瘦炼器师傲然说道，事实上他也只是公会中的炼器学徒，根本不是什么大师！
<br/>　　公会中的炼器师，高贵无比，不可能在一楼大殿中售卖宝器。
<br/>　　“有什么区别吗？”苏莫沉声问道，学徒地位肯定不会高，这点他自然知道，但还是要问清楚具体的区别。
<br/>　　“本大师很忙，没空搭理你！”高瘦青年摆了摆手，转身径直离去。
<br/>　　苏莫见此顿时脸色一沉，心中微怒，不过他还是忍住了怒气。
<br/>　　毕竟，他现在处境堪忧，还是先想办法躲过此劫再说。
<br/>　　他现在有求于人，只能下点本钱了！
<br/>　　“大师留步，在下当年偶的一些高深的炼器之术，一直无法参悟，今日便送给大师！”苏莫一步踏出，立刻追上了高瘦炼器师。
<br/>　　苏莫又准备动用贿赂的手段了，没办法，他必须要留在炼器师公会。
<br/>　　而贿赂炼器师，灵石宝物等效果都不大，唯有高深的炼器之术，才有效果。
<br/>　　“哦？什么级别的炼器之术？”高瘦炼器师闻言，顿时停下来脚步，满脸感兴趣之色。
<br/>　　苏莫立刻从储物戒内取出了一枚玉简，然后用意念括印炼器之术。
<br/>　　很快，苏莫括印完毕，将玉简递给对方，道：“大师请看！”
<br/>　　高瘦炼器师接过玉简，立刻便查看了起来，随即神情一震，面上露出大喜之色。
<br/>　　片刻之后，高瘦炼器师看完了玉简中的内容，立刻满脸喜色的将玉简收了起来。
