　　樱桃，樱桃，白楚燕看着手中鲜艳欲滴如红玛瑙的樱桃，那么的诱人，蕴含浓郁灵气，她却怎么也不敢吃，美眸中反而充满了惊恐之色。
<br/>　　小时候，这个庭院还没有这株樱桃树，曾经白楚燕亲眼见到祖奶奶用一个女人做什么实验，那女人最后死了，死的很惨，全色皮肤都呈现殷红色。
<br/>　　祖奶奶命人在庭院中挖了一个大坑，将一颗樱桃种子放进了那惨死女人的嘴里，再把她放进坑中埋好。
<br/>　　她说那个女人的血液中有一种特殊蛇毒，这样做种的樱桃树以后不长虫子，而且果实也很不容易腐烂，哪怕摘下来，放一个月，樱桃也能鲜艳欲滴，保持新鲜。
<br/>　　最关键，这是一株特殊的樱桃果树，结出来的是灵果，对修武者，吃了有着非常好的辅助作用。
<br/>　　白家的樱桃，可是在温城远近驰名的，曾经她就听过，有人出一百万，为买一颗樱桃的故事。
<br/>　　不管这樱桃如何的不凡跟珍贵，但白楚燕从小到大没有吃过一颗樱桃，她脑海中还记得那女人长着大大的嘴巴，埋在土里的画面，挥之不去。
<br/>　　甚至一度成为她小时候的噩梦。
<br/>　　有些事情不知道还好，一旦知道了，就成了心中的阴影。
<br/>　　她每次看到祖奶奶咬下樱桃果肉，嘴角溢来鲜红果汁，她就觉得是那个惨死女人的鲜血，触目惊心，浑身不寒而栗。
<br/>　　但祖奶奶却吃的非常的愉悦。
<br/>　　“祖奶奶，你就别勉强我吃了，楚儿实在吃不下”白楚燕浮现记忆，脸颊泛着一丝苍白，腹中翻江倒海起来。
<br/>　　“你呀，都不知道说你什么好”祖奶奶点了她一下额头，到也不怪罪，旗袍下，款款莲步，优雅大方，朝着一件房子走去，边走边说：“樱桃不吃，那祖奶奶就带你看只宠物，挺有意思的”
<br/>　　“宠物？”白楚燕脸色缓和了不少，饶有兴致起来，能被祖奶奶看上的，肯定非常不错。
<br/>　　房子进去，有张根木雕成的桌上，桌子上有一个笼子，笼子上盖着一块黑布，祖奶奶二话不说的将黑布被掀开。
<br/>　　只见笼内里面有一只浑身老虎皮的猫萎靡的趴着，当黑布没了，笼中的世界亮了，这只独特的猫骄傲地昂起头，扭动着身体，看到祖奶奶一下子全身毛发炸立，发出嗷嗷的低沉声响，一双眼睛火眼金睛似的，充满了威势。
<br/>　　“这...”
<br/>　　白楚燕被这猫的威势吓得后退了几步，全身如履薄冰，呼吸都紧促起来。
<br/>　　这那里是猫呀，简直是一头小老虎，威风凛凛，浑身有王者之气，不禁让人想冲它臣服。
<br/>　　“咯咯...小家伙，饿了几顿，还那么有精神啊，你要是吓坏了我的小楚儿，看老身怎么收拾你”旗袍下的祖奶奶优雅迷人，却自称老身，让人觉得很别扭。
<br/>　　就好像一位中年男子，管自己叫老夫一样。
<br/>　　那只浑身炸立的猫，周身有很强的气势，它一双眼睛烙印在祖奶奶身上，可她一点都不怕，反而一脸的含笑，那姿态就像欣赏一只宠物猫，愈看愈喜。
<br/>　　“祖奶奶，这真的是一只猫吗，看着好凶呀”白楚燕还从来没见过如此威风凛凛的猫，一时之间，既惊既怕，没来由的反而有点害怕。
<br/>　　这种害怕，让她羞耻。
<br/>　　祖奶奶悠悠然的说：“这只猫，品种无比高贵，可是超等变异生物，而且已经是二阶精怪的层次，难得的很啊”
<br/>　　啊...！
<br/>　　白楚燕闻言就呆滞了，眨了眨自己的眼睫毛，呼吸都沉重了。
<br/>　　二阶的超等变异生物，这也太夸张了吧，这都快属于传说中的神兽级别了，或许说，将来可能成为神兽。
<br/>　　“咯咯...不错吧，这猫算是祖奶奶我捡来的，等你弟弟要是突破三阶，祖奶奶就将这只猫送给你弟弟当战宠”
<br/>　　她继续道：“其实现在送也可以，只是这只猫还是有主的，但相信过了今晚，这只猫的身份就恢复自由了”
<br/>　　话，轻描淡写的说着，她的眼眸闪过一丝得意。
<br/>　　嗷..．
<br/>　　笼中的猫不知为何，凶性大作，猛然的大吼了一声，整个房子都震动了，白楚燕连忙捂住自己的耳朵，脸上有痛色。
<br/>　　“咯咯...小家伙，你还真是调皮”祖奶奶笑呵呵，眼中慢慢寒冷起来。
<br/>　　“黄金瞳，我找到你了”
<br/>　　藏匿在白楚燕兜兜内的涂小安兴奋起来，这黄金瞳被关在笼中，犹如困兽。
<br/>　　这笼子不简单啊，看着很普通，但是能困住一只二阶精怪，就已然是宝物了。
<br/>　　这位祖奶奶的话，涂小安都听的一清二楚，她说过了今天，这猫就自由了，意思就是说，余子童必死无疑，见不到明天的太阳。
<br/>　　事实或许也的确如此，可惜涂小安将那条蛊蛇给弄了出来，余子童现在死不了。
<br/>　　对方的如意算盘自然落空。
<br/>　　嗷..．
<br/>　　笼中的猫再次咆哮，仿佛是被刺激到了。
<br/>　　“怎么，心疼你主人了？”祖奶奶揶揄，区区一个一阶的废人，竟然让这等珍贵的猫认主，她有点想不通。
<br/>　　还如此忠心耿耿，它是知道自己的主人危在旦夕。
<br/>　　“你主人，死定了”
<br/>　　嗷..．
<br/>　　房间剧烈震动，这咆哮宛如音波功，祖奶奶神情自若，一点不受影响，但是白楚燕的嘴唇和面颊惨白而拉长起来，只觉得脊梁上流下一股股的冷汗。
<br/>　　“祖奶奶，我受不了了”
<br/>　　白楚燕修为太低，承受不了这只猫的咆哮。
<br/>　　“你啊，连一只猫的叫声都禁不住，跟你弟弟差太远了”祖奶奶脸上出现不悦，好像被她扫兴了。
<br/>　　白楚燕娇躯一震，惶恐不安道：“祖奶奶，对不起，是楚儿没用”
<br/>　　“白家，可是名门望族，温城的第一世家，你堂堂一个白家二小姐，修为那么低，成何体统”祖奶奶淡淡的看着她，无形中给了白楚燕巨大的压力。
<br/>　　白楚燕扑通一声的跪了下来：“祖奶奶对不起，是楚儿太没用，我日后一定好好修炼”
<br/>　　“你还不如多吃几颗樱桃呢”
<br/>　　一说樱桃，白楚燕死死咬紧牙关，不说话。
<br/>　　“罢了，你下去吧，去看看你弟弟，省的在这给老身堵心”
<br/>　　白楚燕顿时如临大赦，连忙抽身离开，但在她转身离开房间的一瞬间，她兜兜中，一抹光影顺势滑落，无声无息的钻到了角落阴暗处。
