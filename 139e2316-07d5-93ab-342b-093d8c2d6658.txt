>　　    “也不一定吧？我听老人家说，只要被下了绝情蛊的男人，哪怕就是多看两眼别的女人，心里YY点什么就能起作用，灵验的很呢”<br/>　　<br/>　　    “哎呀，果真这么灵嘛？”<br/>　　<br/>　　    “呵呵，惊蛰啊……”<br/>　　<br/>　　    “什么？”<br/>　　<br/>　　    “你怎么流了那么多的汗啊，被吓到了么？”<br/>　　<br/>　　    “没，没有，是热的，热的！”<br/>　　<br/>　　    几天后，小草身上的虱蛊被解开了，黄泉路上走一圈，又回来了。<br/>　　<br/>　　    王惊蛰等到这时候也差不多了，出来一趟将近两月，他还在惦记着川中的事，周深和吴洋两人身上的小账本，也该算算了。<br/>　　<br/>　　    小草留在了寨子里，据她说要是快的话，大概两三年月她就能出徒了，慢的话四五个月到半年也没准，几个月时间一晃就能过去了。<br/>　　<br/>　　    王惊蛰离开苗寨前的一晚，在他俩住的木屋中，小草铺着床，一边收拾着一边说道：“你从这里出山，大概得要两三天的时间，路上你小心一点，多带些吃的和水，出了山之后不远就有镇子，有通往城里的客车”<br/>　　<br/>　　    “咕嘟”王惊蛰瞅着茅小草妖娆的曲线和苗条的背影，咽了咽唾沫。<br/>　　<br/>　　    小草说了一会后，见后面没人搭话，就转过身来，看见王惊蛰绿油油的一双小眼睛，就给吓了一跳。<br/>　　<br/>　　    “哎呀，你要吃人怎么着，眼神这么让人惊恐呢……”<br/>　　<br/>　　    “这地上有点凉，也挺潮的，我睡了好几天了感觉骨头都不舒服了，又酸又疼的，我都快忘了躺床上睡觉是啥滋味了”王惊蛰扣着大腿说道。
