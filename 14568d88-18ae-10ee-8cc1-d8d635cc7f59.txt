夏夕绾僵在原地，她一双澄亮的翦瞳从黑色的洞口落到了男人的俊脸上，现在他的样子很可怕，狭眸里像扑洒开的墨汁，危险如深渊，那一层血红的红血丝更是让他染上几分狰狞的阴鹜。
<br/>
<br/>        现在他拿着枪，掌面的青筋都在暴跳。
<br/>
<br/>        “你…你知道你在干什么么，快点把东西放开！”
<br/>
<br/>        陆寒霆阴沉的盯着她看，薄唇掀动，从喉头里溢出了沙哑的音节，还有决然的狠意，“绾绾，对不起，我不能放你走。”
<br/>
<br/>        刚才夏夕绾只想着长痛不如短痛，说了一些狠心绝情的话，但是她忘了他不能受刺激，一切好转都是假象，他的病情一直在恶化。
<br/>
<br/>        陆茵茵曾经告诫过她，发了病的陆寒霆对自己狠，对别人也狠，她留在他的身边很危险。
<br/>
<br/>        她也看到他在精神病院里的视频，他那样的阴郁嗜血，根本就控制不住。
<br/>
<br/>        所以，现在他拿枪指着她。
<br/>
<br/>        因为，他没有办法放她走。
<br/>
<br/>        “陆寒霆，你现在的行为很危险，如果你真的伤了我，你想过后果么，你的人生会染上污点的。”
<br/>
<br/>        “绾绾，这一切也可以不发生，你不要走，不要离开我，好不好？我真的…不能失去你，我求你了，好不好？”
<br/>
<br/>        夏夕绾鼻尖酸楚，心里又疼又痛，他拿着枪，用最狠的姿态，对她说最软的话，他已经在求她。
<br/>
<br/>        求她不要走。
<br/>
<br/>        他的人生字典里从来没有“求”这个字，后来她出现了，就有了。
<br/>
<br/>        夏夕绾眼眶一热，那些滚烫的泪珠再也无法克制，纷纷砸落了下来，她流着泪点头，“好，那你开枪吧。”
<br/>
<br/>        她转身就走。
<br/>
<br/>        她走了？
<br/>
<br/>        她真的走了！
<br/>
<br/>        陆寒霆这么一刻只觉得睚眦欲裂，心脏被狠狠的撕扯成了两半，他疼的痛不欲生，留不住她，不管用什么方法，都无法留住她，她还是要离开他。
<br/>
<br/>        他盯着女孩渐行渐远的纤柔身影，修长的手指微微蜷了起来，然后缓缓将拿枪的手放下。
<br/>
<br/>        他从来没有想过伤害她。
<br/>
<br/>        他怎么舍得伤害她呢？
<br/>
<br/>        他只是吓唬她一下，想让她留下的。
<br/>
<br/>        陆寒霆觉得自己的人生失去了所有的色彩，好像没有什么再值得他留恋，再值得他撑下去，他知道没有了她，他一秒都不能活。
<br/>
<br/>        “绾绾。”
<br/>
<br/>        夏夕绾走出了几步，这时突然听到了男人沙哑到听不清的嗓音，她缓缓停下脚步，转过了身，“你还想说什么，够了，别纠缠我了。”
<br/>
<br/>        陆寒霆站在原地看着她，他将薄唇勾出了一道浅浅而自嘲的弧线，“如果我还活着，就没有办法做到不去纠缠你，我想我一定会不择手段，折了你的羽翼，将你禁锢在我的身边的，你说过会给我生两个宝宝，我一直记着，也许你又是在开玩笑，但是我当真了。”
<br/>
<br/>        夏夕绾心里突然生出一股恐慌感，现在的他很不正常，好像下一秒他就会对他自己做出什么事情。
<br/>
<br/>        夏夕绾看着他，只见他猩红的狭眸也落在她的脸上，用一种格外偏执而病态的目光炙烈的锁定着她。
