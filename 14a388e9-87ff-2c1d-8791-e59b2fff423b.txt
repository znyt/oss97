　　“神女？”
<br/>　　苏莫闻言一怔，随即脸色沉了下来，面前的这名紫袍青年，看来是不会如实的告知他答案了。
<br/>　　“给你最后一次机会，回答我的问题！”苏莫眸光冷漠了下来，对方不配合的话，那他就只能用强了。
<br/>　　“简直不知死活，以为修为高我一重，就能在我面前张狂吗？”紫袍青年见苏莫强硬了起来，顿时厉声喝道，他丝毫无惧苏莫。
<br/>　　他虽然只有武帝境七重的修为，但是战力强大，比之武帝境八重武者都丝毫不弱，甚至犹有过之。
<br/>　　“既然如此，别怨我了！”苏莫淡漠的说了一声，身上紫雾升腾而起，瞬间便将紫袍青年笼罩了起来。
<br/>　　随即，紫袍青年犹如石化了一般，面色凝固，神情呆滞。
<br/>　　一个呼吸之后，苏莫收回了梦魔战魂，大手一张，直接将目光呆滞的紫袍青年吸了过来。
<br/>　　嘭！
<br/>　　随即，苏莫的手掌直接按在了对方的天灵盖上，立刻施展了搜魂术。
<br/>　　随着搜魂术的施展，紫袍青年的记忆，顿时如同纷乱的雪花，不断的进入了苏莫的脑海之中。
<br/>　　片刻之后，苏莫松开了手掌，紫袍青年的身体直接从高空中跌落了下去，最后嘭一声砸落在地。
<br/>　　先是承受了梦魔空间的折磨，又被苏莫直接搜魂，紫袍青年的神魂已经崩溃，彻底的死亡了！
<br/>　　苏莫神情怔怔，静立的高空中，脸色苍白的可怕，直到十几个呼吸之间，才稍稍有些恢复。
<br/>　　他本来就因为炼制两具分身，而导致神魂受损，精神力减弱，现在施展了搜魂术之后，紫袍青年庞大的记忆冲击，让他更感精神疲惫。
<br/>　　“果然是青璇！”苏莫呢喃自语，通过紫袍青年的记忆，他看到了宏青璇！
<br/>　　那太阴族的神女云悠悠，果然就是青璇！
<br/>　　紫袍青年是太阴族的内族之人，曾有很多次见过青璇，绝对不会有错！
<br/>　　苏莫心中微微松了口气，终于找到了，他的心也定了下来。
<br/>　　只不过，现在摆在他面前的，是一个巨大的难题。
<br/>　　因为他无法带走青璇！
<br/>　　首先，现在青璇成了太阴族的神女，在太阴族地位崇高，他连想见对方都见不到。
<br/>　　其次，就算见到了对方，他也不可能从强大无比的太阴族之中，把青璇顺利带走。
<br/>　　这些还不是最主要的，最让苏莫担心的是，失去了记忆的青璇，可以说完全变成了另外一个人，肯定已经不记得他了！
<br/>　　现在的他，对于宏青璇而言，就是个完完全全的陌生人。
<br/>　　他就算有机会带走对方，对方肯定也不会跟他走。
<br/>　　“哎！”苏莫叹了口气，感觉头大如斗。
<br/>　　此地不宜久留，他也没有过多思索，收取紫袍青年的尸体，飞快的离开了。
<br/>　　几个时辰之后，苏莫来到了一片荒凉的山脉之中，在一处山体上开凿出一个简易的洞府，在洞府中暂时停留了下来。
<br/>　　盘膝坐在洞府中，苏莫面无表情，低头沉思。
<br/>　　现在他有两条路，一是不再去找宏青璇，因为现在的对方，不会再有任何的危险，前途无可限量。
<br/>　　二是继续想办法，接触青璇，然后想办法带走对方。
<br/>　　苏莫心中清楚，其实他现在离开是最明智的选择，因为他很难从太阴族中带走青璇。
<br/>　　而且对方目前在太阴族，地位崇高，前途一片光明。
<br/>　　他若是坚持想要带走对方的话，不仅带不走，更可能会给自己带来杀身之祸。
<br/>　　但是，他不甘心，不甘心就这样抛弃青璇。
<br/>　　因为和对方相处了很久，他的心中也早已生出了感情，无法被替代的感情。
<br/>　　况且，苏莫心中也清楚宏青璇对他的感情，很深很真，不然当初在帝玄宫之时，也不会不顾自身安慰的去救他！
<br/>　　“青璇，你还记得我吗？”苏莫叹了口气，低声自喃，他想要离开，但心中的执念却是不让他离开。
<br/>　　因为放弃，他与宏青璇之间，将彻底不会再有交集。
<br/>　　时间缓缓流逝，苏莫静坐在山洞中，一个时辰又一个时辰。
<br/>　　不知过去了几个时辰之后，他眸中顿时精光四溢，有了最终的决定。
<br/>　　他不能放弃宏青璇，这不仅是为了他自己，也是为了对方。
<br/>　　虽然对方现在不会再有危险，在太阴族受到最好的培养，前途一片光明，但没有了以前的记忆，宏青璇还是宏青璇吗？
<br/>　　没有了以前的记忆，对方就变成了另外一个人，这是苏莫不想接受的，想必也是宏青璇不想接受的！
<br/>　　所以，他不能放弃宏青璇，要想办法先接近对方，以后再慢慢想办法恢复其记忆。
<br/>　　至于说对方现在不认识他了，对他没有感情了，苏莫也不是太担心，再追求回来便是！
<br/>　　不错！
<br/>　　是追求回来！
<br/>　　以前，是宏青璇含蓄的向苏莫吐露心声，可以说是在追求苏莫。
<br/>　　现在，苏莫则是要反过来，去追求宏青璇，把她追回来。
<br/>　　这一刻，苏莫心中坚定了心念，一定要把宏青璇追回来。
<br/>　　对此，苏莫还是有一定信心的，或许目前追求青璇的人很多，但他不会比任何人差。
<br/>　　再说了，以前和青璇相处了那么久，他对于青璇的性格非常的了解，没有人比他更了解了。
<br/>　　他相信，只要青璇没有改变性格，他就有极大的可能，将对方追求回来。
<br/>　　两世为人，苏莫对于泡.妞的诀窍，还是了解的非常多的，远不是这个世界那些只知道修炼的人所能比拟的。
<br/>　　做出了决定，接下来就是考虑如何去追了。
<br/>　　苏莫面露思索之色，他杀了那名太阴族的紫袍青年，倒是可以夺舍对方，然后混进太阴族之中。
<br/>　　就想当初他夺舍巫尘，混进天冥星巫族一样。
<br/>　　不过，当初的巫尘只是一个孤家寡人，而根据夺舍的记忆，这名太阴族的紫袍青年，不仅是太阴族的内族之人，父母居然都是高级武尊。
<br/>　　如此的话，他倒是不能夺舍此人混进太阴族了，不然太容易被识破身份了。
<br/>　　他或许可以瞒过其他的太阴族强者，但很难瞒过熟此人的至亲之人。
