　　从陈家大院离开，涂小安对这附近的路也并不是很熟悉，东走西拐，一个胡同出去，他既然诧异的发现，自己来到了李清照昨天带自己来的古街上。
<br/>　　这条古街人流量异常的旺盛，何种名小吃，何种餐饮店，让人眼花缭乱，好像全国各地的美食，你都能从古街上找到。
<br/>　　涂小安并不知道这附近那里有银行，就只能发信息给李清照询问了一下，可是等了半天，都没有她的回复。
<br/>　　一看时间，现在才早上八点多，稍稍一想，有可能是这丫头昨天折腾了一夜，现在还在跟周公下棋呢。
<br/>　　得...自己慢慢找。
<br/>　　涂小安断定，这条古街的附近一定有银行的，繁华的地段，怎么可能没有银行。
<br/>　　顺着古街行走，涂小安也不着急，准备慢慢的走出去，忽然...他脚步一滞。
<br/>　　前方有一家不大不小的面馆，面馆面前，既然门庭若市，排起了一个长队。
<br/>　　涂小安有点吃惊，这温城人那么喜欢吃面？
<br/>　　大清早就在面馆排队，这是闲的吧。
<br/>　　还是说这家面馆的面非常的好吃。
<br/>　　涂小安的好奇心被勾了起来，走过去，但没去排队，而是看了看外面有一张偌大的菜单，里面有各式各样的面点。
<br/>　　花样不少，但是有意思的是，涂小安一看价格，呆若木鸡，肉丝面，198元，排骨面，298元，三鲜面，398元。
<br/>　　尼玛抢劫呢。
<br/>　　每一种面，都可谓是天价呀。
<br/>　　涂小安重新看了这家面馆一眼，在看着面馆长长的队伍，流露出古怪，大城市还真是一个个都财大气粗，如此贵的价值，既然还有那么多人排队。
<br/>　　都是钱烧的？
<br/>　　涂小安终于知道为什么店主将一张菜单先摆在门口，主要是让大家熟悉下价格。
<br/>　　别到知道吃了面，让你掏了198元，你在那里瞎咋呼。
<br/>　　现在菜单在外，有图有真相，你爱吃不爱。
<br/>　　可偏偏，还就是那么多人吃。
<br/>　　越是如此，涂小安越好奇，什么面，能那么吸引人，如此天价下，还有那么多人甘心排队。
<br/>　　涂小安犹豫了下，也开始排队，同样也留意周围的一些议论声。
<br/>　　“哈哈...等了一个礼拜，今天总算可以吃到这家面馆的面了，隐隐间我都有点激动起来了”
<br/>　　“李哥，真有那么好吃吗，区区一碗面，你还等什么一个礼拜，每天来吃呗。”旁边一位朋友忽然看到门前的菜单价格表乍舌：“卧靠，一碗肉丝面要198那么贵，抢劫呢”
<br/>　　他说着看了李哥一眼，像是知道他为什么要等一个礼拜来吃了。
<br/>　　这面一般人还真不是每天都能吃的起。
<br/>　　李哥斜睨了朋友一眼：“劳资是心疼钱的人吗”
<br/>　　“这店主性情迥异，立了一个非常奇怪的规矩，就是想吃他家的面，一个礼拜只能吃一次，你要想在这个礼拜吃第二碗没门，就算给他在多的钱也没用”
<br/>　　朋友惊奇：“真的假的，天底下还有这样做生意的？我还就不信了，我给他扔个一万，要第二碗，他还能不给”
<br/>　　这话刚说出来，最前面排队的一个青年人大呼小叫起来：“卧靠，什么破店，劳资昨天吃了，今天就不能再来吃吗，你会不会做生意呀”
<br/>　　“还不让我进去，你知道老子是谁吗，劳资给你十万，我就问你让不让我进去”
<br/>　　店门前，有一位高大威猛的汉子，明明面无表情，却给人很是凶悍的感觉，他淡淡的看着青年人一眼，那眼神就跟看一坨狗屎，直接无视。
<br/>　　青年人被看一眼，浑身有股颤栗，不死心的道：“五十万，我就进去吃一碗面，哪怕肉丝面都可以”
<br/>　　哗...
<br/>　　顿时间，周围排队的人都撑着惊骇的目光看着青年人，花五十万吃肉丝面？
<br/>　　青年人看到旁边一个个投来崇拜的目光，得意起来，一副老子就是钱多的样子。
<br/>　　可只见高大威猛的汉子二话不说，大步朝前，直接抓起青年人的领口，一把给丢了出去。
<br/>　　然后冷冷的说：“下一位”
<br/>　　排队的人面面相觑，这店主也太有个性了，区区一个面馆，还请了一个保镖看门，就是为了禁止一个礼拜想吃第二次的人混进去。
<br/>　　五十万都被拒绝，这也太威武不能屈，贫贱不能移了吧。
<br/>　　有个性，太有个性了。
<br/>　　李哥啧啧一笑：“看到没有，我说的不假吧，你现在还觉得面的价格贵吗，那个小子一看就是富二代，出五十万，都进不去”
<br/>　　“李哥，我现在都迫不及待的想吃里面的面了，我还是第一次，有资格进去”
<br/>　　“你小子今天有福了”
<br/>　　涂小安排队，听着看着，到是觉得有趣的很，心中也有点跃跃欲试，什么面能让人如此疯狂。
<br/>　　等，他也准备死等，好奇心害死人。
<br/>　　涂小安从差不多九点开始排队，轮到他的时候，已经11点了，刚好当午餐。
<br/>　　“你可以进去了”
<br/>　　那位面无表情的大汉了涂小安一眼，然后让开了通往里面的路。
<br/>　　“谢谢”
<br/>　　涂小安点了点头，带着好奇心走了进去。
<br/>　　里面不大，也就七八张桌子，除了一张角落的桌子空着，别的都有人坐。
<br/>　　每个人都在很专心的吃面，脸上流露出非常兴奋跟愉悦的表情。
<br/>　　不知道的，还以为他们在吃什么山珍海味，那表情也太夸张了吧。
<br/>　　“你要点什么面”
<br/>　　一位莫约三十多岁的女人走了过来，她脸上有很多的雀斑，更有不少的皱纹，看着涂小安到还算和悦的开口。
<br/>　　这应该就是店主了，不知道是老板，还是老板娘。
<br/>　　但这女人偏老，而且长的非常一般，甚至属于丑女的范围。
<br/>　　看上去有点像寡妇，属于长期没过上女人那种幸福生活的样子。
<br/>　　“我第一次来，也不知道点什么面，你看着上吧”
<br/>　　寡妇脸的老板娘看了看涂小安，见他年轻，像是大学生，道：“那就肉丝面吧，198”
<br/>　　“好的”
<br/>　　涂小安不在乎价格，就是好奇这家面馆的面能好吃到什么地步。
<br/>　　他现在身上还揣着五百万的支票呢。
<br/>　　很快，一碗热腾腾的肉丝面就端到涂小安的桌上，香气扑鼻，让人精神一震。
<br/>　　瞬间，他就暗暗思忖，看来那么多人排队不是没道理，就冲着香味，就知道汤底是非常有讲究。
