        &lt;h3&gt;第三卷第一百二十三章  看不见的硝烟&lt;/h3&gt;
<br/>
<br/>        凌辰在董总的书房，董总对他说：“凌辰，全国总代的这个位置很抢手，你也做了这么多年，今年来找我的很多。我都差点扛不住了。”
<br/>
<br/>        凌辰听董总这样说，心里突然就安定下来了，他静静的听董总说话。
<br/>
<br/>        “何少华是甸城托着上面的关系要拿总代，但是我不能给他，甸城我们的产品每年的销量并不好，但是甸城的产品却很多。我不能将老祖先留下来的传承断了，也不能坏了。”董总叹了口气。
<br/>
<br/>        “今天让你们出去，其实也是考察你们的人品，一个人对女人的态度就能说明一切，一个见异思迁的不尊重人的人是不值得托付的。所以总代的事情我考虑了，这几年都有你来做，明天你和亚子去找办公室主任续签合同，先签五年的吧。”董总皱着眉头说。
<br/>
<br/>        “谢谢董总！”凌辰并没有感觉到很奇怪，如果董总选择何少华他才会感觉奇怪。
<br/>
<br/>        “你也不要谢我，这些是今年的董事会决定的，在你们回来的路上我们就开会研究决定了。你切记，不可让假冒伪劣产品太过张狂，要严查假冒伪劣产品。”
<br/>
<br/>        “我会配合大家一起做好后续工作，还得您这边支持。”凌辰虽然心里很感动，但是脸上还是冷冰冰的神情。
<br/>
<br/>        “我们是会给你大力支持的，只是我需要你后续的力度要比我们更大。”董总殷切的看着凌辰。
<br/>
<br/>        凌辰沉默不语，生意场上永远都是看不见的刀光剑影。董总的意思他明白，要他这个总代担起严打假冒伪劣的责任。
<br/>
<br/>        董总给他透露了一下信息，他们扛着压力将总代给了他，是他要成为他们手中的利器。
<br/>
<br/>        凌辰能说什么呢？他也可以不答应，大不了不做这个总代。但是，面对着这么多年的努力，他不想放弃。
<br/>
<br/>        他需要成长，之前是为了宁雅，现在更是为了宁雅，他知道他和宁雅之前隔着一个目前他还跨越不了的沟，宁雅，你等着看着，我凌辰总有一天会站在顶端。
<br/>
<br/>        这么多年的感情你随随便便的就践踏了，呵，你的男朋友很有钱吗？能配上你们宁家吧？
<br/>
<br/>        此时的凌辰已经坚信宁雅有了男朋友，不但有了而且还给他穿了一身鲜艳的绿衣服。难怪去北京推说工作忙不见他。呵，那一些温柔的话都说骗人的，骗人的！
<br/>
<br/>        “凌辰？”董总见凌辰一直沉默不语，有些不高兴了，难道他看走眼了？这个年轻人担不起责任？
<br/>
<br/>        “我会坚决和厂家站在一条线上，打击假冒伪劣是我们每一个代理商的责任。”凌辰坚定的说。
<br/>
<br/>        “好！你下去吧，后面的路不一定顺利，但是我们一定要共同面对。”董总看着眼前的年轻人，希望他的眼光没有错。
<br/>
<br/>        凌辰从董总的书房出来，手机静静的躺在董总客厅的茶几上。
<br/>
<br/>        他拿起手机，装进了口袋，朝酒店走去。
<br/>
<br/>        董总的院子外面，何少华和几个代理商拦住了他的去路。
<br/>
<br/>        “凌辰，董总和你谈了什么？”何少华问。
<br/>
<br/>        “什么都没有谈，就问我读了一些什么书。”凌辰看着气势汹汹的几个人冷冷的回答。
<br/>
<br/>        “你骗鬼呢？老董是不是想把董亚子送给你啊？”另外一个人看着凌辰不坏好意的问。
<br/>
<br/>        “呵呵呵，凌辰原来是靠女人上位，也难怪，生了这么一张脸，我也喜欢。”几个人哄堂大笑。
<br/>
<br/>        “你们说够了没有？说够了就滚！”凌辰的语气比冬天的冰还冷。
<br/>
<br/>        “凌辰，现在还嚣张呢？当了几年总代就以为没有人能奈何你了？”何少华那双眼睛就像是卒了毒。
<br/>
<br/>        “我觉得我并没有得罪各位吧？做代理期间也一直相处和睦，今天为何如此？”凌辰看着眼前的几个人。
<br/>
<br/>        “就因为你今天和亚子多说了几句话！亚子以后是我的女人！你离她远一点。”何少华“呸”了一声。
<br/>
<br/>        “现在还不是呢，等是的时候再说这话。”凌辰轻蔑的语气让何少华火冒三丈。
<br/>
<br/>        “凌辰，看我今天不把你捏扁。”何少华抬起了手。
<br/>
<br/>        “你试试看，今天谁将谁捏扁！”凌辰一动不动的站着盯着何少华，何少华被盯得心里发毛。
<br/>
<br/>        “哼，今天算你走运，看在是董总家里的份让，饶过你。”何少华手一挥，几个人转身就走。
<br/>
<br/>        何少华以为董总叫凌辰谈话，是要取消凌辰总代的资格，在吃饭的时候，他对着董亚子说的话，董总肯定是听出来了。
<br/>
<br/>        哼，凭着他何少华的关系，娶个董亚子，拿个总代还不是轻轻松松的。
<br/>
<br/>        凌辰，今天晚上就放过你，明天你就等着哭吧！
<br/>
<br/>        何少华志得意满的带着几个代理商出去嗨皮去了。
<br/>
<br/>        凌辰看着远去的何少华，冷冷一笑，何少华，等着瞧吧！
<br/>
<br/>        凌辰回到酒店，叫来了成紫，俩个人商讨下一步的工作计划，这一商量已经是深夜。
<br/>
<br/>        凌辰这些天都没有给宁雅发信息，也没有宁雅的信息，所以他连电话也不肯看，看了也不会有宁雅的信息，何苦让自己又徒增失望？
<br/>
<br/>        第二天，凌辰去厂里，董亚子带着他秘密的续签了合同。
<br/>
<br/>        三天后，厂里召开经销商大会。
<br/>
<br/>        因为凌辰之前签的合同没有到期，而今年销售业绩可观，经过董事会决定宣布凌辰继续为全国总代，代表厂家行使销售、监管、严查假冒伪劣产品的权利。
<br/>
<br/>        何少华听到这个消息后，直接离开会场了。
<br/>
<br/>        凌辰发表了谈话，表达了对厂家和大家的感谢之情，并代表大家宣誓：做一个遵纪守法，恪守本分的代理商。
<br/>
<br/>        接下来的几天，酒店就成了办公室地点，每个经销商需要上报年度计划，凌辰和成紫忙得一塌糊涂，因为今年迟迟没有接到厂家的通知，大家也都不知道该怎么办，之前虽然有些人已经做过了计划，但是还得重新调整。
<br/>
<br/>        凌辰每天例行的问候，凌辰那一些甜蜜的情话再也没有出现的宁雅的手机里，而凌辰似乎想要忘掉宁雅这个人，努力的克制着自己，不去想宁雅。
<br/>
<br/>        宁雅打通了电话却不是凌辰接的，对面的女孩子虽然没有明说她是凌辰的女朋友，但是这么晚又有谁会接听凌辰的电话呢？
<br/>
<br/>        世界上有很多错过，都是一些莫名其妙的误会，如果我们多一份信任、都摊开了心扉，哪怕是打个电话听听对方的解释，也许就少走很多弯路。
<br/>
<br/>        凌辰和宁雅都在自己自以为是的想法里纠结着，疼的不能呼吸，还在安慰着自己，让自己把悲伤转化为动力。
