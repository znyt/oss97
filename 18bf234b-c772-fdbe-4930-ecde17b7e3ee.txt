　　当萧逸回到房间，立刻怀着极高的兴致打开炼药录阅读。
<br/>　　“这个世界的炼药师吗，让我看看你们有多厉害吧。”萧逸满脸自信。
<br/>　　然而，十数分钟后，他暴跳如雷，差点没把炼药录当场撕了。
<br/>　　“我草，这写的什么鬼玩意，晦涩难懂，麻烦无比，还学个屁的炼药术阿。”萧逸愤愤地骂了几句。
<br/>　　除却成为炼药师的苛刻要求，炼药术本身就极为深奥，所以极少有武者能成为炼药师。
<br/>　　当然，萧逸异于常人，作为曾经的形意拳宗师，他的领悟力极为过人。
<br/>　　只要给他一两个月的时间，他绝对有把握将这本炼药录研究透彻。
<br/>　　只不过，他现在想成为炼药师的原因，是想自己炼制丹药，好应对半个月后的家族比武。
<br/>　　所有，他没有时间去钻研炼药录，也在怪责着炼药术的晦涩难懂。
<br/>　　大概将整本炼药录翻了一遍，萧逸发现，里面记载的，大部分是炼药师的基础知识，另外还有三个一品丹药的丹方。
<br/>　　分别是淬体丹，强身丹，养气丹。
<br/>　　淬体丹自不必多说，而另外两种丹药，没有增加武者修为的功效，不过却能增加武者的肉体强度。
<br/>　　丹方上，记载了炼制的办法，以及所需的材料。
<br/>　　而炼制丹药的过程，十分繁琐，又对炼药者的技术要求极高。若是新手炼药，大多是失败、毁炉等情况。
<br/>　　“唉。”萧逸叹了口气，“哪怕是单单钻研这淬体丹，也需要半个月以上的时间，根本来不及。”
<br/>　　其实，萧逸已经很厉害了。
<br/>　　炼药师一道，是出了名的艰难，否则也不会偌大个萧家，仅有四名炼药师。
<br/>　　否则，这个职业，也不会是整个世界最尊贵的职业。
<br/>　　像萧家的其他子弟，乃至普通执事长老，所有人都看过这本炼药录，却没有一人能看懂。更没有谁能有把握钻研透彻，即便给他们数年的时间。
<br/>　　哪怕是萧家的三位一品炼药师，当年也是萧离火手把手教导，培养出来的。若让他们自己学习炼药术，同样是难以钻研透彻，更别说能炼制出一品丹药了。
<br/>　　萧逸有把握在一两个月之内钻研透彻炼药录，已经非常了不起了。
<br/>　　“我就不信你这炼药录能难得了我。”萧逸从来不是个轻言放弃的人。
<br/>　　十数分钟后，当他再一次将炼药录翻了个遍，将这些知识都记在了脑海中，打算慢慢领悟时。
<br/>　　忽然.....
<br/>　　体内冰鸾剑武魂中一阵闪烁，脑海中记载的炼药师知识竟和它产生了共鸣。
<br/>　　萧逸不自觉地将冰鸾剑投影凝聚了出来。
<br/>　　那古朴而锋芒逼人的绝世神剑一出现，萧逸顿时被它吸引了目光。
<br/>　　算起来，这次应该是萧逸第一次真正看到冰鸾剑的全貌。
<br/>　　第一次，是他刚来到这个世界，通过体内，看到它变成了自己的武魂。
<br/>　　第二次，是他对付杨伏，但凝聚出来击杀杨伏后，他心忧依依的伤势，又马上把冰鸾剑散去，没能来得及细看。
<br/>　　萧逸细致而着迷地看着剑身的一分一毫，难以挪动眼睛。
<br/>　　堪称完美的剑身上，点缀着两条游龙，日月星辰环绕其中，栩栩如生，如游龙翱翔天际，俯瞰天地生灵，洞彻万物至理。
<br/>　　密密麻麻的符字遍布剑身，神秘至极。
<br/>　　这些符字，萧逸一个都看不懂。
<br/>　　萧逸的眼睛忽然变了。
<br/>　　又是那熟悉的感觉，左眼冰冷如霜，右眼炽热如火。
<br/>　　太阴太阳之眼毫无预兆地出现了。
<br/>　　当眼睛再次注视到剑身上时，这些符字仿佛有了生命一般，不断游走、组合、分散，玄奥无比。
<br/>　　而同一时间，萧逸忽然涌现出一股阔然开朗的感觉。
<br/>　　那些记载在脑海中的炼药师知识，他竟一瞬间全部悟透。
<br/>　　晦涩难懂的炼药师知识，如今在他眼中，简单得可笑。
<br/>　　数秒后，萧逸脑海‘轰’的一声，出现了一个名为‘九转炼体丹’的丹方。
<br/>　　当冰鸾剑散去，眼睛重新恢复正常，萧逸才反应过来。
<br/>　　“厉害，当真厉害。竟让我瞬间领悟了炼药录上的知识，还自动衍生出一个珍贵的丹方。”萧逸暗暗咋舌。
<br/>　　九转炼体丹，冰鸾剑衍生并创造出来的丹方。功效与淬体丹一致，但效果却强大了许多倍。
<br/>　　“冰鸾剑阿冰鸾剑，你到底强大到什么地步，又还隐藏着哪些秘密呢。”
<br/>　　萧逸一直都知道冰鸾剑的来历极为神秘，必然是华夏上古时期惊天动地的至宝。但关于冰鸾剑的真确信息，他又无从得知。
<br/>　　或许，日后当他武道修为有成，拥有神鬼莫测的手段和本事时，方能解开这些谜团吧。
<br/>　　萧逸摇摇头，甩去心中杂念。
<br/>　　如今最重要的，还是先提升自己的修为。
<br/>　　炼药录上的知识自己已经完全领悟，这代表着，他完全有可能成为一个一品炼药师。
<br/>　　只要将一颗一品丹药炼制出来，他将真正成为一个一品炼药师。
<br/>　　而要炼制丹药，有数个前提条件。一，武者必须具备操控火焰的能力；二，武者要有过人的精神和耐力，炼药是一个不能大意且又枯燥的过程；三，备齐材料和炼药炉。
<br/>　　第一、第二个条件已经解决。
<br/>　　现在只剩下第三个条件，备齐材料与炼药炉。
<br/>　　可这些东西，自己都没有，得去外面购买才行。
<br/>　　萧逸收起了炼药录，出门。
<br/>　　......
<br/>　　“看，是萧逸那废物。”
<br/>　　萧逸几乎是刚离开自己的府邸，已经遭受到其他家族子弟和族人们的指指点点。
<br/>　　“啧啧，这废物竟然还敢出来，换我的话，现在肯定没脸出来见人了吧。”
<br/>　　“哼，一个不要脸的无赖罢了，明明自己无能，却死咬住族规，甚至和若寒表哥打赌，真是自取其辱。”
<br/>　　“等着看吧，半个月后，这个粗鄙的无赖，没用的废物，会被狠狠教训一顿的。”
<br/>　　“到时候，他就会知道什么叫不知天高地厚。”
<br/>　　家族子弟们一向以萧若寒为首，以他为偶像。昨晚的事传开后，萧逸几乎成为了所有年轻子弟的公敌。
<br/>　　对此，萧逸撇撇嘴，懒得理会。
<br/>　　其实，萧逸是个性格极为孤傲的人。
<br/>　　当年在地球，他少年习武，弱冠之年成就大宗师之名；不到二十五岁，已经是震惊世界的杀手之王。
<br/>　　一项项骇人的成就，一个个无敌的光环，铸就了他一身的傲骨。
<br/>　　但，他又是个喜欢率性而行的人，不愿意接受各种束缚，一切跟随自己的心，怎么开心怎么活。
<br/>　　他说话粗鄙。
<br/>　　但这就是他的性格，既然能用一个‘草’字，表达自己的情绪，那他就绝不会用第二个词。
<br/>　　他像个无赖。
<br/>　　但这就是他应对敌人的方式，既然无赖就能让敌人无处对他下手，何乐而不为。
<br/>　　总之，他一切都跟随自己的心，怎么开心，就怎么活着。
<br/>　　当年在地球，他爱着自己的祖国，暗地里竭尽自己的能力，敌视一切试图对祖国不利的势力。
<br/>　　他当然知道，这样会得罪很多人，甚至惹来杀身之祸。
<br/>　　但他不会想太多，他只跟随自己的心。既然自己的心是爱着祖国的，那么，哪怕与全世界为敌，哪怕敌人滔天，他也在所不惜。
<br/>　　哪怕是当时身死的那一刻，他依旧没有后悔。因为，他跟随了自己的心，他认为值得。
<br/>　　但无可否认，他的孤傲，早已烙印到骨子里头。
<br/>　　今天这些嘲讽他，对他指指点点的人，他只当他们是一群傻瓜。
<br/>　　因为，这些人，不值得他放在心上。
<br/>　　起码，这群傻瓜只是单纯指指点点的话，萧逸连对付他们的欲望都没有。
<br/>　　自从来到这个世界后，真正能让他放在心上的，也只有两人了，一个是依依，一个是三长老。
<br/>(本章完)
