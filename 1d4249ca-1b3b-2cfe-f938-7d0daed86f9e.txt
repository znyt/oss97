　　听闻外面的声音，终极神王和紫霄，均是脸色一沉，没想到这个时候，天尘宇居然也来了。
<br/>　　身形一闪，终极神王便掠到了终极大殿之外，抬头望向天穹，便见到一声白衣的天尘宇，伫立在高空之上。
<br/>　　“天小友，殿中一叙！”终极神王大袖一挥，半空之中有无形的阵法立刻裂开了一个口子。
<br/>　　“多谢！”天尘宇抱了抱拳，身形降落而下，随后便跟着终极神王，进入了大殿之中。
<br/>　　走进大殿，天尘宇见到地藏神王、雷火神王和赤云神王，眉头微不可查的一皱。
<br/>　　他没想到，这三人居然也在这里，其他人他倒是不在意，但是，地藏神王是拓跋青云兄长，他不得不注意。
<br/>　　“天尘宇，你来此做什么？”地藏神王微眯着眼眸，盯着天尘宇问道。
<br/>　　现如今，天王宫和地王宫，可不是很太平。
<br/>　　他不太明白，天尘宇来此所谓何事，难道也是为了终极剑道和终极神剑。
<br/>　　“一点私事，不打扰你们，你们继续！”天尘宇摇了摇头说道，既然地藏神王在此，他不会谈论苏莫之事，因为他不能让地王宫知晓苏莫的具体身份。
<br/>　　“坐吧！”终极神王摆了摆手，示意天尘宇就坐。
<br/>　　随后，天尘宇便在边上就坐了下来，故意远离了地藏神王三人，并且目不斜视，显示出不愿打扰众人的模样。
<br/>　　终极神王见此，心中疑惑，难道天尘宇此来，不是为了秦陨之事？不然为何如此姿态？
<br/>　　不过，既然天尘宇没有掺和进来，倒也是好事。
<br/>　　这时，一声蓝裙的绫裳，走进了大殿之中。
<br/>　　“殿主！”绫裳不卑不亢，向终极神王行了一礼。
<br/>　　“嗯！坐吧！”终极神王微微颌首，绫裳便在风雪神王的身侧坐了下来。
<br/>　　地藏神王、雷火神王等人，均是目光在绫裳的身上打量。
<br/>　　他们也都知晓，绫裳在多年之前身死，不知道什么原因，并未彻底的陨落，只是修为暴跌了下去。
<br/>　　“绫裳宫主，根据传闻，秦陨是你绫裳剑宫弟子，我且问你，秦陨是否是紫霄的密门弟子？”地藏神王目光紧紧的盯着绫裳。
<br/>　　其他所有人，闻听此言，亦是全部盯着绫裳。
<br/>　　就连终极神王，亦是如此，他早已私下里询问过绫裳，秦陨和紫霄有没有关系？
<br/>　　当初，绫裳的回答，是她不清楚。
<br/>　　“绫裳，告诉大家真实情况。”紫霄声音低沉的说道，但是，他的眼眸之中，却是蕴含警告之意，明显是提醒绫裳不要诬陷于他。
<br/>　　“诸位，秦陨是我带进终极神王殿的，这点大家应该都知晓，秦陨的天赋无与伦比，紫霄对其很是看重，但是紫霄是否收了秦陨为密门弟子，我就不得而知了！”绫裳面色郑重，沉声说道。
<br/>　　她听从了苏莫的意思，知道苏莫要怎么做，但是，她不能一口咬定，苏莫是紫霄的密门弟子，因为这样太刻意了，不一定让人信服。
<br/>　　而她现在之言，点明了紫霄很是看重苏莫，别人更容易相信苏莫是紫霄的密门弟子。
<br/>　　“绫裳，你胡说什么？我什么时候看重秦陨了？”紫霄闻言，顿时双眉倒竖，他心中担心的事情，果然发生了，这绫裳在颠倒黑白。
<br/>　　“紫霄，我不清楚你有没有收秦陨为徒，但是，此事你不可能独善其身。”绫裳脸色冷漠。
<br/>　　“胡说不到！”
<br/>　　紫霄怒喝一声，长身而起，环视众人，道：“诸位，我和秦陨半分关系都没有，拓跋流明和云沐、古剑山，也不是我杀的，如果我所料不错，三人是死于秦陨之手？”
<br/>　　“紫霄，你当我们如此好糊弄吗？”
<br/>　　地藏神王同样大喝出声，长身而起，道：“秦陨虽然逆天，但是绝对没有能力杀死流明；再说了，秦陨得到终极神剑和终极剑道，若是没有人帮助，能从你们手中逃脱？
<br/>　　地藏神王根本就不信紫霄之言，首先，他对于自己儿子的实力，再清楚不过了。
<br/>　　其次，秦陨如此低劣的修为，得到终极剑道和终极神剑之后，能逃脱神王境强者的追踪？这简直是开玩笑。
<br/>　　“诸位，此事是有人在混肴视线，始作俑者就是秦陨！”
<br/>　　紫霄朗声说眼，言罢，他又看向绫裳，道：“绫裳也和秦陨之间，脱不开干系。”
<br/>　　“紫霄，既然你说你和秦陨没有关系，那当初秦陨杀害你门下几名弟子，你为何没有惩处于他。”赤云神王面无表情的问道。
<br/>　　“这是因为秦陨和绫裳关系较好，我才饶过他一次！”紫霄沉声说道，虽然这一切的一切，让他心中愤怒，但是，他还不至于无法应对。
<br/>　　反正，他没有得到终极剑道和终极神剑，芊寻月也被他藏了起来，别人不可能有证据。
<br/>　　在没有证据之下，任何人都不敢与终极神王殿彻底为敌。
<br/>　　“绫裳宫主，你可知秦陨会去何处？”雷火神王问道。
<br/>　　“不清楚，我虽然和秦陨交情不错，但是并不完全了解此人。”绫裳摇了摇头。
<br/>　　地藏神王、雷火神王和赤云神王三人，闻言沉默不语，他们不在乎终极神殿和终极神剑，他们在乎的是谁杀了拓跋流明三人。
<br/>　　天尘宇坐在不远处，目不斜视，神情淡淡，他不在乎拓跋流明三人死于何人之手，也不在乎终极剑道和终极神剑，他在乎的是，如何能找到苏莫。
<br/>　　终极神王面无表情，静静的听着这一切，现如今，他对于绫裳不信任，对于紫霄同样也不信任了。
<br/>　　“终极殿主，流明无论是死于秦陨还是紫霄之手，此事你必须给个交代！”
<br/>　　少倾，地藏神王目光落在了终极神王的身上。
<br/>　　“不错，紫霄是你终极神王殿之人，秦陨同样也是！”雷火神王点了点头。
<br/>　　“还请终极殿主，给个合理的交代！”赤云神王同样开口道。
<br/>　　“三位放心！”
<br/>　　终极神王点了点头，道：“秦陨，我们会找到，到时候所有的事情，将会水落石出。”
<br/>　　就算三人不来，他也会找到秦陨，毕竟，他必须要拿回终极剑道和终极神剑。
<br/>　　“五年，给终极殿主五年的时间，若是五年之后，终极殿主不能给个合理的交代，我地王宫要用自己的方法，为流明报仇。”地藏神王阴沉着脸说道。
<br/>　　“你在威胁本座？”终极神王双眼微眯了下来，自己的方法？这就是在告诫他，地王宫会向紫霄出手吗？
<br/>　　“不敢，在下只是为子报仇心切！”地藏神王摇了摇头，此事还有待考证，他也不好公然和终极神王翻脸。
<br/>　　“终极殿主，既然你如此说了，那我们便等着你的消息！”雷火神王起身，抱拳说道。
<br/>　　“在下告辞了！”赤云神王提出了告辞。
<br/>　　“不送！”终极神王挥了挥手。
<br/>　　随即，赤云神王和地藏神王、雷火神王，均是离开了终极神王殿。
<br/>　　待三位神王走后，终极神王看向天尘宇，道：“天尘宇，你所来还是为了找秦陨，也就是苏莫？”
<br/>　　“不错！”
<br/>　　天尘宇点头，随即起身向终极神王行了一礼，显得彬彬有礼。
<br/>　　“哎！”
<br/>　　终极神王叹了口气，心中有些无奈，苏莫这一个小小的武者，居然让他们一帮人，如此费神。
