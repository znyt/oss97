　　苏莫在第十三重时光神殿之中，本来可以获得四十多年的时间，即便修炼了二十多年，也还有二十年左右的时间。
<br/>　　但是，现在退到了十重时光神殿，时间流速大大的减缓，剩下能用于修炼的时间，只有五年左右了。
<br/>　　不过，即便是五年左右，对于他来说也是求之不得的事情。
<br/>　　第十重时光神殿之中，苏莫抛却脑中杂念，静修凝练修为。
<br/>　　五年的时间，匆匆而过，时空之城即将再次开启。
<br/>　　而这个时候，苏莫的修为，距离冲击五品的境界，还远远不够，最少好需要五年的时间。
<br/>　　“先将修为提升到四品高级的巅峰再说！”
<br/>　　苏莫睁开了眼眸，开始吞服丹药，他的丹药极多，别说是修炼五品的境界，就是修炼到七八品的境界，也绰绰有余了。
<br/>　　大量的丹药入腹，药力滚滚，源力沸腾，苏莫极力的炼化了起来。
<br/>　　他的修为，再次开始快速的增长了起来，势头仿佛无法抑制。
<br/>　　仅仅是三个多时辰的时间，苏莫的修为，便提升到了四品高级的巅峰，距离无品修为，也只剩下了半步之遥。
<br/>　　但是，这半步之遥，犹如一座大山，他暂时无法跨越。
<br/>　　修为无法增长之后，苏莫将体内的药力炼化完毕之后，便继续凝练修为。
<br/>　　没过多久，九日之期到了，整个时光神殿猛然一颤，外界有光芒照射到了殿中。
<br/>　　所有人都结束了修炼，向外而去，苏莫混迹在人群之中，也向外而去。
<br/>　　公孙嘉也在第十重时光神殿之中，他瞥了苏莫一眼，若有所思。
<br/>　　他心中也有猜测，不过，他并未去证实，这对于他和苏莫来说，都没有好处。
<br/>　　很快，苏莫来到了第一重时光神殿之中，他看到了伫立在神殿门前的贺兰殇。
<br/>　　而贺兰殇，眼眸锐利如鹰，扫视每一个出去之人，尤其是四品高级修为之人。
<br/>　　奈何，十几重时光神殿之中，人数太多，即便是四品高级之人，也有不下千人，他根本认不出苏莫。
<br/>　　“罢了！”
<br/>　　眼见所有人都鱼贯而出，贺兰殇微微的叹了口气，此事只能就此作罢了。
<br/>　　“贺兰兄，我们走吧！”公孙长绝带着几位天才，来到了贺兰殇的身边。
<br/>　　“公孙兄，等你多时了，我们走！”贺兰殇微微一笑，随即和公孙长绝等人，一起离开了时光神殿，返回到了极空城之中。
<br/>　　……
<br/>　　苏莫已经先一步离开了时光神殿，快步而去，很快便走出了极空城。
<br/>　　“贺兰殇到底是何目的？”离开极空城之后，苏莫飞驰在长空之中，暗自思索。
<br/>　　其实，他现在并不惧贺兰殇，虽然他的实力，可能远不如对方，但是他现在可是有终极神王这位护卫。
<br/>　　只不过，他不想把事情闹大，不然，到时候估计天王宫和地王宫的人都要引来了。
<br/>　　所以，他第一时间便选择了离开。
<br/>　　现如今，他的修为，已经达到了四品高级的巅峰，而距离千选战，还是四年多的时间。
<br/>　　虽然时间比较紧迫，但是，他还是有把握，突破到五品修为的。
<br/>　　随着思索，苏莫暗暗猜测，贺兰殇对他出手，是不是与不死神朝的意志有关。
<br/>　　“难道不死神朝已经被天王宫或者是地王宫拉拢了？”苏莫心中暗想，不过，他无法去确定。
<br/>　　唯一能确定的是，这一次千选战，可能会有很大的变故。
<br/>　　“玉虚宗！”
<br/>　　飞驰在长空之中，苏莫准备去玉虚宗看看，因为他知道，绫裳一直惦记着玉虚宗玉虚子。
<br/>　　只是，碍于他的吩咐，对方才没有去。
<br/>　　现在，他就在时空界域，准备去玉虚宗谈谈情况。
<br/>　　若是布下这个阴谋之人，还在玉虚宗，那么，便让终极神王一举击杀。
<br/>　　玉虚宗就在时空界域，是属于永恒神宫的下属势力，苏莫稍一打探，便得知了玉虚宗的所在。
<br/>　　一日之后，苏莫的身形，降落在了一片密林的边缘地带。
<br/>　　抬眼完全，前方数千里之处，有一座大山，大山之上云雾缭绕，有连绵的建筑群，这就是玉虚宗的山门。
<br/>　　“前辈，你现身吧！”苏莫望着远处的玉虚宗，仿佛是自言自语一般。
<br/>　　嗡~~
<br/>　　苏莫话音一落，他身侧的一颗古树之下，一道矮壮的身影，凭空浮现，正是终极神王。
<br/>　　“何事？”终极神王面色冷漠的问道，可以看出，他给予苏莫当护卫，是非常的不爽。
<br/>　　他堂堂高级神王，神域的巅峰强者，给别人当护卫，这个消息若是传了出去，简直是贻笑大方。
<br/>　　“前辈陪我去玉虚宗一趟，若是有六品以上的人向我出手，你便击杀！”苏莫凝视着终极神王，郑重的说道。
<br/>　　虽然知道对方在暗中保护他，但他还是不放心，所以才把对方喊了出来。
<br/>　　“可以！”终极神王面色淡淡的说道。
<br/>　　“那走吧！”苏莫招呼一声，便大步而出，一步千丈，向玉虚宗大步走去。
<br/>　　终极神王也没有再隐藏，跟在苏莫的身后，向玉虚宗而去。
<br/>　　“小子，你在时光神殿之中遇到了敌人？”终极神王一边行走，一边向苏莫遇到。
<br/>　　他并没有进入时门之中，但是，看到苏莫匆匆离开，所以有这个猜测。
<br/>　　“嗯！”苏莫微微点了点头，并未否认。
<br/>　　“你知晓时空之城是什么吗？”终极神王又道。
<br/>　　“是什么？”苏莫反问道，这时空之城，不就是给予武者机缘和造化的地方吗？
<br/>　　“时空之城，乃是离开荒界的通道。”终极神王面带感慨之色的说道。
<br/>　　“什么？”苏莫闻言，顿时心中一震，脚步立刻便停了下来，面带不可思议之色的望着终极神王。
<br/>　　“前辈，你说时空之城，是离开荒界的通道？”苏莫不确定的问道，离开荒界？难道就能去往其它大世界？
<br/>　　传闻，有三千大世界，他所在的这个宇宙，乃至于虚建的神域，都只是在荒界之中。
<br/>　　“不错，时空之城，乃是永恒神宫当年的老祖时空至尊炼制，就是为了打通荒界和其它大世界的通道，不过可惜，永恒至尊失败了，也因此而陨落！”
<br/>　　终极神王叹声说道，当年的时空至尊，何等风采，在没有至强者的神域，就是神域无敌的存在。
<br/>　　但是，时空至尊为了超脱纪元，利用自己的时空之力，妄图打通大世界的通道，可到头来依旧黯然收场。
