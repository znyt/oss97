<br/>    十万大山之中，众多的消息满天飞，妖族们也是惶惶不安，诸位妖族霸主现在正在战场上用鲜血和生命争夺这天下的霸权。
<br/>    
<br/>        在这一片大陆之上，妖族并不强，仙门正道，魔宗几乎各自占据了一半的疆域，妖族在这一片大陆之上，并算不上是一股强大的势力，可以说，十万大山之中，便是妖族的天下，而现在，整个妖族的天下正在狂暴的变更。
<br/>    
<br/>        大妖王横空，搏杀于长空，血战在群山之间，妖族之中，大君之下第一人，金毛吼的领地，紫金山鲜血成河，流淌三日不绝。
<br/>    
<br/>        有人说，他们看到了金毛吼的两颗狮头，如同小山般大小，死不瞑目。
<br/>    
<br/>        有人说，他们看到了燕山大宗主卓青瑶出剑，一剑而过，千军辟易，群山倒悬。
<br/>    
<br/>        在大妖王的战场上，燕山以绝对的优势开始捕杀大妖王，而现在，所有妖族的目光凝聚在妖城，因为在妖城之前，燕山山主，那位一手掀起了蔓延整个十万大山赤潮的神秘存在，正在与妖城对峙。
<br/>    
<br/>        站在百里外的山峰上，陈明看向远处巍峨的妖城，百丈高的妖城，通体铁水浇灌，上面还绘制的有各种阵法，“真是好一座妖城！应该能够为赤潮提供不少的养料。”
<br/>    
<br/>        紫霞仙子问道：“你现在准备怎么办？”
<br/>    
<br/>        陈明笑道：“妖族大君不是想要见我吗，刚好我也想要见他，那就过去见见。”
<br/>    
<br/>        紫霞仙子道：“我也一起去！”
<br/>    
<br/>        陈明道：“你去做什么，你要是也去了，那就是现在就要斩了这家伙了，大君的实力，除了另外一位大君，根本没有其他人能够干扰。”
<br/>    
<br/>        紫霞仙子道：“那你若是去了，妖族大君突然发难怎么办，我还是想直接和你一起过去斩了他，以免夜长梦多！要知道，若是让他发现你的实力在他之上，他可是会向周围的其他鬼仙势力大君求救的。”
<br/>    
<br/>        陈明道：“既然这样，那等我布下一道阵法，以免我们斩杀妖族大君的时候，他派遣兵马袭击血衣剑卫。”
<br/>    
<br/>        实际上，两万五千血衣剑卫，打三十万这种散沙般的妖军，不是没有胜算，不过就算是赢了，也是险胜，自己也会折损不少兵马，这些可都是陈明的宝贝疙瘩，损失太多亏不起，还是布下一道阵法的好。
<br/>    
<br/>        陈明翻手一柄飞剑出现在脚底，行走在周围的大山之中，成为了一个地级阵法师，他已经能够看到更多，什么挥手间大地脉动，群山为之所动，操控整个天地，那是天级阵法师的造诣，虽然还是做不到，但是经过他布置一番之后，还是做的到的。
<br/>    
<br/>        陈明也没想布置下一个什么大杀阵，材料不够，布置下一个迷惑阵法就够了，让对方的三十万大军在阵法之中兜圈子，然后血衣剑卫找到部分敌军，一处一处的歼灭。
<br/>    
<br/>        陈明飞上天空之中，看向周围的山势，虽然看起来杂乱不堪，但是在他的眼中，却是有着一条条混乱的丝线，那是天地大势。
<br/>    
<br/>        山川河流，都是有着自己的势，不过太斑杂，不成规制。
<br/>    
<br/>        陈明要做的，就是把这一条条线整理清楚，形成一座阵法。
<br/>    
<br/>        紫霞仙子跟着陈明，问道：“你在这儿能看出来什么？”
<br/>    
<br/>        陈明飞身到了一座山峰之上，朝着一棵树踢了两脚，紫霞仙子分明的看到，这一座山峰朝着左边移动了一里。
<br/>    
<br/>        紫霞仙子的眼中露出一抹精芒，因为她在陈明的身上，并没有感受到任何法力的涌动，也就是说，陈明完全是靠着自己对阵法的造诣做到的，“阵法，还真是一种奇怪的修行体系。管不得有人说地级阵法师可以堪比王级，看来真的是这样。”
<br/>    
<br/>        紫霞仙子看不明白，陈明却是知道其中的玄妙，刚刚他踢的那棵树，乃是整座山的气势汇聚之地，凡人喜欢管这样的地方叫做龙脉。
<br/>    
<br/>        刚刚踢了两脚，便是改变了此处的龙脉。
<br/>    
<br/>        随着陈明在群山之间走走停停，紫霞仙子可以发现，在周围，竟然是升起了一阵阵的大雾，紫霞仙子道：“只是一阵阵大雾有什么用？一阵风不久刮开了吗？”
<br/>    
<br/>        陈明笑了笑，道：“你试试看。”
<br/>    
<br/>        紫霞仙子毕竟是仙门弟子，所学颇多，随手招引了一阵狂风，却是发现自己的狂风在大阵之中竟然是顺着山涧吹动，还是看不清地势，风一旦过去，瞬间又是大雾笼罩。
<br/>    
<br/>        紫霞仙子有些吃惊地看向周围，“原来阵法在战场上竟然这么厉害。”
<br/>    
<br/>        陈明随意道：“这还只是普通迷幻阵法，让人走进去了很难出来，不是大杀阵，大杀阵一旦展开，进入者都将死无葬身之地，以我现在的造诣，至少王级没得跑。”
<br/>    
<br/>        忙了三天之后，陈明来到苏青阳和雪自悟之前，然后道：“这群山之中有着一座阵法，一旦开启，大雾弥漫，会让冲击来的敌军分不清方向，你们就守候在这唯一的一块没有迷雾的地方，一旦有部分敌军发现了你们，你们就将其斩杀，算了，这样对于你们来说可能太复杂了，我做一块阵盘吧。”
<br/>    
<br/>        苏青阳：......
<br/>    
<br/>        雪自悟：......
<br/>    
<br/>        我们在宗师眼里是有多笨？
<br/>    
<br/>        陈明拿出一块银镜，飞上天空，倒影整个大阵方圆数百里，“天地虽大，一镜容之......”
<br/>    
<br/>        陈明的口中念念有词，将整个大阵之中的地界，全部笼罩在其中，陈明完成之后看向镜子里，已经是包容了方圆数百里，“别说，这玩意儿还有点像摄像头，倒是可以在燕山的统治范围之内重要的地方都弄上一个。”
<br/>    
<br/>        将银镜交给雪自悟和苏青阳，陈明道：“这银镜里可以清楚地看到大阵之中所有的动向，现在开始，这里的大军就暂时交给你们了。”
<br/>    
<br/>        苏青阳接过银镜，拍胸脯道：“放心吧，我们古仙门的天才，指挥个两万五千大军，轻松之极。”
<br/>    
<br/>        雪自悟也是道：“你们燕山，才这么点大军，也太小家子气了。”
<br/>    
<br/>        陈明看了两人一眼，越看越觉得不靠谱，不过现在也就这两货能用了，“要是谁办砸了，以后阴阳和合丹，我这儿没得卖。”
<br/>    
<br/>        苏青阳和雪自悟忽然脸上露出一抹坚毅之色，颇有慷慨赴死之意，“宗师，我等誓死完成任务！”
