　　“苏兄，我们现在怎么办？”聂乾和韩盈来到了苏莫的身边，聂乾问道。
<br/>　　此时，两人心中唏嘘不已，四名武帝境二重的人来敲诈他们，结果却被苏莫反敲诈了，这真是让人感觉有些怪异。
<br/>　　“我们回去吧！”苏莫叹了口气说道，现在得罪了这四人，继续留在魔云海已经不安全了，万一四人找来高手报复，那就麻烦了。
<br/>　　“好！”聂乾两人点了点头，他们也是这个意思。
<br/>　　随后，三人不再逗留，立刻动身返回。
<br/>　　数日之后，三人回到了神武学府，互相告知了自己的住处，约好以后再一起历练，便各自返回。
<br/>　　苏莫回到了自己的阁楼中，盘膝坐在蒲团上，检查华服青年四人的储物戒。
<br/>　　这四人的储物戒内，财富倒是不少，加起来足有九百块下品灵晶，再加上他从廖云剑身上获得的下品灵晶，足有一千两百余块了。
<br/>　　这么多灵晶，还有斩杀魔甲人获得的魔晶，足够苏莫吞噬两三次了。
<br/>　　当然，他现在卡在准帝之境，武魂不进化，就无法提升修为了。
<br/>　　除了灵晶之外，四人储物戒中，各种帝级兵器也有十几件，还有几枚武学玉简。
<br/>　　只不过，无论是兵器还是武学，都没有苏莫需要的。
<br/>　　这些宝物，他暂时全留着，待日后灵晶耗光之后，再售卖宝物。
<br/>　　静静的盘坐在蒲团上，苏莫有些头疼，他的武魂想要晋升到战魂，还有很长的一段路要走。
<br/>　　先不说武魂提升到天级九阶，他需要吞噬多少，就单是这奥义，就够他头疼的了。
<br/>　　不朽奥义，只能作为兼修奥义了，根本不适合凝聚战魂。
<br/>　　但他的吞噬武魂，需要和什么奥义相辅，才能最合适呢！
<br/>　　其实，苏莫现在完全可以尝试凝聚战魂了，但他力求完美，想要最强大的战魂。
<br/>　　因为战魂，也有品质高低之分。
<br/>　　战魂的品质，分为低级、中级、高级和完美。
<br/>　　比如那些天级一阶或二阶的武魂，就算是特殊的武魂，进化为战魂之后，也只能是低级战魂。
<br/>　　一些天级七、八阶的武魂，就算有非常契合的奥义，最多也只是高级战魂。
<br/>　　只有天级九阶武魂，配合最强大，最契合的奥义，才能铸就出完美战魂。
<br/>　　可以说，战魂的威力，不仅取决于是什么类型，还要取决于品质。
<br/>　　苏莫若是现在进化战魂，就算成功了，战魂的品质也不高，很可能是中级战魂，顶天了也就是高级战魂，绝对不可能是完美战魂。
<br/>　　这其中的差别可是非常的大，按照那位孙导师的讲授，同样的战魂，完美战魂要比高级战魂强大数倍，比中级战魂强大十几倍。
<br/>　　所以，苏莫要么就不进化战魂，只要进化便力求进化成完美战魂。
<br/>　　“学府中好像有奥义圣山，倒是可以去看看！”苏莫呢喃自语，奥义圣山是学府**弟子参悟奥义的地方，他可以去找寻合适的奥义。
<br/>　　想到此处，苏莫起身，准备去奥义圣山参悟奥义。
<br/>　　嗡~~
<br/>　　就在此刻，一边骤生，苏莫体内的吞噬武魂，突然波动了起来。
<br/>　　是吞噬武魂之内的那具魔魂，在剧烈的挣扎，想要挣脱出来。
<br/>　　“找死！”苏莫面上闪过狠厉之色，立刻催动了吞噬武魂。
<br/>　　吞噬武魂的吞噬之力，牢牢的束缚者魔魂，让其逃无可逃。
<br/>　　足足挣扎了一刻钟，魔魂也没有逃脱，然后便慢慢的平静了下来。
<br/>　　“可恶！”
<br/>　　蓦地，一声暴喝从魔魂中响起，充满了愤怒与不甘：“吾纵横一世，居然栽在了一个乳臭小儿的手中！”
<br/>　　“哼！谁让你想夺舍我，自找死路！”苏莫闻言，顿时冷笑一声，对于这种敌人，他当然不会手软。
<br/>　　魔魂听闻苏莫之言，沉默了片刻之后，再次出声了。
<br/>　　“小子，吾和汝做个交易如何？”魔魂缓缓的说道，声音变的颇为平静。
<br/>　　当然，魔魂的声音，只是意念中传出，只有苏莫的意念才能察觉得到。
<br/>　　“什么交易？”苏莫心中冷笑，但还是出言问道。
<br/>　　“放了吾，吾送你造化！”魔魂说道。
<br/>　　“你当我傻吗？放了你，你再夺舍我怎么办？或者直接逃走了，我到哪找你去？”苏莫不屑的说道，但他心中却是一动，自己说不定能从这魔魂身上套点好处。
<br/>　　毕竟，这魔魂可不简单，估计身前修为极高。
<br/>　　“汝大可放心，吾重创未愈，实力千不足一，无法伤害汝！”魔魂道。
<br/>　　“任你舌绽莲花，我也不信你，除非你先拿出好处！”苏莫轻笑了起来，完全不信对方。
<br/>　　魔魂闻言再次沉默了，良久之后，问道：“吾如何信汝？”
<br/>　　“你不信？那就在我武魂中好好待着吧！过不了多久，我就能将你彻底吞噬融合！”苏莫淡淡的说道。
<br/>　　“混账！”魔魂闻言怒喝一声，几乎要气疯了。
<br/>　　此刻的魔魂，心中悲凉不已，他若是在巅峰之时，吹口气都能将苏莫灭杀。
<br/>　　而现在，他虎落平阳，居然被一只小蚂蚁威胁，当真是世事无常。
<br/>　　“你没有选择，不想死就乖乖的拿出好处！”苏莫耸了耸肩，再次盘膝坐了下来。
<br/>　　他心中也是有些期待，这魔魂以前定然是超级强者，他若是能从对方身上获取一些好处，那真是再好不过了。
<br/>　　魔魂沉默不语，他本想出言糊弄苏莫，但苏莫根本就是个人精，完全让他没有半点主动权。
<br/>　　但他现在受制于人，却是不得不妥协。
<br/>　　因为他现在极为的虚弱，一旦苏莫日后突破到武帝境之后，他必然难以存活了！
<br/>　　“小子，吾可以指导汝修炼，这个造化如何？”良久之后，魔魂开口道。
<br/>　　他没有给予苏莫什么实质的好处，因为一旦给了好处，就是泼出去的水，不仅收不回来，还制衡不了苏莫。
<br/>　　但指导修炼就不同了，他有的是方法制衡苏莫。
<br/>　　“指导我？”
<br/>　　苏莫闻言，心中一动，好奇的问道：“你以前是什么修为境界？”
<br/>　　若是有一个强者指导，苏莫当然是非常愿意，毕竟他的修炼，很多都是自己摸索，知识太浅薄了！
<br/>　　“圣境！”魔魂淡漠的道。
<br/>　　苏莫闻言顿时心中一震，不免倒吸了一口凉气，居然比他猜测的还要恐怖！
<br/>　　对方，居然是武圣境的超级强者！
