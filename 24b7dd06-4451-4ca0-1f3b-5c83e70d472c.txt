　　“苏莫，你即将成为我的女婿，我自然会帮你，不过，具体事宜还要仔细商议！“帝释毫不犹豫的开口了，表示愿意一起对付黄泉魔宗。
<br/>　　这让苏莫有些意外，这个未来的老丈人，正是够可以的。
<br/>　　之前对方说要考虑几日，现在他拿出帝级功法的甜头，对方立刻就答应了。
<br/>　　“苏宫主，不瞒你说，老朽的确对帝级中品功法非常渴求，所以，也愿意全力助你！”
<br/>　　帝释表态之后，海明江也立刻表态了。
<br/>　　相比于帝释，海明江倒是更为坦荡一些，直言渴求帝级中品功法。
<br/>　　“好，有两位的相助，大事可期！”
<br/>　　苏莫面露微笑，继续道：“待灭了黄泉魔宗，帝级上品功法，我自然会双手奉上！”
<br/>　　帝释和海明江闻言，均是点了点头。
<br/>　　随即，海明江提醒道：“苏宫主，对付黄泉魔宗之事，还要妥善商议，切不可鲁莽！”
<br/>　　“这是自然，一切就等大婚之后再决定吧！”苏莫道。
<br/>　　随后，苏莫又和两人聊了片刻，询问了一番最近苍穹世界各大势力的动向。
<br/>　　最后，便命人将两大势力之人安排住处，带众人回去休息。
<br/>　　婚期还有两日，苏莫也没有再去修炼，他去了一趟水月宫。
<br/>　　夕儿已经出关，得到了水月大帝传承的她，气质稍稍有些改变。
<br/>　　这种气质无法形容，仿若夕儿拥有水一般的灵性。
<br/>　　“夕儿，两日后就是我们大婚之日了！”
<br/>　　苏莫和夕儿坐在水月大殿之外的台阶上，抓着夕儿的手，苏莫满面的柔情。
<br/>　　“嗯，苏莫哥哥，我等这一日，已经等了很久了！”夕儿俏脸上带着迷人的红晕，轻声说道。
<br/>　　“呵呵！你还记得我们第一次见面时的情景吗？”苏莫笑了笑，眸中露出回忆之色。
<br/>　　他也没有想到，当初的一次善举，就救了自己未来的老婆！
<br/>　　“当然记得！”
<br/>　　夕儿靠在苏莫的肩上，轻声道：“那时候我只是个没有修炼武道的普通人……！”
<br/>　　夕儿也是陷入了回忆之中，口中轻声述说着当初的一切。
<br/>　　苏莫静静的听着，从对方的话语中，他能感受到对方的心声。
<br/>　　最后，两人都不再说话，只是静静的坐着，相互依偎在一起，享受着悠然的时光。
<br/>　　陪伴了夕儿一整天，苏莫才回到自己的住处，随后他又进入了虚界神图之中，来到了芊寻月所在的山洞中。
<br/>　　芊寻月没有任何变化，依旧是静静的躺在玄天灵柩之中。
<br/>　　苏莫检查了一番，发现对方并没有任何好转，没有半点生命气息。
<br/>　　“我要成亲了！”望着眼眸紧闭的芊寻月，良久之后，苏莫轻声开口。
<br/>　　他不知道自己为什么在这个时候来到这里，但他觉得此事有必要告诉芊寻月。
<br/>　　对于芊寻月，苏莫心中除了伤感之外，还有些愧疚。
<br/>　　“不过你放心，我答应过你的事，就一定会做到，我会救活你！”
<br/>　　“只是希望你给我救活你的机会，你能撑到那个时候！”
<br/>　　“以前我的确是毁了你的清白，但一切都是为了救你！”
<br/>　　“……”
<br/>　　苏莫仿佛是无意识一般，有一句没一句的说着，良久之后，他才离开山洞。
<br/>　　苏莫没有注意到，在他离开之后，芊寻月的眼角，有微微的湿润。
<br/>　　……
<br/>　　苏莫大婚之日，终于如期而知。
<br/>　　清晨，天刚蒙蒙亮，整个苍穹神宫便沸腾了，苍穹神宫所有弟子，都结束了修炼，前来参加宫主苏莫的婚礼。
<br/>　　司空炎、闻人天都、姜枫然等几位传人，也都结束了闭关，纷纷走了出来。
<br/>　　白玉广场上，已经搭建了高高的礼台，以及在四方搭建了环形的观礼台，足以轻易容纳数万宾客。
<br/>　　礼台很大，前方摆放了一尊大鼎，后方则是摆放了两张华贵的座椅。
<br/>　　而在座椅上，端坐着苏洪和帝释，两人是这次大婚主角的高堂。
<br/>　　此刻苏莫和夕儿还没有出现，大婚由金一等几位傀儡人主持。
<br/>　　“太玄宗宗主到！”
<br/>　　随着一声高喝响起，一大群太玄宗的强者来到了苍穹神宫。
<br/>　　太玄宗带队之人，正是太玄宗宗主，究楚北。
<br/>　　苍穹神宫的水幕阵法，已经打开了一条宽广的通道，由石一和火一带领上百名弟子在通道口迎接宾客！
<br/>　　“战神殿殿主到！”
<br/>　　“独孤世家家主到！”
<br/>　　“天火门门主到！”
<br/>　　“……”
<br/>　　随着太玄宗的到来，战神殿、灭世剑宗等超级势力，以及很多略次一些的势力纷纷到来。
<br/>　　其实，很多势力都提前来了，不过没到大婚之日，众势力之人都在不远处的通阳城中。
<br/>　　不得不提的是，因为苍穹神宫的山门，就悬浮在通阳城外的空中，这段时间通阳城也因此繁华了十倍。
<br/>　　无数各方的武者都慕名而来，几乎将通阳城挤爆了。
<br/>　　更有不少年轻的武者，在等待苍穹神宫遭受弟子，但却是迟迟等不到。
<br/>　　一个个势力不断的到来，全部来到了广场上，在金一等人的招呼下，就座于观礼台之上。
<br/>　　每个到来的势力，都带来了贺礼，交给了金一等人。
<br/>　　快到中午时分，足足有数百个势力来到了苍穹神宫，前来参加苏莫的婚礼。
<br/>　　这些势力都是苍穹世界有名的大势力，而一些小势力，根本没有资格进入苍穹神宫。
<br/>　　白玉广场上，一片欢腾，嘈杂的声音充斥四方。
<br/>　　而这个时候，苏莫还没有现身。
<br/>　　此刻的苏莫，正伫立在厚土宫的厚土塔之中，透过塔楼的窗口，静静的凝望着这一切。
<br/>　　苏莫穿着一身大红长袍，头戴高冠，显得器宇轩昂，气度不凡。
<br/>　　“北堂世家、殷家、阴阳神宫！”苏莫遥望着远处的白玉广场，低声自喃。
<br/>　　因为整个苍穹世界的大势力，除了黄泉魔宗之外，唯有这三个势力没有派人来参加他的大婚。
<br/>　　北堂世家和殷家，应该就是和黄泉魔宗同流合污的两大超级势力，这一点应该不会错了！
<br/>　　现在，则又加上了阴阳神宫！
<br/>　　阴阳神宫虽然不是人族九大势力，但也不可小觑，其并不比天涯海阁弱多少。
<br/>　　当初宏青璇被拍卖，便是阴阳神宫所有的拍卖行所为。
<br/>　　不过，让苏莫有些意外的是，独孤世家居然来了！
<br/>　　独孤世家和他有些仇怨，而且据海明江所言，独孤世家最近和黄泉魔宗多有接触。
<br/>　　苏莫本以为独孤世家也与黄泉魔宗合作了，不会来参加他的婚礼，但对方却是来了。
<br/>　　当然，即便来了，也不能百分百的表明独孤世家没有与黄泉魔宗合作。
<br/>　　片刻之后，苏莫见时间不早了，而该来的宾客都来了，不该来的也不会来了，他便离开了厚土塔，虚空踏步，向白玉广场而去。
