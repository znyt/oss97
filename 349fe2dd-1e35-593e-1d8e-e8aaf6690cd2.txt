　　第537章
<br/>　　落凤城。
<br/>　　李行哉回到了城主府，又将那份刚收到的情报看了一遍，之后扣放在帅案上。
<br/>　　“消息刚刚传过来，十天前，他在兖州出现，与兖州兵遭遇，之后便下落不明。”
<br/>　　李行哉手指敲击着桌面，脑海里思索着这件事。他来落凤城的时间还不算长，暂时稳住了形势。当然，情况还谈不上有多理想。之后，他便选出人专门负责收集情报，但能用的人手还不多，所以消息传到这里的时间要稍晚一些。
<br/>　　“能够确认是他么？”李行哉不放心的问了一句。
<br/>　　“暂时已可确定，兖州王已经派出高手，兴师动众。看这动静，敌人的来头当是不简单。青州徐州豫州冀州相应都有安排，招贤馆都派出了高手。”
<br/>　　“啧，他怎么来了呀。”李行哉叹了一声，呐呐自语道：“不好好在凉州窝着，跑东海干什么。”
<br/>　　“属下已下令去查，一旦有消息，立刻禀报王上。”
<br/>　　“罢了，罢了。”李行哉挥挥手：“城里事情多，已分不出人手专门做这件事。戚将军，你也莫要事事躬亲，该歇息还是歇息，毕竟身体要紧啊。”
<br/>　　李行哉是真担心戚继光有个头疼脑热，三长两短。自从戚继光来到落凤城后，李行哉顿时如虎添翼，原先无处着手的糟糕问题，颇有几分迎刃而解的架势。
<br/>　　不过，落凤城的人手还是太少了，诸般事压在戚继光一人身上，也无人能与他分担。
<br/>　　戚继光眼中一热，咚地一声跪倒：“士为知己者死，能替王上纷忧，属下在所不辞。”
<br/>　　“好啦好啦，快起来说话，你我不需如此。”
<br/>　　李行哉忙将他扶起，又亲自给戚继光倒了杯茶，当然，城主府也没有可用的下人。
<br/>　　“程大雷的问题先不去管他。还是我们自己的事情要紧。自从城中的守军交给戚将军整治后，军容一改从前，连续几次对海盗的剿灭战，都是我们获胜。今天又消灭了一小波海盗，我估计之后，便不会有海盗轻易扰乱我落凤城。”李行哉顿了顿，道：“但是，我们的兵力还是太弱，城中守军算上最近招募的，也不过一千余人。要想打仗，手上还是得有兵啊。”
<br/>　　“这个问题，属下已经想到了，也琢磨出个法子。”
<br/>　　“喔，戚将军当真是想本王之未想，忧本王之未忧，你有什么法子，快说来听听。”李行哉赶忙道。
<br/>　　“落凤城土地贫瘠，周围又有强敌环伺。但是落凤城也同样有一件其他地方没有的特产。”
<br/>　　“什么特产？”李行哉心里有些困惑：“是铁还是铜啊，我来也有段时间了，怎么从没听说过。”
<br/>　　“是海盗。”
<br/>　　“呃……”李行哉无语。
<br/>　　“海盗活跃在大海之上，时而上岸劫掠，时而打劫来往船只。在他们盘踞的一座座小岛上，都有大量财富。”
<br/>　　“你的意思是，海盗抢劫客商，我们抢劫海盗，让一伙伙海盗，成为我们的钱包？”李行哉睁大眼睛，然后立刻摇摇头：“不行不行，这件事说说容易，可要出海扫平海盗，所花费可不是一笔小数目，既需要银子来造船，也需要兵力来打仗。而关键是，这两样东西我都没用。”
<br/>　　“有了兵就不愁没钱，而有了钱，就不愁没兵。”
<br/>　　“但如果两样东西都没有呢？”李行哉没好气的问。
<br/>　　戚继光没用回答，确切来说，是没有办法回答。纵然他天赋异禀，对练兵一道极有天赋，但巧妇难为无米之炊，有些事也并不是只靠脑袋就可以办到的。
<br/>　　“好啦，好啦。”李行哉安慰道：“咱们不要求快，没有钱就慢慢想办法。没有海盗，还有山贼么，抢他们也是一样。一步步发展，等有了钱造船，我们再去海盗。”
<br/>　　“王上言之有理。”
<br/>　　“报！”
<br/>　　一名小卒从外面过来，冲李行哉行过礼后，道：“启禀王上，门外有人求见。”
<br/>　　“喔，什么人呀，带上来见我。”
<br/>　　李行哉最近也在招兵买马，虽然没有财力做支撑，但免不了有眼瞎的投到他麾下。比如这戚继光，不就是上天送到他身边的名将么。
<br/>　　他也很期待，这次会不会又是个如戚继光一样，眼睛比较瞎，但脑袋很好使的人前来投靠自己。
<br/>　　眼巴巴瞅着，就见一队人从门外涌进来，为首一个白发无须老者，走进大厅便立刻跪倒：
<br/>　　“六王子在上，请受老奴一拜，老奴可算见到殿下了。”
<br/>　　李行哉有些懵，他见这队人有七八个。人人锦衣华服，一脸豪横，看他们身上的穿着，应该都很有钱。
<br/>　　“老丈快快请起，这是怎样话说，为何如此这般？”李行哉走出帅案，想要将老者扶起。
<br/>　　可这老者的情绪明显很激动，脸上老泪纵横，李行哉很不容易，才算将他扶起。
<br/>　　“老奴御前从三品太监孙德龙，参见六王子，求殿下宽恕老臣之罪。”
<br/>　　李行哉还是有些懵，不知道是如何一回事。
<br/>　　这老者便是扬州千舟城漕帮当家，原先随明帝逃出京师的老太监孙德龙。虽然明帝已经死了，但孙德龙对帝国的忠心却是从未变过。现在得知李行哉被封逍遥王，于是就带着手下人前来投奔。
<br/>　　那已是三十几年前的事情了，那时候李行哉还未出生，根本不知道这孙德龙是谁。他断断续续听着孙德龙的话，算是明白事情的原委。
<br/>　　尤其一个词，令李行哉的心狠狠颤了一下：漕帮。
<br/>　　漕帮很有钱。
<br/>　　“当年与先帝分开，已过了三十余年，谁料先帝爷竟已归天。”孙德龙语带哽咽：“我漕帮大小儿郎三千人，都愿在殿下身前效力，为殿下粉身碎骨，在所不辞。”
<br/>　　“言重了，言重了。”李行哉赶忙拉拢人心：“帝国有你这般忠臣，何愁不能匡扶社稷，再造一个盛世。”
<br/>　　“殿下，前些年，您曾派人去江南取粮。之后，事情还顺利么？”
<br/>　　李行哉眨眨眼睛：“前些年，什么事情？”
<br/>(本章完)
