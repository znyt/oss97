　　苏莫和司空炎、清雅等人，商议了一个时辰。
<br/>　　这一个时辰之中，苏莫告知了清雅，人王宫的大体情况。
<br/>　　也与姜枫然、闻人天都、战乱三人，详细的探讨了一番具体挑唆天王宫和地王宫的办法。
<br/>　　更告知了司空炎，终极神王殿的大体情况，以及紫霄、绫裳等人的关系。
<br/>　　随后，他便命龙腾回龙都，准备特制传讯符。
<br/>　　而他，则带着李峰，进入了一个偏殿之中，准备赐予李峰元神。
<br/>　　苏莫的元神不多，除了他自己的混沌吞噬元神，两种混沌万化神剑元神，混沌不败战气元神之外，还有一个千幻元神和天罚神瞳元神。
<br/>　　千幻元神自己要使用，也不好给予李峰，只能给予对方天罚神瞳元神了。
<br/>　　此元神乃是云沐的元神，极为强大，但对于苏莫来说，用处并不大。
<br/>　　毕竟，他的精神力有些，无法催动太多的超级元神，此天罚神瞳元神自己留着，也没有机会使用。
<br/>　　最主要的是，他将来还能得到更强的元神，所以也不在乎天罚神瞳元神了。
<br/>　　“李峰，我们开始吧！”苏莫和李峰相对盘膝而坐，沉声说道。
<br/>　　“嗯！”李峰点头，随即闭上了眼眸。
<br/>　　随后，苏莫开始试着为李峰融合元神，但是，仅仅是片刻，他就放弃了。
<br/>　　因为，李峰的神魂实在太弱了，别说是融合元神，就算是融合战魂都不可能成功。
<br/>　　融合乍一开始，李峰的神魂就极度不稳，脸色瞬间惨白。
<br/>　　“你没事吧？”苏莫面色凝重的问道，他不敢尝试了，若是继续融合下去，对方的神魂会瞬间崩溃。
<br/>　　“没事！既然无法融合，那就算了！”李峰摇了摇头说道。
<br/>　　“李峰，你去霸天神朝之后，肉身修炼先放一放，先把修为提升起来，待你修为提高，神魂壮大，我再给你融合元神。”苏莫告诫道。
<br/>　　李峰也有武魂，虽然只是人级的武魂，非常的弱小，但修炼还是没有问题的。
<br/>　　在神域，只要是有武魂，在大量上等资源的堆积之下，李峰达到武皇乃至武帝，并不是多么的困难。
<br/>　　再者，李峰的肉身极为强大，体质早已不是凡体，修炼起来应该会更快不少。
<br/>　　在宇宙之时，数十年或许很漫长，但是在神域，数十年时间，对于高级强者而言，只是弹指一瞬间。
<br/>　　“如果霸天神朝，有肉身的修炼大道，我还是会专修肉身！”李峰郑重的道。
<br/>　　“但是，你若是专修肉身，寿元很快会耗尽！”苏莫无奈的摇了摇头。
<br/>　　如今的李峰，虽然依旧年轻，但是寿元并不会太多。
<br/>　　“其实，我自己已经摸索出了肉身之道大概的走向，但是还需要一番印证，去霸天神朝，或许能让我豁然开朗！”李峰道，他的肉身极度强大，肉身之中产生了浑厚的气力，这气力同样妙用无穷。
<br/>　　他所要做的，是如何运用这肉身气力，逆天改命，以力证道。
<br/>　　“好吧，我帮不了你，不过，我想说的是，若是走不通，立刻回头！”苏莫告诫道。
<br/>　　“放心吧，我自有打算！”李峰点了点头。
<br/>　　随后，两人也没有多谈，再次回到了大厅之中，和众人汇聚在一起，一边互相闲谈，一边等待龙腾回来。
<br/>　　众人并未等待多久，约莫两个时辰之后，龙腾回来了。
<br/>　　“怎么样？”苏莫急忙起身问道。
<br/>　　“我在龙族之内，取来了龙族特制的传讯符，虽然不是最好的，但是也是顶级的！”龙腾手掌一番，掌中出现了九枚传讯符。
<br/>　　这九枚传讯符，全部呈纯金之色，只有孩童手掌大小，其上雕刻着栩栩如生的金龙。
<br/>　　“这传讯符，传讯距离有多远？”苏莫沉声问道。
<br/>　　“三百万里左右！”龙腾道。
<br/>　　苏莫闻言，有些失望，三百万里，这个距离实在是太短了。
<br/>　　就算在同一个神朝之内，距离一旦过远，都无法正常传第消息。
<br/>　　但是，他也没有办法，若是在宇宙之中，他还能通过在众人的识海之中，留下一缕精神意念，用来远距离传讯。
<br/>　　但是在神域，精神意念的力量，被极度的压制，精神感知力暴降万倍，这个方法目前行不通。
<br/>　　“大家一人一枚传讯符，日后，这座府邸就是苍穹神宫的山门，若是无法传递消息，那就来此府邸，或者去找龙腾！”苏莫收起一枚传讯符，向众人吩咐道。
<br/>　　“明白！”众人纷纷点头，随即均是收起了一枚传讯符。
<br/>　　“好了，大家可以行动了！”
<br/>　　苏莫郑重的说道，言罢，他手掌一番，掌中出现了七枚储物戒，然后一一分发给司空炎七人。
<br/>　　“这储物戒之中是五行源石，数量不多，每人十万中品源石，也足够你们消耗一段时间了，若是源石与自身修炼的属性不同，可以到外面兑换！”苏莫沉声说道，他并未给予龙腾源石，因为龙腾并不需要，但是其他人却是需要。
<br/>　　众人闻言，纷纷检查了一番自己的储物戒，司空炎和战乱、清雅三人，一人修炼火源之道，一人修炼土源之道，一人修炼木源之道，源石与自身吻合，其他人就不吻合了。
<br/>　　“苏莫，保重！”
<br/>　　姜枫然向苏莫抱拳，郑重的说道：“这一次，我定然天王宫和地王宫彻底大乱！”
<br/>　　“一切小心！”苏莫点了点头，之前他已经详细的和对方商议了具体的计划，只要不出大问题，基本上行动能成功。
<br/>　　当然，事实如何，还要看具体情况。
<br/>　　“告辞！”姜枫然深深的吸了口气，随即转身大步而去。
<br/>　　“等等！”就在此刻，苏莫突然出声，喊住了姜枫然。
<br/>　　“从今日起，你们几人不能暴露真实名讳！”苏莫告诫道，他此刻才想起来，当初帝一魂在下界，可是有万界山，肯定知晓他很多消息。
<br/>　　所以，姜枫然等人的存在，帝一魂也可能知晓，所以，为了安全起见，大家还需要改名换姓。
<br/>　　“明白！”姜枫然点头，随即大步离开了府邸，向天王宫而去。
<br/>　　“苏莫，我们也走了！”
<br/>　　“保重！”
<br/>　　“我们一定会完成任务！”
<br/>　　众人纷纷向苏莫提出了告辞，苏莫一一向众人抱拳，片刻之后，大厅之中，只剩下了苏莫和龙腾两人。
<br/>　　“苏莫，真的能成功吗？”龙腾面带忧愁，实际上他并不看好苏莫，这倒不是苏莫计划不好，而是敌人太强了。
<br/>　　无论是帝一魂和拓跋青云，亦或是那位紫霄，均是神王级别的强者，而且均是中级神王。
<br/>　　实力差距太悬殊了，在绝对的力量面前，有时候任何的计谋，都不值一提。
<br/>　　“走一步看一步吧！”苏莫叹了口气，他也不知道将来如何。
<br/>　　事实上，适才他也考虑过，要不要把司空炎等人牵扯进来。
<br/>　　毕竟，此事极为危险，一个不慎就是生死魂灭。
<br/>　　不过，他也算了解司空炎等人，绝对不会置身事外，所以就这么安排了。
<br/>　　好在，司空炎等人，天赋均是被他调教的极度妖孽，超越万古天骄，只要时间足够，他们都能成长起来。
<br/>　　“需要我做什么？”龙腾亦是叹了口气，他终不能坐视不理吧。
<br/>　　“你就安心在龙族修炼吧，若是将来能控制龙族一部分势力，那就再好不过了。”苏莫沉声道。
<br/>　　“这个问题不大，但是需要大量的时间！”龙腾道，他的天赋在龙族之中，虽然不是数一数二的存在，但也是年轻一代之中，最顶尖的一批人之一。
<br/>　　所以，只要有足够的时间，控制龙族一部分势力，还是能做到的。
<br/>　　但是，他却是清楚，苏莫没有足够的时间了。
<br/>　　“有机会的，抓一条龙魂给我！”苏莫沉吟着说道。
<br/>　　“好！”龙腾点了点头，他也没有询问，苏莫为何要龙魂，但只要苏莫需要，他必定做到。
<br/>　　随后，龙腾又何苏莫聊了片刻，也告辞离去，这座府邸之中，便只剩下了苏莫一人。
<br/>　　苏莫在府邸之中，找到了一座密室，盘膝而坐，便准备开始了修炼。
<br/>　　同时，他也没有坐以待毙，心神沟通了自己的分身。
<br/>　　他在外界还有一具分身，这具分身的身上，有他的天梦元神。
<br/>　　这是苏莫当初留下的后手，防止本体陨落，而彻底死亡，所以才留下的这具分身。
<br/>　　这具分身，目前就在天龙神朝的边缘之地，距离他并不是很远。
<br/>　　沟通了分身之后，苏莫向分身，也吩咐下去了任务。
<br/>　　随后，他才开始安心的修炼了起来。
