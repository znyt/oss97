　　苏莫在雅翠居静等了三天，古天意才再次来到了雅翠居。
<br/>　　宫殿中，苏莫和古天意相对而坐。
<br/>　　“上官兄，已经安排好了，现在你就可以去赤龙营！“古天意笑着说道。
<br/>　　“那好，我立刻前去！”苏莫沉声说道。
<br/>　　“上官兄，莫急！”
<br/>　　古天意摆了摆手，道：“我先和你说一番赤龙营的情况，到时候你到了赤龙营，也好行事！”
<br/>　　“你说吧！”苏莫点头。
<br/>　　“是这样的，赤龙营是一个特殊的地方，所有将士均是来自圣朝各地的天才，竞争颇为激烈，希望上官兄到了赤龙营之后，勇于表现，能获得更高的职位！“古天意沉声说道。
<br/>　　“更高的职位？”苏莫闻言面带疑惑之色，他刚进赤龙营的话，如何能获得更高的职位？
<br/>　　“不错，赤龙营共有将士三千，分为三个大队，三十个中队，三百个小队，以你的实力，先夺得一个小队长，没有任何的问题！”古天意说道。
<br/>　　“如何夺得？”苏莫问道。
<br/>　　“挑战！”
<br/>　　古天意微微一笑，继续道：“你加入一个小队之后，只要能打败小队长，就能取而代之。”
<br/>　　“明白了！”苏莫点了点头，看来是以实力来定职位。
<br/>　　“还有，赤龙营之中，有一些大哥和三哥的人，他们知道你是我的人，一定会刁难你，希望你有心理准备！”古天意继续道。
<br/>　　“嗯！”苏莫继续点头，同时心中不免无奈，看来古天意给他安排的职位，也不是什么安生的地方。
<br/>　　对方这是要让他打前锋，深入龙潭虎穴啊！
<br/>　　“好，你现在就去赤龙营吧！”
<br/>　　古天意言罢，取出了一枚玉简，递给了苏莫，道：“这是入营文书，赤龙营所在之地，就在城西！”
<br/>　　“你不去？”苏莫接过玉简，诧异的问道。
<br/>　　“不去了，我现在就返回神武学府，过段时间将你的两位朋友一起带来！”古天意说道。
<br/>　　“好！”苏莫点了点头，随即便没有再多言，告辞离开。
<br/>　　古天意派了一位仆人，为苏莫引路，没过多久便走出了皇宫。
<br/>　　离开皇宫之后，苏莫一步百丈，直奔城西而去。
<br/>　　……
<br/>　　赤龙营，是一处军营，坐落在圣皇城的西部位置，占地方圆二十里。
<br/>　　整个赤龙营的地域，都用高达十数丈的围墙围了起来，犹如是一座小城。
<br/>　　赤龙营大门之前，摆放着两尊巨大的赤色石雕。
<br/>　　这石雕就是两条神龙，每一条都长达数十丈，栩栩如生，而又狰狞恐怖。
<br/>　　两座石雕屹立在赤龙营门前，无形的气势散发而出，森然霸道，让得任何人都不敢靠近。
<br/>　　两名身穿赤色铠甲的年轻士兵，伫立在大门之前，眼神冷漠，气息澎湃。
<br/>　　嗒！嗒！嗒！
<br/>　　脚步声响起，一道身影由远及近，很快便来到了赤龙营的门前。
<br/>　　“站住，赤龙营重地，闲杂人等严禁靠近！”两名守门的士兵，见到快速到来的身影，立刻厉喝出声。
<br/>　　唰！
<br/>　　瞬息之间，来人停下了脚步，停在了赤龙营大门之前。
<br/>　　来人自然便是苏莫。
<br/>　　苏莫抬眼打量了一番赤龙营，以及门前两只巨大的赤龙石雕，神情讶然，这赤龙营倒是颇具威势。
<br/>　　随即，他手掌一翻，取出了玉简，向两名守门士兵说道：“我来加入赤龙营，这是入营文书！”
<br/>　　言罢，他手掌一抛，玉简便向其中一名守门士兵。
<br/>　　“加入赤龙营？”
<br/>　　士兵接过玉简，稍一查探，便点头道：“你在此等候，我去为你通报！”
<br/>　　言罢，这名年轻的士兵便转身走进了赤龙营中。
<br/>　　“多谢！”苏莫微微抱了抱拳，便静心等待了起来。
<br/>　　同时，苏莫亦是暗暗惊讶，这两名守门的赤龙营士兵，修为均是高达武帝境七重，真是不简单。
<br/>　　看来这个赤龙营，真的非同一般。
<br/>　　约莫过去盏茶的功夫之后，那名前去通报的士兵回来了。
<br/>　　“你跟我来！”士兵对苏莫喊道。
<br/>　　随即，苏莫便跟着这名士兵，走进了赤龙营之中。
<br/>　　赤龙营中，首先就是一个巨大的校场，占地广阔，方圆近万丈，边缘处耸立着一座高高的点将台。
<br/>　　点将台上，插着一根巨大的旗幡，旗幡上绣着一条赤龙栩栩如生。
<br/>　　赤龙营中的建筑，全部都是古朴的石殿，连绵成片，一眼看去除了一小部分略大之外，其它的都没有任何的区别。
<br/>　　不多时，苏莫被带到了一座较大的石殿之前。
<br/>　　石殿大门敞开，其内是一个大厅，主座上端坐着一名面容俊朗的青年，同样身穿赤色铠甲。
<br/>　　而在俊朗青年的下首，左右两边还各坐着一名青年，同样的身穿赤色铠甲，一名身材壮硕，一名身材高瘦。
<br/>　　这三人之中，除了俊朗青年是武尊境修为之外，其它两人都只是武帝境九重武者。
<br/>　　“禀中队长，人带来了！”带路的士兵向俊朗青年抱拳说道。
<br/>　　“进来吧！”俊朗青年目光望着苏莫，随即对带路的士兵挥了挥手。
<br/>　　那名带路的士兵直接转身离去，苏莫则是大步走进了石殿中，不过，此刻他不禁眉头微皱了起来。
<br/>　　因为主座上的那名俊朗青年，他居然认识，正是曾经在黑山要塞的祝英。
<br/>　　当初在黑山要塞，祝英不让他们一众神武学府的弟子进要塞之中，让他们去镇守环山矿脉。
<br/>　　后来他盗取了环山矿脉，祝英赶来之后，气的吐血，还要调查他们所有人。
<br/>　　当时，他为了不被调查，还和祝英顶了几句，惹得对方大怒。
<br/>　　苏莫心中暗凛，立刻收敛了全身的气息，虽然祝英不认识他，但毕竟有过一面之缘，虽然他改变了面目，还是很容易将他认出来。
<br/>　　苏莫不敢大意，祝英可是三皇子的人，若是知道了他的身份就麻烦了。
<br/>　　“你叫上官昊？八皇子推荐而来？”待苏莫走进石殿，祝英打量了一番，淡淡的问道。
<br/>　　“正是！”苏莫点了点头。
<br/>　　“你曾经是神武学府的弟子？”
<br/>　　祝英继续问道，他感觉对苏莫有些熟悉之感，而且因为古天意是神武学府的圣子，其推荐的人多是神武学府的弟子，所以他有此一问。
<br/>　　“不是！”苏莫立刻摇头否认。
<br/>　　“嗯！武帝境八重境界，修为还不错！”
<br/>　　祝英微微颌首，便也没有多想，随即看向下首的两名青年，道：“你们两人的小队，均是各缺一人，这位上官昊，你们谁愿意要？”
<br/>　　那两名青年闻言，仔细的打量了苏莫一番，其中那名高瘦青年立刻起身，轻笑道：“中队长，让他来我们第29小队吧！”
<br/>　　“好，赖羽，他就加入你29小队了！”祝英没有多言，立刻就同意了。
<br/>　　“是！”高瘦青年赖羽点头，随即对苏莫道：“上官昊，跟我来，去29小队！”
<br/>　　言罢，赖羽大步的走出了石殿，苏莫没有反对，便跟着对方而去。
<br/>　　望着苏莫离去的背影，祝英的嘴角一勾，露出了一丝嘲弄之色。
<br/>　　“八皇子居然也想插足赤龙营，真是天真啊！”祝英低声自语，他不可能让八皇子的人留在赤龙营！
<br/>　　“中队长，你无需操心，赖羽会整死这家伙的！”另一名壮硕的青年笑着说道。
<br/>　　“这是自然！”祝英闻言一笑，赖羽的手段，他还是相信的。
