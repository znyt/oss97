<br/>　　然后一个医护人员拿着个仪器从头到脚把林峰给扫了足足三遍。
<br/>　　随后示意林峰可以进去。
<br/>　　“这是你们的临时身份证。”

<br/>　　林峰是888888.

<br/>　　“你们看里面居然还有个墙壁，不过这个要矮一些，只有十米的样子。”
<br/>　　众人可以看到尽头还有个墙壁树立起来。远远的看过去就好像是一条黑色的黑色大蛇一样匍匐在地上。
<br/>　　“看来最里面住的都是些大佬。”
<br/>　　林峰道。
<br/>　　在这个基地内。
<br/>　　街道两边有许多商贩摆地摊，卖什么的都有。
<br/>　　卖衣服的，卖鞋子的，或者是卖食物的，甚至还有卖武器的，当然卖的都是些砍刀扳手水管之类的。

<br/>　　“我要这个，这瓶水我要了，多少黄金！”
<br/>　　一个看起来像是个富商的大胖子拎着个箱子道。
<br/>　　“三根金条！”
<br/>　　商贩伸出三根手指头道。
<br/>　　“三根？这么贵？你这瓶水还只有半瓶而已。”
<br/>　　大胖子顿时愣了。
<br/>　　“爱要不要，不要拉倒。”
<br/>　　商贩呵呵一笑。压根不担心自己的东西卖不出去。
<br/>　　“我要了。”
<br/>　　大胖子一咬牙拿出了三根金条给了对方。
<br/>　　另外一处。
<br/>　　“你这砍刀能不能便宜点。”
<br/>　　“不能，少一个子都不卖。”

<br/>　　“你把砍刀卖我，我给你法拉利，这是我爸给我的，刚开了不到半年，一辆四百万的跑车换你的破砍刀。”

<br/>　　“跑车啊，你难道不知道吗？”年轻人愣了。

<br/>　　“你……”
<br/>　　年轻人满脸难看，没想到他用自己的爱车换一把武器都没有办法换到。
