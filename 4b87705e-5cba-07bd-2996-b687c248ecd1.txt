　　离开大殿后，萧逸边走回后山，边思索着。
<br/>　　他有些不明白，为何自己要去东荒十八城，长老们会反应如此大。
<br/>　　他大概能想到是刚才和十位内门首席战斗时，自己展露出五百丈真气漩涡的缘故。
<br/>　　但，自己就算天赋再怎么高，也只是个天才罢了。
<br/>　　受师门长辈重视，很应该。
<br/>　　但像刚才那样，九大长老齐刷刷地盯住聂如山，这种反应，有些过了。
<br/>　　萧逸摇摇头，想不明白也懒得想。
<br/>　　现在，还只是早晨时分。
<br/>　　今日和十位内门首席的战斗，早上开始，但并没有持续多长时间。
<br/>　　距离大长老说的今晚才作决定，还有很多时间。
<br/>　　萧逸不会放过任何一丝修炼时间，在后山上盘膝坐下，开始一日的修炼。
<br/>　　只不过，还未修炼多久，两个时辰后，中午时分。
<br/>　　一个内门执事便忽然来了后山。
<br/>　　“萧逸，大长老有请，现在立刻去大殿。”
<br/>　　这位内门执事，语气中，带着平等和友好。
<br/>　　“好。”萧逸点点头，结束了修炼。
<br/>　　……
<br/>　　不多时，萧逸来到大殿。
<br/>　　十大长老，有九位都在，易老也赫然在此。
<br/>　　只有剑堂长老并没有来。
<br/>　　“参见众长老。”萧逸行了一礼。
<br/>　　“免礼。”大长老摆摆手，示意萧逸坐下。
<br/>　　萧逸坐到了易老身旁。
<br/>　　这时，大长老率先出言道，“你很想去东荒？”
<br/>　　“是。”萧逸回答道，“如今兽潮告急，东荒危在旦夕。
<br/>　　“作为武者，弟子有义务前往猎杀妖兽。”
<br/>　　“保绿光城安危。”
<br/>　　当然，对于武者来说，这也是一次不可多得的历练。
<br/>　　“好，我允了。”大长老点点头。
<br/>　　“听十一说，你掌握了五行炼药录，是吗？”
<br/>　　“是。”萧逸再次点点头。
<br/>　　“临行前，众长老给你准备了些相助之物。”大长老说着。
<br/>　　众长老接连拿出一个乾坤袋递给萧逸。
<br/>　　“谢过诸位长老。”萧逸接过九个乾坤袋，一一道谢。
<br/>　　“不打开看看吗？”大长老淡笑道。
<br/>　　“额。”萧逸微微尴尬。
<br/>　　长老们才刚给，出于礼貌，没理由立刻打开。
<br/>　　再者，他其实也能猜到乾坤袋里有什么东西。
<br/>　　不过，既然大长老这样说，他便打开来看了看。
<br/>　　乾坤袋内，都是天材地宝和妖兽内丹。
<br/>　　而且数量极其庞大。
<br/>　　最重要的是，里头竟还有五级妖兽内丹。
<br/>　　足足有500颗。
<br/>　　萧逸微微一惊，各位长老，好大的手笔。
<br/>　　就算是北山慕容家，倾尽盛宝商行的商物，也绝对凑不齐这个数。
<br/>　　裂天剑派，果然底蕴过人。
<br/>　　“谢过众长老。”萧逸站起身，重重地鞠了一躬。
<br/>　　大长老摆摆手，道，“此次兽潮，很严重，也很危险。”
<br/>　　“你要去，便先把修为提上来。”
<br/>　　“另外，我和三长老还有其它防身之物给你。”
<br/>　　这时，三长老拿出一物，交给萧逸。
<br/>　　道，“这是五火傀儡球，不过只是残次品。”
<br/>　　“内有五头破玄六重实力的傀儡。”
<br/>　　“嘶。”萧逸倒吸一口凉气。
<br/>　　虽说是残次品。
<br/>　　但，五头，破玄六重实力的傀儡，就是去攻打一个大势力都足够了。
<br/>　　三长老道，“此物，便暂时赠予你，让你到东荒时可有更高的自保之力。”
<br/>　　萧逸刚想答谢一句。
<br/>　　易老却抢先说道，“暂时？一个残次品，也好意思收回去。”
<br/>　　三长老苦笑一声，“我在北山郡，也是有数的五品炼药师。”
<br/>　　“但，五火傀儡球有多难炼制，十一你又不是不知道。”
<br/>　　“这已经是我的巅峰之作了。”
<br/>　　“再说了，这东西就算给了萧逸，萧逸也无法为它注入真气…”
<br/>　　易老打断道，“萧逸小子不行，我可以。”
<br/>　　三长老无奈道，“既然十一你这么说了，这颗五火傀儡球，我送给萧逸就是。”
<br/>　　萧逸接过。
<br/>　　半晌，大长老也拿出一物，道，“此乃五行铠甲，乃是剑派的重宝之一。”
<br/>　　“位列上品灵器。”
<br/>　　“以你现在的修为，穿上之后，破玄六重以下，休想伤你分毫。”
<br/>　　边说着，大长老紧张地看了易老一眼。
<br/>　　“十一，我可说明了啊，五行铠甲，乃是剑派历代传承的东西。”
<br/>　　“只能暂借给萧逸，可不能赠予。”
<br/>　　易老撇撇嘴，道，“堂堂裂天剑派大长老，如此小气，啧啧。”
<br/>　　“你不用激我。”大长老认真道，“我也随你怎么说，反正要归还。”
<br/>　　说罢，大长老将五行铠甲交到萧逸手上。
<br/>　　“谢大长老。”萧逸接过，行了一礼。
<br/>　　“不必谢。”大长老摆摆手，道，“你这师傅，狮子大开口，要送你是不可能的。”
<br/>　　“但，如果哪天你真正成为剑主了，这东西便随你处置。”
<br/>　　……
<br/>　　之后，大长老认真地嘱咐了萧逸几句后，众人纷纷离开大殿。
<br/>　　萧逸手中拿着九个乾坤袋，心头的疑惑却更浓郁了。
<br/>　　今天，算得上‘大丰收’。
<br/>　　无论是天材地宝、妖兽内丹，还是五火傀儡球，以及五行铠甲。
<br/>　　这些东西，加起来的价值，高得无法想象。
<br/>　　就是长老，本身也无权调动这么多东西。
<br/>　　可他一个弟子，却得到这些馈赠。
<br/>　　剑派对自己似乎太好了。
<br/>　　一直来到外门宝库，易老见他满脸疑惑的样子，便出言相问。
<br/>　　萧逸将心头疑惑道出。
<br/>　　易老淡淡一笑，道，“小子，你是不是拥有道体？”、
<br/>　　“而且还是比先天道体更强的道体，可以让你气泉内的真气一次性全部调动出来。”
<br/>　　“嗯。”萧逸如实道，“大半年前发现的，我姑且称之为冰火道体。”
<br/>　　萧逸自然不会道出是九粒五行丹所致，随便说了句自己发现的。
<br/>　　“那就是了。”易老点头道，“五百丈气泉，加之强大道体。”
<br/>　　“又领悟了10成极界碑。”
<br/>　　“一旦你真正成为剑主，你便是剑派历史上最强的一位剑主。”
<br/>　　“长老们自然对你青睐有加，更是不舍得让你去涉险。”
<br/>　　“最强的剑主？”萧逸皱了皱眉头。
<br/>　　“剑主二字，我不只一次听说过，这到底是什么？难道就是掌教？”
<br/>　　萧逸问道。
<br/>　　“当然不是。”易老回答道。
<br/>　　“无论是剑主还是掌教，都已经许多年没有出现了。”
<br/>　　“你不知道也是正常。”
<br/>　　“我这样跟你说吧。”
<br/>　　“剑主，是整个北山郡的剑主；而掌教，则只是裂天剑派的掌教。”
<br/>　　“两者的差别，你可想而知。”
<br/>　　“整个北山郡的剑主？”萧逸一愣。
