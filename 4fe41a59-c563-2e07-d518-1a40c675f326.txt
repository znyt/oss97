        刘婶点了点头：“烧早就退了，只是她看起来身子有些虚。不过，大少奶奶这一日三餐倒是吃得很勤快，晚上还加了一餐夜宵。只不过，每餐都必须是营养均衡搭配，所以，大少奶奶的口味有些叼了。”
<br/>
<br/>        连华生听后，一脸狐疑，顿了一下，才问道：“她人呢？”
<br/>
<br/>        “在餐厅里吃香菇鱼片粥，连医生要不要也来一碗？”刘婶接着说道。
<br/>
<br/>        连华生点了点头，然后只身往餐厅走去。
<br/>
<br/>        童筱颖坐在餐桌前，安安静静地喝鱼片粥。
<br/>
<br/>        这女人还真就很淡定啊！
<br/>
<br/>        连华生心里不爽，走过去后，随之在童筱颖的对面坐下。
<br/>
<br/>        刘婶端了一碗香菇鱼片粥过来，放在了连华生跟前的桌面上。
<br/>
<br/>        “……”童筱颖淡漠地抬眸看了连华生一眼，什么话也没说，继续喝自己的粥。
<br/>
<br/>        连华生舀了一勺放入嘴中，冷嘲热讽地说道：“这鱼片粥了，一般都是选用鳕鱼，或者其他少骨肉嫩的海水鱼来做。而你却选则鲫鱼来做，怎么，你这是怀孕了吗？”
<br/>
<br/>        童筱颖身心一怔，手上的动作定格了下来。
<br/>
<br/>        “你该不会是怀了傅昼景的孩子吧？”连华生忽然间替傅夜沉感到心塞起来。
<br/>
<br/>        傅夜沉离开的这七天里，她童筱颖对傅夜沉可谓是不闻不问，再说得不好听点就是漠不关心。
<br/>
<br/>        反倒是落以柔，天天打电话询问他关于傅夜沉的安危。
<br/>
<br/>        她童筱颖这当妻子的，还不如傅夜沉身边的一个女秘书。
<br/>
<br/>        一想到这，连华生就忍不住要挖苦她童筱颖。
<br/>
<br/>        童筱颖白了连华生一眼，冷淡地回答道：“我怀了谁的孩子，跟你有什么关系？”
<br/>
<br/>        “我和阿沉是出生入死的兄弟，你要是敢给阿沉戴绿帽，别怪我对你肚子里的孩子下手！”连华生目露凶光地呵斥。
<br/>
<br/>        童筱颖倒是被他这眼神里的戾气和话语里的锋锐给吓住了。
<br/>
<br/>        她垂下眼帘，讷讷地开口：“我还不知道我有没有怀孕，才七天而已。为了以防万一，还是提前替自己补补，想给大哥生个健健康康的孩子。”
<br/>
<br/>        “……”顿时，连华生被童筱颖这句话给堵得语塞了。
<br/>
<br/>        完全没料到，他家阿沉会这么腹黑！
<br/>
<br/>        临走前，还想着要给他自己留个种。
<br/>
<br/>        难道，他真怕她童筱颖跟别的男人跑了不成？
<br/>
<br/>        “你还要对我肚子里的未知数下手吗？”童筱颖深吸了口气，淡定地反问。
<br/>
<br/>        连华生咽了咽口水，十分别扭地回答道：“阿沉的孩子，就是我连华生的孩子。只要你肚子里怀的是阿沉的孩子！我就必定会护你们母子俩的周全！”
<br/>
<br/>        “你是在告诉我，他傅夜沉是有多可笑吗？自己的妻子怀孕了，竟然需要他的拜把子的兄弟来照顾。”童筱颖嘴不饶人。
<br/>
<br/>        连华生被她怼得哑口无言。
<br/>
<br/>        要不是傅夜沉不让他对她说出真相，他才不会让她逞口舌之风。
<br/>
<br/>        “童筱颖，我发现你还真就是一根筋。”连华生不打算跟她在傅夜沉的事情上绕弯子，反而跟她翻起旧账来，“三年前了，我就明里暗里地暗示过你，傅昼景不是你值得托付终身的男人。你当时还一根筋地一心向着那个渣男，什么事都帮着那个渣男，替那个渣男说话。现在吧！我怎么就没见你一心向着阿沉？不给阿沉找麻烦，你就已经很不错了。还有，我说你就不会多学学落以柔吗？落以柔对阿沉可谓是百依百顺，体贴入微。你再看看你……”
<br/>
<br/>        “三年前，傅昼景除了季思妍的那件事情以外，他对我的意义，不是你随随便便一个‘渣’字就能去批判的。你压根就不了解我和傅昼景之间的事情，你凭什么单方面的去批判他。至于我对大哥，你说我的不是，我无话可说。我也承认，我不及落以柔。但是，我和落以柔不一样。她对大哥的爱是很深，但这份深爱里还充斥着崇拜与目的。但我没有。我对大哥的爱里没有崇拜也没有目的。我爱他，爱得很纯粹，但可能只是爱得不深……可是，一个人对另一个人的爱，其实压根就无法去丈量的吧！”童筱颖慢条斯理地把话说得很认真。
<br/>
<br/>        “所以，你就仗着阿沉对你的爱，在为所欲为喽？”连华生铿锵有力地质问道。
<br/>
<br/>        童筱颖不假思索地否认：“我没有！”
<br/>
<br/>        “没有？没有的话，你要阿沉去救季雅珊？”连华生气恼道。
<br/>
<br/>        童筱颖反驳道：“救？！你们需要去救吗？你们只需要放人即可！是你们的人抓了雅珊！是你们的人对雅珊严刑逼供！对，你们是不会要了雅珊的命，你们确实也会留雅珊一条活口！但是，你们会用各种法子去逼雅珊说出你们想要知道的秘密。雅珊深爱着十爷，她不会背叛十爷，你们那么逼她，雅珊会怎么做？”
