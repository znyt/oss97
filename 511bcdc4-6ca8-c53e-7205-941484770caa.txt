　　苏莫三人，在云顶传送大殿中，乘用星空传送大阵，离开了古周星。
<br/>　　接下来得一段时间中，三人的身影，辗转一颗又一颗星辰。
<br/>　　由于有神武学府的身份令牌，有大半以上的星空传送阵，都是十三个超级势力掌控，所以三人花费的灵晶极少。
<br/>　　足足百日之后，三人到达了一颗名为青石星的星辰上。
<br/>　　这颗星辰是一颗废星，其上没有半点灵气，一片荒芜，全部是杂乱的青石。
<br/>　　这颗青石星，虽然荒芜，但却是古灵星河中一处很大的传送中转站。
<br/>　　一片辽阔的山谷中，足足布置了百余个星空传送大阵，通向不同的目的地。
<br/>　　山谷中人来人往，非常的热闹，除了人族之外，其它异族随处可见。
<br/>　　“苏兄，我们到了青石星，下一站就是巫族的天冥星了！”苏莫三人走出传送大阵，韩盈笑着说道。
<br/>　　这时，刚刚从传送阵中走出的苏莫，面部蠕动，瞬间便变换了容貌，变成了一个面容儒雅的青年。
<br/>　　此青石星中转站，来往的巫族之人不少，苏莫也是担心被人认出来，所以，刚踏出传送大阵，他就改变了容貌。
<br/>　　“苏兄，你这是……？”韩盈和聂乾见此，顿时面带不解之色。
<br/>　　苏莫眼眸扫视四方，见无人注意到自己，便意念向两人传音，道：“我与巫族有大仇，所以不能被认出来！”
<br/>　　“什么？”韩盈和聂乾闻听此言，顿时心中一惊。
<br/>　　“苏兄，你与巫族有仇，还敢去天冥星，那里可是巫族的地盘，一旦被发现，绝无逃生的可能！”聂乾同样意念传音。
<br/>　　若苏莫所言为真，那这次天冥星之行，可就危险了。
<br/>　　“我知道，但我有朋友被巫族之人抓走了，必须前去相救！”苏莫叹了口气。
<br/>　　聂乾和韩盈两人沉默了，原来苏莫来此，并不是单纯的历练。
<br/>　　不过，苏莫能为朋友只身涉险，这份情意倒是可敬可佩。
<br/>　　“两位，这次天冥星之行，可能非常的危险，你们还是去其它星辰历练吧！”苏莫见两人沉默不语，传音告诫道。
<br/>　　“苏兄，你这说的哪里话？你为朋友不惜只身涉险，我们也是一样！”聂乾沉声道。
<br/>　　“不错！苏兄你是我们的朋友，我们自然不能让你一人前去！”韩盈传音道。
<br/>　　两人均是面色真诚，苏莫对他们推心置腹，他们自然不会让苏莫一人去冒险。
<br/>　　苏莫目光凝视着两人，见两人面色坚定，便也没有多劝，此行虽然危险，但他们身在暗中，只要谋划得当，完全可以全身而退。
<br/>　　“能交到两位这样的朋友，是苏某之福，既然如此，我们就动身吧！”苏莫点了点头，向两人传音说道。
<br/>　　而后，三人没有在青石星过多停留，缴纳了灵晶之后，便踏上了前往巫族天冥星的传送阵。
<br/>　　因为这个传送中转站，是其它势力掌控，所以三人都需缴纳灵晶。
<br/>　　片刻之后，星空传送大阵启动，苏莫三人消失在了青石星。
<br/>　　……
<br/>　　一阵晕眩感过后，苏莫眼前恢复了清明。
<br/>　　抬眼看向四方，天空有些阴暗，乌云密布，而他出现在了一座古城之中。
<br/>　　这座古城很小，四方均是低矮的城墙，城内没有任何建筑，有的只是大量的星空传送大阵。
<br/>　　大阵数量极多，粗略一扫，不下四、五百个。
<br/>　　此古城，居然是一个专门进行星际传送的城池。
<br/>　　城内一座座星空传送大阵不断的闪烁，大量的武者来到此地，或离开此地。
<br/>　　这些武者有普通的人族，也有巫族之人。
<br/>　　“我们走吧！”苏莫招呼身边的聂乾和韩盈一声，便向城门之处飞去。
<br/>　　聂乾两人急忙跟上。
<br/>　　片刻间，三人便来到了城门之处。
<br/>　　城门处，有数十名巫族强者把手，个个气息浑厚，修为最低的都有武帝境五重境界，其中更是有几名尊者。
<br/>　　这些巫族人，眉心之间均是巫族的族印，一个三叉戟标准，所以非常的好认。
<br/>　　苏莫见此心中一凛，急忙收敛了自身气息，现在到了巫族的地盘，他不得不小心。
<br/>　　“所有外星之人，来到天冥星需缴纳十块下品灵晶，所有人排队依次缴纳！”一名中年的巫族强者朗声高喝。
<br/>　　苏莫闻言松了口气，原来只是要缴纳灵晶，这倒无妨！
<br/>　　不过，这巫族可真会敛财，其它星辰之人到此，一人十块下品灵晶，这真的是在抢劫。
<br/>　　但苏莫虽然心中腹诽，却丝毫不敢多言，他不能引起任何人的注意。
<br/>　　很快，苏莫三人便排队缴纳了灵晶，走出了这座微小的古城。
<br/>　　古城之外，是一望无际的荒原，充满着荒凉的气息。
<br/>　　“苏兄，我们接下来去哪里？”聂乾向苏莫问道。
<br/>　　“黑耀城！”苏莫轻声说道，巫天御便是居住在黑海耀城，他必须要去此城。
<br/>　　“黑耀城在何地？”韩盈问道。
<br/>　　“不清楚！我们先离开这里，再搞清楚黑耀城的所在！”
<br/>　　苏莫摇了摇头，身形冲天而起，向远处飞去。
<br/>　　聂乾和韩盈仅随其后。
<br/>　　三人一路疾行，飞行了两日之后，来到了一座叫‘五华’城的城池之中。
<br/>　　苏莫准备在此城暂留两日，搞清楚天冥星和黑耀城的情况，再做打算。
<br/>　　若是冒冒失失的前往黑耀城，实在太危险了。
<br/>　　五华城中，巫族之人奇多无比，大街小巷之中，巫族之人成群结队。
<br/>　　当然，普通的人族也不少，所以苏莫没有多少担心。
<br/>　　只不过，苏莫能清晰的感觉到，此地的人族之人，面对巫族之人，明显的挨了一截。
<br/>　　很多人族之人，对巫族之人均是恭敬无比，不敢有丝毫的不敬。
<br/>　　而且很多巫族强者的随从、下人，均是人族武者。
<br/>　　“聂兄、韩姑娘，我们找个客栈，先住下来吧！”苏莫向聂乾和韩盈说道。
<br/>　　他们对天冥星一无所知，客栈则是打听消息最好的地方。
<br/>　　“好！”聂乾两人点了点头，他们对此没有什么意见。
<br/>　　很快，三人便找到了一家大型的客栈。
<br/>　　“掌柜，来三间上房！”走进客栈中，苏莫对柜台后的掌柜说道。
<br/>　　掌柜是一名中年巫族，身材肥胖，大腹便便。
<br/>　　“人族只能住中房和下房！”中年掌柜撇了苏莫三人一眼，淡淡的说道。
<br/>　　客栈的房间，一般都是分为下房、中房和上房，档次越高则越是精致豪华。
<br/>　　苏莫闻言眉头微皱，他没想到人族在此地，地位居然如此之地，居然连上房都不能居住。
<br/>　　但他对于房间的档次无所谓，只是栖身之所而已。
<br/>　　他有丝毫迟疑，立刻改口道：“那就三间中房吧！”
<br/>　　很快，苏莫缴纳了灵晶，三人便在客栈中暂住了下来。
<br/>(本章完)
