　　第172章
<br/>　　黑石城，大军欲发。
<br/>　　“十鼓之内，有未到者，斩！”
<br/>　　赤眉骑在马上，眉头拧了起来，脸上那撮红眉更加鲜艳。
<br/>　　昨夜追击的战士直到现在都没回来，结果只有一个，他们已全军覆没。上千人的死亡，在戎族是极其罕见的事情。而且这支队伍环伺由赤眉亲自统率。这不是一件令人开心的事情，赤眉忍不住去想，是不是自己平时对这帮人太好了。
<br/>　　“小山部！”
<br/>　　“在！”
<br/>　　“犬丘部！”
<br/>　　“在！”
<br/>　　“牙泉部！”
<br/>　　“在！”
<br/>　　“孟……”
<br/>　　一声喝便有一部落头目答应，十通鼓后，赤眉率领的戎族十几个大小部落已到齐。
<br/>　　昨夜的惨败令所有人都有些不开心，脸上憋着一股劲儿想去杀人。
<br/>　　“出发！”
<br/>　　赤眉干净利落的撂下一句话，万马扬蹄，队伍却保持自有的队形，丝毫不见杂乱。戎族骑术天下无双，队伍行进的速度是极快的。
<br/>　　行不多时，队伍已进入青牛山范围。
<br/>　　牛角峰上，一棵杨树的树梢偏移，指向北方。
<br/>　　“到了，到了。”程大雷看得仔细，瞬间来了精神。
<br/>　　刚行到青牛山，赤眉的队伍便被堵住了，只见大道上横着巨木、山石、以及挖得深深浅浅的土沟。
<br/>　　以这些是拦不住一支万人军队的，一万人的队伍，一人抓一把土，就可以将沟填满。
<br/>　　几乎不用赤眉吩咐，已经有人下得马来，在前方开路。将大木搬开，山石挪走，土沟填满……自然，队伍的速度也被拖慢了。
<br/>　　山林内，程大雷背着大斧，掌中握剑，拳头攥紧又松开。
<br/>　　“放！”
<br/>　　百人队的弓箭手同时放箭，铁箭若雨，由高处射向大道上开路的戎族。
<br/>　　这突如其来的箭雨令人猝不及防，瞬间已有十几人被射伤，一时慌乱竟没有反应过来。
<br/>　　“有埋伏！”赤眉冷喝：“杀了他们！”
<br/>　　其实不用赤眉吩咐，已经有戎族的汉子提着弓箭弯刀冲了过去。
<br/>　　“走喽，走喽！”
<br/>　　程大雷大吼，转身就向大山里面逃。
<br/>　　戎族骑术无双，箭术也没得说，这算是人家的种族天赋，而程大雷率领的这些山贼，其实根本无所谓箭法，能拉得开弓就不错了。这些人更擅长的是锄头，而非战斗。
<br/>　　当然，他们也有他们的天赋，在青牛山生活这么多年，早已对这里的地形无比熟悉，翻山越岭对他们来说如履平地。而戎族在山地……没听说过谁放羊放牛赶到山上去的。
<br/>　　一伙人在山上转了几个圈就消失得无影无踪，赤眉派出去得追兵无功而返，还有两个因跑得太急而崴着脚，疼地满脸涨红。
<br/>　　有些尴尬，也有些丢人。
<br/>　　“继续前进，注意警惕四周！”
<br/>　　大军继续前进，因要不停开路，队伍的速度被拖慢。
<br/>　　“来了！来了！”
<br/>　　高飞虎手掌摁着胸口，距离戎族这么近，他吓得心惊胆战。
<br/>　　“放！”
<br/>　　“走喽，走喽！”
<br/>　　赤眉脸上得表情越来越不好看，仗着地理之便，山林中时不时会放出一波羽箭。而这些放箭得人也极其可恶，放一波箭就走，一旦被斥候提前发现，也绝对没有恋战得意思，拔腿就跑。
<br/>　　而在山地……自己得战士也的确跑不快。
<br/>　　其实伤亡并不大，几十人的受伤或死亡对队伍来说算不得什么。但这种随时都有冷箭射过来的感觉，实在太恶心人了。
<br/>　　如此，队伍的行军速度大大减慢。
<br/>　　可再远的路，终也有抵达目的地的一刻。
<br/>　　距赤眉出发三天后一个黄昏，赤眉的兵已经进入青牛山中心。他掏出一副潦草的地图观察周围的地形，然后长长松了口气，视线望着前方道：“我要的东西就在里面！”
<br/>　　队伍前进，峰回路转，赤眉没有看到自己想象中的山谷，而是看到一座拔地而起，点满火把的城墙。
<br/>　　事情和自己想得有些不一样啊，这里这么会驻扎着军队，他们又是谁的士兵？
<br/>　　不管那么多了，打下这这座城门，那些突变的就会回到它本来该有的样子。
<br/>　　没有片刻停留，赤眉下令攻城。
<br/>　　帝国百战百败，戎族百战百胜，直到现在，帝国输习惯了，戎族也赢习惯了。
<br/>　　在戎族的认知中，帝国战士是瘦弱的，胆小的，帝国的城池也像纸糊的一样，铁蹄踏出，不攻自破。
<br/>　　“大当家……他们攻来了！”
<br/>　　“这么急！”程大雷也吓了一跳，刚过来就攻城。难道他们不用休整，不用恢复体力，不用调查周围地形？
<br/>　　“准备，准备！”程大雷大叫：“要打仗喽！”
<br/>　　四架投石机从队伍后推到前方，随着第一枚石弹砸到城墙上，双方间的第一场战斗就这样展开了。
<br/>　　赤眉没有给程大雷喘息应对的时间，哪怕他现在都不知道自己的敌人是谁。正如某个登山家所说，山在那里，所以我要登山。
<br/>　　如果问赤眉，赤眉肯定会说，城在那里，所以我要破城。
<br/>　　“他们怎么有投石机，他们怎么有投石机！”程大雷在城头大叫。
<br/>　　赤眉不仅有投石机，也有冲车、云梯等攻城器械。自然，这不是他们自己制造的，而是从幽州王那里抢来的。
<br/>　　以投石机为掩护，战士们冲向城墙，大木撞门，借着云梯向城头攀爬，弓箭朝天，愤怒射去。
<br/>　　程大雷倒抽一口凉气：防不胜防呐，本以为戎族不擅长攻城，原来攻起城来以如此犀利。
<br/>　　“放！”程大雷愤怒大吼。
<br/>　　重弩早已蓄势待发，五十弩箭射向城下。
<br/>　　双方的第一场战斗就在互相杀戮的状态下进行，从黄昏开始攻城，三更方才收兵。
<br/>　　赤眉没有打下蛤蟆寨。
<br/>　　趁夜，对方在离蛤蟆寨三里处扎营，在才是真正的兵临城下，互相都能看见彼此吃饭喝水的样子。
<br/>　　程大雷没有趁对方扎营而夜袭，他也忙着修补城墙，补充弩箭，清点伤亡。
<br/>　　或许是因为不擅长使用投石机的缘故，石弹有些没砸到城头，有些搁着城头飞了过去，最终的结果，也只是有一枚石弹砸到城头上，伤着了七八个人，没有人牺牲。
<br/>　　但程大雷知道，这仅仅是第一场战斗。充其量也仅仅算得上大战之前的开胃菜。
<br/>
