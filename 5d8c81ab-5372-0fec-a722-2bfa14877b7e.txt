　　第640章
<br/>　　程大雷登高望远，借着熹微夜色，黑风寨的地形尽收眼底。
<br/>　　整座山寨三面环水，背靠大山，围着河滩一圈栅栏，只有一座吊桥与外相通。
<br/>　　寨门上有站岗的喽啰，山寨四座箭楼，此刻亮着灯笼，上有警戒的山贼。
<br/>　　程大雷微微点头，委实不可小看了这混江龙。黑风寨易守难攻，当真是一座险地。当然，混江龙能聚集三千人马，在此地为非作歹多年，自然不是什么易于之辈。
<br/>　　蛤蟆城的人马扎下，与黑风寨隔河相望。阿喜摊开一份地图，道：“大当家，这座吊桥是黑风寨正面向外的唯一通道，现在我们被水拦着，如果要强行渡河必然会被山寨的岗哨发现。不过，喜字队打听到后山有条小道，是山贼偷偷下山用的，我们可以从那里通过。”
<br/>　　此刻绕到后山，所消耗的时间太多，但从正面突破明显危险许多。
<br/>　　程大雷皱起眉头，盯着黑暗中的黑风寨，脑海里思考着对策。他自然不可能被一道河拦住，不过，强行突破必定伤亡惨重，程大雷在思考一个将伤亡减少到最低的办法。
<br/>　　“大当家，我倒是有条计策。”此刻杨国忠突然开口道。
<br/>　　程大雷看了他一眼，道：“你有什么办法？”
<br/>　　杨国忠是一个很识时务的人，假若自己挣脱不了程大雷的控制，那么就该及早表现自己的价值，如此才能换得更好的待遇。
<br/>　　“黑风寨背靠大山，三面环水，以稳妥起见，当绕到后山从小道突袭。若大当家想要强攻，我不信黑风寨的防御没有破绽。我们当找到它的防御死角，选一二百水性好的人泅水而过，搭好浮桥，一千人强攻而入，杀他们一个措手不及。”
<br/>　　程大雷格外看了他一眼，不得不对杨国忠刮目相看，不愧是出场便绝世，当真不可小觑。杨国忠委实算不上什么奇思妙想，可这便是战争，审明局势之后，选择最优策略。
<br/>　　“就按杨先生的法子办，阿喜，黑风寨哪处防守有缺？”
<br/>　　阿喜也没辜负程大雷的信任，关键是没在杨国忠面前丢人，他指着地图上某处地方，道：“这里只有一座箭楼盯着，只要将箭楼上的哨子除掉，这就是一处防守空白地带。”
<br/>　　程大雷点点头，挥手下达了命令：“转移，阿喜带队。”
<br/>　　一千人开始默不作声转移，在阿喜的带领下去往合适战斗的地方。杨国忠回头看这支队伍，上千人沉默着，除了草木噼啪被踩碎的声音，没有任何噪杂声响，甚至呼吸都趋于一致。
<br/>　　这是一支怎样恐怖的队伍，而当这一千人扩充到一万人、十万人又该可怖到怎样程度。程大雷在江湖上恶名远播，却也不仅仅是因为他的凶恶，另外还因为他手上有一支战无不胜的蛤蟆军。
<br/>　　天下大乱，诸侯相争，这天下会落在谁手中还说不好。目前程大雷还未露出麟角之象，可手中有这样一支兵马天下未必没有他的份儿。若自己能从中相助，有辅龙之功，未来封疆裂土，重整家族荣光也不算痴心妄想。
<br/>　　相比之下，做了山贼的骂名也就显得没那么重要了。
<br/>　　只用了半个时辰，队伍便转移到位，过程中并没有惊动黑风寨的守卫。不知该说黑风寨疏忽大意，还是该说程大雷的队伍周转如意，上千人在黑风寨外围绕圈子也没有被站岗的哨子发现。
<br/>　　“大当家，那座箭楼是我们唯一的麻烦，只要除掉上面的山贼，这里就无人防守。”阿喜指着远处给程大雷看：“关键是如何除掉他。”
<br/>　　程大雷顺着他手指的方向看过去，见箭楼上亮着灯笼，他晃晃脑袋道：“这有什么麻烦，我去除掉他，你们等我信号。”
<br/>　　“大当家，小心……”
<br/>　　阿喜下意识道出声，却见程大雷已经窜了出去。两三丈的河面他点水而过，几个纵跃已跨过河面，翻身跃过栅栏，潜入黑风寨中。
<br/>　　程大雷刚刚还觉得单雄信好逞匹夫之勇，没有大将之风，然而事实上，他自己也没有好到那里去。
<br/>　　一个人单枪匹马便闯入敌军老巢，也亏得他有一身好本事，否则后果不堪设想。只见程大雷如一直黑鸦，盘旋着便上了箭楼，那箭楼上瞭望的山贼根本没有发觉。这一幕却惊坏了远远看着的杨国忠。
<br/>　　“大当家……竟还有这样本事。”
<br/>　　阿喜没有搭理他，挥手指挥手下士兵准备渡河工具。无非是将草木搭成筏子，泅水而渡。上千人一起动手，这些事倒也没有什么难度。相比之下，阿喜更像这次行动的指挥官，单雄信握着大槊只等着冲入敌寨大杀一番。
<br/>　　程大雷此刻已默不作声的爬上箭楼，拔剑送了那探子的性命，抄手接过他手中的灯笼，在空中晃了三晃。
<br/>　　收到信号后，阿喜离开展开行动，将筏子推入水中搭成浮桥，一队人一队人从浮桥上穿过。
<br/>　　拔掉这座箭楼后，此地已经成了防守的空白地带，稍微动静大些也不惧被人发现。
<br/>　　等人都登上河岸之后，立刻用大刀砍开栅栏，一千人长驱直入。
<br/>　　程大雷站在箭楼上观战，见涌入山寨后，一千人迅速分为几队，一队人去占领粮仓，免得黑风寨的山寨走火入魔，一把火扫了粮仓。
<br/>　　一队人去占领营房，另外单雄信带着一队人去找混江龙单挑。他也明白擒贼先擒王的道理，只要除掉他，黑风寨的山贼必定大乱，赶他们便也不必赶几只猪困难。
<br/>　　一时间，黑风寨内喊杀声四起，一众人一边喊杀，一边高声喊着：
<br/>　　“老子是蛤蟆城蛤蟆大王的队伍，不想死的都把手举起来。”
<br/>　　混江龙刚刚睡着不久，就听得外面乱哄哄一片，『蛤蟆大王』四个字令他大惊失色，衣服没穿好便从床上跳了起来，抓起自己鱼叉就往外走。
<br/>　　正好与找过来的单雄信撞个正着，单雄信赤发蓝脸，当真如怪物一般。
<br/>　　“呔，你可知道那混江龙在何处！”
<br/>(本章完)
