　　“没问题，但是，我只怕不是他们的对手啊，毕竟是阐教金仙。”
<br/>　　“他们久居空界，不习惯人间气息的浓度，实力不能完全施展，而且你最近实力又提升了少许，再找几个人一起，不怕事情不成，况且，我不要你把他们怎么样，只要抢到召唤东岳大帝的通灵信物就行。”
<br/>　　叶少阳点点头道：“行吧，我带兄弟们一起。”
<br/>　　“我跟他们，不能撕破脸皮，你去抢夺信物，不要承认动机！”
<br/>　　“那怎么行得通，难道我是看他们不爽去打他们一顿？”
<br/>　　“我给你想到了一个借口，这次我得到线索，苏沫也跟着黎山老母一起来了人间，芙窕仙子不是一心想炼化她吗，这就是借口，见面之后，黎山老母必然不肯把苏沫给你们，那就只有打了。你对东岳大帝的信物感兴趣，答应了顺手牵羊而已……”
<br/>　　叶少阳怔怔地看着他，林三生现在的样子，像极了一个阴谋家。
<br/>　　虽然不太喜欢这个计划，但叶少阳没法不答应，因为这是林三生的事。
<br/>　　“什么时候行动？”
<br/>　　“大概需要几天，我会派人来告诉你，你等我消息便是。”
<br/>　　叶少阳点点头，说道：“我只有个疑问，随口问下哈，东岳大帝入真到了空界……在对付尸族的战争上，是有好处的吧。毕竟他有实力，也有号召力。”
<br/>　　林三生一笑，“你想什么呢，真到那一步，妖族和邪灵的几个山头，都是不服管的，到时候空界内乱，最后的结果，只能是被大帝用武力征服。”
<br/>　　“征服之后，不就拧成一股绳了吗？”
<br/>　　林三生沉默半晌，道：“少阳，如果有一天，跳出来一个人，实力冠绝天下，在你之上，他想领导人间法术界一起对抗天劫之乱，但前提是你和你的茅山必须臣服他，你做得到吗？”
<br/>　　叶少阳一下子怔住，摇摇头，“开始肯定不想，但如果……他真的是更好的领导人选，我愿意让他当老大。”
<br/>　　“你呀！”林三生用扇子点了点他的头，“你首先得相信自己是真命天子，除了你，一切都是假的。”
<br/>　　“如果我不是真的呢。”
<br/>　　“实力，实力能解释一切。真与假，只不过是一瞬间的，但历史是胜利者书写的，我们看到的历史，也只不过是前人想让我们看到的那部分，好了，扯远了。总之，你先准备一下，不几日便要行动。”
<br/>　　叶少阳自然没话说，跟他打听起道风和芮冷玉的近况，两人聊起来。
<br/>　　天黑透了，瓜瓜突然想起陈悦还没回来，提出去山上找。
<br/>　　“你真把她当成你娘亲了？”林三生皱眉。
<br/>　　“对啊要你管啊。”瓜瓜就要出去。
<br/>　　“回来你！”林三生试图阻拦，但瓜瓜已经跑了，在门外丢下一句话，让他有什么事回来再说。
<br/>　　从道观出去，瓜瓜突然有一种不详的预感，干脆飞起来，俯瞰峰顶。
<br/>　　陈悦正在山路上狂奔，在她身后，有三道黑影正在追赶，就在他看的这一瞬间，三道人影迅速贴近，朝陈悦后心抓去。
<br/>　　“不！”瓜瓜大叫，但是他担心的事并未发生：陈悦的脖颈上突然亮起了一道幽蓝色的光环，打在三人身上，直接震飞了出去，陈悦自己也惨叫一声，向前扑倒在地上。
<br/>　　从附近的山林中，又跳出来七八个黑衣人，朝陈悦冲了过去。
<br/>　　“住手！！”
<br/>　　瓜瓜大吼一声，化出真身十二年蝉，俯冲过去，凌空用力扇动翅膀，卷起一股又一股的阴风，朝那几个黑衣人挂过去。
<br/>　　几人被吹的东倒西歪，勉强站住身形，为首一人下令道：“不管他，快杀了这女子！”
<br/>　　几人一拥而上。
<br/>　　陈悦仰面倒在地上，根本没有还手之力，茫然无措地看着这些杀手靠近。
<br/>　　突然一只巨大的蝉虫落在面前，瞪着两个铜铃大小的眼睛，近距离看着她，瓮声瓮气地道：“娘，不要怕，我保护你！”
<br/>　　什么怪物啊！陈悦直接吓昏了。
<br/>　　几个黑衣人顿了一下，为首一人喝道：“你们缠住他，我来！”
<br/>　　从丛林中又飞出十数道人影，朝瓜瓜扑了过来，瓜瓜能感知到他们身上强烈的鬼气，这些都不是人类，而且实力都不弱。
<br/>　　瓜瓜疯了一般，施展出了一切手段，对付这些神秘人，但这些神秘人很聪明，不跟他死斗，只是努力缠着他，只要瓜瓜一动，他们立刻上来袭扰，那边，为首的黑衣人已经朝陈悦冲了过去……
<br/>　　“吱……”一声蝉鸣，震耳欲聋，所有人心神都颤了一下，气息被震散，瓜瓜抓住机会，冲出了包围，朝陈悦冲去。
<br/>　　这也是情急中不顾一切的行为。那些黑衣人，很快又重新提气，朝瓜瓜后心扑去。
<br/>　　瓜瓜眼看着陈悦要被抓到，情急之中，完全顾不上自己了，拼着被重伤一击的风险，加速朝陈悦冲过去。
<br/>　　两个黑衣人的手已经摸到他的后心，犹豫了一下，居然收手了。
<br/>　　瓜瓜俯冲下去，抓住为首的黑衣人，用力掼在地上，立刻爬上去，尖锐的口器立刻就要对着脑门刺下去，黑衣人突然哀求道：“瓜二爷！”
<br/>　　瓜瓜一怔，伸手扯下他脸上的黑布，是个三十来岁的汉子，有那么一点面熟，但想不起来，继续要动手，这人抬手挡着脸，道：“瓜二爷切莫动手！”
<br/>　　“你是谁！”
<br/>　　男子不说话。
<br/>　　他身后那些黑衣人也都落下来，不知所措地站在瓜瓜身边，瓜瓜提着黑衣人来到陈悦身边，继续呵斥那人：“你是谁啊，你怎么认得我！不说我立刻杀了你！”
<br/>　　黑衣人还是不说话。
<br/>　　瓜瓜一怒之下就要动手，突然远处传来一声呵斥：“住手！”
<br/>　　瓜瓜抬头，看到叶少阳和林三生一起走来。
<br/>　　是他之前情急中通过魂印给叶少阳传递了紧急信号，他们立刻就赶来了。
<br/>　　看到他们，瓜瓜也就放心了，按着黑衣人，等叶少阳过来。
<br/>　　“怎么回事？”叶少阳看着面前这么多黑衣人，问道，心中也很好奇，这些黑衣人行刺失败，为什么都不跑呢，难道还要帮他们首领抢回去不成？
