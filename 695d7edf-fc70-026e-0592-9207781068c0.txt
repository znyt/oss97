　　　　“尔敢！”
<br/>　　千钧一发之际，忽闻九天之上，一声大喝传来，随之一道流光乍现，如长虹贯日，破空而来，想要强行阻断玄天那道斩仙之力。
<br/>　　“哼！吾想让他死，就凭你们也想阻拦？今天你们谁也拦不住！”
<br/>　　只听玄天一声冷哼，看着那道飞来的流光，一拳轰出。
<br/>　　“轰！”
<br/>　　两道强烈的光芒绽放，一声巨响，天地动荡，最激烈的交锋，玄天一拳将那道流光破灭，同时，那道斩仙之力毫无阻碍的斩下了火之祖巫祝融的头颅。
<br/>　　“你……”
<br/>　　祝融脸上的表情凝固了，之前因为岁月轮与生死轮的原因，祝融已经被极致削弱了，如在再加上玄天的斩仙之力，瞬间便在别人截挡之前，斩杀祝融。虽然十二祖巫之一的火之祖巫祝融肉身已经堪比准圣后期了，但玄天连帝俊都能差点灭了，对付一个祝融，还是废不了多大力。
<br/>　　“哐当！”
<br/>　　随着一阵声响，祝融的头颅摔落下来，在地面之上滚动起来，接着，他那雄伟的身体在此刻倒了下来，在地上砸出一个巨坑。
<br/>　　风云涌动，乾坤震荡，熊熊烈焰瞬间便燃烧起来，直通九天。
<br/>　　“祝融！”
<br/>　　十二祖巫中其余的十一道身影在此刻破空而来，看着地上倒下的祝融，他们脸上充满悲怆、愤怒之色，看向玄天的目光之中，充满冰冷的杀机。周边空气仿佛都在此刻凝固了，时间与空间都彻底凝滞。
<br/>　　“人族玄天，你竟敢杀我祝融兄弟，我跟你拼了！”
<br/>　　水之祖巫共工满脸悲怆，眼中杀机森森，仿佛能够将空间冰冻，挥手之间，便召唤出无尽黑水，滔天黑水，滚滚而来，汇聚在他身前，凝聚成一个黑水世界。
<br/>　　“哈！巫族背弃盟约，屠杀人族，该死！祝融只是一个小小的开胃菜，放心，你们别急！我不止要杀了祝融，说不得等会儿还要将你们也宰了！”
<br/>　　玄天凭立虚空，冷眼看着眼前的这十一道身影，脸上满上森然冷笑，看向这十一个祖巫，眼中充满冷意。
<br/>　　巫族屠杀人族之事，他心中充满愤怒，若是他没有及时出手，人族岂不是要被巫、妖两族全部屠杀了？这是玄天不允许的。
<br/>　　“狂妄！人族玄天，你人族算什么东西，生来不就是为了给我巫族做奴隶的吗？不就是杀了几个人族吗，杀了就杀了，有何了不起的？你人族竟敢以下犯上，屠杀祖巫，真是该死！”
<br/>　　时间祖巫烛九阴看着玄天冷冷道，语气之中，充满对人族的不屑。
<br/>　　听到此话之后，玄天气急反笑道：
<br/>　　“杀了就杀了？好！好！好！巫族，自认为高人一等，今天本祖倒是要看看，你们比别人高贵在哪里？”
<br/>　　此刻，他眼中充满冰冷之色，对着人族下命令道：
<br/>　　“人族听令，巫族无情屠戮人族，今后人族与巫族势不两立，以后见到巫族，杀无赦！”
<br/>　　“杀杀杀！……”
<br/>　　人族经过一番屠杀，早已不是之前的那个人族了，更何况巫族屠杀人族的那些场景还历历在目，人族一直都没忘，他们心中早已憋着一股气，对巫、妖两族充满杀机。此刻听到玄天的话，毫不犹豫地便喊了出来。
<br/>　　“找死！”
<br/>　　帝俊脸色冷峻，看向人族的眼神之中，充满杀机，之前是因为人族表现出来的潜力，他便已经下定决心要屠杀人族了，现在眼前这些人族，竟然还狂妄自大，想要挑衅巫族，不给他们点颜色看看，他们是不知道巫族的厉害。
<br/>　　“杀！”
<br/>　　帝江祖巫一动，一拳轰出，破碎虚空，乾坤震荡，破空拳带着一击必杀之势轰向人族玄天。
<br/>　　帝江一动，一旁早已杀气沸腾的共工祖巫，一样冲了过来，杀向玄天，想要为祝融祖巫报仇。虽说水火不容，祝融与共工平时闹得最凶，但他们同样是最了解对方的人，祝融死后，十二祖巫之中他的心中最为难受，此刻，他恨不得撕碎玄天。
<br/>　　看着杀向他的两大祖巫，玄天怡然不惧，抬手就是两掌，拍向两大祖巫。巨掌遮天，势比天高，此为遮天掌。
<br/>　　两掌拍向两人，就仿佛是两座大山镇压了下来，要将帝江与共工镇压。两大祖巫当然不会如他所愿，对轰了过去。空间震荡，风云变色，日月倒垂，不断有黑洞衍生，吞噬着周边的一切。
<br/>　　瞬间，三人便大战在了一起，一时之间，不分胜负。
<br/>　　见帝江祖巫动手，其他祖巫自然也没有停着的道理，时间祖巫烛九阴冷笑一声，身形闪动，就要对眼前的人族屠杀。
<br/>　　但，就在此时，一直悬在玄天脑后的三道神轮飞了出来，化为三道流光，阻击烛九阴，三道转轮无形无形，只是一种道则神韵，并不实质的，触摸不及。其中更有岁月轮，更是与烛九阴的时光轮有些类似
<br/>　　时光、生死、命运的道则闪动，三道转轮一下子就与烛九阴纠缠起来，烛九阴也无法将之轰碎，因为那根本就不是实质的。
<br/>　　一时间，两者纠缠在一起，分身无暇。
<br/>　　剩下八大祖巫，自有人族燧人氏三祖带着九大武皇对接上。因为人族只有武皇初期，堪比准圣初期，但十二祖巫却又堪比准圣中期甚至后期的肉身，所以人族之中，都是两人或者三人对上一个祖巫。
<br/>　　燧皇燧人氏与盘皇盘王对上雷之祖巫强良，巢皇有巢氏对上木之祖巫句芒，衣皇缁衣氏与风皇风后对上风之祖巫天吴，剑皇赤松子与雨皇雨伯对上金之祖巫蓐收，太皇太昊氏与少皇少昊氏对上电之祖巫弇兹。
<br/>　　六处战场，十数位堪比准圣的强者，大混战起来，一时之间，谁也顾及不上谁？
<br/>　　那么，问题就来了，巫族有十二祖巫，除了被玄天斩杀的祝融与大战之中的八位祖巫，还留下三大祖巫，一为天气祖巫奢比尸，二为雨之祖巫玄冥，三为土之祖巫后土。
<br/>　　就在天气祖巫奢比尸要动手屠杀人族之时，玄天又动了。他屈指之间，又是几道通天剑气飞出，化为一片缥缈的光雨，飞向奢比尸，光雨锋利如剑，这是斩仙之力。
<br/>　　显然，玄天一人大战三大祖巫，依旧显得不慌不乱，还能再战一位。
<br/>　　“也罢，既然你要战，那便别怪我以多欺少。更何况，先杀了你，再去杀其他人族，也是一样。”
<br/>　　就这样，天气祖巫奢比尸也加入到玄天四人的战斗之中。
