　　“刚才你也听到了，本座会找到他的。”终极神王叹道，他虽然不知道天尘宇一直在找苏莫所为何事，但是，他也不想深究。
<br/>　　“终极神王，我相信你们会找到苏莫，我想要说的是，找到苏莫之后，我不要他任何宝物，只要他这个人，只要你将苏莫交于我，百亿极品源石相送。”天尘宇沉声说道。
<br/>　　终极神王殿寻找苏莫，比他要简单一些，毕竟，终极神王有名正言顺的理由，可以倾所有力量寻找。
<br/>　　而他，为了不让苏莫的身份，被地王宫知晓，只能暗中寻找，力量有限。
<br/>　　终极神王等人闻言，心中暗暗惊讶，百亿极品源石，这对终极神王殿和天王宫这种顶级势力，都几乎是大半的家底了。
<br/>　　天尘宇，为了得到苏莫，居然愿意花费这么大的代价，这是整个天王宫的意思吗？这苏莫有何秘密？
<br/>　　“此事不成问题，待找到苏莫之后再说吧！”终极神王沉声说道。
<br/>　　“嗯，既然如此，我便告辞了，需要帮助的话，晚辈义不容辞！”天尘宇起身，提出了告辞。
<br/>　　“嗯！”终极神王微微颌首。
<br/>　　天尘宇转身离去，至始至终，他都没有与紫霄和绫裳交谈，因为他知道不可能问出结果。
<br/>　　不过，在离开之前，他的目光有意无意的扫视了一眼紫霄和绫裳两人。
<br/>　　他不会将希望，全部寄托在终极神王殿的身上，想要找到苏莫，还需要从紫霄和绫裳身上下功夫。
<br/>　　紫霄的实力太强，不好对付，但是绫裳却是修为不高。
<br/>　　“殿主，秦陨确实与我无关！”
<br/>　　天尘宇离开之后，紫霄立刻向终极神王表明自身的清白，言罢，他冷视着绫裳，怒道：“绫裳，你故意诬陷于我，居心何在？你与秦陨之间，到底是什么关系？”
<br/>　　对于绫裳，紫霄心中有些愤怒，不过，同样也心有忌惮。
<br/>　　毕竟，他将芊寻月从秦陨的手中抢了过来，此事不知道绫裳是否知晓？
<br/>　　总之，芊寻月之事，不能暴露。
<br/>　　一旦暴露的话，他将无法独吞终极剑道和终极神剑，也会因为此事彻底与终极神殿决裂。
<br/>　　毕竟，他一直在隐瞒，就是为了独吞终极剑道和终极神剑，这无异于背叛。
<br/>　　不过，只要等得到终极神剑和终极剑道，这一切都是值得的。
<br/>　　以他的修为和实力，只要得到终极神剑和终极剑道，不仅整个神域，没有什么人能奈何的了他，将来超脱纪元也不是不可能。
<br/>　　“紫霄，你的那些勾当，别以为我不知晓？”绫裳面带寒霜，冷冷的说道。
<br/>　　“什么勾当？”终极神王问道，现如今，对于绫裳和紫霄，他心中已经有了怒气。
<br/>　　“殿主，不瞒你说，这一切都与紫霄有关！”绫裳向终极神王抱拳说道。
<br/>　　“绫裳，休要胡言乱语！”紫霄闻言大怒，一巴掌将身下的座椅，震成了齑粉。
<br/>　　“绫裳，你说来听听。”终极神王对于紫霄的愤怒，视而不见，面无表情的说道。
<br/>　　“殿主，外界的传闻，均是属实，秦陨的爱人，被紫霄挟持了，紫霄想要以此女要挟秦陨，独吞终极剑道和终极神剑！”绫裳沉声说道。
<br/>　　此言一出，无论是终极神王，还是一旁默不作声的风雪神王和毁灭剑王，均是面色一沉。
<br/>　　“胡说八道！”
<br/>　　紫霄怒喝，面色微变，他没想到将绫裳将此事都抖了出来，难道不管那棺椁中女人的安危了吗？
<br/>　　不过，他也没有因此恼羞成怒，毕竟，那秦陨的爱人，早已为他藏在了隐秘之处。
<br/>　　没有证据，谁都不能证明，此事与他有关。
<br/>　　终极神王闻言，沉默不语，并没有什么表示，因为，他早已查过紫霄，根本没有找到所谓的秦陨的爱人。
<br/>　　现在，无论是紫霄的话，还是绫裳的话，他都不太相信。
<br/>　　“我胡说八道？”
<br/>　　绫裳冷笑一声，道：“现如今，秦陨估计早已被你杀了，终极神剑和终极剑道，也已经被你得到了。”
<br/>　　现如今，绫裳也不再客气了，要将所有的事情，都扣在紫霄的头上。
<br/>　　“一派胡言！”
<br/>　　紫霄怒斥一声，向终极神王道：“殿主，绫裳自从再次归来之后，早已心性大变，这是在诬陷于我，不知有何诡计？”
<br/>　　“这……！”
<br/>　　风雪神王和毁灭剑王两人，面面相觑，均是有些无语。
<br/>　　在他们的了解之中，紫霄和绫裳，关系非常的好，似道侣、像兄妹，而现在居然在互相指责，大有撕破脸皮的局面。
<br/>　　这到底是怎么了？
<br/>　　这两人，怎么会闹到如此的局面？
<br/>　　“你们两人！”
<br/>　　终极神王环视着紫霄和绫裳，略一沉吟，眸中闪过凌厉的光芒，沉声道：“从今日起，没有本座的允许，谁若是胆敢离开终极神王城一步，视为背叛终极神王殿。”
<br/>　　凌厉的声音，从终极神王口中响起，任谁都能听出终极神王的愤怒。
<br/>　　紫霄和绫裳闻言，心中微颤，他们知道，殿主这次是真的怒了。
<br/>　　“你们两人，都下去吧！”
<br/>　　随即，终极神王挥了挥手，示意两人离去。
<br/>　　紫霄和绫裳，也未再言，纷纷向终极神王行了一礼，便躬身告退。
<br/>　　走出大殿，两人互相凝视一眼，目中的冷漠，不言而喻。
<br/>　　对于绫裳而言，她就是要配合苏莫，想办法从紫霄的手中，救走芊寻月。
<br/>　　而对于紫霄而言，他知道绫裳的目的，是想要救走芊寻月。
<br/>　　但是，这是根本不可能的事情，任何的诬陷都是苍白无力。
<br/>　　现在对他来说一切都不重要了，只有得到终极剑道和终极神剑才是最主要的。
<br/>　　事情走到了这一步，他已经无法回头了。
<br/>　　“两位，你们怎么看？”
<br/>　　终极大殿之中，终极神王看着剩下的风雪神王和毁灭剑王，沉声问道。
<br/>　　“殿主，依我看，无论是绫裳还是紫霄，都与此事脱不了干系。”毁灭剑王沉吟着道。
<br/>　　“想要找到秦陨，或许还要通过他们两人。”风雪神王美眸闪过一丝精光。
<br/>　　“说说你的想法。”终极神王望着风雪神王，他自然知晓，此事和紫霄与绫裳，肯定有一定的关系。
<br/>　　但是，现如今，拿回终极剑道和终极神剑，才是重中之重。
<br/>　　“殿主，此事的源头是秦陨，一起都有人推波助澜，虚虚实实，真真假假；我有一个计划，或许可以解决此事。”风雪神王微微一笑，眸中闪过睿智的光芒。
<br/>　　随即，风雪神王立刻将计划，详细的告知了终极神王。
