　　“苏莫，这次争夺宝物，有十件是极品的宝物，你能分辨出是哪些吗？”司空炎望着身前的一堆宝物，皱眉问道。
<br/>　　这次的争夺的宝物，总共有五百一十件，其中五百件是上品的宝物，十件是极品的宝物。
<br/>　　但是，他却是完全分辨不出来。
<br/>　　“首先，这些兵器可以排除在外！”
<br/>　　苏莫面色郑重，继续道：“若是有极品宝物的话，肯定在这些丹药和神果、神草之中。”
<br/>　　这些兵器的品阶，明显不是极品神器，那么，极品的宝物，肯定是丹药或者神草、神果。
<br/>　　这么多宝物之中，若说每有一件极品宝物，他们根本不信。
<br/>　　“这三块晶石是什么东西？”司空炎一把抓起三块晶石，疑惑的打量了起来。
<br/>　　这三枚晶石，一模一样，全部是乳白之色，比源石要大不少，气息晦涩，让人无法辨认。
<br/>　　“不清楚！”苏莫摇了摇头，这个时候，他深深的感觉到，他们两人知识的匮乏。
<br/>　　在天赋方面，他们可以冠绝神域，但是，在这些方面，他们却是见识浅薄。
<br/>　　“那这些丹药，都是什么丹药？”司空炎又问道。
<br/>　　“不清楚！”苏莫再次摇头。
<br/>　　司空炎无语了，双手一摊，道：“这些宝物，我们一件不识，那我们怎么用，不能贸然使用，不然很容易浪费了。”
<br/>　　苏莫闻言，心中沉吟，此言在理，必须搞清楚这些是什么宝物，有什么效用。
<br/>　　“要不，我去将龙腾叫来？他或许会知晓！”司空炎道。
<br/>　　“他也不太可能都知晓！”
<br/>　　苏莫要来摇头，他们都太年轻了，见识有限，这是正常的情况。
<br/>　　略一沉吟，苏莫道：“我将这些宝物的特征，传讯给人德宫主，他应该会都知道！”
<br/>　　“这样最好！”司空炎闻言，顿时眼眸一亮，他倒是没想到，苏莫能直接给人德宫主传讯。
<br/>　　人德宫主，这种老一辈的高级神王，活了无数的岁月，见识肯定极广。
<br/>　　随即，苏莫取出人德宫主给予他的传讯符，开始给对方传讯，将此地的这些丹药、神果、神草、晶石等宝物，一一向人德宫主描述，询问对方这些宝物的具体效用。
<br/>　　传完消息之后，人德宫主并未回消息，苏莫和司空炎，静静的等待了起来。
<br/>　　两人大约等待了半刻钟的时间，传讯玉符，顿时微微颤抖了起来。
<br/>　　苏莫心中一喜，意念一动，便开始查看传讯符中的消息。
<br/>　　“苏莫，那三枚晶石，乃是极品宝物，名为道晶，说白了就是神王级别人族武者的武道传承，那些丹药分别有渡心神丹、破虚神丹……！”
<br/>　　传讯符之中，人德神王，将所有的宝物，以及它们的效用，全部向苏莫述说了一遍。
<br/>　　查看完了人德神王的传讯，苏莫心中了然，但亦是有些许失望。
<br/>　　因为，三件极品宝物，是神王级别强者的传承，对于他来说，并没有太大的用处。
<br/>　　毕竟，他修炼的道统，比较特殊。
<br/>　　不过，剩下的宝物之中，无论是丹药、神果、还是神草，有很多都对他都有大用。
<br/>　　这些宝物，能增强修为、加快修炼速度，或者是辅助冲击修为壁障、甚至是有助于法则的参悟。
<br/>　　“苏莫，怎么样？”司空炎问道。
<br/>　　“都清楚了！”苏莫点了点头，随即向司空炎说明，这些宝物的情况。
<br/>　　“原来如此！”
<br/>　　司空炎恍然，略一沉吟，他继续道：“这些丹药、神果和神草，你拿七成，剩下的三成和这些兵器留给我吧！”
<br/>　　司空炎心中清楚，苏莫比他更需要宝物，所以他只要三成。
<br/>　　至于那些兵器，对于苏莫而言，肯定无用，所以他就要了，日后可以换成源石。
<br/>　　“嗯！”苏莫点了点头，随即又道：“这三枚晶石，你也拿去吧，研究一番，看看是什么传承？若是对你无用，日后可以给姜枫然他们！”
<br/>　　“好！”司空炎同意。
<br/>　　随后，两人将宝物分了，便各自进入了密室之中，准备修炼。
<br/>　　浑天草！
<br/>　　密室之中，苏莫手中拿出一株蓝色的小草，仔细的打量，根据人德神王的之前的传讯，此草名为浑天草，对于人族的武者修炼，有极大的好处。
<br/>　　此草的效力，能让武者在两个月之内，修炼的速度暴增二十倍以上。
<br/>　　最主要的是，无论是修炼某种属性的道统，都没有问题。
<br/>　　少倾，苏莫将这株浑天草吞服了下去，开始了修炼。
<br/>　　神草入腹，苏莫整个人都变得燥热了起来，全身气血滚滚，源力如同兴奋的野马，疯狂的在体内经脉之中咆哮。
<br/>　　苏莫屏弃杂念，静心的修炼了起来。
<br/>　　……
<br/>　　终极神王殿，紫霄剑宫，一处密室之中。
<br/>　　紫霄盘膝坐在密室的中央，脸色冷漠，他的身前，伫立着一名身穿藏蓝色长袍，面色古板的青年。
<br/>　　“天墟，有发现了吗？”紫霄的目光，凝视着身前的楚天墟。
<br/>　　最近这些年，终极神王殿在到处追查苏莫的下落，紫霄同样在暗中追查。
<br/>　　只不过，现如今，他在终极神王殿之中，已经失去了殿主的绝对信任，所以都是让楚天墟暗中调查。
<br/>　　“宫主，上次一役，那秦陨是经过了精心的安排，一步步将您和整个终极神王，玩弄在了鼓掌之中，此子心思缜密，手段无双，想要找到他，非常的困难！”楚天墟面无表情。
<br/>　　“这我自然知晓，但是，无论是多困难，也要找到他！”紫霄脸色阴沉的说道，若不是他不方便，早已亲自出去寻找了。
<br/>　　现如今，终极神王对方猜忌不已，他只能让楚天墟暗中追查。
<br/>　　对于那个秦陨，也就是苏莫，紫霄可谓是恨之入骨，不找到苏莫他誓不罢休，心中的一口闷气，怎么也无法消散。
<br/>　　“宫主，想要找到秦陨，只能从绫裳宫主身上下手！”楚天墟眸中闪过犀利的精光，沉声说道。
<br/>　　“这还用你说，但现在绫裳也无处可寻！”紫霄叹了口气，若是能找到绫裳，终极神王早已找到了。
<br/>　　绫裳目前，虽然修为不是很高，但毕竟当初是神王，手段很多，天机都无法衍算得到。
<br/>　　“宫主，经过我近几年的调查，我找到了一个可行的办法！”楚天墟面色郑重的说道。
<br/>　　“什么办法？”紫霄闻言，眸光一亮，急忙出声问道。
