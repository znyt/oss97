　　第120章
<br/>　　千舟城。
<br/>　　帆船在视线中慢慢消失，孙德龙立在岸上却还是久久不肯离开。十虎八犬立在其身后，有人过来在孙德龙肩上披上大氅，猛然发现此刻的孙德龙已经老泪纵横。
<br/>　　“老爷子，水边风大，咱们先回去吧。”
<br/>　　“回去，回去。”
<br/>　　“还有一船子货，怎么处理？”
<br/>　　孙德龙自然知道程大雷留下的东西，在他的印象中，这些东西都是不值什么钱。他帮程大雷，也并不是为了程大雷的货，这对他来说，是一个尽忠表心意的好机会。
<br/>　　“随便处理掉吧，卖得出去就卖，卖不出去就想办法卖掉。”
<br/>　　孙德龙心里还在想着别的事。这次倾漕帮之力帮程大雷凑齐粮食，但漕帮也是元气大伤。甚至，孙德龙还欠下一些债。如何让漕帮恢复元气，是一件足以令孙德龙头疼的事情。
<br/>　　这时候有人迎过来，此人是城里的一个大户，和孙德龙也算是熟识。
<br/>　　“孙老爷子，您辛苦，听说您在一天之内凑齐四十万斤稻米，垄断了辣椒，老爷子大手笔啊……”
<br/>　　“怎么你是来笑话我的么？”
<br/>　　“笑话什么？”此人一头雾水：“我是佩服老爷子的气魄，老爷子，这辣椒能卖给我一些么？”
<br/>　　咦，事情怎么感觉和自己想的不一样。在孙德龙心里，这一船辣椒本就是不值钱的。他试探着问：“你能出什么价钱？”
<br/>　　“价钱方面自然能让老爷子满意，银子我都已经带来了，只有老爷子肯卖给我一些就行……”
<br/>　　孙德龙目光环视一圈，突然发现还有很多人看向这边。看他们的意思，像也是对船上的货感兴趣。
<br/>　　如果都能按照刚才的价格，似乎这笔买卖并不吃亏，好像还能不少的赚上一笔。
<br/>　　孙德龙目光转向大海，忽然跪了下来：“六皇子，您可怜老奴呐。”
<br/>　　戎族西狄部，一个蓬头垢面的牧羊人重重的打了个喷嚏，扭头朝向南方，露出一双澄清的眼睛。
<br/>　　“是谁又想我了，最近老是打喷嚏，或者……是说我染了风寒。”
<br/>　　……
<br/>　　“为什么要当牛做马，为什么不是以身相许，呀，杀呀。”
<br/>　　程大雷大吼，一剑穿过女人胸脯，血从胸腔涌出来，女人呼吸越来越急促，她难以置信的盯着程大雷。
<br/>　　难道对方就因为这个理由杀死自己，这个疯子般的理由。
<br/>　　所有人都被这一幕搞得有些懵，在这个世界，杀个把人委实不算什么，只是眼前的行为，实在有违程大雷过往的行事风格。
<br/>　　现在他的行事风格像个疯子一样。
<br/>　　“杀呀杀！”
<br/>　　在他们发愣的时候，程大雷又一剑一个杀了两个，剩下另外两个一男一女，也立刻被制住。
<br/>　　程大雷脸上的疯狂渐渐消失，他望着两个活口，道：“落到我手里，我想你们也知道自己活不成，我也并不相信你们有什么气节之类的东西。不过，这世上有个词叫做生不如死。我问你们几个问题，谁回答得好，回答得快，我让他去得痛快些。”
<br/>　　“第一个问题，你们是什么人？”
<br/>　　这二人被压得跪在甲板上，互相对视一眼，却又都不吭声。
<br/>　　程大雷撇撇嘴，一剑将女人的耳朵斩下，换来一阵凄厉的惨叫。
<br/>　　“这是第一次，如果再有第二次，我会刺你们的眼睛，不要让我那么做，我并不喜欢，相信你们也一样。”
<br/>　　“我们是骷髅岛的人。”
<br/>　　徐神机诸人一楞，这几个人果然有问题。
<br/>　　“海盗么，你们这次有多少人动手？”
<br/>　　“岛上的弟兄有二百人，我们寨里的二当家带队。”
<br/>　　男人回答，程大雷在女人身上刺了一剑：“你是听不清楚问题么，下次回答问题再不积极，你就没有机会了。”
<br/>　　女人抬起头，以极其恶毒的眼神看着程大雷。
<br/>　　程大雷视若无睹，继续询问自己想要的问题。他对海贼的事情不清楚，更没有听过所谓的骷髅岛，自然要问得详细些。
<br/>　　“你们为什么要对我们动手？”
<br/>　　“大当家说你刚进城就杀人，霸气外露。”
<br/>　　“咦，你们大当家见过我？”
<br/>　　“我们大当家外号红胡子，在城里化名耿星河。”
<br/>　　“咦，是他！”
<br/>　　程大雷等人大吃一惊，谁也没想到，那和和气气，见人三分笑容的耿星河竟是骷髅岛海贼红胡子。
<br/>　　他们对千舟城这里的情况不了解，红胡子一伙海贼，抢劫海上商船。而红胡子却坐镇千舟城，以做生意的名头，打听海上商船来往的消息。
<br/>　　“他脸上可没胡子？”
<br/>　　“红胡子脸上不一定要有胡子。”
<br/>　　“说得也有道理。”程大雷点点头，一剑帮对方抹了脖子。
<br/>　　“你是什么人？”女人问，程大雷的所作所为，绝不像一个做生意的公子哥。
<br/>　　“别太吃惊，咱们是同行，阎王爷那里也别告错了状，杀你的是青牛山蛤蟆大王。”
<br/>　　在对方吃惊的表情中，程大雷给了她一个痛快。
<br/>　　徐神机头顶冷汗唰唰滴落，幸亏程大雷及时发现这五人不对，否则若真让他们上了船，和海贼里应外合，一伙人怕都要死在海里喂鱼。
<br/>　　“大当家，你是怎么识破他们破绽的？”
<br/>　　“如果真是遭了海贼落难，在一个五人组成的小社会中，自然是谁拳头大谁掌握话语权。所以一开始与我说话的，绝不该是个女人。”
<br/>　　徐神机眼前一亮：“大当家高呀，他们绝想不到，大当家靠此识破了他们。他们伪装得太像了，我差点都上当了。”
<br/>　　“差点儿？”
<br/>　　“是呐，就算大当家没发现，我也会提醒你的，毕竟想糊弄我，他们还委实得不够。”
<br/>　　“……”程大雷。
<br/>　　“大当家你也别气馁，纵然你比我稍有不足，但也的确蛮高的。”
<br/>　　“高什么高啊。”程大雷摇摇头：“现在的情况，就算遇到的人没有任何问题，也是要杀的。咱们船上装着的是五千条人命，没资格冒险。”
<br/>　　“弟兄们，忙活起来，来买卖了。”程大雷忽然举剑高喊一声。
<br/>　　“好！”
<br/>　　两艘船一起回应。
