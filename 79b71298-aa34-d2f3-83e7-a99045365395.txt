　　没过多久，苏莫便来到了宗府门前。
<br/>　　宗府，距圣巫广场不远，位于天冥圣城的中央位置，是一片浩大的府邸。
<br/>　　府邸占地方圆足有百余里，院墙高大，气势磅礴，井然就是一座小城。
<br/>　　“见过巫尘少司！”
<br/>　　“见过巫尘少司！”
<br/>　　苏莫刚来到宗府门前，大门前的两名护卫便向他抱拳行礼，
<br/>　　“嗯，我来见大祭司！”苏莫点头说道。
<br/>　　“大祭司有交代，让你在‘程殿’等候，请随我来！”一名护卫说道，随即便引领着苏莫走进了宗府之内。
<br/>　　宗府之内地广人稀，建筑也不是很多，但每一座建筑均是气势恢宏，大气磅礴。
<br/>　　程殿，是一座偏殿，护卫带着苏莫穿过林荫小道，没过多久便到达了程殿。
<br/>　　殿中装饰古朴，是一个宽敞的客厅，墙壁上绘满了巫族的族印标志。
<br/>　　随后，护卫让苏莫在殿中等候，便大步离开了。
<br/>　　苏莫坐在殿中，打量了一番，便静静的等待了起来。
<br/>　　足足等了两个多时辰，到了中午时分，殿中空间一阵波动，大祭司的身影突然出现在了主座之上。
<br/>　　“见过大祭司！”苏莫见此，连忙起身抱拳。
<br/>　　“嗯！坐吧！”大祭司微微颌首，示意苏莫就坐。
<br/>　　少倾，大祭司沉吟一番，目光灼灼的望着苏莫，问道：“巫尘啊，你有几种战魂？”
<br/>　　苏莫闻听此言，顿时心中微震，望着大祭司的目光，他感觉自己仿佛被对方看穿了。
<br/>　　心念急转，苏莫急思如何回答对方，与巫人王一战之时，他动用了梦魔战魂，也不知道有没有被对方发现？
<br/>　　这个他不敢隐瞒，万一对方发现了，而他又隐瞒的话，他必然让对方生疑。
<br/>　　“禀大祭司，我有……三种战魂！”苏莫沉声说道，他本来想说两种战魂，那就是梦魔战魂和剑之战魂。
<br/>　　但他刚一开口突然想到了一个重要的问题，立刻改成了三种战魂。
<br/>　　这个重要的问题，便是巫尘本身的战魂。
<br/>　　巫尘在巫族之中，虽然是名声不显，但毕竟是武帝境武者，在巫族生活多年，黑耀城不少人都知道其战魂。
<br/>　　只不过巫尘的妖豹战魂，当初死亡之时已经消散了，并没有被苏莫吞噬。
<br/>　　但是，苏莫却不能忽略这个问题，所以只能硬着头皮说自己有三种战魂。
<br/>　　“哪三种战魂？”大祭司闻言，深邃的眼眸之中闪过一缕璀璨的精光，三生战魂这在天冥星巫族一脉之中，可是从未有过的。
<br/>　　“云煞豹战魂、剑之战魂、还有天幻战魂！”苏莫面不改色的说道。
<br/>　　巫尘的妖豹战魂，便叫云煞豹战魂。
<br/>　　至于天幻战魂，完全是他瞎扯的名字，他当然不能告诉对方，自己的战魂是梦魔战魂。
<br/>　　反正这种特殊的战魂，不真正感受到战魂之力，根本分不清具体是什么战魂。
<br/>　　“据老夫所闻，你以前只显露过云煞豹战魂，为何隐藏其它战魂？”大祭司淡淡的问道，显然其对于‘巫尘’有些许疑惑。
<br/>　　苏莫心中紧张到了极点，看来对方调查过自己，或许对方现在没有怀疑他的身份，但他若是露出一丝破绽，难免会引起对方的疑心。
<br/>　　不过，他也有一些底气，反正现在巫烈和巫玉儿已经被他杀了，死无对证，没有人知道真正的巫尘早已死了。
<br/>　　“回大祭司，我以前在族中没有什么地位，深知太强的天赋会引起他人的嫉妒之心，所以一直没有展露！”
<br/>　　苏莫努力的保持着淡定，微笑道：“这几年我一直憋着一股劲，就是要在少司选拔上一飞冲天，现在我做到了！”
<br/>　　大祭司闻言微微颌首，淡笑道：“你的确做到了，成为族中少司，才是你展露天赋的开始。”
<br/>　　言罢，大祭司手掌一番，掌中出现了一块巴掌大的黑色铁牌，随即抛给苏莫。
<br/>　　“这是我族天冥星一脉的少司令牌，持此牌你将拥有无上权力，可以号令无数的族人！”大祭司说道。
<br/>　　苏莫伸手接过令牌，稍一打量，只见令牌上刻着一个硕大的巫族族印，族印之下还有四个古朴的小字“天冥——少司”！
<br/>　　这少司令牌不知用何物打造，入手滚烫，仿佛是一块烙铁一般。
<br/>　　“多谢大祭司！”苏莫将少司令牌收进储物戒，然后向大祭司郑重抱拳。
<br/>　　“嗯！你日后就住在宗府吧！宗府内有通往祖地的通道，你可以随时前往祖地！”大祭司说道。
<br/>　　“多谢大祭司栽培，晚辈没齿难忘！“苏莫再次感谢道。
<br/>　　祖地，是巫族的禁地，只有族中族长、祭司、少司、核心长老才能进入，其内是巫族的传承之地。
<br/>　　可以说，只要能进入祖地，便能得到天大的造化。
<br/>　　当初巫天御，便因为立了功劳，得到进入祖地的机会。
<br/>　　不过，巫天御在祖地内待的时间太短，得到的造化并不大。
<br/>　　但是，进入祖地的机会虽然重要，但苏莫却是完全不在乎，他现在在乎的却是如何查探到青璇的消息。
<br/>　　“嗯，你有此心意便好！”大祭司微微颌首，面露满意的笑容。
<br/>　　“大祭司，晚辈有个问题，不知当不当问？”略一沉吟，苏莫暗暗咬了咬牙，开口说道。
<br/>　　“有何问题？”大祭司问道。
<br/>　　“大祭司，不知我族查到了那个苏莫的行踪了没有？”苏莫沉声问道，他准备从大祭司口中探听宏青璇的消息，虽然这样无比的危险，但他只要有理由，也不会太过唐突。
<br/>　　“苏莫的行踪？”
<br/>　　大祭司闻言，眸中寒光一闪，淡漠的道：“有些头绪了！”
<br/>　　言罢，大祭司目光凝视着苏莫，问道：“你为何问及此事？”
<br/>　　“上次试炼的族人中，我的两位好友惨死在苏莫的手中，所以有些关心此事！”苏莫叹道，他并不是信口开河，上次的试炼者之中，确实有两位是巫尘的朋友。
<br/>　　只不过，那两人和巫尘的关系很一般，根本不算是好友。
<br/>　　“此事你无需操心，很快便能找到苏莫！”大祭司目光幽幽，淡淡的说道。
<br/>　　苏莫闻言心中暗凛，不知道对方此言是否为真，难道对方已经查到他栖身古周星神武学府？
<br/>(本章完)
