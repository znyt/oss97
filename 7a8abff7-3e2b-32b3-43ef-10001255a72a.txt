　　第386章
<br/>　　长安城。
<br/>　　当年的天下第一城，如今多了些肃杀味道。正义教的十万大军包围了这座城池，从城头往下望去，到处都是飘舞的天蟾吞月旗。
<br/>　　其中最显眼的一张大旗上，写着一个『义』字，所有人都明白，义字旗下，就是正义教的首领，当年因为刺杀明帝被擒住，差点被砍掉脑袋的林少羽。
<br/>　　当然，义字旗再显眼，和城头那副龙旗相比还是差了很多。黄底红纹，上面绣一个李字，此刻竖立城头，迎风飘展。
<br/>　　百二十年前，当初的大武开国皇帝，就是打着这面旗，东征西战，打下整个天下都姓李。
<br/>　　时隔上百年，今天再挂出这面旗，不是为了打江山，而是为了守江山。
<br/>　　今天这面旗下站立的，是帝国皇子，李乐天。
<br/>　　从正义教兵临城下那一日，明帝便病倒了，的确，他的身体是一天不如一天，一刻不如一刻。在最后关头，他将长安的兵权交给了李乐天。所有人都明白，如果能守住这场危机，李乐天将会是帝国下一任皇帝，九万里江山的主人。
<br/>　　当然，现在说这话有些为时过早，因为今天之后，这万里江山姓不姓李都不好说。
<br/>　　今日，正义教鸣鼓发兵，有人在城关下叫阵。
<br/>　　环长安皆山，从外界进入长安只有一个入口，就是伏虎关。把守这里的是帝国老将西门俊，伏虎关易守难攻，只要打不破伏虎关，就无法靠近长安城。
<br/>　　同样的道理，只要打得下伏虎关，援兵也过不来。
<br/>　　在四十天前，正义教就打下了伏虎关，西门俊的脑袋挂在伏虎关头。伏虎关告破，长安无险可守，只能仗着精兵坚城。
<br/>　　正义教并没有着急进军，守伏虎关却不进攻长安城，只是日日在长安城下叫阵，单枪匹马决斗。
<br/>　　过去的这段时间，已经发生了十几场战斗，由帝国的龙虎之将应战，却无一例外败北。
<br/>　　林少羽已经连斩帝国十三员大将，这份战绩足以震惊天下，甚至超过他那位被冤杀的父亲。
<br/>　　帝国不能再败下去了，再败下去，丢失的不仅是颜面，还有人心。
<br/>　　长安城内早已经人心惶惶，有传言说林少羽是天命之将，暗中有神仙保佑，已有些大户，悄悄联系正义教，愿意私开城门，放他们进城。
<br/>　　对于这些人，李乐天直接处死，过去的这一个多月，长安城有大片大片的人被砍头。令李乐天无语的是，他们在临死前还高喊着正义不死，邪魔必亡。
<br/>　　而他们崇拜的神明是……蛤蟆大神。
<br/>　　这是最令李乐天崩溃的一点。
<br/>　　“大皇子。”
<br/>　　一个戎装男人的声音令李乐天回过来神，此人是将军府的门客，名叫田枝疆，年方三十，却是一个人精力最旺盛的时候。虽然还未在人前露过面，却是将军府着力培养的年轻人才。
<br/>　　“此战，许胜不许败，你有多大把握？”李乐天。
<br/>　　“请殿下放心，我若出马，必取林少羽狗头。”田枝疆昂着头道。
<br/>　　“不可大意，接连已有十三员帝国名将被他打败，他们的本事，未必弱于你。”
<br/>　　“谁强谁弱，通过这一战就可以见分晓。”田枝疆道。
<br/>　　李乐天点点头，大手一挥：“擂鼓，开城门。”
<br/>　　擂鼓则进，鸣锣则退，随着震天的鼓声，长安城的城门开启，一席戎装的田枝疆骑马踏出城门。
<br/>　　“乱臣贼子，请速速过来受死！”
<br/>　　李乐天在城头看着这一幕，表情有些无语。这一战的胜负，极有可能决定谁会是帝国未来的主人。然后，现在却搞得像是两个顽童打闹。于是，李乐天有些哭笑不得，千万人的生死，大武一百多年的气运，真的就如此滑稽的决定么。
<br/>　　只希望，田枝疆不仅口气大，本事也大，能赢了林少羽，否则，不管如何，总是输的人比较难看。
<br/>　　鼓声急鸣，忽见敌营中杀出一人，黑衣黑甲，手中一杆长枪。鼓声越急，马蹄越快，最后如同一道黑色闪电。
<br/>　　很快，在上万人的注视下，两马相交，长枪如黑蛇一般探出。
<br/>　　田枝疆没见过这么快的枪，瞳孔中只剩下一道电芒，越来越急，越来越大，随之，长枪洞穿他的心脏，惊愕的表情永远留在他脸上。
<br/>　　噗通一声响，田枝疆的尸体坠地，只有他的坐骑唿律律奔回城关。
<br/>　　当着两军的面，林少羽只用一招，就结果了田枝疆。帝国未来的将星，再一次被他扼杀。
<br/>　　李乐天对这一幕，并不感觉意外，失败得太多，多到他已经麻木的程度。
<br/>　　这一幕已经不断重演，反正正义教也不进攻，当然，也攻不下长安城。
<br/>　　忽然，城下的林少羽横起长枪，长枪指向城头的李乐天。
<br/>　　李乐天感觉到了杀机，果然，林少羽的野心不小，只杀一个田枝疆没办法满足他的胃口。他所贪图的是自己的脑袋，甚至是自己那位父王的命，才能满足他的胃口。
<br/>　　他冲城下的林少羽拱拱手，口中吐出两个字。
<br/>　　“收兵。”
<br/>　　今日的战斗就落下了帷幕，双方已经耗了这么久，谁也不介意继续耗下去。
<br/>　　……
<br/>　　琴川关。
<br/>　　程大雷在书房和阿喜碰面，不得不说，阿喜精神了很多。自从程大雷把情报部门交给他后，他憋着劲儿要干出点事来，给所有人证明自己。
<br/>　　程大雷今天叫他过来，也是看他最近做得怎么样。说了放手不管，也不能完全不过问。
<br/>　　简单聊了一下，阿喜做得不错，他是有脑子的，只要认真做事，总也不会辜负了程大雷。经过这段时间的苦练，这帮人的草莽气已经去了很多，渐渐像真正的军人。
<br/>　　“大当家，这是什么呐？”
<br/>　　程大雷看到阿喜拿起书案上的纸钞，他不愿和别人提起这件事，自己只是偷偷摸摸的研究。现在阿喜问起来，戳中了程大雷的痛处。
<br/>　　“哈，也没有什么了，随便折腾玩的，你喜欢就送给你了。”
<br/>　　“我听人说大家准备发行纸币，是不是就是这东西啊。”
<br/>　　程大雷白了他一眼，你真是哪壶不开提哪壶。
