　　看了一眼，九叔便也没管三人，在他眼中，就算自己的徒弟再笨，这么简单的事情，他们也应该不会出什么纰漏才对。
<br/>　　见祭台上的香已经快要燃烧完毕，九叔拿出三根崭新的香来，放在蜡烛上点燃，给祭台续上了香火。
<br/>　　“人分好人坏人，尸分僵尸死尸，任老太爷的尸就是快要变成僵尸的尸。”将香插在香炉上，九叔缓缓开口，不知为何，声音却是有些低沉。
<br/>　　正在帮棺材弹上墨斗的刘哔眼中精光一闪，就在九叔刚才插香的时候，他感应到一股天地灵气，随着九叔的动作，涌进了香炉之中。
<br/>　　怪不得，九叔之前一直嘱咐他们，停尸间的香不能灭掉，先前刘哔一直不懂，现在才察觉到有什么作用。
<br/>　　是因为九叔将灵气灌进香炉，所以插在上面的香，才有了一些特殊的作用，这应该也是防止停尸间尸变的最根本原因吧。
<br/>　　在此之前，没凝聚出灵种的刘哔根本察觉不到灵气的存在，现在却观察到了一丝端倪。
<br/>　　事实上在九叔之前做法事的时候，他就察觉到一股不太正常的波动，但因为他修行尚浅，并没有从中观察到什么，现在联想起来，之前九叔做法，估计也没有那么简单。
<br/>　　“尸怎么会变成僵尸呢？”秋生手中忙活着，随口发问，文才也笑嘻嘻的接道：“是啊，人怎么会变成坏人呢？”
<br/>　　拍了拍手，九叔叉着腰，来到棺材旁边，注视着三人给棺材弹上墨斗：“人变成坏人，是因为他不整齐，尸变成僵尸，是因为多了一口气啊。”
<br/>　　心中一动，刘哔手上忙活着，耳朵却是竖了起来，听九叔在旁边继续说道：“一个人在死之前，生气，憋气，闷气，到死了之后，就会有一口气，聚在喉咙那。”
<br/>　　点了点头，秋生认同的符合道：“所以我说人活着要争气，人死了最要紧的是断气，如果不断气，就只会害人害己。”
<br/>　　“让你弹就弹，别满嘴的怨气，弹好了就告诉我，千万别漏了啊。”眉头微微一皱，九叔伸手拍了一下秋生的后脑勺，他似乎有些理解错了秋生的意思。
<br/>　　秋生顿时低下了头，口中应了一声，九叔这才满意的走了出去，忙活了这么久，他觉得有些口渴，想要去沏一杯茶，润润嗓子。
<br/>　　片刻之后，三人弹完了线，秋生和文才嘻嘻哈哈的打闹着追逐了出去，刘哔看着棺材，脸上浮现一抹犹豫。
<br/>　　看过原片的他，知道棺材底部是没有弹上墨斗的，再加上二十多年的时间，棺材已经有些腐烂，这才被任老太爷变成的僵尸生生给抖散了。
<br/>　　也是为此，任发才在第二天夜里死在了自己父亲口中，也是因为吸食了至亲之人的鲜血，这才变的强横无比，以至于九叔和师弟联手，方才险险将其制服。
<br/>　　如果想要顺利将任务完成，将棺材底弹上墨斗，然后彻夜守护在棺材旁边才是最好的选择，但这样一来，无疑是让刘哔失去了一次获取积分的机会。
<br/>　　要知道，僵尸也算是能量体，是会产生积分的。
<br/>　　虽说，以现在刘哔的实力，是肯定对付不了原著中那只已经进化了的僵尸，但是如果在任发没死之前，让他和这头僵尸撞上，还是有很大的胜算。
<br/>　　刚刚尸变，充其量也就是前几天那几头僵尸一样的实力，先前刘哔就能和其打成平手，如今实力提升了这么多，自然也不会有什么问题。
<br/>　　按照这个世界的等级划分，他之前碰见的僵尸，连最低级的行僵都算不上。
<br/>　　犹豫了良久，刘哔下定了决心，便将墨斗收起，随手将停尸间的大门关上，走了出去。
<br/>　　他知道今晚任家还不会有事，明天清晨的时候，这具尸体才会变成僵尸，况且，因为刚刚尸变，白天的时候根本无法行动，也只有晚上才会出现在任家。
<br/>　　但是为了预防事情有什么变化，他还是将棺材周围洒上了一圈糯米，还在糯米中加入了一些灵气。
<br/>　　虽然不知道会有什么效果，但是聊胜于无吧，今晚，他还有一件事情需要去做，
<br/>　　……
<br/>　　出了停尸间，外面天色已经开始黑了下来，刘哔确认了一下秋生还没走，便径直走出义庄，隐藏在外面的房屋旁等候着。
<br/>　　大概半个多小时的时间，秋生和文才走了出来，一边絮絮叨叨的说着什么，秋生手中，还握着一捧点燃的香。
<br/>　　眉毛一挑，刘哔舔了舔口中的獠牙，这段时间刻意的想要最大限度保护原著剧情，为的，也只是今天这一刻。
<br/>　　他非常清晰的记得，在原著中有一个女鬼，一直在缠着秋生，最重要的是，这并不是一头恶鬼，而且实力还不怎么强劲。
<br/>　　对于刘哔来说，这简直就是到手的积分，他是绝对不会放弃的。
<br/>　　虽然说这个世界的鬼怪僵尸众多，但寻常也不是十分容易碰见，而且碰见了，他也不一定打得过。
<br/>　　像这样几乎没有风险的机会，已经不多了，借此事件，他还可以验证一下自己的吞噬技能。
<br/>　　虽然对于吃这种人形生物有些抗拒，但为了提升自己的实力，他也顾不上那么多了。
<br/>　　末世十年，为了生存吃人的都不计其数，更何况是吃鬼，虽然刘哔没有尝试过，但也不是很难接受。
<br/>　　以至于为了让食材更加可口，他还从厨房拿了一点孜然和盐，要不是时间有限，他甚至还要带上一口铁锅。
<br/>　　这边的刘哔准备已经齐全，而秋生此时也是浑然不知，依旧按照原著中的剧情，将香插在自行车头上，慢悠悠的骑着向家中赶去。
<br/>　　确定了一下四周的情况，刘哔从藏身地走了出来，紧紧跟在自行车后面不远处。
<br/>　　虽然一个骑车，一个步行，但经过强化后的体质，根本没让刘哔感觉到吃力，没过多久，他就跟随秋生出了城镇，来到镇外有些阴森的小道上。
<br/>　　四周树木林立，借助着微弱的星光，秋生慢悠悠的向家中骑着，四周逐渐，弥漫起了一层浓郁的雾气。
