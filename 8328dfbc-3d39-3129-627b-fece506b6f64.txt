　　“我要将这万年间的剑术，再次凝聚在一起，取其精华，足以令我的剑术，再进一步！”
<br/>　　楚风眠心神一动，一下子通悟过来。
<br/>　　看向这雕像，仔细的参悟出其中的剑术来。
<br/>　　这雕像是主人，是一名名为剑极真人的武者，实力踏入了生死台阶，却是没有将九道台阶全部踏过。
<br/>　　他的剑术，便是名为剑极剑术，这是一种极端速度的剑术。
<br/>　　在楚风眠看来，虽然嘈杂无比，但是其中的一些玄妙，也知道楚风眠参悟。
<br/>　　“我要凝聚所有剑术，化为最为强大的剑术，我在九域，便为九域之王，名为九域剑术！”
<br/>　　楚风眠心神一动，他所领悟的所有剑术，同时融合，开始来化为这九域剑术。
<br/>　　将来，楚风眠必将是以这剑术，称霸九域。
<br/>　　“去更里面看看。”
<br/>　　几分钟的时间，楚风眠便是将这剑极真人的剑术，全部参悟。
<br/>　　他看了一眼旁边的白萱，还在参悟剑术，楚风眠便是没有叫醒白萱，而是直接向着众圣堂深处走去。
<br/>　　这越是靠近众圣堂深处，其中雕像主人的实力，也就越发强大。
<br/>　　其中蕴含的武道，也越发的晦涩，难懂，虽然强大无比，但是北邙学院历代弟子中，真正能够将其领悟出的，却是百不足一。
<br/>　　不过这些却是都难不倒楚风眠，前世之中，楚风眠的剑道领悟，可是就连剑道第一人，他的师尊剑道之主都自叹不如。
<br/>　　这其中的剑术，无论是多么隐晦，楚风眠都足以全部领悟。
<br/>　　所有剑术的精髓，都全部是被楚风眠凝聚到了他的九域剑术之中，令他的九域剑术，开始升华。
<br/>　　每凝练出其中一种剑术，楚风眠也就感觉到，他的九域剑术变的更加强大，玄妙。
<br/>　　“这北邙圣域***奉了如此之多先祖雕像，上面蕴含的剑术，武技，秘术，其他人很难参悟，却没想到居然是给我送上了这样一份大礼！”
<br/>　　楚风眠看着众圣堂中，还有着无数他还没有参悟的雕像，脸上也是不由大喜。
<br/>　　“将这所有雕像的剑术参悟，我的九域剑术，至少足以达到小成，倒时候足以让我的实力，再进一步！”
<br/>　　寻常剑修参悟剑术，一般只能参悟其中一种。
<br/>　　因为这每一道剑术，其实其中的奥秘，也都不尽相同，尤其是有些剑术的奥秘，都是有着冲突的。
<br/>　　一般剑修若是修炼好几种剑术，只怕是实力会不进反退。
<br/>　　但是楚风眠不同，楚风眠本身的剑道领悟，就在这些人之上，这些剑术楚风眠参悟，也只是参悟其中精华，而并非是剑术本身。
<br/>　　他参悟的越多，楚风眠对剑道的理解，也就会更深一步。
<br/>　　足足七日时间。
<br/>　　楚风眠一直都在这众圣堂中，连连参悟了无数剑术，令他的九域剑术，已经是小有所成。
<br/>　　这众圣堂中的无数雕像，楚风眠都已经参悟。
<br/>　　如今只剩下这众圣堂中，最深处的那一尊雕像。
<br/>　　据说是这北邙学院的创始人，便是一尊极为强大的剑修，楚风眠也想要见识一下，这一手创造出北邙学院的，到底是一位什么人物。
<br/>　　楚风眠在众圣堂中，一直前行，一直是来到了其中最为庞大的一尊雕像面前。
<br/>　　这雕像，仅仅是靠近，其中凌冽的剑意，都要将人撕裂一般。
<br/>　　这北邙学院创始人的雕像，就算是北邙圣域中的种子弟子，都不敢轻易靠近，只有其中的佼佼者，才能够靠近，去参悟。
<br/>　　“这雕像……”
<br/>　　楚风眠看向这雕像，感觉到其中的剑术，突然之间，居然是感觉到了一丝丝熟悉的感觉。
<br/>　　这雕像上的剑术，气息，都让楚风眠，感觉到极为熟悉。
<br/>　　尤其是在雕像中心，刻画的一把灵剑印记，更是让楚风眠脑海轰的一下炸开。
<br/>　　极道剑印！
<br/>　　这雕像上的印记，正是极道剑印，曾经剑道之主的印记。
<br/>　　“这北邙学院的创始人，到底是谁？”
<br/>　　这雕像的样貌，楚风眠倒是极为陌生，但是其中蕴含的剑意，还是这极道剑印，都代表这，这雕像跟楚风眠，跟曾经的剑道之主，脱不开关系。
<br/>　　很有可能，是楚风眠极为熟悉的一尊存在。

<br/>　　楚风眠扫视了周围一圈，在确认没有任何人后，便是悄然之间，凝聚出了一股剑意。
<br/>　　这一股剑意，也化为了一把跟那雕像印记一模一样的灵剑，猛然刻了上去。
<br/>　　“轰隆！”
<br/>　　灵剑归位，顿时这雕像上，一下子发生了极大的变化。
<br/>　　周围的空间，顿时开始天旋地转起来，仿佛是周围的一切都发生了一种诡异的变化。
<br/>　　从这雕像之上，散发出来的剑意，却是仿佛将周围的空间都给直接封锁了一般，令周围空间，发生了如此变化。

<br/>　　起码楚风眠的大师兄剑凌，便是活了下来，不过却隐姓埋名，建立了这北邙学院。
