　　　　拜托，这是二阶精怪级别的超等生物，岂是你一个还未觉醒的人能碰。

<br/>　　坐到客厅的一张沙发上，涂小安看着又在认真梳理九彩色羽毛的青枭道：“进化也进化完了，将你送回去吧”

<br/>　　大清晨，涂小安也没什么事情可做，打开系统商城，就跟看淘宝一样的在里面闲逛着。
<br/>　　心中不禁的想，自己是不是应该去弄件什么武器，他现在三种元素属性能力加身，强是很强，但动不动就用元素属性能力，太过招摇了。

<br/>　　点进去一看，顿时呆若木鸡。
<br/>　　《金蛇剑》此剑金光灿烂形状奇特，剑身犹如是一条蛇盘曲而成，价格：50000000（RMB）。
<br/>　　《火麟剑》剑中邪神，剑上的鳞甲可自生一股不可思议的力量，价格：10000000（RMB）。
<br/>　　《倚天剑》锋利无比，削铁如泥，倚天不出，谁有争锋，绝世神兵，（自带剑气）价格：1000000000（RMB）。
<br/>　　.......
<br/>　　剑的兵器很多，多不繁几，涂小安从排名最高的上方看了几眼，既然都是武侠小说中的神兵利器。

<br/>　　只是，涂小安看了一眼后，激动归激动，也不说贵不贵的，但你后面写的RMB是几个意思。
<br/>　　莫非兵器界面中，需要用真实的RMB来购买，而不是信仰点数了。

<br/>　　像这种绝世神兵，花上一个亿貌似并不贵，但是突然需要RMB，还是让涂小安有点不能接受。

<br/>　　雪饮狂刀，乃天下间至寒之物，刀长三尺七寸，锋刃无瑕，（自带刀气）价格：100000000（RMB）
