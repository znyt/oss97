<br/>				今天的早饭依旧是小米粥和凉菜，以及两片锅盔。<br/>　　<br/>　　锅盔是西秦非常有特色的食物，人称“睁眼锅盔像锅盖“，即象头盔状，顾名思义又叫锅盔。锅盔制作工艺精细，素以“干、酥、白、香“著称。干硬耐嚼，内酥外脆，白而泛光，香醇味美。<br/>　　<br/>　　锅盔的制作，是用温水和面，先和成死面块，放在案头上用木杠压，边折边压，压匀盘好，然后切成两块，分别加入酵面和碱水再揉压，视面的软硬程度，如面软可加些干面再压，直至面光色润，酵面均匀时，用温布盖严发酵一下。<br/>　　<br/>　　把面块分成几块，看自家烙馍锅的大小，推擀成圆形，大约两指到三指厚的饼，上锅勤翻转，俗称“三翻六转“，烙得火色均匀，皮色微鼓时即熟。<br/>　　<br/>　　这是一种非常好吃的馍馍，在古时作战的时候，都可以作为军粮使用，而且能有比较长的保质期，还顶饿。<br/>　　<br/>　　陈默去上市上学的时候，基本也就没有吃过，这不是回来后，昨天晚上让小舅烙了几个解解馋。<br/>　　<br/>　　自从回来后，每天变着花样吃，怎么也吃不烦，还是家乡的饭香啊！有句俗话说的好：金窝银窝，不如自己的狗窝。人在怎么样，总是忘不掉自己的家乡，尤其是家乡的食物。<br/>　　<br/>　　正在陈默吃的呼噜的时候，两只小狗站起来，直接跑到了门前，看来是有人来了。<br/>　　<br/>　　将两只狗叫回来，就见德林叔走了进来。<br/>　　<br/>　　“二娃，你养的这两只狗真不错，我还没有走到呢，它们就到门口迎接了！哈哈！”陈德林笑着说道。<br/>　　<br/>　　陈默顿时满头黑线，大灰和大黄那是迎接你？别搞笑了吧，如果不是自己叫住两个小家伙，走进院门绝对会咬上去的！叔！就别逗了好不！<br/>　　<br/>　　“德林叔，你早饭吃了么？”陈默问道。<br/>　　<br/>　　“吃过了，才过来的。”<br/>　　<br/>　　村里一般早饭都吃的早，所以陈默也就不再客气，引着陈德林坐下。<br/>　　<br/>　　“那我还有点饭没有吃完，你等等！”<br/>　　<br/>　　“你吃你的，不用管我！”看到桌子上放的香烟，就自己动手掏出一支抽起来。这是陈默放在凉亭这里，招待客人用的，都是好烟，所以陈德林也不去客气什么。<br/>　　<br/>　　陈默三两下将饭扒拉完，然后将碗收拾了一下，放到小舅厨房哪里。为了自己的方便，他在栅栏哪里开了一个小门，这样进出就方便些。<br/>　　<br/>　　“德林叔，你找我来是什么事情啊？”<br/>　　<br/>　　“没事就不能来找你？”<br/>　　<br/>　　陈默听了只能腹诽，这不就是废话么！<br/>　　<br/>　　“呵呵！德林叔，怎么可能，你那个时候过来都没有问题。”陈默只能笑着回答。<br/>　　<br/>　　“屁话!”陈德林也是笑了笑，他是看着陈默长大的，所以也没有什么忌讳，无论他有钱还是没有钱，有能力还是没有能力，都是自己的子侄辈，自己笑骂几句无伤大雅。<br/>　　<br/>　　“我找你还真有事情！”陈德林说道。<br/>　　<br/>　　“哦！你说！”陈默将茶壶烧上水，然后拿出一个干净杯子，准备给德林叔倒点茶喝。<br/>　　<br/>　　陈德林却摆摆手，将自己带的茶杯拿出来，指了指后说道：“续到这里就成！”<br/>　　<br/>　　“放不放茶叶？”<br/>　　<br/>　　“不用，早上才泡的茶！”<br/>　　<br/>　　“二娃！今天找你来是化缘来的。”<br/>　　<br/>　　“化缘？”<br/>　　<br/>　　“嗯！村里要修建一下道路，另外要改造一下电力电缆，还有修建一些路灯，但是有些资金不足。”陈德林说道。<br/>　　<br/>　　“怎么会想到修建道路这些呢？”<br/>　　<br/>　　“主要是背街上的一些路现在还是泥土路，村里人走着也不方便，所以我想着修一下。另外就是上次陈银水和陈二虎两个人出事后，但是一直没有查出来是怎么回事，再说出事的时候，村里也是黑灯瞎火的，所以村民也反应说是不是修个路灯什么的，能避免一些危险。<br/>　　<br/>　　可是修建路灯什么的，就会敷设电力电缆，这个不是电力公司负责，钱需要村里掏！”陈德林说道。<br/>　　<br/>　　陈默心中一愣，上次将陈银水和陈二虎这两个人给弄残废了，没有想到这倒是兜兜转转的又找到了自己。<br/>　　<br/>　　“德林叔，那两个人现在怎么样了？”<br/>　　<br/>　　“废了！只能躺在床上，不过很惨！尤其是陈二虎，老婆也跑了，现在就他一个人，也不知道能挺多长时间！”<br/>　　<br/>　　“哈！算了，不说这些糟心事情了！德林叔，出钱什么的，没有问题，你说个数！”陈默并没有计较什么，直接询问道，这是好事，自家也能用上不是。<br/>　　<br/>　　“差的不多，也就十万元！”陈德林说道。<br/>　　<br/>　　“行！我给你取过来！”转身去了房间，其实是从乾坤珠内取出十万元，直接拿给德林叔。虽然可以通过转账，但是德林叔除了会用手机打电话外，还真没有怎么操作过，所以对于陈德林来说，还是现金方便些。<br/>　　<br/>　　“二娃！你知道我为什么找你，让你出这个钱呢？”陈德林看了看桌子上的十万元，有点了支烟抽着，询问陈默。<br/>　　<br/>　　“嘿嘿！我哪里知道，但是我知道德林叔你不会坑我就是了！”陈默当然不知道，但是却和他说的一样，陈德林不会坑他就是了。<br/>　　<br/>　　“知道就好！”陈德林撇了陈默一眼。<br/>　　<br/>　　“二娃子，你告诉我，你是不是得罪什么人了？”<br/>　　<br/>　　“嗯？怎么说？”<br/>　　<br/>　　“镇里有人在查询你的事情，上次还把我叫去询问来着。”<br/>　　<br/>　　“查询我什么事情？”<br/>　　<br/>　　“承包合同有没有什么违法条款，还有你回来后有没有什么违法或者违反规定的事情。”<br/>　　<br/>　　“那就奇怪了，我回来后也没有怎么出去过，怎么会得罪人呢？”陈默想了想，还真没有想到得罪过谁。<br/>　　<br/>　　陈德林看了看陈默不是作伪，也就点了点头，说道：“没有什么就好，镇里的事情我给怼回去了。所以你放心好了。”<br/>　　<br/>　　“另外让你出钱，就是想村里的人都知道，你是做了好事的了，如果谁在听了什么谣言之类的跳出来，我绝对让他好看！”<br/>　　<br/>　　陈德林没有说到底是谁，但是陈默也知道能将德林叔叫去问话的，也就那么几个人。另外他也明白了，让自己出钱，是为了安定村里的人心，让大家都知道，陈默也是出了钱，让大家都落到了好处，那么谁要是心里不忿或者有意见，他德林叔就不放过。<br/>　　<br/>　　攘外必先安内，这也是陈德林的考虑。既然外面有人想要找陈默的事情，那么自家子侄就要保护着，看看谁能怎么着。<br/>　　<br/>　　陈默殷勤的将德林叔送走以后，不大功夫就听到村里的广播将自己出了十万元的事情广播了好几遍，也让村里人知道，陈默他还真是发财了不说，不过也出血了不是。<br/>　　<br/>　　但是陈默却有些警醒，究竟是谁想找自己的麻烦？<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
