        傅夜沉离开后，童筱颖用力闭了闭双眼，一直在压下自己那不争气的眼泪，然后装作若无其事地继续吃饭。
<br/>
<br/>        而当天下午，她正坐在办公室里工作，傅夜沉真就替她请了一个律师团队过来，对她做婚内强奸的取证。
<br/>
<br/>        但是律师还未开口提问，就被她给毫不客气地轰走了。
<br/>
<br/>        结果，这件事情，被以讹传讹。
<br/>
<br/>        童筱颖去上厕所的时候，刚准备从隔间里出来，就听到门外有女同事在背后议论她。
<br/>
<br/>        “我听说，设计部那个外聘进来的童筱颖，要告我们傅大Boss强奸她呐！”
<br/>
<br/>        “我去！她还真是不要脸啊！也对！她本来就是照着女明星苏漫雪的样子整容的！哪里还记得自己原来的脸长什么样子。”
<br/>
<br/>        “我想想都觉得好笑啊！她都不照镜子的吗？”
<br/>
<br/>        “她怕是想钱想疯了吧！”
<br/>
<br/>        “我觉得吧！她一定是其他公司派来给咱们的傅大总裁下套的！”
<br/>
<br/>        “搞不好，那女人真的就是商业间谍。太可怕！”
<br/>
<br/>        “你补完妆了没有？走吧！赶紧回去做事吧！”
<br/>
<br/>        “补完了！走吧走吧！”
<br/>
<br/>        她们离开后，童筱颖才从隔间里出来，整个人的脸色惨白到了极点。
<br/>
<br/>        就在她返回设计部的途中，半路被落以柔给拽住了手腕，强行拉到了安全通道里。
<br/>
<br/>        “童筱颖，我说你脑子要是有病，就去治好吗？傅总亲自外聘了一支律师团队过来，说是要帮你打官司告他。你要告傅总什么？强奸罪？你可不可笑？”落以柔只知道傅夜沉去请了律师的事情，但是至于为什么要请律师，她也只是听到了一些流言蜚语。
<br/>
<br/>        今天中午在餐厅里，她和傅夜沉的谈话内容，一定被一旁的员工偷听到一些断字断句的内容，才会以讹传讹。
<br/>
<br/>        面对落以柔的质问，童筱颖显得有些无动于衷。
<br/>
<br/>        每一次都是这样，不管发生什么事情，到头来都是她童筱颖的错。
<br/>
<br/>        “童筱颖，我警告你。这件事情的流言蜚语，我会替你扼制在傅氏集团内部，并让那群喜欢八卦的女员工们闭嘴。你不替傅总着想，我得替他着想。傅总若是因为你这事情而出了事，就代表整个傅氏集团都得跟着出事。而你自己最好守口如瓶。还有那个律师团队，你最好给我撤了！这事就当什么也没发生过！”落以柔双拳紧握，疾言厉色地呵斥。
<br/>
<br/>        童筱颖沉默不语，任由她劈头盖脸地训斥自己。
<br/>
<br/>        原本火大的落以柔，此刻也因为她童筱颖的沉默，而闹到最后没点脾气了。
<br/>
<br/>        她童筱颖到底是一个什么样的女人，完全分不清一件事情的利弊，还可以做到如此心如止水，事不关己高高挂起的姿态。
<br/>
<br/>        “你好自为之。”落以柔斜睨了童筱颖一眼，甩手离去。
<br/>
<br/>        当时，童筱颖不知道落以柔的这句“你好自为之”是什么意思，直到一个月后，傅氏集团外聘的CEO上任，她才真正明白。
<br/>
<br/>        这一个月来，傅夜沉不回家，白天晚上都见不着人，一个电话不给，一条短信也没有的原因，是他带着郑忠威和落以柔，以及手下一支精英团队，去了国外，拓展海外市场。
<br/>
<br/>        明明他是她的丈夫，得知此事时，她还是从部长孙莉莉口中听说的，而且她居然是最后一个得知的，是不是很可笑？
<br/>
<br/>        他傅夜沉不会为了儿女情长，而止步不前，也没时间陪她过度地沉浸在悲愤与气恼之中。
<br/>
<br/>        同时，她说，要他离她越远越好，他真就这么做了。
<br/>
<br/>        她不知道他到底是不是故意而为之，而是因为拓展海外市场的公事刚好撞到她这件事情的风口上而产生的巧合。
<br/>
<br/>        不管是哪一种，她的世界彻底清静了。
<br/>
<br/>        一个人上班、一个人下班、一个人吃饭、一个人睡觉、一个人逛街……
<br/>
<br/>        她把这个世上，最后一个爱她、宠她又伤她的人，给彻底地气走了。
<br/>
<br/>        他连个音讯，都没有给她。
<br/>
<br/>        好几次，她都拿起手机，想要给他打电话，想要给他发短信，最终还是因为自己的倔强而放弃了。
<br/>
<br/>        他都不主动给她打电话、发短信，那么她为什么要主动？
<br/>
<br/>        手机一扔，她翻了个身，将被子拉过头顶，蒙头大睡。
<br/>
<br/>        其实，也就只有她自己心里最清楚。
<br/>
<br/>        这一个月来，她夜夜失眠，日日夜夜都在想他，想他什么时候回来陪她，还告诫自己，下一次，一定不再对他说那些口是心非的狠话了。
<br/>
<br/>        可是，事实证明，她想也没用，或许大概可能，他真的不会再回到她身边了……
