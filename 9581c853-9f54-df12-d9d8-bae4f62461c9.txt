　　奇异空间内。
<br/>　　萧逸与寒冰傀儡鏖战着。
<br/>　　但，一直处于下风。
<br/>　　同境界，同实力的战斗中。
<br/>　　萧逸从来都是无敌，从来都是他碾压别人。
<br/>　　而这一次，自己的肉体力量，已经飙升到地元八重了。
<br/>　　实力比这头只有地元五重力量的傀儡，高出三重。
<br/>　　结果反倒是他自己被碾压。
<br/>　　自己一个地元八重，被地元五重碾压？
<br/>　　实在是有些匪夷所思。
<br/>　　而归根结底，实在是这头傀儡的极致寒冰太过厉害。
<br/>　　它的剑道水平，则是堪称恐怖。
<br/>　　嘭…嘭…嘭…
<br/>　　萧逸一次次被轰飞。
<br/>　　胸膛上，已然多出了几条狰狞血痕。
<br/>　　“游云杀步。”
<br/>　　迫不得已下，萧逸只能将刚刚炼至小成不久的游云杀步使用出来。
<br/>　　脚步玄妙挪动。
<br/>　　朝寒冰傀儡一剑刺出。
<br/>　　锵的一声。
<br/>　　寒冰傀儡持剑挡下。
<br/>　　但，这一次萧逸没有给寒冰傀儡反击的机会。
<br/>　　一剑不成，萧逸脚步立刻后退。
<br/>　　接着，萧逸再寻机会，再次攻击。
<br/>　　锵的一声。
<br/>　　再次被挡下，便再次后退。
<br/>　　游云杀步，一剑一杀。
<br/>　　精髓，便是在那一剑之下。
<br/>　　一剑与一剑的微小间隔中，会有破绽。
<br/>　　萧逸不能让这些破绽被寒冰傀儡捉住。
<br/>　　战斗，忽然成了僵局。
<br/>　　萧逸与寒冰傀儡，战成了平手。
<br/>　　但，他却没有丝毫喜悦。
<br/>　　此刻凝聚在他心头的，只有浓浓的惊讶和挫败感。
<br/>　　对，挫败感。
<br/>　　寒冰傀儡，拥有的力量，只有地元五重。
<br/>　　而自己，地元八重肉体力量，手持极品灵器血戮剑。
<br/>　　外加地阶巅峰步法，游云杀步。
<br/>　　再加剑势。
<br/>　　以及霸道之剑的精妙剑道。
<br/>　　多番增幅下，竟还只是平手。
<br/>　　而且对手的力量还比他低那么多。
<br/>　　这让他如何不感到挫败。
<br/>　　“寒冰傀儡上，到底烙印的是哪位剑宗长老的剑宗知识。”
<br/>　　“竟强悍如斯。”
<br/>　　萧逸自问，以他如今多番增幅下的实力。
<br/>　　击杀地元九重都轻而易举。
<br/>　　战天元一重都有一定把握。
<br/>　　难怪这第二层的最难试炼，连天元境都难以通过。
<br/>　　时间，在一分一秒过去。
<br/>　　萧逸，始终连寒冰傀儡的身躯都伤不了分毫。
<br/>　　数个时辰后。
<br/>　　萧逸开始着急和烦闷。
<br/>　　虽说战斗一直是平手，处于僵局。
<br/>　　但，寒冰傀儡，怎么打都不会累。
<br/>　　而自己的修罗之力，可是会一步步耗光的。
<br/>　　待得修罗之力耗光，自己的肉体力量跌回地元五重时。
<br/>　　他必败无疑。
<br/>　　再这样下去，他只剩一条路可走。
<br/>　　便是使用其他底牌，如冰鸾剑，如紫炎、十界灭生火，如崩界拳。
<br/>　　但，这些底牌，他都不想用。
<br/>　　准确来说，是不敢用。
<br/>　　他终究是初初来裂天剑宗，不知这里的情况。
<br/>　　这些底牌，若是暴露了，被发现了，自己将十分麻烦。
<br/>　　至于肉体力量，这不溢于表面，倒是不会被发现。
<br/>　　在这着急和烦闷，以及深深的挫败感之下。
<br/>　　时间又过了半个时辰。
<br/>　　这时，萧逸莫名的眼前一亮。
<br/>　　似是遇到了什么让他兴奋的事。
<br/>　　“我的剑道水平，竟在提升。”萧逸心头一喜。
<br/>　　明显感觉到，原本只能真正掌控三成的裂天剑魔的剑道知识。
<br/>　　现在，已经达到四成。
<br/>　　萧逸这才发现。
<br/>　　与寒冰傀儡战斗的这数个时辰中。
<br/>　　自己配合游云杀步，以及每次刺出一剑都是霸道之剑。
<br/>　　无数次挥剑，无数次压迫自己潜能，以最高水准施展霸道之剑。
<br/>　　他的霸道之剑，也在不断融会贯通。
<br/>　　裂天剑魔的剑道知识，他早已参悟。
<br/>　　缺的，只是真正将其为己所用，真正做到完全掌控的地步。
<br/>　　同时，他的武道瓶颈，也在不断提升。
<br/>　　“此战，无论胜败，单单能提升我的剑道水平，就是最大的收获。”
<br/>　　萧逸心头乐观地想着，再次战意大增。
<br/>　　……
<br/>　　奇异空间内，萧逸正与寒冰傀儡鏖战。
<br/>　　而剑阁第二层中。
<br/>　　流行剑主、猛虎剑主、戒墨和钟无忧四人，正满脸着急地等待着。
<br/>　　蓦然地，一道身影凭空而现。
<br/>　　正是感知到几人的剑主令气息后，连忙赶来的剑宗大长老。
<br/>　　见大长老来到。
<br/>　　几人连忙将萧逸和万山剑主的事告之。
<br/>　　剑宗大长老，当即皱起了眉头，并上前看了一眼那小光球。
<br/>　　“可是这个光球？”大长老问道。
<br/>　　“嗯。”几人点点头。
<br/>　　大长老脸色一变，随后叹息一声。
<br/>　　“北山剑主的试炼之路，结束了。”
<br/>　　“这是第二层最难的光球，没有之一。”
<br/>　　“天元境一重，都无法通过。”
<br/>　　众人闻言，道，“此事，错在万山剑主，能否…”
<br/>　　“不可。”大长老打断道，“规矩就是规矩。”
<br/>　　“不过，北山剑主在里头不会有危险的。”
<br/>　　“剑主令可保他平安，将他送到阁楼之外。”
<br/>　　说罢，大长老看向四人。
<br/>　　“走吧，我带你们回第一层，继续试炼。”
<br/>　　大长老双手一挥，光芒一闪，众人已经离开了第二层。
<br/>　　……
<br/>　　奇异空间内。
<br/>　　萧逸仍旧与寒冰傀儡鏖战着。
<br/>　　时间已经过去了一天。
<br/>　　萧逸体内修罗之力，不断在消耗。
<br/>　　但，剑道水平，也在不断提高。
<br/>　　裂天剑魔的十成剑道知识，他已经能掌控5成。
<br/>　　此时的萧逸，感到更为震惊。
<br/>　　因为，面对寒冰傀儡的剑法。
<br/>　　他发现，自己竟然仍旧无法理解，仍旧看不透。
<br/>　　萧逸实在想不到，裂天剑宗，到底有哪位长老，剑道水平如此高超。
<br/>　　裂天剑魔，作为地极境的霸主人物。
<br/>　　其5成剑道知识，已经是极高水准的了。
<br/>　　萧逸摇了摇头，想不明白，懒得多想。
<br/>　　游云杀步，一步一杀。
<br/>　　仍旧在不断挥剑，与寒冰傀儡交手。
<br/>　　……
<br/>　　时间，来到第二天。
<br/>　　今天，便是各试炼弟子的最后一天了。
<br/>　　待得外头夕阳西下。
<br/>　　一众剑主和弟子，纷纷离开了剑阁。
<br/>　　剑宗大长老，看着面前众人，忽然皱起了眉头。
<br/>　　“嗯？怎么少了一个？”大长老问道。
<br/>　　“北山剑主还未出来。”戒墨等人说道。
<br/>　　“还未出来？”大长老面露疑惑之色。
<br/>　　“哈哈。”万山剑主笑道，“这么久没出来，恐怕是死在里面了。”
<br/>　　“万山剑主，你是不是欠收拾了？”钟无忧冷冷地盯向他。
<br/>　　“你这个废物都出来了，我北山郡的天骄，会出不来？”
<br/>　　“哼。”万山剑主冷哼一声。
<br/>　　有剑宗大长老在，他可不怕钟无忧对他出手。
<br/>　　“这都两天有多了，如果没死，早该出来了。”万山剑主冷笑道。
<br/>　　“除非，他有把握应付那个试炼吧。”
<br/>　　“不过，这是不可能的。”
<br/>　　“据我猜测，肯定是他不自量力，想挑战那最难的试炼。”
<br/>　　“孰料，根本不是对手，直接就被秒杀了。”
<br/>　　“连剑主令都来不及用。”
<br/>　　“大长老，你就不去看看情况吗？”九江剑主周月瑶说道。
<br/>　　大长老摇了摇头，沉声道，“不可能的。”
<br/>　　“剑阁，之所以区分开楼层。”
<br/>　　“就是因为我们这些长老的实力，若进入一、二层的禁制试炼，会导致禁制破坏。”
<br/>　　“我们不能进入那些试炼。”
<br/>　　“再等等吧。”
<br/>　　大长老再次说道，“距离试炼结束，还有10分钟。”
<br/>　　“若10分钟后，北山剑主还未出来。”
<br/>　　“我再想办法。”
<br/>　　……
<br/>　　奇异空间内，此刻的萧逸，脸上写满了喜悦。
<br/>　　因为，他终于有了击败寒冰傀儡的实力。

<br/>　　裂天剑魔10成剑道知识，他能掌握8成。
<br/>　　同时，武道瓶颈，也提升到地极境以上。
<br/>　　他不确定是地极几重。
<br/>　　因为，那是更高深范畴的东西，他无法猜测。
<br/>　　但，必定是地极境以上。
<br/>　　另外，他已经能真正看透寒冰傀儡的剑法了。
<br/>　　自然有办法对付。
<br/>　　“喝。”萧逸大喝一声。
<br/>　　一股霸道威势，凭空从血戮剑上出现。
<br/>　　那是剑势。
<br/>　　剑势，直接将寒冰傀儡压迫住，让其动弹不得。
<br/>　　掌控了裂天剑魔8成剑道知识，所衍生出的剑势。
<br/>　　威力强大如斯。
<br/>　　“破。”萧逸再次大喝。
<br/>　　寒冰傀儡，瞬间在剑势压迫下毁坏，最后化作一阵光芒，消散无踪。

<br/>　　木台上，是一本典籍。
<br/>　　萧逸连忙上前观看，他想知道，到底是哪位剑宗长老的剑道知识。
<br/>　　竟让寒冰傀儡如此强。

<br/>　　翻开书籍，看了几眼，瞬间脸色大变。
<br/>　　“寒冰剑罡，天阶中级功法，所创者…裂天剑姬。”

<br/>　　一是裂天剑魔，二是裂天剑姬。
<br/>　　这两人，不曾当过宗主。
<br/>　　但在他们那个时代，却是公认的至强者。
<br/>　　‘裂天’称号，并非宗门授予，而是天下强者，尽皆承认。
<br/>　　拥有称号者，一人，代表整个剑宗。
