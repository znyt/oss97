
武林中文网最快更新天命赊刀人最新章节！<br/>　　<br/>　　八拐里坡其实就是个山沟子然后里面有八道弯，总共八里长左右，天下间知晓这地方的人不太多，知道还敢来的就更没几个了，通风水走阴阳的人都把这里当成是了不能踏足一步的禁地，因为这八拐里坡都是有进无回的。<br/>　　<br/>　　王惊蛰从竹筏上跳下来上了岸以后，往里又走了一段路，就进了一个山沟子里，瞬间就感觉到山沟上方乌云密布，山沟里鬼哭狼嚎起来，扑面而来的阴气冷的人直打哆嗦，原本之前的天气还是星光闪耀月头高挂的呢。<br/>　　<br/>　　“咔嚓”王惊蛰刚迈了几步，没留意到脚下踩碎了什么东西，他低头一看右脚下面是个碎了好几块的头骨，一条青色的小蛇正从骨头下面爬出来，蛇头吐着信子朝着王惊蛰望了一眼后就快速的溜走了，头骨旁边还散落着几根白骨，上面布满了虫洞，被啃食的千疮百孔了，明显这人死了都不知道多少年了。<br/>　　<br/>　　在往远点，地上的白骨更多了，他仿佛一脚就踏进了个乱葬岗里。<br/>　　<br/>　　又往前大概走了一里地左右，此时他已经算是深入到八拐里坡里面了，山沟子的林中弥漫着刺鼻的瘴气，闻着就让人有股想要昏昏欲睡的感觉，王惊蛰咬了下舌尖，嘴中念着道家清心咒来驱散四周的瘴毒，免得人还没走几步呢，就得中毒倒地了。<br/>　　<br/>　　清心咒刚吟了一段，林中忽然一闪即逝了好几道的黑影，几双幽幽的绿光“唰，唰”的望向他这边，树上枝叶无风自动抖个不停，瞬间响起好几声鬼夜哭来。<br/>　　<br/>　　鬼夜哭可和鬼哭狼嚎不是一回事，后者是鬼吓人的一种方式，全都是由孤魂野鬼营造出来的一种慑人心弦的氛围，主要诉求就是为了吓人一跳。<br/>　　<br/>　　而鬼夜哭则是一种道行高深的厉鬼自带的出场音效，就好像陈近南一登场脚踏飞剑天降花雨的感觉似的，其凸显出来的诉求就是，我很牛逼，不太好惹。<br/>　　<br/>　　这鬼夜哭叫的王惊蛰都有点心慌慌的，耳膜子被震的生疼，他再次咬破舌尖吐出一口精血让脑袋清明过来，朗声说道：“各位，天道有轮回，人鬼是殊途，我和列位阴阳两隔无冤无仇，本无意打扰贵宝地，但此番前来实有要事相求，还行各位能行个方便，若是能通融一番此次情分惊蛰肯定铭记在心，你们若有生前未了的心愿，我在此立下心誓，尽我所能为你们了结了……”<br/>　　<br/>　　王惊蛰这叫先礼后兵，在他的心里从生物的角度而言是没什么人鬼之分的，都是一个样，普天之下能喘气的东西最多分出三六九等，但绝对不分贵贱，他从来也不以人间正道是沧桑为行事准则，见鬼就要打，所以在八拐里坡他也不想无缘无故的和这些玩意儿结怨。<br/>　　<br/>　　林中无鬼相应，反倒是鬼夜哭的动静越来越浓密了起来，王惊蛰皱了皱眉，翻手从包中抽出一把菜刀攥在手里，然后张嘴喷了口带血的涂抹在刀上，用手指快速的写了一串符文，原本带着锈迹刃口也不是太锋利的菜刀，在这时候显得特别沉重起来，古朴的刀身上似乎瞬间就透出股凛冽的气势。<br/>　　<br/>　　这把菜刀和以往王惊蛰送出去的那几把，都是大有不同的。<br/>　　<br/>　　以往他包里的刀都是赊刀人走街串巷摆摊做买卖时所用的，而他手里拿的这一把是不知从哪一代赊刀人那传下来的，简单来说就是这是一把挺有故事的刀。<br/>　　<br/>　　王家宝贝不多，这刀肯定能算一个，自从不知在哪一代老祖宗那里传下来后，经过历代赊刀人的手里，这把菜刀走过岁月长河的同时，也被浸淫成了一个法器，颇有镇家之宝的架势。<br/>　　<br/>　　要不是因为自己身上带着这把刀，王惊蛰来八拐里坡也得哆嗦。<br/>　　<br/>　　“你走的是人间道，现在踏上的是阴阳路，你都说了人鬼殊途，八拐里坡不喜有人来，既然来了也就别回去了……”见王惊蛰抽出那把菜刀，鬼影重重的林中忽然有道沙哑的嗓音传来，敌意明显。<br/>　　<br/>　　鬼夜哭，瞬间又拔高了一个调，林中无风自动的枝条晃动的更剧烈了不少！<br/>　　<br/>　　王惊蛰心知，自己上了八拐里坡就别想善了了，真要是这么容易被他两句话就给说通了，那八拐里坡禁地的名头也太名不副实了，这里没一个善茬子，他也不是舌战群儒的诸葛亮，用嘴并不能好使。<br/>　　<br/>　　“唰”王惊蛰抬手，刀指林间。<br/>　　<br/>　　“瘦”一抹泛着幽绿的鬼火，不知从何处急射而来，奔着他的面门就砸了过去。<br/>　　<br/>　　王惊蛰平伸出去的手一挽就耍出了一道刀花，刃口上突然就爆出一团金光来，这菜刀刀身上被王惊蛰画了一道惊雷符，此咒专刻阴物，能引动雷声滚滚，专门就是降妖伏魔来的。<br/>　　<br/>　　“啪”刀花闪出来后，王惊蛰迎着鬼火就拍了过去，刀身正中那幽绿的火光上，顿时就四散开来。<br/>　　<br/>　　“八方雷动，九曜顺行，元始徘徊华精茔，明元灵散开……”王惊蛰嘴中快速吐出一串符咒，脚尖点地，人如离弦之箭一般，突兀的就冲进了前方鬼影重重的林间。<br/>　　<br/>　　“轰隆隆……”于此同时，八拐里坡上空雷声滚滚，阴云翻滚个不停。<br/>　　<br/>　　八拐里坡外面，澜沧江上送王惊蛰过来的船家，惊愕的回头看着他上岸的地方，就看见那一片林子上骤然间涌动起来：“闹得好大的动静，可比以前厉害多了”<br/>　　<br/>　　墓崖村里，余生婆的小院中她和李红拂都坐在屋外，远远的看到村子西边天气突变之后，两人都“呼”的一下站了起来。<br/>　　<br/>　　余生婆说道：“他还真是敢进去了？”<br/>　　<br/>　　“咔嚓”一道闪电劈下，细细小雨忽然下了起来。
