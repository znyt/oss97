　　耀眼的金色剑光闪掠虚空，如同从天而降的金色闪电，向苏莫当头斩下。
<br/>　　冷峻青年名为郭剑青，实力比之赵城丝毫不弱。
<br/>　　而且，这一剑出的极为巧妙，正是苏莫与赵城硬撼一击之后，被震的气血虚浮之时。
<br/>　　这个时候，苏莫体内旧力刚去，新力未生。
<br/>　　可以说，郭剑青时机把握的极为精准，就是要趁着这个巧妙的时机，展开必杀的一击。
<br/>　　咻！
<br/>　　金色的剑光平滑的切开空间，速度快到极致，瞬间跨越了百里距离，斩到了苏莫的身前。
<br/>　　“不好！”苏莫心中大惊，没想到这两人居然配合的如此巧妙。
<br/>　　剑光已经来到身上，躲避已是来不及了，电光火石之间，苏莫只能用尽全力，向剑光狠狠的一斩。
<br/>　　长剑划过优美的弧线，重重的斩击在了金色剑光之上。
<br/>　　轰！
<br/>　　一声爆响，惊天动地。
<br/>　　金色的剑光强大无比，瞬间斩飞了苏莫。
<br/>　　嗖！
<br/>　　苏莫的身形，如同飞驰的炮弹，急速倒飞了出去。
<br/>　　轰隆隆！
<br/>　　巨响连连，苏莫的身形撞在了千里之外，一座千丈高峰上，顿时将山峰顶部撞的炸裂开来。
<br/>　　这时，他才止住了退势，喉头一甜，嘴角留下了一丝猩红。
<br/>　　这一轮攻击，他被两人打了个措手不及，衣衫偻滥，很是狼狈。
<br/>　　轰！
<br/>　　咻！
<br/>　　还没来得及喘口气，一道硕大无比的拳印，和金色的剑光，便一左一右向他袭杀而来。
<br/>　　拳威盖世，剑威漫天，恐怖的杀机，笼罩了苏莫的身心。
<br/>　　“找死！”苏莫脸色铁青，这两人果然强大，而且联手之下配合的相当默契。
<br/>　　嗖！
<br/>　　毫不犹豫，他身形立刻向天穹之上冲去，与此同时，体内二十八柄本命灵剑，立刻化为大星宿剑阵，裹挟浩大无比的剑气风暴，向拳印和剑光迎击而去。
<br/>　　轰隆隆！！！
<br/>　　爆响连连，大星宿剑阵虽然强大，但根本无法匹敌拳印和剑光的攻击，立刻被击的四分五裂，二十八柄本命灵剑四处崩飞。
<br/>　　而后，拳印和剑光咆哮长空，没有击中苏莫，击到了万里之外。
<br/>　　不过，大星宿剑阵虽然挡不住拳印和剑光，却是成功的阻挡了一瞬，为苏莫争取了喘息之机。
<br/>　　“看看今天到底是谁杀谁？”
<br/>　　苏莫身形冲上了高空之中，眸光如电，杀机森然。
<br/>　　狂暴的气息从他体内冲天而起，巨大无比的吞噬战魂直接释放了出来，悬浮在天穹之上，犹如天穹塌陷了进去。
<br/>　　轰隆隆！
<br/>　　吞噬战魂催动了起来，巨大的漩涡急速的旋转。
<br/>　　刹那之间，恐怖到极致的吞噬之力，从吞噬旋涡中咆哮而出，如同万里汪洋，席卷九天十地。
<br/>　　这股吞噬之力极为强大，远超之前十倍不止。
<br/>　　战魂的力量，远非以前的武魂可比，况且，还是最高级的完美战魂。
<br/>　　霎时之间，方圆数千里之内的一切，灵气、白云、碎裂的空间碎片，全部都向吞噬战魂飞了过去。
<br/>　　赵城和郭建青两人，亦是在吞噬之力的笼罩之中，顿时感到全身剧震，身形不受控制的向吞噬战魂飞了过去。
<br/>　　“这是什么战魂？”
<br/>　　“怎么可能？”
<br/>　　赵城和郭建青面色大变，立刻大吼了起来，两人望着巨大的吞噬旋涡，眸中均是露出不可思议之色。
<br/>　　他们活了这么久，还没有见过如此恐怖的战魂。
<br/>　　在吞噬之力中，他们不仅身体受到强力的拉扯，体内的血液、玄力都剧烈的沸腾了起来。
<br/>　　甚至，两人体内的战魂，都是不断的颤动。
<br/>　　不过，两人并不是弱者，急忙稳住了身形，立刻开始后退。
<br/>　　只不过，两人受到吞噬之力的影响太大，后退的速度非常的慢，还不如真灵境武者飞行的快。
<br/>　　“杀！”
<br/>　　厉喝声响起，只见苏莫状若杀神，身形如一道疾电，带着吞噬战魂，直接向赵城和郭建青冲过去。
<br/>　　轰！轰！轰！
<br/>　　吞噬战魂之中，光影闪烁，一道道虚影绽放开来，悬浮在了苏莫的背后，赫然是一种又一种武魂。
<br/>　　这些武魂大多都是剑武魂，但还有两种刀武魂，一种枪武魂，加起来足足有十三种武魂。
<br/>　　十三种武魂，加上吞噬战魂，此刻，苏莫一次性催动了十四种武魂。
<br/>　　在没有突破到武帝之前，他自然无法同时催动如此多的武魂，同时催动十种，差不多已经到了极限。
<br/>　　但现在他修为大增，精神力自然也随之增长，同时催动十四种武魂，也勉强可以做到了。
<br/>　　十三种器武魂乍一出现，苏莫身上的气息简直凌厉到了极致，冲天的锋芒洞穿了九霄。
<br/>　　他身上逸散的一丝气息，仿佛都化为了神兵利器，将空间切开。
<br/>　　“斩！”
<br/>　　苏莫快速的临近了赵城和郭剑青，手中长剑连斩，瞬间击出了两剑。
<br/>　　咻！咻！
<br/>　　两道长达万丈的三色剑气，风驰电逝，从天而降，裹挟惊天杀机，向赵城两凌空袭杀而去。
<br/>　　“怎么可能？”赵城和郭剑青两人，仿佛看到了这个世界上最可怕的事情，眼珠子几乎要瞪出了眼眶。
<br/>　　十几种武魂？
<br/>　　这可能吗？
<br/>　　双生武魂已经是超级天才，三生武魂已经万年罕见，至于四生武魂，则根本就是不可能的事情。
<br/>　　但苏莫现在是什么情况，居然拥有十几种武魂。
<br/>　　一瞬间，赵城和郭剑青几乎吓傻了。
<br/>　　直到惊天的三色剑气临近了两人，凌厉的杀机让两人全身冰寒彻骨，两人才反应了过来。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　面对生死危机，两人顿时疯狂怒吼了起来，立刻出招抵挡。
<br/>　　他们受到吞噬之力的影响，速度慢入蜗牛，根本避无可避，只能竭力抵挡。
<br/>　　一瞬间，两人几乎同时释放出了自身的战魂，动用了战魂之力。
<br/>　　赵城的战魂是一只猿猴，名为金刚猿，身形高大，气势非凡。
<br/>　　而郭剑青的战魂，则是一柄白色的长剑，剑气冲销，威势同样浩大。
<br/>　　不过，两人的战魂，在吞噬之力下，不断的颤动，气势瞬间便荡然无存。
<br/>　　但赵城和郭剑青根本来不及理会自身的战魂，电光火石之间，只能极力抽调体内的玄力，加持战魂之力，向迎面而来的三色剑气迎击而去。
<br/>　　轰！
<br/>　　咻！
<br/>　　赵城再次击出了一拳，耀眼的拳茫瞬间凝结成了拳印，震碎空间。
<br/>　　郭剑青则是斩出了自己最强的一剑，刺目的金色剑光一往无前，狠狠的斩向三色剑气。
