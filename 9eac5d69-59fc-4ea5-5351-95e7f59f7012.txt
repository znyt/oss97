　　叶少阳绕着山寨一直走，一直没找到翻越进去的机会，虽然强行翻越也没问题，但他完全不知道山寨里的情况，怕翻越进去之后碰到土匪，这些人都有枪，自己再想出来就麻烦了。
<br/>　　后来走到了一处绝壁前，围栏在绝壁千前面到头了，叶少阳扫了一眼，这是是一块巨大的岩石，矗立在山顶的右侧，半边凌空，下面就是万丈深渊。
<br/>　　可能是考虑到这种绝壁人爬不上去，所以土匪们也觉得没有在岩石上修筑围栏的必要。
<br/>　　叶少阳认真打量了一会，认定这岩石直接爬是爬不上去的，不过办法也不是没有。回到树林里，寻找到若干长条形状的石块，堆积在栅栏的下方，一直推了有一两米高，就算是有这些石块垫着，正常人想要从这个高度翻过围栏，也是做不到的。
<br/>　　不过叶少阳不是正常人。从小修炼茅山体术的他，虽然不能飞檐走壁，但比一般人要灵敏的多。
<br/>　　深吸了一口气，叶少阳一个箭步踏上了垫起来的石块，再用力一跃，人已爬上围栏，伸出脑袋朝山寨里看去，一眼就看到几个土匪在建筑物之间游荡，身上都挂着枪，顿时明白为什么围栏这边不太设防了：里面到处有人巡逻和放哨，而且建筑物之间也没什么遮拦，想要不被发现地闯进去根本不可能。
<br/>　　还好叶少阳所在这一带围栏处在最边上，附近无人把守，正好地势也高，叶少阳小心地爬上围栏，观察山寨里的情况，将那些建筑物大致的位置记住。
<br/>　　在山寨最中间，有一座最大的建筑物，很长，看上去像是个大瓦房，有一道很阔的大门，门厅上挂着一块横匾，因为距离太远，看不见写的什么，估计什么“聚义厅”之类的，以这些土匪的文化，肯定不会弄什么风雅的名字。
<br/>　　叶少阳观察的时候，这两扇门打开了，一队人从里面走出来，看打扮都是土匪，头上都扎着白布，四个人抬着一口棺材出来，边上有两队穿长衫的，手里各自都提着一支唢呐，一出门就吹奏起来。
<br/>　　不用说，一定是给那个什么三当家的送葬的。叶少阳只是好奇这人才刚死怎么就发丧了，而且这棺材是从哪里来的，难不成是这些土匪生怕死人，在山上常备了棺材不成？
<br/>　　叶少阳想也觉得不可思议。
<br/>　　最后面出来的，是一个高头大马的土匪，头戴毡帽，身上披着一块白布，被一群土匪簇拥着，叶少阳猜测着这大概就是土匪窝里的大当家的。大当家的身边跟着一个穿着白色长衫的，打扮的像个过去的书生，可惜距离太远，都看不到脸。
<br/>　　这一行人抬着棺材，吹吹打打地朝后山进发，估计是去下葬。
<br/>　　那个疑似大当家的，叫来两个土匪，吩咐了几句，然后两人朝着山寨大门方向走去。
<br/>　　叶少阳灵机一动，开始往回走，顺着山林一直跑到大门附近，等了不一会，那两个土匪果然也出来了，牵着马，朝山下方向走。
<br/>　　叶少阳从树林里小心翼翼地下山，听着马蹄声，一路跟随，快到之前那个破庙附近的时候，这俩土匪才骑上马，叶少阳恍然，估计是因为最上面那些路太陡，不便骑马，因此才牵着走一段。
<br/>　　等他们骑马走了，一切都来不及了。叶少阳飞快地从树林里闪了出去，大喝一声：“站住！”
<br/>　　两个土匪正在交流逛窑子的经验和心得，正说到激动的地方，突然听见这么一声大喝，吓了大跳。
<br/>　　“干什么的！”一人回过神来，翻身下马。
<br/>　　“此山是我开，此树是我栽，要想从此过，留下买路财，若是不给钱，我一刀一个，管杀不管埋！”叶少阳一时犯二，把小时候从小人书上看到的一段土匪拦路时候的台词搬了出来。
<br/>　　两个人一听就懵了，互相看了一眼，憋不住大笑起来，其中一个甚至对叶少阳竖起了大拇指，“你兄弟特么的牛比，牛比！哥哥我当土匪这么久，从来都是我打劫别人，还是头一回撞着敢劫我的！”
<br/>　　说完，两人一起拔出了盒子枪，刚对准叶少阳，叶少阳一抬手，打出了两枚铜钱，刚好打在两人手背上，两人手一软，手中的枪自动滑落。
<br/>　　叶少阳一个箭步上前，啪啪两脚，将两人踢飞出去。
<br/>　　他这一手摸钱击打的本事，不光能对付邪物，打人也没问题，之前在破庙里，他虽然拿出刘老头给的护身符，不过当时也做了两手准备，左手心一直攥着一把铜钱，只因为翠云和陈三在场，让他不敢放手拼一把。
<br/>　　叶少阳从地上捡起一把盒子枪，轮流瞄准两个土匪。
<br/>　　“别别，好汉别开枪，我们给钱！”两个土匪立刻跪在地上，摆着双手，一般的这种喽啰型的土匪，都是欺软怕硬，非常的识时务。
<br/>　　叶少阳当场有点郁闷，这俩笨贼，真把自己当成跟他们一样的土匪了。
<br/>　　“不要钱，你们随我来，问你们几句话！把马牵上！”叶少阳生怕在路上遇到别的土匪，于是催促两人牵马进树林，两人见叶少阳不杀，急忙跟了过去。
<br/>　　进入树林，走了一段，叶少阳感觉距离差不多安全了，这才让他们停下，蹲在路边一棵树下，双手抱头，用枪指着两人，问道：“第一个问题，你们老大派你们下山是干什么的？”
<br/>　　“是……是去买红纸，红烛之类的办喜事用的东西……”一个土匪颤巍巍地说道。
<br/>　　叶少阳一听有点懵，“办喜事？你确定没说错？你们不是在办丧事吗，怎么改办喜事了？”
<br/>　　另一个土匪抢着回答道：“回大爷的话，现在是办丧事，等发落了三当家的，晚上我们大当家的要娶夫人，所以要办喜事。”
<br/>　　叶少阳听了之后，愣了好一会，真是有点哭笑不得。这个大当家的也是够了，白天办丧事，晚上办喜事，也真是亏他干的出来。
