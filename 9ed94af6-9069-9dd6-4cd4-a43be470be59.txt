　　第303章
<br/>　　当程大雷再看到严笛时，见他完全像变了个人似的，英气不在，头发枯槁，整个人似乎老了好几岁。
<br/>　　啧啧……也不知道他在刘悲手里受了怎样折磨，该不会落下什么心理阴影吧。看来，自己以后和刘悲还是少接触。
<br/>　　“事情都问明白了。”程大雷坐在聚义厅，手指敲着桌面。
<br/>　　“都在这儿呢。”
<br/>　　刘悲拿出一份审讯记录，厚厚一叠，都是严笛陈述的琴川关情况。
<br/>　　程大雷点点头：“可以问问那些俘虏，看这人有没有撒谎。”
<br/>　　“我已经问过了，一字不差，这人很老实。”刘悲。
<br/>　　啧，程大雷心底忍不住赞了一声，刘悲办事就是靠谱，省了自己多少事。
<br/>　　“那现在可以放严军师走了。”程大雷递给严笛一封信：“回去交给莫将军。”
<br/>　　严笛捧着信，见程大雷没有阻止，便打开信看了看，只是一眼，他整个人就愣住了。
<br/>　　『互不侵犯条约』
<br/>　　在这封信中，程大雷详细定了许多细则，大意是双方都在琴川混饭，没理由你吃饭，就要砸我的碗。以后两家保持和平状态，你不来打我，我不来打你。
<br/>　　这是要和莫鸣米平起平坐的意思呐。
<br/>　　严笛将信收好，道：“莫将军是不会答应的。”
<br/>　　“所谓谈判么，就是互相退步的过程，若都不退步，你要砸我的碗，我就有本事踹你的锅。”程大雷笑着道：“我相信，莫将军会掂量好其中利害的。至少，我们还有严军师么……”
<br/>　　“我？”严笛一楞。
<br/>　　程大雷抄起那份口供晃了晃，道：“你觉得莫鸣米看到这些东西会作何感想，他会不会认为你已经将他出卖！”
<br/>　　“你！”
<br/>　　严笛眼睛睁大，意识到自己陷入一个圈套中，如果自己不听程大雷的话，这份口供就会成为自己是叛徒的证据。可自己对莫将军是忠心耿耿呐，好吧，好像自己也没有那么忠心，但不能这么做人吧。
<br/>　　“无耻！”最后，严笛口中吐出这样两个字。
<br/>　　“好啦，好啦，我都是山贼了，无耻对我是褒义词。”程大雷挥挥手：“派人送严军师离岛。”
<br/>　　从鬼门关上逃出一条命，严笛不知道自己该喜该忧，他被送出岛后，就拖着憔悴的身体返回琴川关。
<br/>　　“咦，是严军师么，军师怎么变成这个样子？”
<br/>　　守门的小兵接住了他，见严军师落魄得紧，整个人已经累得气喘吁吁。
<br/>　　“找将军，快找将军……”严笛虚弱道。
<br/>　　“好好。”小兵急忙答应，恍惚觉得这一幕似曾相识。
<br/>　　严笛在病榻上见到了莫鸣米和牧长庚，这一幕何其相似，不久前，牧长庚倒在病榻上，现在是牧长庚站着，他自己坐在病榻上。
<br/>　　莫鸣米心里也有些奇怪，怎么从蛤蟆寨的人都会大病一场，蛤蟆寨究竟有如何魔力。
<br/>　　严笛将自己如何被风浪吹到蛤蟆寨，又在蛤蟆寨经历了什么与莫鸣米说了，自然忽略交待口供的事情。
<br/>　　最后，他将怀中的信取出来：“将军，这是程大雷让我带过来的，也因为这封信我才能活着回来见将军。”
<br/>　　“回来就好，回来就好。”
<br/>　　莫鸣米安慰两句，将那封信接过来，刚看了一半，他就勃然大怒，手重重的拍在桌子上。
<br/>　　“他有什么资格，他有什么资格与我平起平坐！”
<br/>　　如此愤怒的莫鸣米却是很少见，严笛很大一会儿不敢吭声，想起自己留在蛤蟆寨的口供，才试探道：“将军，那可是程大雷呐。”
<br/>　　只用三个字，就点明这件事的要害。要和莫鸣米平起平坐的不是旁人，是程大雷。不提他以前做的事，就是最近，能够在几次攻击下挺过来，其人怎么可能是泛泛之辈。
<br/>　　这不是程大雷配不配的问题，而是莫鸣米有没有资格和程大雷平起平坐。
<br/>　　见莫鸣米神色稍缓，严笛慢慢道：“将军，我被关在蛤蟆寨这些日子，也对他们山寨做了些了解。以我看来，蛤蟆寨人数虽少，但战斗力绝对不弱。我们想要除掉程大雷，不是不可以，但定会付出惨重代价，现在我们刚经历一场失败，人心惶惶，宜静不宜动呐。”
<br/>　　听罢严笛的话，莫鸣米沉思良久，忽然长叹一声：“军师说的话，我如何不明白。程大雷刚来时我不动手，不就是忌惮他这个人么。若真与他共分琴川，对我也不是丢人的事情，毕竟对方是程大雷。可军师，你是还不知道，最近琴川关发生了些什么事？”
<br/>　　“又怎么了？”
<br/>　　“落在程大雷手里的俘虏回来很多，他们到处与人说程大雷如何如何仁义，如何发给他们路费，我对这些人都重重惩罚了。可是我关得住他们的嘴，却关不住人心。军师，你觉得这样一个人，肯要的只是平起平坐这么简单么？”
<br/>　　“这……将军该如何处置？”
<br/>　　“如军师所说，宜静不宜动，我们还是暂且答应他，可但凡有机会，决不可让此人活着。”
<br/>　　……
<br/>　　嘟，十次抢劫任务完成，奖励三级山寨建设图纸一份。
<br/>　　这天中午，程大雷正在休息，忽然收到一条系统提示。他这些日子，一直派出手下去抢劫，算下来，这任务也当完成。
<br/>　　琴川关那边莫鸣米也传过来信，他答应了程大雷提出的条件。而且信中极其客气，说我和程兄弟不打不相识，以后这琴川就是咱们弟兄的天下等等之类的话。
<br/>　　程大雷回去的信也是极其客气，互相都藏着刀，互相都面带微笑。
<br/>　　而按照程大雷的计划，自己也该开三级基地了。只要把山寨升到三级，将系统商店开出来，程大雷就是首先冲莫鸣米亮出刀的人。
<br/>　　原先在蛤蟆寨，程大雷已经升过三级本，倒不是特别期待，关键是三级后面的四级山寨。二级本开启点将台，三级本开启系统商店，四级山寨又该开什么呢？
<br/>　　当然，这是一件大工程，在开始建设山寨前，程大雷还有必须的几件事要办。
<br/>　　首先一件，就是派出人去，在浩甲城开了一间酒楼，做情报收集工作。
<br/>
