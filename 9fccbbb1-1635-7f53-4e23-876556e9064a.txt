    夜幕降临，车队停了下来。工人司机们三三两两地蹲在一起，抽烟闲聊。厨师架起大锅，点燃煤油炉，开始煮饭烧菜，为数不多的几名佣兵在营地外例行巡查，排除可能存在的危险。<br/>    <br/>    这里是北部荒野的西南端，位于龙石镇和乌鸦市之间。由于靠近龙涎河流域，草木相对茂密一些，但因为地质问题，还是无法种植作物。<br/>    <br/>    作为佣兵之一，陈兴在营地外围游荡了一圈。北面有个土山包，算是个安全隐患。如果袭击者，肯定会在那里居高临下地狙击。不过土山上没多少植物，隐蔽效果很差。陈兴拿出瞄准镜瞄了瞄，把几个可能的射击位记在心里，返回了营地。<br/>    <br/>    “恰饭咯，恰饭咯！”<br/>    <br/>    绕过一辆卡车，就听见厨师操着一口南部方言喊大伙吃饭。这厨师生得五大三粗，手上的汗毛又黑又粗，据说是青国人，父亲得罪了一个小贵族，携家带口逃到红国，一晃就是三十年。<br/>    <br/>    红土大陆素有“一洲二河三国四野五山”之称。<br/>    <br/>    一洲是指红土大陆南部的“大方洲”，戈壁地形，辽阔无边，是整个红土大陆最大的平原。<br/>    <br/>    二河是指红土大陆两大河流，“龙涎河”与“堕星河”，堕星河源起瑞嘉尔斯群山，由南至西北流向，在大陆西部出海，是青国的母亲河。<br/>    <br/>    三国是指红土大陆三大古国，有红国之称的龙涎河联合王国，青国之称的瑞嘉尔斯神山帝国，黄国和沙国之称的格拉塔木部落联合王国。<br/>    <br/>    四野是指四大荒野，分别是北部荒野、西部荒野、东部荒野和中部荒野。北部荒野在红国国境内，西部荒野分属青国和红国，东部荒野分属红国和黄国，中部荒野则分属多个独立小王国。<br/>    <br/>    五山就是五大山脉，东北部的诺兰雪山，西北的巨龙群山，西南的瑞嘉尔斯群山，中部的红岩火山，东南部的格拉斯丘陵，其中瑞嘉尔斯群山最为庞大，素有“神山”之称。<br/>    <br/>    晚餐是土豆牛肉大白菜杂烩面，撒了些胡椒粉，味道还算不错，但比起在水站的时候，还是有很大的差距。毕竟在那边是自己做老大，厨房都是开小灶的，偶尔还能自己下下厨，弄点儿家乡风味的。<br/>    <br/>    吃完晚餐，陈兴用配给的水洗了食盒，将行李包往帐篷里一扔，取出便携屏幕，靠在包上看小说。<br/>    <br/>    他的行李包是寸步不离身的，里面装着从65号地下求生所里私藏的东西，有两把金枪，一快怀表、三个少校铜章和一个红宝石。<br/>    <br/>    如果价格卖得合适，起码有六七千金币。有了这个钱，他就能晋升精英级了。<br/>    <br/>    成为精英级，就等于跨入强者行列的第一步，以后无论到哪里，起码也是个小队长，不需要从底层做起。只是陈兴特别命苦，到现在为止，注射的药的价值已经超过一万金币了。<br/>    <br/>    精英级的雇佣费用是月薪50~100金币，以陈兴的投入来说，差不多是血本无归了。普通人想达到精英级，只要天赋不太差，两千来金币就差不多了。比如亚伦，不需要很多钱就能上去。<br/>    <br/>    不过话又说回来，陈兴的作战能力已经远超普通精英级，光是一个三重奏就能完爆大部分人，多花点儿钱也是正常的。<br/>    <br/>    思绪之间，外面忽然传来脚步声，陈兴眉头微微一皱，但没有进一步的动作，还是保持半躺看书的姿势。<br/>    <br/>    “嗨，Mr秦，你好啊。”<br/>    <br/>    帐篷的布帘被很不礼貌地掀开了，一张黑黝黝的脸出现在眼前，咧着嘴巴，露出一口白牙。<br/>    <br/>    对于这种自来熟的表现，陈兴很是不感冒，只是盯着对方有些外凸的眼睛，一言不发。<br/>    <br/>    这是一个黑人男子，名叫“卡西”，是同团的几个佣兵之一，约莫二十七八岁，穿着破损的耐磨布军服，黝黑的肌肉十分扎实，一看就知道是个近战好手。<br/>    <br/>    红土大陆和地球上的人种有些相似，也有黄种人、白种人、黑种人之分，人种特征也十分相似，只是更为突出。<br/>    <br/>    黄种人灵活，敏捷倾向，白种人力气大，力量倾向，黑种人耐力好，力量和敏捷都很优秀，但灵能感知能力较差。<br/>    <br/>    除此之外，红土大陆还有两个不同于地球的人种，分别是“沙民”和“蓝血人”。<br/>    <br/>    沙民们主要分布在大方洲，通常以游牧为生，肤色介于黄种人和黑种人之间，呈现出古铜色，是黄国的主要人种。<br/>    <br/>    相较于其他人种，沙民没有非常突出的四项能力，但特别耐旱耐热。普通人只要三天不喝水就会出问题，沙民可以十几天不喝一滴水，依然能保持活力。<br/>    <br/>    学者们发现，沙民的细胞中多了一个“水囊”，能够额外储存水分，其功能就和骆驼的驼峰一样。<br/>    <br/>    大方洲辽阔无边，但土地普遍贫瘠干旱，缺少矿物。因此，黄国的国力是三大古国中最差的。可千百年过去了，沙民们依然统治着片土地，靠的就是这点。<br/>    <br/>    中古时期，红国一度强盛，曾挥军百万，进攻黄国。高头大马，坚甲利剑，势不可挡。<br/>    <br/>    面对入侵家园的百万大军，黄国却毫不抵抗，躲进大漠深处，任其长驱而入。<br/>    <br/>    等红国的部队深入到一定程度，黄国的部队才开始反击。他们也不硬抗，采取迂回战术，不断骚扰伏击，拉长战线，分化部队。结果红国的百万大军被拖在沙漠里几个月，进又进不得，退又退不得，最后被活生生渴死。<br/>    <br/>    自那以后，黄国就成了其他国家眼中的硬骨头，除了疆域够大外，一无是处。哪怕是土地抢回来，也用不上，只有沙民才适合那里。<br/>    <br/>    而另外一个人种，蓝血人，顾名思义，就是蓝色血液的人种，主要分布在瑞嘉尔斯群山，是青国的主要人种。<br/>    <br/>    较于其他四类人种，蓝血人的身体素质较弱，但灵能感知能力极强，是天生的灵者。因此，青国超级强者的数量也是三大古国中最多的。<br/>    <br/>    “出来找点乐子啊，一个人待着会发疯的。”<br/>    <br/>    卡西朝陈兴晃了晃手中的扑克牌，目光有意无意地扫过陈兴背后的包裹。<br/>    <br/>    “下次吧。”陈兴冷淡地说道。<br/>    <br/>    一直以来，佣兵被人们戏称为“三大背后捅刀子职业”，其余两个分别是“刺客”和“行刑者”，由此可见一斑。所以没什么事，大伙最好少接触点儿。<br/>    <br/>    “Mr秦……”卡西用着一口不太流利的大陆通用语，有些不快地说道，“你这样太不给面子了。”<br/>
