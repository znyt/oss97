<br/>				    走了一段距离，越发的觉得不对劲，尤其是，当她顺着小路走，来到一处空地时，看到的却是三五成群围坐着的一些魔修和邪修，以及一些通身充斥着煞气的修士。<br/>　　<br/>　　    魔修与邪修且不去论他们，但，那些通身充斥着煞气的修士，也是一身的血腥气味，一身的杀气弥漫着，怎么看，都不像是正常的散修。<br/>　　<br/>　　    她经过的这地方，周围散坐着的人加起来约莫三四十人，有的独自坐着，有的两三人围坐着，也有的身边趴着一头凶兽，而这些人，男的居多，女的则只有一两个，而且还是衣着暴露，依附在一些实力强大的修士身边。<br/>　　<br/>　　    她的出现，自然而然的引起了那些人的注意，无论是邪修还是魔修，或者是那些周身充斥着煞气和杀气的修士，一个个带着异色的目光都落在她的身上，神色不明的打量着她。<br/>　　<br/>　　    看在众人的眼中，这名红衣女子仿佛妖孽降世，她背着月光缓步而来，红衣妖娆，墨发披散，绝美的容颜上端着的是冷漠淡然的神情，周身半点气息也不显，可，偏偏就让所有的人忌惮起来。<br/>　　<br/>　　    在这样的地方，岂是没有本事的人生存得了的？<br/>　　<br/>　　    这女子容颜倾城，一袭在夜色中依旧耀眼的红衣不见干净得不沾血迹，这样的人，自是有古怪的，更何况，自她出现，这周围的空气便冷了几分，那种阴嗖嗖的气息自周围弥漫而来，让每一个人都能感觉到那股森寒的冷嗖气息。<br/>　　<br/>　　    可，偏偏这女子却仿佛没有察觉一般，她的目光在他们众人的身上淡淡的掠过，步伐也没有停留便继续往前走去，居然，所有的人都只是看着她，没有人出手，更没有人出声。<br/>　　<br/>　　    没有人知道，凤九此时内心是紧绷着的，她就那样粗粗看了一眼，发现这里的这些人，实力皆是仙圣级别，其中还有几个是仙圣巅峰的实力修为，那些人身上的血腥味极重，煞气和杀气也是极重，就算是她一手培养起来的人，也没有这些人身上的那股摄人的嗜血杀气。<br/>　　<br/>　　    在这里，她将一身的玄力气息敛了起来，并没露出分毫，因为她知道，若是让这些人知道她是玄修，估计也会陆续对她出手，甚至极可能是联手。<br/>　　<br/>　　    在这些人的身边经过，她只能摆出淡然冷漠的面孔，端着强者的姿势，一步一步，不紧不慢的自他们身边走过，稍有不慎，便会引起他们的质疑与探究，从而将自己陷于危险之中。<br/>　　<br/>　　    若是落入这些人的手中，死，可不是最轻松的事情，若是落入这些人的手中，那么，极有可能会被废去修为沦为他们的玩物。<br/>　　<br/>　　    因为那些人，无论是邪修还是魔修，或者是那些浑身充斥着煞气与血腥气味的修士，每一个盯着她的目光，都像几年不曾闻过肉香尝到肉味的人，在盯着一块肥美可口的肉食，垂涎欲滴，恨不得马上扑上前将她撕了吞下肚。<br/>
