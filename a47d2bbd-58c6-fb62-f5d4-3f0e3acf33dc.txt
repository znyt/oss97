　　涂小安蛇眸盯着这座山洞，倏然地，他还诧异的发现里面居然有不少人类的气息。
<br/>　　莫非那些失踪的捕蛇人还有存活？
<br/>　　这股念头一出，涂小安身形一动，骤然的闪现进去。
<br/>　　这个山洞不算深，纵深大概三百米左右，很快涂小安就看到三十多个年龄不等的人蜗居在一个昏暗的角落，他们精神不震，状态颓靡，双手交叉抱胸的坐在肮脏的地面，时不时的还浑身颤栗。
<br/>　　好像受了什么天大的刺激。
<br/>　　冷淡无波的一眼后，涂小安蹙眉起来，吸血蛇不在，这山洞中并没有吸血蛇的踪影。
<br/>　　之所以此山洞有那么强烈的吸血蛇气息，应该是此蛇在这山洞待过不少时日。
<br/>　　气息久了不散，凝固在一起。
<br/>　　如今洞中有那么多失踪的人口，想必是吸血蛇准备将这些人先圈养起来，留着慢慢的享受。
<br/>　　短短几日，有超过五十多人失踪，吸血蛇在好的胃口，也不可能将他们一次性全给吃了。
<br/>　　这时，涂小安还留意到里面居然还有不少沈穿警察制服的人，不过他们的神情同样充满的恐惧，一副等死的样子。
<br/>　　“吸血蛇不在山洞，为什么他们不跑”
<br/>　　一个疑惑刚冒出，涂小安蛇眸闪烁精光，他在一个隐秘的角落居然看到了一条盘踞的大蟒蛇。
<br/>　　不，它足足可以称之为巨蟒了，这蟒有十五米长度，通体麻花色，有云状斑纹，背面有一条黄褐斑，外貌凶煞可怖。
<br/>　　“这是...”涂小安看了一眼后，笃定：“这呀的是一条正宗的黑斑蟒呀”
<br/>　　黑斑蟒算是常见的蟒蛇，但是此蟒恐怖之处居然是成长到了整整十五米。
<br/>　　要知道涂小安十七级的时候，才十二米，做了系统任务，等级提升十八级，体型再次暴涨，也才十五米长度。
<br/>　　这蟒是蟒，本身就是蟒蛇品种，但这种蟒是不太可能成长的那么大。
<br/>　　十五米什么概念呀，直立起来，有三层楼那么高。
<br/>　　这要是放到外面，会震惊世人的。
<br/>　　况且，此蟒是普通类，并不是觉醒生物。
<br/>　　一个洞察下，涂小安仿佛将这条黑斑蟒看透了一般，等级才不过十六级。
<br/>　　怎么会这样呀，这有点说不通呀。
<br/>　　涂小安实在搞不明白一条黑斑蟒为什么会达到十五米，吃什么吃的呀。
<br/>　　看来蛇沟不愧是蛇沟，白镇捕蛇人的禁地，一条普通品种的黑斑蟒，居然能成长的那么大。
<br/>　　与此同时，涂小安也明白了为什么这些人不逃跑了，有这条十五米的巨蟒看着。
<br/>　　谁敢乱动，找吃呢？想必这条黑斑蟒被吸血蛇收服了，替它看门呢。
<br/>　　“完了完了，没想到我张某人一生捕蛇，最后却沦为蛇的食物，命运弄人啊”
<br/>　　蜗居在角落，其中一位中年人满眼恐惧跟唏嘘的说了出来。
<br/>　　“那条会说话的大蟒要是回来，我们中间又会有几个人会被吃，我真的受不了了，你们谁帮帮忙，杀了我吧”
<br/>　　对于他们来说，死或者是一种很好的解脱，活着就是等待被巨蟒活生生吃的命运，那样太残忍。
<br/>　　洞中周围黑漆漆的，光线昏暗，他们一分一秒的活在无尽的死亡恐惧之中。
<br/>　　不是被笼罩在身旁的黑暗吞噬，就是被一条凶煞的巨蟒吞噬。
<br/>　　、
<br/>　　“我们逃吧，现在只有这条黑斑蟒看门，我们那么多人，要是一起冲，总会跑出去几个”
<br/>　　这是一位身为制服的警员提议，他还保持独有的情醒理智。
<br/>　　立刻有人附和：“左右都是死，还不如拼一次”
<br/>　　在死亡的恐惧面前，人是会被激发最大的潜能。
<br/>　　况且这里面不是捕蛇人，就是警察，都不是普通人。
<br/>　　但依然有不少人犹豫起来，该逃吗?逃失败了呢，会不会立刻被这条黑斑蟒给吃了，不逃或许还能苟延残喘几日。
<br/>　　上空岩壁上有一滴冰凉的液体滑落到他们惊恐的脸颊上，冰冷渗骨，像是在点醒他们。
<br/>　　“拼了，等下三二一，我们那么多人一起冲出去，谁能终归在这条黑斑蟒的血口下逃生，就是他的造化”
<br/>　　众人最后压着牙的点头，眼中迸发强烈的求生欲。
<br/>　　因为他们生生的明白，等那条会口吐人言的巨蟒回来，他们将无一丝生机。
<br/>　　意见统一下来，众人开始做准备，一个个深深的吞了一口气，相互的眼神对碰一下。
<br/>　　一位制服警察大喝一声：“跑”
<br/>　　霎时间，所有人如潮水一般，朝着山洞涌去，那远远的微光，将是他们活下来的曙光。
<br/>　　他们不想死，他们要活着，因为外面有等待他们的家人。
<br/>　　“吼...”
<br/>　　一声撼天震地的咆哮声旋即冲天而起，回荡在山洞之中，震的两边岩壁泥土碎石簌簌而落。
<br/>　　一圈圈盘卷的黑斑蟒变的凶煞无比，一条粗长的蛇尾横扫出去，将一起冲的众人直接全部掀翻在地。
<br/>　　顿时间，有不少人张口便是鲜血喷出。
<br/>　　好强大的力量。
<br/>　　紧接着，十五米的黑斑蟒扶摇直立起来，血盆大口冲着众人不断的咆哮，一股股作呕的血腥之气弥漫出来，让人产生晕眩之感。
<br/>　　他们绝望了，十五米的巨蟒挡在他们的面前，就好像是一座永远翻不出去的大山。
<br/>　　“继续跑”
<br/>　　其中一位警察制服的中年男子，一个原地空翻的站起来，还未冲，黑斑蟒的蛇尾就横扫回来，顿时间，这人飞了出去，狠狠撞击在墙壁上，一口血喷出，空中绽放嫣红的花朵。
<br/>　　此时，只见黑斑蟒铜铃般的蛇眸看蝼蚁一样的看着众人，像是在嘲笑他们，让他们死了这条心。
<br/>　　“逃不出去的，我们还是等死吧”
<br/>　　不少人心生绝望了，这十五米的黑斑蟒太恐怖了，力大无穷，而且这山洞的两边很窄，很难从它的巨无霸蛇躯下逃脱。
<br/>　　山洞入口在前，那丝曙光仿佛触手可及，但这条十五米的黑斑蟒却对他们来说，宛如鸿沟，迈不过去。
<br/>　　这便是他们的命，等待被巨蟒吞噬的命运。
<br/>　　突然...
<br/>　　这山洞中的温度骤然下降，宛如寒冬腊月，一股强大而阴冷的气息不知何处席卷出来，众人紧张得汗一股脑儿往外冒，心“扑冬，扑冬’，地跳，脸色煞白起来。
<br/>　　完了，更没希望了，想必是那条口吐人言的巨蟒回来了。
<br/>　　这凶煞的黑斑蟒也低沉的嘶吼起来，那双铜铃般的凶眸既然不可思议的流露出惧怕之色。
<br/>　　好像看到了什么可怕的存在。
<br/>　　众人确信，是那恐怖的存在回来了，要回来吃他们了。
