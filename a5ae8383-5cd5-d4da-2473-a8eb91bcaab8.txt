的人，所以就反馈上来的信息也是比较准确的。两边都需要很多的蔬菜，但是就数量而言，还真是有些少了。<br/>　　<br/>　　有时候郝仁这边，都想自己雇人过来替陈默将地种植好。就是因为大海的拒绝，才没有做个事情。<br/>　　<br/>　　将山坡用地的事情定了下来，陈默又交代按照现有的蔬菜区域，做好防护。<br/>　　<br/>　　“那就这么着！大海，你去县里农机站买两台大马力的拖拉机！要皮实耐用的，然后问问他们，有没有二手农机卖，要是有直接就买了！要是没有你就让他们打听打听，尽快将这个事情办好。”陈默说道。<br/>　　<br/>　　“好！”大海点头答应。<br/>　　<br/>　　“金贵叔，那你就带着丁兰花，去镇子上的农技站，买那种有机化肥。一定要去正规的销售部门，千万不要为了图便宜，去那种私人开的店铺。他们买的有机化肥根本就用不了，心好的往里面弄些不伤地的东西，黑心的家伙，可是会弄些乱七八糟的东西！”<br/>　　<br/>　　陈默说的，并不是全部，但是也有很大一部分私人就是这么干的。一些不知名的品牌化肥，就是因为便宜，所以里面可以说，有的连点化肥的功效都没有，甚至有些还使用后烧苗！<br/>　　<br/>　　这个假化肥因为便宜，就全部都销往这些农村的私人农资店，可以说一些农民为了省一点点的钱，被坑了整整一年。<br/>　　<br/>　　陈默可不想自己被坑，而且陈金贵还是那种图便宜的人。所以他让丁兰花跟着，就会好些。<br/>　　<br/>　　“金贵叔，大海！你们忙着这个事情，那个葫芦谷的事情也不能拖！西边的地，已经都差不多整理好了，都可以种东西了。”<br/>　　<br/>　　陈默打算将葫芦谷哪里的山坡地用起来，要不然整理出来后，却荒废在哪里，真的有些浪费。<br/>　　<br/>　　“那你准备种什么？”大海问道。<br/>　　<br/>　　“我准备种些葡萄、草莓、西瓜等，另外还可以种植一些花！”陈默说道。<br/>　　<br/>　　“要是葡萄草莓西瓜的话，可能今年有些来不及了，要等到明年才行！”大海走着眉头说道，再好的东西，也要遵守自然规律。<br/>　　<br/>　　“你一说我还想起来了，等过几个月，将这里山坡用地，还有平地的2百亩地，全部搭建成温室。这样就不耽误蔬菜的生长。”陈默说道。<br/>　　<br/>　　“可行!等我去买化肥的时候，正好问问农技站有没有大棚。”陈金贵说道。<br/>　　<br/>　　“那就成，金贵叔你记得问问！”陈默说道：“另外就是葫芦谷的山坡地，等我弄来种子和秧苗后，你们就先安排工人种植吧！至于说天气气候，先不用管这么多！”<br/>　　<br/>　　陈默想到自己能制作阵法，也就不再担心植物冬天越冬的事情。身为修士，自然能让一小块土地改变本身的气候，手段不要太简单！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
