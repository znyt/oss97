　　“走，我们去看看！”苏莫招呼秦不死一声，随即立刻向那座城池飞去。
<br/>　　途中，苏莫眼眸扫视四周，确认没有危险之后，便撤去了硕大的玄力气罩。
<br/>　　没过多久，苏莫和秦不死两人，便飞到了那座城池之前。
<br/>　　宽阔的城门前有不少武者进出，显得颇为热闹。
<br/>　　这让苏莫松了口气，这些人都是人族，并不是什么异类。
<br/>　　“我们进城！”苏莫拉着巫娴的手臂，大步而行，走进了城中。
<br/>　　秦不死则是拉着两名巫族天才，紧跟在苏莫的身后。
<br/>　　走进城池之中，苏莫和秦不死都不禁有些震撼，此城之大自然不必多说，但此城内的建筑，同样是规模巨大。
<br/>　　城中的街道宽达数百丈，城中高达千丈的建筑比比皆是，那些街道旁的商铺，均是修建的颇为高大，一间小小的铺面，都建有五丈之高。
<br/>　　至于这些商铺为何会建造的这么高大，苏莫不用问也已经知道了。
<br/>　　因为他在城中看到了不少‘巨人’，这些人有男有女，个个身高三丈以上，有的更是达到四丈出头，非常的高大。
<br/>　　除了这种巨人之外，城中还有一些颇为奇异的人，比如说有的人全身长满了黑色的鳞片，还有的人居然长有一对翅膀。
<br/>　　当然，这些怪异的人终归是少数，大多数人还是和苏莫一样，是正常的人类。
<br/>　　这些并不是让苏莫两人最震惊的地方，毕竟他们早已知道外界种族无数，有些异类再正常不过了。
<br/>　　最让他惊讶的，是这城中武者的实力。
<br/>　　一眼望去，武王境武者多如狗，武皇境武者遍地走，武帝境武者都有很多。
<br/>　　“这里果然不是苍穹世界可比！”苏莫心中暗叹，他在苍穹世界拥有无敌的实力，但在此地只是个普通的不能再普通的武者了。
<br/>　　走在城中的街道上，苏莫一边打量一边凝神戒备，心中更是在暗暗思量，到底如何才能摆脱两名巫族的尊者。
<br/>　　他有种直觉，那两人一定在暗中跟着他们，寻找出手的时机。
<br/>　　只不过，出于担心伤及巫娴，而不敢贸然出手而已。
<br/>　　事实上，只要能安然脱身，放了巫娴对于苏莫来说，也无不可。
<br/>　　只不过，他不敢放，一旦放了此女，他必死无疑。
<br/>　　“苏莫，我们怎去哪里？”秦不死向苏莫问道。
<br/>　　苏莫闻言略一沉吟，嘴唇微动，向对方传音道：“去一个巫族尊者不敢肆意妄为的地方。”
<br/>　　秦不死闻言，眼眸一亮，此法的确可行，只要去了一处有超级强者坐镇的地方，他们就能暂时安全了。
<br/>　　只不过，哪里有超级强者呢？
<br/>　　随后，苏莫和秦不死大步而行，在城中不断的闲逛了起来。
<br/>　　“苏公子，其实你们已经安全了，飞行了这么久，想必我族两位尊者也不可能追到这里！”
<br/>　　这时，巫娴开口了，低声说道。
<br/>　　苏莫根本没有理会对方，安全？他若认为自己已经安全了，那离死也就不远了。
<br/>　　另外两名巫族的天才，向巫娴使了使眼色，示意她不要多言，免得惹恼了苏莫。
<br/>　　他们只需跟随着苏莫两人便可，他们都知晓，以巫娴的身份，两位尊者定然会相救她们。
<br/>　　苏莫和秦不死在城中逛了片刻，最后来到了一座高大的建筑之前。
<br/>　　这座建筑高达千余丈，形似高塔，又仿若阁楼，大门上刻着五个龙飞凤舞的古字‘炼器师公会’。
<br/>　　看到炼器师几个字，苏莫的面上露出了一丝笑容，或许此地是他的机会。
<br/>　　“我们进去！”
<br/>　　毫不犹豫，苏莫拉着巫娴，和秦不死一起走进了公会中。
<br/>　　……
<br/>　　距炼器师公会万余丈远处，一处小巷之中，两道身影伫立与此。
<br/>　　这两人不是别人，正是巫梵和巫尹两位尊者。
<br/>　　“这两个土著居然进了炼器师公会！”巫梵尊者眼眸望向远处，皱眉说道。
<br/>　　“我们等等吧！想必要不了多久他们就会出来了！”巫尹尊者低声说道。
<br/>　　“刚才他们在进城之时，我们就应该出手的，却白白错过了一次机会！”巫梵摇头叹道。
<br/>　　他们放苏莫两人离开，就是准备在暗中出手，这样就会容易许多。
<br/>　　但跟踪了苏莫四个月，苏莫的警惕心一直非常的高，让他们一直寻不到机会。
<br/>　　在星空之时，本以为苏莫两人会耗尽玄力和资源，他们就能出手救下巫娴三人。
<br/>　　但苏莫撑着巨大的玄力气罩，居然支撑了四个月，让他们感觉到匪夷所思。
<br/>　　苏莫每一次补充玄力，他们都看的清清楚楚，补充一次玄力，居然就能支撑半个多月！
<br/>　　这简直让两人目瞪口呆，那种玄力气罩，就算是一般的武帝境二、三重修为的武者，也不一定能支撑半个月之久吧！
<br/>　　而苏莫，不过是武皇境九重的修为。
<br/>　　现在，两人算是知道了，他们巫族试炼的子弟，为何会败的那么惨了！
<br/>　　小小的苍穹世界，居然出现了一个妖孽人物。
<br/>　　之前在苏莫两人进城之时，苏莫撤掉玄力气罩的那一刻，巫梵本想动手，但被巫尹制止了。
<br/>　　“刚才你有把握瞬间制住那个土著吗？”巫尹反问道。
<br/>　　“当然，以我们的实力，足以在他们反应过来之前，瞬间将他们两人制止。”巫梵自信的说道，苏莫和秦不死的实力，在他眼中不值一提。
<br/>　　“你能瞬间制住他的身体，但能制住他的意念吗？”巫尹摇了摇头，继续道：“别忘了，他意念一动，娴小姐必死无疑！”
<br/>　　巫梵闻言暗叹一声，这才是他们最无奈的地方。
<br/>　　以他们两人的实力，足以轻松镇杀或制住苏莫和秦不死，但却无法瞬间制住两人的意念。
<br/>　　若是镇杀了两人，巫娴和另外两名族中天才，会立刻死去。
<br/>　　而若是制住的话，两人也会用意念引爆巫娴三人识海中的意念玄力。
<br/>　　所以，他们忌惮不已，迟迟不敢动手。
<br/>　　“等吧！总有机会的！”
<br/>　　驼背的巫尹尊者同样叹了口气，浑浊的眼眸中寒光一闪，阴冷的道：“这两个土著残杀我族近千名天才子弟，罪恶滔天，整个天荒星域，都不会有他们的容身之地！”
