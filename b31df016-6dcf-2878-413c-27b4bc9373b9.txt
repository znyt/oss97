　　“修罗之力？”萧逸先是眉头一皱。
<br/>　　而后内视体内，却惊讶地发现，那玄奥图案的样子，已经变了。
<br/>　　看起来，那像极了一滴微小的水滴。
<br/>　　充斥着血色。
<br/>　　那便是修罗之力凝结而成的血色水滴。
<br/>　　这时，又是两道信息传入脑海中。
<br/>　　“修罗变，崩界拳。”萧逸自语着，脸色涌现出了狂喜。
<br/>　　修罗变，是一门功法。
<br/>　　效果，有些类似于升龙。
<br/>　　只不过，修罗变是增幅肉体力量；而升龙，则是增幅修为。
<br/>　　形意五绝，乃是内家拳法。内力在经脉中，通过特殊的游走之法。
<br/>　　刺激奇经八脉，达到增幅的效果。
<br/>　　既是内家拳法，升龙所增幅的，自然是真气。
<br/>　　但，随着自身修为的不断突破增加，升龙，如今的效果已经很小了。
<br/>　　升龙，是萧逸形意五绝中，唯一没有大成的一门绝技。
<br/>　　其它四绝，虎形、豹形、蛇形、灵鹤。
<br/>　　皆有实物。
<br/>　　观其样貌，悟其真意。
<br/>　　形意拳，要悟出真意，万不存一，玄奥至极。
<br/>　　能悟出真意，证明萧逸本身就是一个武学天才。
<br/>　　而升龙，龙，那是传说中的东西，谁都没有见过，哪怕是萧逸。
<br/>　　他前世只是通过想象，古画等东西，强行悟出真意。
<br/>　　越是强大的东西，本来就越难参悟透彻。
<br/>　　更别说只靠想象了。
<br/>　　可想而知，他的领悟力高到何种地步。
<br/>　　但也因没有真正见过龙，他的升龙一绝，始终没有大成。
<br/>　　不，甚至说，小CD达不到。
<br/>　　以前在凡境、后天境、先天境时，尚且能增幅两重修为。
<br/>　　现在，达到洞玄境，怕是顶多能增幅不足一重。
<br/>　　要想发挥出升龙的真正威力，只有将其大成。
<br/>　　萧逸毫无把握。
<br/>　　这也是自从离开剑派历练以来，他几乎没有用过升龙的原因。
<br/>　　言归正传。
<br/>　　修罗变，是一门功法。
<br/>　　而崩界拳，则是一门武技。
<br/>　　脑海中的信息，并没有阐明它们是哪个等级。
<br/>　　因为，它们是因修罗战体而衍生出来的功法武技。
<br/>　　但很显然，它们都很不凡，等级也绝不低。
<br/>　　萧逸之前见识过木龙降世的威力。
<br/>　　按照他的猜测，它们绝对比木龙降世的等级更高。
<br/>　　也就是说，它们起码是地阶高级以上的功法武技。
<br/>　　突破第三层修罗战体，需要如此庞大数量的妖兽精血。
<br/>　　很大程度上是要激活血色水滴，还有这两个武技功法。
<br/>　　这是第三层赋予的东西。
<br/>　　也是实力质的飞跃的原因。
<br/>　　很显然，无论是修罗变和崩界拳，都极难。
<br/>　　只不过，对于萧逸来说，再难的武技功法，都难不住他。
<br/>　　他的领悟力，高得吓人。
<br/>　　洞中，他参悟着脑海中信息，刻苦修炼着。
<br/>　　十天后，他才破洞而出。
<br/>　　全力一拳轰出，轰的一声。
<br/>　　整个大地开始颤抖。
<br/>　　“好强，使用修罗变后，我的肉体力量，竟飙升到破玄三重巅峰。”
<br/>　　萧逸暗暗一惊。
<br/>　　同时，他再次内视了一下体内。
<br/>　　心脏位置，那血色水滴，稍微少了一丁点儿。
<br/>　　修罗之力，是修罗战体特有的力量。
<br/>　　它等同于真气，却又比真气强大得多。
<br/>　　更简单来说。
<br/>　　就像施展武技需要耗费真气。
<br/>　　施展修罗变，则要耗费修罗之力。
<br/>　　且，修罗之力，是唯一能支撑修罗变运行的力量。
<br/>　　施展修罗变时，修罗之力，就会自动化出力量，而后游走于身体各个部位。
<br/>　　修罗之力，本身就极强，加以修罗变的玄奥运行方式，便成了一门强大的增幅肉体力量功法。
<br/>　　按照萧逸的猜测，这一小滴修罗之力。
<br/>　　足以支撑他运行修罗变一天时间。
<br/>　　仅仅一小滴，内头却蕴含着极庞大的能量。
<br/>　　……
<br/>　　施展崩界拳也一样，需要耗费修罗之力。
<br/>　　只不过，需要多少，萧逸还不知道。
<br/>　　他现在便打算试验一番。
<br/>　　“崩界拳。”萧逸大喝一声。
<br/>　　心脏处的血色水滴，瞬间暴走。
<br/>　　一股澎湃的血色，陡然间凝聚在拳头。
<br/>　　“好恐怖的力量。”萧逸顿时一惊，而后心头涌现了一股不妙的念头。
<br/>　　轰的一声。
<br/>　　整个天地，竟产生了一股震天轰鸣。
<br/>　　同时，他的脑袋，忽然空白一片。
<br/>　　拳头上，身体中，立刻传来一股锥心般的扭曲疼痛。
<br/>　　血色水滴，几乎是肉眼可见的速度在减少着。
<br/>　　“收…噗…”萧逸猛地吐出一口鲜血。
<br/>　　噗…噗…噗…
<br/>　　连续吐了几口腥血，他才稳下体内伤势。
<br/>　　他的拳头，竟扭曲得变形。
<br/>　　“该死，施展崩界拳，竟是要瞬间调动所有修罗之力。”
<br/>　　“这已经超出我的肉体承受限度了。”
<br/>　　萧逸暗暗咬牙。
<br/>　　但下一秒，当他看到面前的景象时，顿时震惊得无以复加。
<br/>　　面前，原本不断流淌的河流，硬生生断流。
<br/>　　中间一段的水流，凭空变成了虚无。
<br/>　　河的对面，本有一小片森林。
<br/>　　此刻，空荡荡一片，连枝叶都没有留下一根一片。
<br/>　　再前方，本有一座高山。
<br/>　　此刻，被夷为了平地，只剩一堆粉尘。
<br/>　　“好…好强…”萧逸震惊着。
<br/>　　他无法估计这一拳的威力有多强。
<br/>　　施展修罗变，一滴修罗之力，足以支撑一天。
<br/>　　施展崩界拳，却是一瞬间就要将一滴修罗之力完全耗尽。
<br/>　　而这，是萧逸如今的肉体远远无法承受的。
<br/>　　所以，崩界拳他并没有真正打出。
<br/>　　不过是刚刚施展，感觉到身体承受不住，已经立刻收回了拳头。
<br/>　　一滴修罗之力，也只是消耗了三分之一左右。
<br/>　　但仅仅如此，其产生的威力，却强得惊人。
<br/>　　面前的满目疮痍，完全超出了他的想象。
<br/>　　若是真正完全打出崩界拳，一滴修罗之力耗光。
<br/>　　其威力，该是达到何种地步。
<br/>　　恐怕，那已经是超越破玄境的范畴了吧。
<br/>　　当然，他也只敢想想。
<br/>　　他毫不意外，若自己真的完全打出；恐怕敌人还未死，自己就先肉体崩坏而亡了。
<br/>　　“咕。”萧逸不禁咽了口口水。
<br/>　　他现在算是明白过来了。
<br/>　　难怪修罗变和崩界拳，会在第三层才激发。
<br/>　　要使用修罗之力，自身的肉体必须承受得住。
<br/>　　仅仅是施展修罗变，肉体就必须不亚于破玄一重。
<br/>　　要想施展崩界拳，肉体破玄一重，只是开始，还远远不够。
<br/>　　“修罗变，崩界拳。”萧逸喃喃自语着。
<br/>　　无论是施展修罗变还是崩界拳，都需要耗费修罗之力。
<br/>　　这一小滴修罗之力，若是消耗光了，体内会以缓慢的速度，慢慢恢复。
<br/>　　只不过，这需要很长一段时间，起码十天半月。
<br/>　　另外，也可以通过吸收妖兽精血，在短时间内恢复。
<br/>　　只不过，所要耗费的妖兽精血，数量很多。
<br/>　　萧逸盘膝坐下，稳住刚才反噬所受的伤势。
<br/>　　半响，他淡淡地站了起来。
<br/>　　“修为，达洞玄五重。”
<br/>　　“肉体力量，达破玄一重。”
<br/>　　“是时候离开了。”
<br/>　　萧逸自语着，身影一闪，化出紫炎火翼，瞬间飞跃离去。
<br/>　　这大半个月修炼，他实力大增。
<br/>　　但，想来，那一小滴修罗之力，才是他的最大收获。
