　　第2245章
<br/>　　嘭！
<br/>　　仅仅是呼吸的时间，苏莫停了下来，因为他撞在了第一重神殿之上。
<br/>　　“果然如此！”苏莫睁开了眼眸，眸光闪亮。
<br/>　　适才，他固守心神，不感知外界任何存在，只是单纯的飞行，便来到了神殿之前。
<br/>　　随即，苏莫深深的吸了口气，大步的走进了神殿之中。
<br/>　　进入神殿，入眼处一片朦胧，光晕万千，没有任何人影，也没有任何的实物。
<br/>　　“人呢？”苏莫心中惊疑，他伫立不动，仔细感知查看四方。
<br/>　　他没有任何的发现，唯一的发现，便是此地充满了玄而又玄的奥义之力。
<br/>　　不！
<br/>　　这不是奥义，是法则，天地法则，亦被称为大道规则。
<br/>　　“时间法则笼罩的地方，会有什么改变呢？”苏莫心中思索。
<br/>　　说是三十三重神殿之中，能掌握时间流速，甚至是逆转时空。
<br/>　　这虽然有些夸张，但是，苏莫相信，这并不是虚言。
<br/>　　“时间流速，逆转时空!”苏莫低声自喃，随即缓缓的盘膝坐了下来。
<br/>　　他的精神感知力，弥漫四方仔细感知这处空间的时间法则。
<br/>　　法则，乃是比奥义更高深的大道，想要参悟，几乎是不可能。
<br/>　　不过，事到如今，苏莫只能屏弃凝神，静静的感悟，感悟周身时间法则的变化。
<br/>　　时间缓缓的流逝，苏莫好像把握住了什么，但又像什么也没有把握住。
<br/>　　“时间……规则……控制……改变……！”
<br/>　　苏莫心中，千回百转。
<br/>　　他发现，这无数的规则，形成了一片独立的世界，将一切都隔绝了。
<br/>　　在这片空间之中，时间被重塑，而且变得非常之清晰。
<br/>　　好似，时间变成了有质之物，他手掌一抓，便能抓住时间。
<br/>　　这种感觉，无比的玄妙，无法言喻。
<br/>　　“我明白了，此处的时间法则，并不是让人参悟，而是有既定的规律，需要控制此规律，然后去控制这里的法则！”
<br/>　　苏莫心中空灵，暗暗猜测，这就仿佛是，一个本来无法抓住的虚幻之物，已经被别人凝实，他只要抓住实物便可。
<br/>　　也就是说，他只要控制了这个规则，就能掌控此地的时间。
<br/>　　当然，这也仅限在此地而已，对于他在外界参悟时间大道，没有多大的好处。
<br/>　　苏莫试着去抓住这规则，当然，并不是真的动手抓去，而是用意念去控制。
<br/>　　他的精神意识，彻底的辐散而开，笼罩着这片空间。
<br/>　　感触规则……亲近规则……控制规则！
<br/>　　约莫用了一个时辰的时间，他便坐到了这一点。
<br/>　　唰！
<br/>　　苏莫睁开了眼眸，眸中精光一闪，他已经成功了。
<br/>　　“这是？”
<br/>　　看到眼前的场景，苏莫顿时一呆，因为此刻，眼前场景已经大变。
<br/>　　第一重时光神殿之内，朦胧的光晕已经消失，彻底一目了然，只是一个空旷的大殿。
<br/>　　此时此刻，大殿之中，只有数十人，或伫立不动，或盘膝而坐，显然也在感悟。
<br/>　　至于其他人，显然已经进入了第二重时光神殿。
<br/>　　而在这第一重大殿之中，地上有十几具尸骨，均是已经腐朽，不知道存在了多久。
<br/>　　“控制时间！”
<br/>　　苏莫没有理会这一切，他的心中已经怦然心动。
<br/>　　因为，掌控了第一重神殿的时间规则，他已经能控制这里的时间了。
<br/>　　只是不清楚，能控制到何种地步？
<br/>　　手掌一番，他取出了一块剩余的无用灵识，向神殿之外抛了出去。
<br/>　　灵石飞出神殿的殿门之后，便径直向下落去，只需半个呼吸的时间，便能落地。
<br/>　　就在此刻，苏莫意念一动，调动了神殿之中的规则力量。
<br/>　　然后，苏莫清晰的看到，灵石下落的速度暴降，便的极为缓慢。
<br/>　　嘭！
<br/>　　约莫三个呼左右，灵石才落在了神殿门前，由虚空之力凝结的地面上。
<br/>　　“六倍左右！”苏莫心中惊喜，第一重神殿之中的时间流速，在他的控制之下，加快了六倍左右。
<br/>　　换言之，他在神殿中待六日，外界才过去一日。
<br/>　　“好玄妙的时间规则！”苏莫心中惊叹，不过，他发现，他能控制的时间规则，只在他深处的半丈范围之内。
<br/>　　殿中的其他人，完全不受到影响。
<br/>　　“有了如此快速的时间流速，这里六日，外界一日，简直是修炼福地！”苏莫眸中闪过欣喜的光芒。
<br/>　　他的时间，一直不多，导致很多绝学都没有时间修炼。
<br/>　　而在这里，他就能凭空多出数倍的时间。
<br/>　　深深的吸了口气，苏莫没有再第一重神殿停留，向第二重神殿走去。
<br/>　　第二重神殿，可能比第一重神殿更加玄妙，他自然不会留在第一重神殿。
<br/>　　神殿的另一端，有一个宽敞的通道，同样是由虚空之力凝聚而成，直通第二重神殿。
<br/>　　通道只有数十丈长，穿过通道，苏莫便来到了第二重神殿的门前。
<br/>　　第二重神殿之中，同样是漆黑一片，看不到任何的眉目。
<br/>　　唰！
<br/>　　苏莫一步未停，立刻便进入了第二重神殿之中。
<br/>　　和进入第一重神殿时一般无二，他再次来到了一个朦胧的空间，眼前光晕流转，难辨方向。
<br/>　　随即，苏莫又用同样的方法，感知时间法则，掌控时间法则。
<br/>　　第二重神殿之中，时间法则略有改变，让他多耗费了半个时辰的时间。
<br/>　　然而，第二重空间的时间规则，却是时间流速变缓，而且是足足慢了十几倍之多。
<br/>　　这对于苏莫来说，没有任何的用处，他需要的是，时间流速暴增。
<br/>　　九天的时间，能让他多出千倍万倍的时间。
<br/>　　或者是，真的能逆转时空，虽然他觉得这不太可能，但也要去试一试。
<br/>　　第二重神殿之中，驻留的人更多，足有百人左右。
<br/>　　地面上，同样有不少的尸体，数量远超第一重神殿。
<br/>　　苏莫不清楚这些人死于什么时候，是怎么死的，他进入了第三重时光神殿之中。
<br/>　　同样的一幕再次出现了，这一次，时间规则大变，让得苏莫振奋了起来。
<br/>　　耗费了更长的时间，苏莫终于掌控了第三重时光神殿的规则。
<br/>　　随即，他控制规则之力，稍稍一动用，立刻让他心神大震。
<br/>　　因为，不可思议的一幕发生了！
