　　第238章
<br/>　　距离七月十四已经不远，古语说这一天盂兰盆会，鬼门开启，逝去的亡者会重新来到人间。适合放河灯，悼念逝去的亡人，追忆旧情人，也适合诛杀极恶凶徒。
<br/>　　程大雷被关进天牢后，没有审，也没有问，每日三餐具备，要酒有酒，要肉有肉，还有青叶佛这德高望重，字字珠玑的罗汉僧帮程大雷做精神安慰。
<br/>　　待遇是相当不错的，原因是，不需要审，也不需要问。杀之一字，是早已确定的。关键是怎么杀，最后大家一商量，如程大雷这样的魔头，砍头之类的太仁慈了，干脆凌迟吧，三百六十五刀，在天下人面前活剐了他，也让天下人看看，和帝国做对是一个怎样的下场。
<br/>　　皇城，御书房内。
<br/>　　明帝整理着下面呈上来的奏折，这是一份江南地区的情报，江南今年的皇粮只收上来一半，据说因为连绵大雨，犯了涝灾，可据明帝得到的情报，江南今年风调雨顺，并无灾情。
<br/>　　在他脚下跪着一个长发女子，长发垂落，如同黑瀑。
<br/>　　时间一点一滴过去，明帝桌上的茶已经换了三次，那女子仍旧跪着，不吭不响。
<br/>　　良久，明帝将奏折合上，目光才肯落在女子身上。
<br/>　　“回去吧。”
<br/>　　“父王，请……”
<br/>　　“朕知道你的来意，所以你也不要说。你不仅是朕的女儿，也是帝国的公主，给自己留点体面。”
<br/>　　因为知道说了无用，便不要说了么。李婉儿知道自己是帝国公主，所以不管她再胡闹，也在有限的范围内，对于朝堂之事，从不妄发一言。
<br/>　　可这次的事情不一样。
<br/>　　“父王，程大雷罪不该死。”
<br/>　　明帝继续翻看奏折，他既然已不让李婉儿说了，那么不管李婉儿说什么，他都不会听见。
<br/>　　“您也说程大雷乃是帝国出类拔萃，不拘一格的人才，他活着，远比他死了有价值。”
<br/>　　奏折是一份西北军求粮的，那里马匪横行，西北军需要一笔粮草剿匪。
<br/>　　“程大雷于国有功，杀死功臣，却让奸臣横行，怕是会让天下人寒心，长此以往，恐李氏将失去天下。”
<br/>　　明帝忽然睁开双目，那眼中的威压如同实质。
<br/>　　李婉儿知道自己触怒了明帝，可相比毫无反应她更愿意明帝暴怒。
<br/>　　“你为何一定要救他，他是那个你愿意给他生儿育女的男人？”
<br/>　　明帝似乎没有因刚才的话生气，反而提了一件不太适合此刻提起的事情。
<br/>　　李婉儿沉默，借着摇摇头：“女儿觉得，他可以算作我的朋友。”
<br/>　　“帝皇之家，没有朋友。”
<br/>　　明帝合上了奏折。
<br/>　　……
<br/>　　长安城，这几日可以说鸡犬不宁，但也可以说万籁俱寂。
<br/>　　自从那一日事情过后，城防军便开始大肆搜捕，城内的贩夫走卒，江湖术士都遭到缉拿，一时间城内风声鹤唳，大牢内人满为患。当然，这时程大雷的人已经撤出去，而其他大盗却没有这样的好运，他们罩着的人七七八八都落入大牢中。
<br/>　　满福楼，在长安城以烧鹅闻名，已传了好几代，大家都说，你若没有吃过满福楼的烧鹅，你就不算来过长安。
<br/>　　福满楼的东家周老板来到后院，指挥着伙计劈柴烧水，他一个人走进了地窖。
<br/>　　地窖存储着很多食材，空气中漂浮着一种食物腐烂的味道。他搬开几个堆在墙角的酒坛，之后地上又出现一个向下的梯子。
<br/>　　因为三十年前戎族来过长安，在长安城烧杀抢掠，经过那次事情的人，都会在家里修一个藏身之地，以避免戎族再杀进长安。
<br/>　　这个地窖内的地窖是福满楼上一任东家修的，三十年来没有用过，此时却发挥了作用。
<br/>　　里面像一个老鼠洞，聚着七八个人：阿苦，阿喜，徐神机，徐灵儿，秦蛮，林冲，飞天蛤蟆……
<br/>　　“周老板，今次的事情多谢你了。”
<br/>　　“诸位放心，大哥对我有恩，现在他落难，我怎么能不管，我们长安的人都是讲义气的。”周老板慷慨激昂，心里想的却是：当然，如果不是你们把我女儿带走藏起来，我也没有这么讲义气。
<br/>　　“现在外面什么情况？”
<br/>　　“我听来酒楼吃饭的人说，朝廷已经颁下旨意，七月十四，要在朱雀大街将大哥凌迟处死。”
<br/>　　七月十四，留给诸人的时间没有几天，阿苦挥挥手让老板离开，地窖内响起轻轻的叹息声。
<br/>　　如何办……
<br/>　　“诸位可有什么高见？”阿苦。
<br/>　　“还能怎么办，只有劫法场一条路。”徐神机道：“叫上兄弟，咱们劫了大当家，反出长安城。”
<br/>　　阿苦有些发愣：长安城有城防军十五万，皇城有禁卫军三万，凭几百江湖人就想反了长安城，这是没睡醒吧。
<br/>　　常听程大雷说，徐军师如何神机妙算，如何足智多谋，作为同行，阿苦还是很崇拜他的，只是，今天听来，事情好像不是这样。
<br/>　　“劫法场怕是不行。”林冲道。
<br/>　　阿苦心里点点头，林冲不愧是曾经长安城有名的高手，还是看得懂其中利害的。
<br/>　　“得劫天牢啊！”林冲。
<br/>　　呃……看来自己还是误会了什么。也许，自己排除掉他二人的建议，就能更靠近正确答案。
<br/>　　不得不说，阿苦已隐隐约约摸索到徐神机的用法。
<br/>　　“依秦蛮看，不管是劫法场还是劫天牢，都太过冒险，到时候未必能救出大当家，反而会损掉很多弟兄的性命。”
<br/>　　阿苦看着这位相识不久的汉子，他有一丝欣慰，在这场考试中，自己还有可以互相抄袭的人。
<br/>　　“干脆杀进皇宫，把明帝劫了，用他换大当家。”
<br/>　　好吧，又排除掉一个错误选项，自己距离正确答案又近了一步。
<br/>　　仔细想想，还有些小兴奋呢。
<br/>　　阿苦看向与自己相熟的阿喜，满脸期待。来吧，说出你的答案，不用羞涩，用你最大的声音，再告诉我一个错误答案。
<br/>　　可是，这道题下不止四个选项，并不是排除掉三个后，就可以得到正确答案。
<br/>　　更或者，这道题的所有选项下，根本没有正确答案。
