　　接下来一个月的时间，刘哔都是早出晚归的忙碌着，一边是为了搜刮生存所必须的物资，一边也在猎杀丧尸，用以赚取晶核和积分。
<br/>　　贫民区的那种环境，隐私性实在是太差了，他想换个地方居住，要不连晚上修炼的空间都没有。
<br/>　　毕竟他的修炼方式和这个世界的异能者不同，不用晶核就可以直接修炼，这个消息万一传出去，还不知道要引来多少麻烦。
<br/>　　就在日子这样一成不变的缓慢推进着，系统久久不来的提示，却又突然之间响起了。
<br/>　　“距离传送倒计时还有三个小时，即将进入下一个世界，请宿主提前做好准备。”
<br/>　　此时的刘哔正在一座城市里，像往常那样一般搜索物资，听到系统提示的时候，整个人瞬间懵了。
<br/>　　这一个月的时间里，系统一直没有出现过，他几乎都要以为是不会再进行穿越了。
<br/>　　但今天系统突然给出提示，让他瞬间仓促了起来，他还有很多事情没来得及准备。
<br/>　　“能不能晚一些，我需要一些准备时间。”顾不得自己现在还在城市中，刘哔站在十字路口的交叉处，皱着眉头向系统问道。
<br/>　　心中有些焦急，他忘记了用意识也能和系统对话，周围的丧尸被吸引，皆是凑了上来，将刘哔围得水泄不通。
<br/>　　“相关功能暂未开通。”系统冰冷的提示音从刘哔脑海中响起，让他打消了自己心中原本的打算。
<br/>　　既然这样的话，那也没什么办法了，不过好在还有三个小时的时间，也足够他做一些事情了。
<br/>　　看着周围围绕着他嗅来嗅去的尸群，刘哔有些烦躁，伸手将几只丧尸扒拉到一边，整个人冲天而起。
<br/>　　为了增加效率，在外出搜索物资的时候他都不会收起翅膀，当然脸上的面具也从来没有摘下来过。
<br/>　　一路飞到基地附近，刘哔没有开车，他肯定不能用正常方法入城，否则的话，光是禁闭时间就不止三个小时。
<br/>　　围绕着高耸的城墙转了几圈，刘哔从空间戒指中取出一个麻袋，随手填充了一些晶核进去，这是他给刘莽准备的，有这些晶核，想必在基地生存还是没有什么问题的。
<br/>　　虽然基地的制度并不算完善，但只要你有足够的晶核，又不主动去招惹事情的话，安全还是足以得到保障的。
<br/>　　将一切准备好之后，刘哔等待了一会，看到城门附近一个带着鸭舌帽的男人。
<br/>　　这是走私犯，专门负责帮人运送违禁品或者其他不方便检查的东西进入基地，而他们从中赚取一定数量的晶核，当然，如果晶核到位的话，人他们也可以运送进去。
<br/>　　这类人的存在在基地中并不是什么秘密，奇怪的是基地高层虽然也对此进行了几次打压政策，但每次都是虎头蛇尾的收场。
<br/>　　故此，有流言传开，这群走私犯在基地高层有相当强硬的关系，或者压根就是基地高层培养出来的特殊势力，在基地附近，几乎所有人对他们都不敢有丝毫不敬。
<br/>　　“我赶时间，不能按流程进入，你能不能把我送进去？”来到鸭舌帽男子身边，刘哔从衣服里掏出一把晶核，塞入了男子怀中。
<br/>　　看到怀中亮晶晶的晶核，鸭舌帽男子眼睛明显一亮，但也没说什么，只是点了点头，随后将晶核收好。
<br/>　　因为是第一次经历这种事情，刘哔也不知道具体该怎么操作，只是手中提着麻袋，静静的站在原地，等待鸭舌帽男子的指示。
<br/>　　但似乎并不需要他来做什么，鸭舌帽男子拉着刘哔来到一处远离基地的地方，手中亮起一道灰色的光芒。
<br/>　　随后，一股强烈的眩晕感从脑海中传来，刘哔脸色一冷，立即用灵气护住脑部，警惕的看着鸭舌男子。
<br/>　　鸭舌男子对此没有丝毫反应，依旧保持着手中的动作，刘哔似乎是想到了什么，忍住了心中想要动手的欲望。
<br/>　　大概过了十几分钟，四周的场景一阵变换，两人传送到了一个简陋的工厂中。
<br/>　　“谢谢。”认出了这里是基地内部，刘哔微微有些讶异，但脸上也没有露出什么表情，抬脚就要离开。
<br/>　　鸭舌帽男子伸出手臂，阻住了刘哔的动过，声音低沉的说道：“衣服脱掉，检查一下身上有没有伤势。”
<br/>　　闻言，刘哔脸上的表情逐渐开始冰冷了下来。
<br/>　　“怎么，我怎么没听说过，找你们走私还要进行检查啊？”将手中的麻袋放下，刘哔转过身来，整理了一下衣服。
<br/>　　四周，几个身材彪悍的壮汉将工厂封锁，从他们身上流露出来的气息判断，任何一个都不是好相与之辈。
<br/>　　这样的变故让刘哔心情变得有些糟糕，他倒不在乎战斗，只是现在时间已经不多了，他要尽快赶回家里。
<br/>　　“走私物品不用，我们从来不关心货物是什么。”男子将头顶的鸭舌帽取下，露出了他的真实面目。
<br/>　　这是一个三十多岁的中年男子，面色刚毅，留着一头精神的短发，但脸上一道斜贯了整个脸庞的狰狞刀疤，破坏了男子脸上的阳刚之气，让人莫名有些觉得不太舒服。
<br/>　　“但是，人，我们要检查，要不出了事情，我们也不好交代。”将鸭舌帽随手扔到一边，刀疤男子继续向刘哔解释道。
<br/>　　看着一眼周围将工厂封锁住的彪形壮汉们，刘哔脸色阴沉，大脑飞速旋转着。
<br/>　　这个规矩，他倒是从来都没有听人说起过，不过从情理上来讲，刀疤男子的话语好像没有什么问题。
<br/>　　而这些人虽然将工厂封锁住，但也没有主动对他露出什么敌意，更没有主动进攻，如果是针对他而布下陷阱的话，刀疤男没有必要和他啰嗦这么多。
<br/>　　犹豫了片刻，刘哔心中有了定计，对刀疤男微笑了一下，伸手脱下了身上的衣服。
<br/>　　外衣，衬衫，裤子，鞋，一直脱到只剩下一条内裤之后，刘哔停止了动作，不耐烦的看着刀疤男子。
<br/>　　“快点，别耽误我时间，我还有事。”估算了一下时间，他逐渐开始失去了耐心。
<br/>(本章完)
