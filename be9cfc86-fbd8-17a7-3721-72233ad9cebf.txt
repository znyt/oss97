　　　　蛇博士陈默的脸色有点难看，要白家真的有一条成年的金蚕蛊蛇，那说明已经有不少无辜的人惨死。
<br/>　　“这条蛊蛇可否让我拿回去做研究”忽然，陈默看着涂小安开口。
<br/>　　创造一条金蚕蛊蛇太过繁琐，还要用大量的活人血，可现在有一条现成的摆在面前，就算这是条邪蛇，但也蕴含着巨大的宝藏，蛇博士按捺不住渴望的心。
<br/>　　涂小安闻言，深深看了蛇博士一眼，迟疑了下，但最后还是点了点头。
<br/>　　这条蛇，他要来没什么大用。
<br/>　　蛇博士看到涂小安点头，顿时大喜，连忙感谢。
<br/>　　蛊蛇有非常大的研究价值，纵容没成年，也无妨。
<br/>　　“袁儿在古学院还好吧”此时，蛇博士终于想到了东方袁身上。
<br/>　　涂小安笑了笑：“你放心，纵然我不在古学院，也能保证他无碍，不受欺负”
<br/>　　出来的时候，涂小安已经让副院长老头多关照一下东方袁，还有林凡这个生物掌控者跟他混迹在一起。
<br/>　　另外，涂小安收了罗子杰为战仆，这位大神学员以后就是东方袁最好的依仗了。
<br/>　　“那便好，那便好”蛇博士欣慰的点了点头，拿着蛊蛇跟涂小安告辞。
<br/>　　至于白家的事，蛇博士也没多问，力所不能及。
<br/>　　陈默走后，涂小安看着骨瘦如柴的余子童道：“余大师，你的黄金瞳还想要吗”
<br/>　　余子童病怏怏的，眼中惊恐之色：“怎么要，我的猫现在落到白家祖奶奶手中，任由我有天大的本事，恐怕也要不回来，我拖您的福，现在能保住一条小命，已经感恩戴德了，不敢在有非分之想”
<br/>　　“你若不要，那我去要，但是我要回来了，黄金瞳就属于我了”涂小安妖治的眼眸精光一闪的开口。
<br/>　　“你要？？？”余子童有点诧异，但转念一想，苦笑连连，表态道：“涂大师你要是有本事尽管去要，要到了，余某人我绝无二话，全当报了你的救命之恩”
<br/>　　余子童经过这件事情，已经明白君子无罪，怀璧有罪，超等生物，岂能是他区区一阶的修武者能拥有。
<br/>　　说句难听的话，他不配，门不当户不对。
<br/>　　突破二阶的超等生物黄金瞳在他的身边，不但不能给他带来什么好处，还会祸事缠身，被人觊觎。
<br/>　　余子童的识相，让涂小安很欣慰，这家伙总算是开窍了，不枉费自己救他一命。
<br/>　　白家的祖奶奶对余子童下蛊蛇，有两点厉害之处，可谓一石二鸟。
<br/>　　第一，黄金瞳跟余子童有血之契约，想真正得到黄金瞳，那么余子童这个主人就必须死，死了契约就不复存在，第二，对方并不想让自己的手染血，就用蛊蛇要了他的命，还能帮助蛊蛇成长，简直就是一举二得。
<br/>　　等余子童死了他体内的蛊蛇也自动会回去。
<br/>　　很厉害，很毒的手段，将自己的好处最大化，也充分的将余子童废物利用。
<br/>　　“这十颗小果实你每天吃一颗，对你的身体有帮助，能让你胖点回去”涂小安给了余子童十颗蛇莓，以后那只猫，跟这个人再无任何关系。
<br/>　　接下来，涂小安在问了一些相关的事情，就让苏羽带着余子童去休息了，这家伙现在太弱鸡了，说几句话，都把他累的够呛。
<br/>　　而涂小安带着牛妖王离开了味鲜阁。
<br/>　　“主人想去白家？”牛妖王听到涂小安跟余子童问了白家的地址。
<br/>　　涂小安笑了笑：“想去看看，他那只猫我挺喜欢的，另外我对成年的金蚕蛊蛇也有点兴趣，最关键，白家跟我渊源颇深”
<br/>　　深的下午他才将白家的少主，白君然给打趴下。
<br/>　　“你不用跟着我去了，有事我自会召唤你过来”
<br/>　　涂小安在黑夜中自行离开，朝着余子童给的白家位置摸去。
<br/>　　莫约在晚上十点钟，涂小安站在了一栋气派不凡的巨大别墅不远处，抬头望去，就算是夜幕笼罩，也无法隐藏这栋建筑物的强大气场。
<br/>　　“不愧是温城第一隐藏家族，够有钱的”涂小安眼眸闪烁了下，这栋大别墅内，他感受到了很多股强大的修武者气息，绝对的藏龙卧虎。
<br/>　　“想要进去，恐怕要抛弃人形态了”
<br/>　　涂小安现在要去的地方，可不是普通人家，不可能一个翻墙就进去，稍稍一眼，就知道门庭森严。
<br/>　　要是翻墙进去，恐怕第一时间就会被发现。
<br/>　　他今天的目的很简单，将黄金瞳给带出来，顺便看看是不是真的有一条金蚕蛊蛇的存在。
<br/>　　至于白家的人，能不惊动就别惊动，这种大家族，肯定有涂小安洞察不出实力的强者存在。
<br/>　　不过就算如此，涂小安也不怕，他现在有底气闯这种藏龙卧虎的大家族，为什么呢，他有无条件收服能力。
<br/>　　遇到白家的掌舵人更好，直接给收服了，白家岂不是属于他的？
<br/>　　当然，涂小安就这样一想，不要把他逼急就好。
<br/>　　此时，涂小安摇身一变，变成了一条迷你的紫金蛇，准备偷偷摸摸的游进去，可忽然，白家门庭前，一辆艳红色的敞篷跑车，颜色十分耀眼，造型也非常拉风的停了下来。
<br/>　　跑车门打开了，一个装扮与穿着都极为时尚的极品美女迈着雪白长腿下车了。
<br/>　　“白楚燕？”
<br/>　　涂小安一眼就认出了这位大美女，只不过没有穿警察制服的白楚燕到是更加的漂亮跟时尚，穿着极短的热裤，露出一双雪白修长，高挑笔直之极的超级大美腿，非常的吸引人眼球。
<br/>　　只见她下车，就直径的朝着白家门庭走去，看样子还挺着急，行色匆匆。
<br/>　　当警察，开这种车，是不是太嚣张了点。
<br/>　　涂小安恶趣味一想，不过现在是大晚上，警察也要下班，拥有自己的生活。
<br/>　　看来有人可以带自己光明正大的进去了，不用偷偷摸摸。
<br/>　　涂小安的蛇躯再次无限缩小，大概缩小到小泥鳅状，骤然化为一抹微光射去，射到白楚燕的上衣兜兜内隐藏。
<br/>　　“二小姐”
<br/>　　白家大院的门打开了，有两个精壮的门卫朝着白楚燕尊敬的开口。
<br/>　　白楚燕冷若冰霜，直接朝着别墅的内部走去，然后很快来到了一处古老而庄严的宅院。
<br/>　　“我听说我弟弟受伤了，严重吗”
<br/>　　宅院中，一位老管家出来迎接，白楚燕立刻边走边开口询问。
