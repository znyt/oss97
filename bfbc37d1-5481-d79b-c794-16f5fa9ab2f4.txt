        “你有点贫血，我带你上去开药！”连华生连忙将童筱颖的身子捞起来，横抱在怀中，疾步往楼上走去。
<br/>
<br/>        这童筱颖可是傅夜沉的心头肉，万一被他这次和傅老太爷联手试探把她童筱颖给整出什么大毛病出来了，若是被傅夜沉知道，他一定会死得很惨！
<br/>
<br/>        若说兄弟和最爱的女人之间，傅夜沉会选择谁？
<br/>
<br/>        连华生用脚趾头想想都知道，傅夜沉十有八九会选择童筱颖。
<br/>
<br/>        所以，连华生是打心底地希望童筱颖不要出事。
<br/>
<br/>        连华生将童筱颖送回房间里后，就只身离开，打算去给她拿药过来。
<br/>
<br/>        童筱颖蜷缩着身子，捂着自己的腹部，侧卧在大床上。
<br/>
<br/>        头发湿漉漉的，一绺一绺的像一条条黑色的水蛇，蜿蜒在她的肩头。
<br/>
<br/>        连华生拿了药过来，在童筱颖的床边坐下，将药递给了她。
<br/>
<br/>        童筱颖身心难受着，看到连华生，直接转过身去背对着他，更加不想吃他开的药。
<br/>
<br/>        “喂！大小姐！你把药吃了吧！好吧！我知道这次是我不对，但是我也是受命于傅老太爷！傅老太爷决定的事情，我也没法改变啊！虽然，你这次和大少奶奶都过关了。只不过，傅老太爷现在对大少奶奶似乎有想法了。起初，你两的表现，傅老太爷通过远程监控，看得一清二楚。”连华生耐着性子解释道。
<br/>
<br/>        起初，苏漫雪的确是带那个“歹徒”去拿那把金钥匙，但是苏漫雪只不过是借着拿金钥匙的幌子，趁机在自己的行李箱里翻出一瓶发胶喷雾，喷到了那个“歹徒”的眼睛上以此逃脱。
<br/>
<br/>        但是她这个行为很明显地激怒了“歹徒”，“歹徒”直接拿刀“捅死”了她。
<br/>
<br/>        当然，既然只是试探，那把匕首自然是假的，苏漫雪算是虚惊一场。
<br/>
<br/>        “我坦白跟你说了吧！阿沉的命，关系到很多人的生死。所以说，阿沉的命比你和大少奶奶更重要。身为阿沉身边的女人，必须得扛得起这些……”连华生接着说道。
<br/>
<br/>        然而，他的话才说到一半，童筱颖突然坐起身来，疾言厉色地呵斥道：“大哥身边的女人？呵！你们要试探，就试探苏漫雪一个人就行了！为什么要搭上我？难道，真的像苏漫雪所说的那样。爷爷、大哥还有你们，都在怀疑昼景对大哥不利？昼景也是爷爷的亲孙子，昼景也是大哥的亲弟弟。昼景那么为傅氏集团卖力，为了傅氏集团日夜加班，昼景拿的只是他该有的工资，你们到底还想怎样？你们真的好自私！”
<br/>
<br/>        “你——”连华生欲言又止。
<br/>
<br/>        就连傅老太爷都看得出，傅夜沉喜欢的女人是她童筱颖，她童筱颖怎么就不明白？
<br/>
<br/>        连华生剑眉微微扬了扬，似懂非懂地笑了笑。
<br/>
<br/>        童筱颖见连华生只笑不语，心里极其不爽地反问：“你笑什么？”
<br/>
<br/>        “你一心护着你的丈夫，你又真正地了解过你的丈夫吗？”连华生话中有话地说。
<br/>
<br/>        “我当然了解他！”童筱颖斩钉截铁地回答。
<br/>
<br/>        连华生不再多说，而是把药和水杯放在了一旁的床头柜上，站起身来，接着说道：“药我放这里了！大小姐，看人不是用眼睛去看，而是用心。表面上所呈现出来的，不一定是真的！你是个不错的女人，希望，你的丈夫能好好珍惜你。”
<br/>
<br/>        “……”童筱颖有些无言以对。
<br/>
<br/>        连华生说完，便起身离开了，他还得去楼下把刘婶弄醒。
<br/>
<br/>        身体是自己的，自己好起来，才不会成为昼景的负担。
<br/>
<br/>        连华生离开后，童筱颖便把药给吃了，然后去冲了个热水澡，洗掉身上那一身鱼腥味。
<br/>
<br/>        这件事情，让童筱颖心生芥蒂。
<br/>
<br/>        苏漫雪后来被傅老太爷的手下带走了，至于带去了哪里，连华生没说。
<br/>
<br/>        傅夜沉回来后，确实对这件事情一无所知，就连刘婶也在傅老太爷的命令下都对傅夜沉守口如瓶。
<br/>
<br/>        晚餐的时候，童筱颖低垂着眸子，面无表情地吃饭。
<br/>
<br/>        傅夜沉总觉得气氛很怪，但又说不出哪里奇怪。
<br/>
<br/>        吃过晚饭后，童筱颖帮着刘婶一起做完家务，就自顾自地回房间。
<br/>
<br/>        哪怕在走廊上遇到了刚从房间里出来的傅夜沉，童筱颖也视而不见，一个招呼也不打地去了自己的房间。
<br/>
<br/>        明明同住在一个屋檐底下，又同时上下班回家，童筱颖依旧未跟傅夜沉说过一句话，连个笑脸都不再给。
<br/>
<br/>        一天、两天也就算了，连着三四天，她都对他不理不睬，视而不见，听而不闻。
