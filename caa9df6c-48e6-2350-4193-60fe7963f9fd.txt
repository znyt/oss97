        &lt;h3&gt;第三卷第一百八十八章  陈虞落姸&lt;/h3&gt;
<br/>
<br/>        团团和嘟嘟将张沛尧和弥青云带着参观了他们的书房，然后带着他们又到了妈妈的书房，指着里面的书说：“这些我们都看过。”
<br/>
<br/>        对于团团和嘟嘟的书房，他们并没有感觉到很惊奇，都是孩子们看的读物，只是到了陈子昂的书房，他们才感到了惊奇，陈子昂的书房很大，陈子昂的房子是在三环之外，算是一个小型的庄园，环境优雅，光住房大概占地面积大概三百多米。
<br/>
<br/>        陈子昂的书房大概有八十多平米，里面有很多书，简直是一个小型的书库，团团说他们都看多？
<br/>
<br/>        “你们几岁开始看书？”
<br/>
<br/>        “我们在会说话的时候，妈妈就教我们读诗，我们上幼儿园的时候，已经能读很多的诗了，而且还认识很多字。所以，这么多年，我们基本都看了很多书，唉，只是看了一遍而已，妈妈看书基本都会标注。”团团似乎觉得妈妈有了标注有点可惜。
<br/>
<br/>        “妈妈有了标注不好吗？”弥青云问。
<br/>
<br/>        “妈妈标注了就是妈妈的思想，人不应该有自己的思想吗？”嘟嘟反问。
<br/>
<br/>        “那你们觉得妈妈的标注不好吗？”张沛尧感觉他的思路更不上了。
<br/>
<br/>        “当然好啊，妈妈的三观很正啊，要不然怎么能教育出来我们这么优秀的孩子？”嘟嘟说
<br/>
<br/>        “是啊，妈妈要是那种趋炎附势，拿着自己儿子幸福谋福利的人，我们怕是丢垃圾桶都不会惹人注意的了。”团团那萌萌的童音突然就冷了下来。
<br/>
<br/>        张沛尧和弥青云相视一眼，都是藏不住的震惊，这哪是俩个孩子啊？陈子昂教出来了什么样的孩子啊？
<br/>
<br/>        “爷爷，想不想看我们小时候的照片？”团团问。
<br/>
<br/>        “想啊，快拿出来我们看看。”弥青云很想看看孩子儿时的照片。
<br/>
<br/>        “爷爷，那上面有个相册，你帮我拿下来吧。”团团指着书架说。
<br/>
<br/>        张沛尧拿下相册，弥青云就抢过来了，拿在手中就翻起来了。看着俩个孩子儿时的照片，弥青云笑着流出了眼泪。
<br/>
<br/>        他们当初真是瞎了眼，捡了芝麻丢了西瓜啊。
<br/>
<br/>        翻到最后，看到了张函和陈子昂的照片，照片上俩个年轻人的笑容晃得弥青云的眼睛疼，她从来都没有见过她儿子笑得这么开心。
<br/>
<br/>        “哈，这个是我妈妈和爸爸，不过吧，这个爸爸算是半道捡来的，可有可无，我们认他他就是爸爸，不认他，他嘛什么都不是。你说是不是啊，老奶奶？”团团说。
<br/>
<br/>        弥青云一惊，手中的相册掉落在地上。
<br/>
<br/>        张沛尧饶有兴趣的看着团团，俩个孩子带着他们参观书房是假，有话对他们说才是真的。
<br/>
<br/>        他们突然而来，要说是陈子昂教的那完全不可能，这俩个孩子的心智完全的超华了常人。
<br/>
<br/>        团团弯腰捡起了相册，小心的擦了擦，摊开来看了看，合上，递给张沛尧，张沛尧将相册放上去。
<br/>
<br/>        “我认识的爷爷奶奶是一直带着我长大的爷爷奶奶，这几天放假了，妈妈让爷爷奶奶回家去陪他的儿子过新年去了，对于你们，是妈妈让我们叫，我们才叫的，我姓陈，妹妹姓落，和那个半道捡来的爸爸没有任何关系，所以，请你们不要为难我妈妈！”团团看着张沛尧和弥青云说。
<br/>
<br/>        “团团，嘟嘟，无论你姓什么都改不掉你是你爸爸的孩子的事实，我们今天来只是想看看你们，并没有要为难你们的意思，对于我们给你和你妈妈造成的伤害我们深表抱歉和遗憾，所以我们会想办法弥补，但是，大人的世界你还是不懂。我们已经错过了一次，所以我们不希望我们在错一次。”张沛尧开口了，他觉得面前的俩个孩子太成熟了。
<br/>
<br/>        有好处也有坏处，现在他们的思想还不太成熟，不能让他们对张函和他们有意见，要不然以后怕是不好相见。
<br/>
<br/>        “我们是不懂啊，但是不妨碍你们懂啊，你告诉我为什么要拆散我爸爸和我妈妈？就因为我妈妈是农民吗？”嘟嘟开口了。
<br/>
<br/>        “我妈妈是农民没错，你家里有这么多书吗？有吗？”嘟嘟指着满屋子的书问。
<br/>
<br/>        张沛尧和弥青云看着孩子摇摇头。
<br/>
<br/>        “哇，没有啊，没有你凭什么要拆散他们？”嘟嘟又问。
<br/>
<br/>        “嘟嘟，你个小傻瓜，在他们的心目中，读书有什么用，权利才是最重要的。”团团一语惊人。
<br/>
<br/>        “哥哥，原来是这样啊，我明白了，原来妈妈没有权利，别急，等我们以后有权利了再和你们聊天，不过不知道你们能不能等得住呢？”嘟嘟言语犀利，咄咄逼人。
<br/>
<br/>        张沛尧和弥青云汗水都出来了，陈子昂这个妖孽生了俩个妖孽。
<br/>
<br/>        他们的眼睛确实是被老鹰啄了。
<br/>
<br/>        “我们......”张沛尧被俩个不到十岁的孩子说的羞愧不已。
<br/>
<br/>        “张先生、弥女生请不要为难我妈妈，也不要为难我们，莫欺少年穷！”团团一字一句的说。
<br/>
<br/>        “我们没有为难你们，我们只是想来看看你们。”弥青云解释。
<br/>
<br/>        “请，书房已经参观完了。”团团很绅士的做了一个“请”的手势。
<br/>
<br/>        张沛尧和弥青云算是见识了自己没有抱过一天的亲孙子的犀利，原来他们张家的孩子如此优秀，嗯，不亏是张家的孩子。
<br/>
<br/>        虽然被俩个孩子数落了一顿，但是张沛尧的心情竟然无比的开心，还好，这俩个孩子是被陈子昂带着，要是被陆玉带着，怕是养成了混世小魔王，会成为第二个陆玉。
<br/>
<br/>        想到陆玉，张沛尧心里又很堵。
<br/>
<br/>        现在局势不明，只有等机会啊。
<br/>
<br/>        俩个人算是被团团赶出了书房，俩个小家伙在后面相视一笑，彼此给对手竖了一个大拇哥。
<br/>
<br/>        哼，想欺负他们，门都没有！
<br/>
<br/>        当四个人走出书房，到客厅的时候却没有看到陈子昂和张函。
<br/>
<br/>        “妈妈！”俩个孩子没有看见妈妈心里有点慌。
<br/>
<br/>        “在这里呢。”陈子昂打开厨房的门，张函系着围裙在厨房忙碌着。
<br/>
<br/>        “今天是除夕，听张总说你们家里也是三个人，不如一起吃顿饭？”陈子昂微笑着看着张沛尧和弥青云。
<br/>
<br/>        “太好了，我还担心晚饭不知道在哪里吃呢。”弥青云高兴的说。“那我也来帮忙。”
<br/>
<br/>        “要你帮个啥忙呢，过来陪我和孩子说说话。”张沛尧拉着弥青云和团团嘟嘟到客厅去了。
<br/>
<br/>        “好好，其实我这人最不喜欢做饭了。”弥青云又尴尬的笑着说。
<br/>
<br/>        她好像永远都没有眼色。
