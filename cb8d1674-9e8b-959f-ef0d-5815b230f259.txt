　　终极神王城，是终极神王殿新建的山门，占地方圆三千里，城高雄伟。
<br/>　　终极大殿，乃是终极神王殿的主殿，此时此刻，主殿之中，终极神王和风雪神王、毁灭神王，端坐在大殿之中。
<br/>　　“两位，你们怎么看？”
<br/>　　主座之上，终极神王环视风雪神王和毁灭剑王，沉声问道。
<br/>　　现在，外界都在谈论，说秦陨是紫霄的弟子，终极剑道和终极神剑，被紫霄得到了。
<br/>　　“殿主，此事有些诡异，我已经派人查过了，秦陨当初的确是杀死了几名紫霄剑宫的弟子，此事也的确是被紫霄压下去了，并未公开！”毁灭剑王沉声说道。
<br/>　　“你认为，传言有可能是真的？”终极神王面无表情的道。
<br/>　　“不一定，但是，紫霄却是可疑，但现在紫霄和绫裳，都没有回来，也无法印证！”毁灭剑王摇了摇头。
<br/>　　“殿主，你派绫裳去天王宫找天尘宇，让紫霄暗中跟随，以他们的速度，应该早在一月之前，就能回来了，但现在居然迟迟未归！”风雪神王声音轻柔，眸中带着疑惑。
<br/>　　终极神王闻言，沉默不语，他是不太相信传言的。
<br/>　　毕竟，紫霄和绫裳一样，都是他教导出来的弟子，虽然没有弟子之名，但是算是半个师尊。
<br/>　　不过，此事的确处处透漏着可疑之处，首先，紫霄和秦陨，绝对是有关系。
<br/>　　不然的话，秦陨杀了几名紫霄剑宫的弟子，紫霄为何不惩处？
<br/>　　其次，紫霄和绫裳两人，出去了这么久，按理说早该从轮回神朝回来了。
<br/>　　“殿主，你准备怎么处理？”毁灭剑王问道。
<br/>　　“等吧！等紫霞回来！”终极神王道。
<br/>　　“若是……若是他不回来呢？”风雪神王美眸微眯着问道，因为，倘若传言是真的，那紫霄可以算的上是背叛了终极神王殿，可能就不会回来了。
<br/>　　这倒是很有可能，毕竟，紫霄是无上神王，实力非常强大，接近高级神王。
<br/>　　若是得到了终极剑道和终极神剑，虽然不能说无敌了，也有余高级神王一战之力了。
<br/>　　所以，倘若传言是真的，紫霄也不在乎终极神王殿一个剑宫之主的位置了。
<br/>　　“倘若他不回来，本座会清理门户！”终极神王面无表情的说道。
<br/>　　风雪神王和毁灭剑王闻言，顿时心中一凛，看来此事是碰到殿主的逆鳞了。
<br/>　　殿主如今的具体实力，他们两人也不清楚，因为，已经十几万年没有见过殿主出手了。
<br/>　　……
<br/>　　“可恶”
<br/>　　一处荒山之中，紫霄盘膝坐在一处山洞中，眸中闪过一丝怒火。
<br/>　　这座荒山，距离终极神王城不是很远，他一直在此静修。
<br/>　　上次，他威胁过绫裳之后，便一直在此，并未会终极神王城。
<br/>　　因为，终极神王让他去跟踪绫裳，他总不能自己先返回吧。
<br/>　　所以，他就在此地静修，一边静修，一边等待绫裳回来。
<br/>　　若是绫裳能将秦陨带回来，那就再好不过了，当然，绫裳估计也不知晓秦陨在哪，但没有带回来也无妨，毕竟，他有筹码在手，不担心找不到秦陨。
<br/>　　可是没想到，就在刚才，他庞大的神念扫视方圆两万里，居然发现不远处的城池之中，有很多人在谈论他。
<br/>　　而谈论的内容，几乎让他吐血。
<br/>　　秦陨是他的弟子？
<br/>　　他得到了终极剑道和终极神剑？
<br/>　　这简直就是一派胡言，明显是有人在混肴视线。
<br/>　　“秦陨？绫裳？还是其他人？”
<br/>　　紫霄的眼眸之中，闪过怒火，传出这个消息的人，最有可能是秦陨，其次便是绫裳，当然，两人合谋也有可能。
<br/>　　少倾，紫霄压下心中的怒气，心中沉吟了起来。
<br/>　　目前绫裳还未回来，他也不好回终极神王殿，还是再等等吧。
<br/>　　他也没有太过在意，这若是秦陨的手段，也未免太小儿科了！
<br/>　　“秦陨啊秦陨，希望你别不知死活！”紫霄冷笑一声，随即闭上了眼眸，再次开始静修。
<br/>　　……
<br/>　　天龙神朝，云龙城，苏莫修炼了一个多月之后，便悄然离开了府邸。
<br/>　　他离开了天龙神朝，进入了轮回神朝的境内。
<br/>　　来到距离人王宫不是很远的地方，他给清雅传讯，询问对方的情况。
<br/>　　很快，清雅便回讯，已经顺利的加入了人王宫，只不过，并未见到人德宫主，但是见到了玄天神王。
<br/>　　随后，苏莫再次传讯，将自己的计划，简单的告知了清雅，也交代给了清雅任务。
<br/>　　做完这一切，他便返回了天龙神朝云龙城，再次开始修炼。
<br/>　　大概的计划，他已经心中有数了，接下来，就看事情具体的走向了。
<br/>　　密室之中，苏莫取出大量的五行源石，一边吸纳源石，一边参悟吞噬法则。
<br/>　　悟道神丹的药效，可以撑二十年，药效还没有过去，万万不能浪费。
<br/>　　时间流逝，苏莫的修为缓慢的增长，法则之力也在缓缓的增强。
<br/>　　而此时此刻，轮回神朝天王宫，天尘宇再次来到了帝一魂闭关之地。
<br/>　　宽敞的密室之中，地面上依旧是亘古不变的漩涡图案，浓郁的星光布满了整个密室。
<br/>　　一声蟒袍的帝一魂，周身环绕着迷蒙般的光晕，缓缓流动。
<br/>　　“最近，又有苏莫的消息了？”一身白衣的天尘宇，伫立在密室的门前，望着光晕中盘坐的帝一魂背影，朗声说道。
<br/>　　“又有什么惊人之举？”帝一魂开口了，声音略显沙哑，平淡而悠远，没有任何的感情波动。
<br/>　　“的确是有惊人之举，他在万道剑冢得到了终极剑道，还取走了终极神王殿的终极神剑。”天尘宇道。
<br/>　　“果然是厉害，比拓跋青云要强！”帝一魂微微叹了口气。
<br/>　　虽然他没有去过万道剑冢，但是却知道拓跋青云当年去过，都没能得到终极剑道。
<br/>　　“现在这小子，又消失无踪了，他在和我们玩追迷藏！”天尘宇脸色淡漠，告诫道：“必须早日解决这小子，他若是成长起来，比拓跋青云还要难以对付。”
<br/>　　帝一魂闻言，陷入了沉默之中，少倾之后，问道：“具体有什么线索？”
<br/>　　“线索倒是有一些，但是，不知是真是假！”
<br/>　　天尘宇点了点头，道：“传言，苏莫和紫霄有关系，还传言苏莫取得的终极剑道和终极神剑，已经被紫霄获得，具体情况我已经在派人调查了。”
<br/>　　“无风不起浪，你再辛苦走一趟吧！”
<br/>　　帝一魂叹了口气道，言罢，他声音沙哑的继续道：“必要的话，我可以出关。”
