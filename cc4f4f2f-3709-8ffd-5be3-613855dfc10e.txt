　　半个月的时间，悄然而过。
<br/>　　这一日，天刚蒙蒙亮，密室的石门大开，苏莫大步的走了出来。
<br/>　　他面色有些沉重，眸中露出无奈之色，因为，他消耗了大量的丹药和源石，依旧还是没能突破到五品的修为。
<br/>　　今日，就是出发前去混沌神朝的日子，他也只能无奈的出关了。
<br/>　　走到殿外，苏莫望着泛亮的天空，长长的吐出了一口浊气。
<br/>　　唰！
<br/>　　就在此刻，人影一闪，一人出现在了苏莫的面前，正是司空炎。
<br/>　　“苏莫，你怎么会收了一个徒弟，这几年没把我烦死！”司空炎立刻开始抱怨了起来，满脸的不忿之色。
<br/>　　“怎么了？”苏莫闻言疑惑，不就是指点指点池儿吗？有这么大的怨气？
<br/>　　“你是不知道，那个小丫头整天缠着我，满脑子的古怪问题！“司空炎无语的说道。
<br/>　　“呵呵！”苏莫笑了笑，司空炎居然也有被人制住的时候。
<br/>　　“师叔，你说我什么坏话呢？”
<br/>　　就在此时，一声轻盈的声音响起，只见不远处的宫殿之中，池儿走了出来。
<br/>　　“呀，师尊，你居然出关了？”池儿看到苏莫，很是惊疑，立刻如同闪电般飞了过来。
<br/>　　“修为长进不错！”苏莫打量了池儿一眼，微微点头，对方已经达到了虚神第二变。
<br/>　　“嘻嘻，师叔给我弄了很多好东西！”池儿笑着说道。
<br/>　　“我担待不起师叔之名，你是我的师叔！”司空炎翻了个白眼道，这个小丫头，虽然年纪不大，但是，脑子想法奇多。
<br/>　　最近几年，不少龙族的子弟，都被这丫头坑了。
<br/>　　苏莫见此，无奈的摇了摇头，这两人倒是一对活宝。
<br/>　　“苏莫，你的修为有没有达到五品？”司空炎打量了苏莫一眼，不确定的问道。
<br/>　　他的修为才二品高级，所以，看不出苏莫的具体修为。
<br/>　　这几年，他也从龙腾那里，了解了千选战的情况，所以，对此也比较关心。
<br/>　　“没有！”苏莫叹了口气说道。
<br/>　　“你也不要太过担心，有龙腾在，应该不会有事！”司空炎拍了拍苏莫的肩膀道。
<br/>　　“嗯！”苏莫点了点头，不过，他还有机会，距离千选战还有半个月。
<br/>　　“走吧，龙腾让我带你去龙帝的府邸，在那里集合！”龙腾道。
<br/>　　“好！”苏莫点头，随即，一行三人，便向龙帝的府邸而去。
<br/>　　这一次千选战，事关重大，司空炎虽然不参加，但也会前去观看，而池儿自然不会独自留下来。
<br/>　　没过多久，三人便来到了龙帝的府邸，在一名龙族子弟的引领下，来到了府邸中的一处小广场之上。
<br/>　　此刻，由于才刚刚清晨，广场上空无一人。
<br/>　　“我们在此等待片刻吧！”司空炎道。
<br/>　　“龙腾呢？”苏莫疑惑的问道。
<br/>　　“龙腾这段时间，一直都在大长老的府邸中。”司空炎道。
<br/>　　“原来如此！”苏莫恍然。
<br/>　　“师尊，你这次出关了，等千选战以后，可要多多指点我。”
<br/>　　这时，池儿睁着大眼睛，凝视着苏莫，面带希冀的说道。
<br/>　　这几年，她在龙都城之中，算是将苏莫的一切，都了解的非常清楚了，心中更加渴望苏莫亲自指导她。
<br/>　　不过，苏莫一直在闭关，而且龙腾严词嘱咐不得打扰，她只能一直等待。
<br/>　　“好！”苏莫笑着说道，对方一口一个师尊，他不指点一番，也说不过去了。
<br/>　　“嘻嘻，一言为定！”池儿嬉笑，举了举小拳头。
<br/>　　“一言为定！”苏莫苦笑一声，随即便在广场上盘膝坐了下来，等待其他人到来。
<br/>　　随着时间的流逝，渐渐的有人开始来到广场，越聚越多。
<br/>　　“苏兄！”
<br/>　　“苏莫老弟！”
<br/>　　“几年不见，苏莫兄弟修为大涨啊！”
<br/>　　一些来到广场的龙族之人，纷纷向苏莫打招呼，即便是修为远高于苏莫的强者，也不例外。
<br/>　　“各位，苏莫有礼了！”苏莫起身，向诸位龙族之人见礼，他身在龙族，得龙族庇护，自然要和龙族之人保持良好的关系。
<br/>　　“苏莫，在下龙伊，我观你修为还在四品高级，怕是参加不了千选战了！”一位高高瘦瘦的龙族青年向苏莫微微抱拳，沉声说道。
<br/>　　这位龙伊，修为奇高，气息无比的浑厚，比之楚天墟都不遑多让。
<br/>　　千选战，一个大势力只有十个名额，而龙族的十个名额，有一个留给了苏莫，这是众所周知的事情。
<br/>　　关于名额的所属，龙族高层之前专门商议过，依龙腾之言，预留了一个给予苏莫。
<br/>　　若是苏莫无法达到五品的修为，那就给予另外一位龙族天骄。
<br/>　　“还有半个月的时间，我尽力而为吧！”苏莫叹了口气道。
<br/>　　“苏莫，千选战对于你而言，没有什么意义，我们会展现龙族的荣光。”龙伊笑着说道，他并不知道苏莫参加千选战的原因。
<br/>　　苏莫笑了笑，并未答话，他对于这些龙族天才，并不看好。
<br/>　　整个龙族之中，也就龙腾一人可堪大任而已。
<br/>　　龙伊见苏莫不答，也没有和苏莫多聊，便与其它人闲谈了起来。
<br/>　　“苏莫，这个龙伊不简单，九品高级的修为，天赋在龙族天骄之中，是顶尖的存在！”司空炎站在苏莫身边，向苏莫传音说道，他在龙族之中待的时间久，这两年也没怎么闭关，认识的龙族之人，远比苏莫要多的多。
<br/>　　“嗯！”苏莫点了点头，并未多问，在龙族天骄之中顶尖存在？连祖龙塔第八层都登不上，又能有多厉害。
<br/>　　要知道，登上祖龙塔第八层之人，在龙族只有五人，分别是龙帝，龙腾、龙裴霖、龙羽和龙双双。
<br/>　　所以，在龙族之中，年轻一代之中，除了龙双双、龙羽和龙腾之外，基本上都是庸才。
<br/>　　至于那个龙裴霖，苏莫没有见过，但此人是无上神王。
<br/>　　不大的广场之上，龙族之人越聚越多，很快便超过了百人，全部是年轻一代。
<br/>　　修为最高者是九品，最低者是一品，高低不等。
<br/>　　龙双双、龙城、龙烨、龙羽等人，全部都来到了广场。
<br/>　　千选战参加之人，只有十人，至于其他的人，全部是前去观战之人，而且全部都是天才人物。
<br/>　　不是天才，连前去观战的资格都没有，至于池儿，因为是苏莫的弟子，现在在龙族人尽皆知，所以自然有资格前去观看。
<br/>　　广场之上，人声鼎沸，热闹非凡，所有人都在谈论关于这次千选战的情况，猜测谁最有希望获胜。
<br/>　　“哎，苏莫没有突破到五品修为，真的是太可惜了！”
<br/>　　“是啊，以苏莫的天赋，要是参加这次千选战，毕竟如鱼得水，能为我龙族一大助力！”
<br/>　　“这也不一定，他虽然天赋无敌，但是就算达到了五品，修为也太低太低了！”
<br/>　　“千选战，修为高并没有太大的优势，一切还是看天赋和气运！”
<br/>　　“关键是，目前具体的规则还不明朗，在千选战开始之前才会公布。”
<br/>　　各种各样的议论之声，在广场上涌动，嘈杂不已，其中掺杂了大量的叹息之声。
<br/>　　苏莫没有突破到五品修为，众人自然都第一时间知晓了，不由得纷纷惋惜。
<br/>　　若是苏莫能参加，对于龙族而言，绝对是一大利器，但是，苏莫没有突破，那就无法参加了，这是龙族的损失。
