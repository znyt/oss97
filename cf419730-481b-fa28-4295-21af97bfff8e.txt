　　“吼！”
<br/>　　妖猴怒吼，不知从何处掏出一根金色棍棒，瞬间变大，他双手握棒，向着前面的九鼎狠狠地砸落下去。
<br/>　　“给老子碎！”
<br/>　　妖猴竭力一击，阻拦在他前方的那坚不可摧的九鼎，竟然有五个被那金色棍棒给生生击碎了。大鼎被击碎，碎片化为流光，飞回夏禹体内，这需要时间蕴养，才能恢复过来。
<br/>　　“混账！妖猴，还敢放肆！”
<br/>　　夏禹见此，大吼一声，龙行虎步，手中拳头紧握，瞬间在拳头上浮现一个大大的山河鼎，拳镇山河，向着妖猴砸落下去。
<br/>　　说时迟，那时快，妖猴见逃跑不行，瞬间改变策略，做出应对。面对向他击来的山河拳鼎，他手中金棒翻转而出，棒尖用力一顶，轰向那山河鼎。
<br/>　　“轰！”
<br/>　　大地震动，空间微微一震，金棒竟然与那拳印所化的山河鼎僵持住了，并没有一击击碎。
<br/>　　“给老子碎！”
<br/>　　妖猴大吼一声，手中将金棒用力一挺，法力输出，从金棒上传来一股巨大的威力，带着破灭一切的威势，传到山河鼎上。
<br/>　　夏禹感受到那股破灭巨力，瞬间做出反应，他放弃施拳，不知从何处取出一拳头大小的青铜元鼎，那元鼎顷刻间变得巨大无比，鼎的形状与之前的九鼎相似，只不过这次鼎身上并没有刻录神兽的图案，而是刻满了九州的山河地形图，一个鼎，就仿佛包含了整个九州，囊括了宇宙。
<br/>　　这鼎好似是青铜鼎，其实不然，这并非是那普通的青铜，而是传说中的顶级仙材镇世青铜所铸之鼎，鼎名：九州鼎！
<br/>　　镇世青铜九州鼎！
<br/>　　“当——”
<br/>　　那带着破灭气息的金棒轰击在九州鼎上，只是发出一道清脆悠扬的响声，鼎身震动了片刻便停止了，没有倒退半步。
<br/>　　“杀！”
<br/>　　夏禹目露神光，一手抓住九州鼎的一脚，狠狠地向着妖猴砸去。巨大的力道，带着一股无可匹敌的气势，冲击着妖猴的心神。
<br/>　　不可力敌？妖猴心中一动，将手中金棒顺势收回，迅速横挡在胸前，以此来对抗那巨大的九州鼎。
<br/>　　“轰！”
<br/>　　无匹的力量，骇人的气势，坚不可摧的九州鼎，狠狠地砸落在金棒之上，金棒微微弯曲，但瞬间便反弹回去，借助这股反弹之力，妖猴顺势弹了出去，直直撞到数千丈之远的大山之上，才狠狠地砸落下去。
<br/>　　妖猴见不可力敌，便想要逃走，只是夏禹怎么可能将这个祸害天下的幕后黑手放走，他瞬间追击而上，妖猴无奈，只得抓起手中金棒迎了上去。
<br/>　　妖猴一个跳跃，跃起数万丈之高，手中金棒同时也迅速变大，狠狠向着夏禹砸去，同时，他双眼暴瞪，双眼之中，金光闪闪，两道巨大的金色光柱从他眼中射出，带着毁灭的气息，轰击向夏禹。
<br/>　　“哼，妖猴，哪里逃？”
<br/>　　夏禹冷哼一声，他眼神冷厉，看到妖猴那双目之中的金光向他射来，他亦是心中一惊，因为他能够感受到那金光之中带着浓浓的毁灭气息，若是不加以抵挡，恐怕能够轻易击穿一个大罗金仙。
<br/>　　夏禹手持九州青鼎，将鼎口对准两道金光，护持在胸前，同时脚步不停，向着妖猴砸落下来的金棒迎了上去。
<br/>　　九州鼎中仿佛是一个无底洞，不管来什么，它都能将之吞没，金光进入鼎口，瞬间便没了声息，鼎内依旧黑漆漆的一片，什么都望不见。同时，九州鼎带着无上威势，对轰上那变的巨大的金棒。
<br/>　　“当——”
<br/>　　两者相交，谁也没能奈何得了谁。这时，妖猴的身形瞬间涨大，高达数万丈，变得如同一句巨山一样高大，同时手中金棒，也随着变形，变得如同一根天柱，向着大禹镇压下来。
<br/>　　夏禹看着那高大的身影，也瞬间施展出法天象地的神通，变得同样高大，他的九州鼎，也仿佛知道主人的心意，变得如同一座高山，静静地悬浮于夏禹头顶，九州青鼎大口朝下，三脚朝天，缓缓旋转，散发出丝丝青色玄光，镇压下来，任凭妖猴如何攻击，都不能鸡婆其防御，有点像天地玄黄玲珑宝塔的万法不侵。
<br/>　　于是，就这样，一人一妖边战边走，随时转换着战场，两人交战的地方，都变成了废墟。如此交战，只是一会儿的功夫，两人便战到了淮河。
<br/>　　站立于淮河之上，妖猴似乎是找到了自己的场地一般，随着手中金棒挥舞，那淮河之水就如同是他的臂膀一般，任由他驱使。
<br/>　　他舞动手中金棒，卷起滔天水浪，滔滔大河之水在这妖猴的神通下，如臂指使，对着夏禹无情地轰击而去。
<br/>　　“嗖！”
<br/>　　夏禹见此，眉头一皱，张口便吐出一口黑色小鼎，对着那无情水浪轰击而去。小鼎见风而涨，瞬间涨得巨大，正是之前的九鼎之一的水鼎。
<br/>　　黑色水鼎倾斜，将鼎口对着水浪，任由那无尽水浪冲击而来，它都无情地吞噬一空。
<br/>　　“杀！”
<br/>　　夏禹头顶九州青鼎，挥动手中拳头，继续向前杀去，妖猴亦操控水势，向着夏禹击去，一人一妖，便如此大战起来。
<br/>　　眨眼间，两人便交战数千回合，谁也没能奈何得了谁。夏禹借着九州鼎的威势，保持自己不败，而妖猴借着操控水的神通，虽然略显下风，但一时间也不会败下阵。
<br/>　　而这时，夏禹的老师玉玲珑与他的大臣们已经到了。
<br/>　　“就是这妖猴在兴风作怪，掀起无尽水浪，造成洪水爆发吗？”
<br/>　　“真是该死，原来是妖族在背后作怪，真当我人族好欺负吗？”
<br/>　　“这妖猴好生厉害，竟然能与大王交战如此之久，而且他那御水神通也好精妙，简直近乎入道，若论控水，恐怕就是应龙也比不上，也只怕是上古时期的共工才胜得过他。”
<br/>　　“这妖猴到底是何来历，背后又有什么人在支持他？”
<br/>　　“妖猴手中那金棒似乎是如意精金吧？”
<br/>　　“不错，的确是如意精金，如意精金是顶级仙材，可自成法宝，而且能够随着人的心意随意改变法宝形态及大小。”
<br/>　　“……”
<br/>　　群臣见此情况，纷纷大惊，顿时开始议论纷纷，这妖猴实在厉害，恐怕就是一般的准圣都不能拿下他，而且仗着这御水神通，在大江大海之中，便可堪比数尊大罗金仙，完全不怕群战啊。
<br/>　　“应龙先生可知此妖是何来历？”
<br/>　　一大臣见到这么厉害的御水神通，便向同样是控水行家的应龙问道。
<br/>　　这应龙正是轩辕时期的大将，大战之后便一直住在人族之中，相当于人族的长老之类的，有重要的事情便会请他帮忙。不过，大多时候都请不到他，因为人族也不可能随时遇到解决不了的问题。
<br/>　　应龙此刻并没有回答他的问题，而是双眼一眯，静静地看着水中大战的两人，良久，他才开口赞叹道：“好厉害的御水之道，龙族在这方面恐怕都不如他。”
<br/>　　接着又说道：“我看他的神通之中隐隐有龙族神通的影子，这妖猴的来历，恐怕是与龙族有关。”
<br/>　　“什么，龙族？！”
