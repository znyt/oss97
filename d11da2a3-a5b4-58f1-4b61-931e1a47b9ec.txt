个是一个。这个世兴文虽然只是一个病人，但是谁没有个朋友不是，所以还是表现的大方些好。<br/>　　<br/>　　等陈默走进去后，世兴文就转头看见了他。<br/>　　<br/>　　“你怎么有过来了？不是昨天给你说了么，我拒绝你的治疗。”世兴文有些皱着眉头对陈默说道，此时李普河和常文斌，还有两个救护车上的护理人员也走了进来。<br/>　　<br/>　　李普河想说什么来着，但是却被陈默阻止了，然后上前对世兴文说道：“昨天可能是我不对。作为一名医生，首先应该以治病救人为主，而不是意气用事。你的病我想看看，然后能治疗我会用心治疗，如果不能治疗你在转院！”<br/>　　<br/>　　“呵呵！不用你费心。再说我也不相信你的医术，你这么年轻，有个什么技术可言！”世兴文有些反感的说道。<br/>　　<br/>　　“我虽然年轻，但是也是特管局的人请来给你们看病的，那么请我来的领导难道说不把你们当回事么？”陈默有些皱眉，这个家伙就是个茅坑里的石头，这是又臭又硬的！<br/>　　<br/>　　“这也就是我想说的，正所谓久病床前无孝子。我们也是一样，时间长了就不会再将我们当回事！”世兴文有些低沉的说道。<br/>　　<br/>　　“我感觉你有些嫉世愤俗了，对于特管局有些以小人之心度君子之腹！”<br/>　　<br/>　　“呵呵！我不想和你说话！”世兴文说道。<br/>　　<br/>　　“唉!世兴文，作为一名医生，我想最后一次询问你一下，因为今天如果你还是拒绝接受我的治疗，那么就要转回到特管局去。所以这是最后的询问，你的意愿是什么？”<br/>　　<br/>　　陈默内心对世兴文早就放弃了，不过场面上还是要做的，尤其是在特管局的人员在现场的情况下，自然要做最后的争取。<br/>　　<br/>　　其实陈默也并不都是为了表现或者取得一个好的印象。对于世兴文他还是从心里想争取一下的，毕竟他也是为国受伤的，那么为这样的人，即使打脸一回也是没有关系的。<br/>　　<br/>　　“不接受！我不接受！听到没有？不要在我耳边在唧唧歪歪好不好，你他么的很烦知不知道？”世兴文突然的爆发，也让现场的人都有种莫名的同情。既有对他的同情，也有对陈默的。<br/>　　<br/>　　如此好的医生，如此的心胸，既然受到这样的待遇，真是活该躺病床。而且对于陈默昨天的治疗成果，李普河他们也没有义务给世兴文解释，心中都是为陈默感到暗暗不愤！<br/>　　<br/>　　“唉！”陈默叹了一口气，转身对李普河他们挥挥手，然后缓缓走了出去。心中没有所谓的得意或者什么，对于这样的人，真的有些无奈。自己都已经打自己的脸，说过的话已经不算数，今天腆着脸又来询问，结果和昨天一样，心中真的有些哭笑。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
