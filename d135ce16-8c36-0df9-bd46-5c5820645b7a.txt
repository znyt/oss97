　　第530章
<br/>　　安顿好程大雷与刘发财后，黄为善被引到后宅，见到了自己的老母。
<br/>　　他母亲病重，如此才着急从青州赶回来。老母病卧在床上，形销骨瘦，看上去已经命不久矣。黄为善跪在地上，也不敢太过悲伤，无非说些安慰的话。
<br/>　　“好啦，好啦，你母亲身体不好，你就不要打扰她歇息，你出来一下，我有话与你说。”黄老汉走进来道。
<br/>　　黄为善擦掉眼泪，随着老父走出房间，来到外面的堂屋。
<br/>　　黄老汉摆摆手，道：“坐，坐下谈。”
<br/>　　“父亲……”
<br/>　　“随你回来的两个，他们的来历你清楚么？”黄老汉问。
<br/>　　“他们一个姓程，一个姓刘，孩儿幸亏在路上他们遇上他们，不然就见不到父亲了。”
<br/>　　“我是问他们的来历，不是问他们救了你，你聊这些做什么。”黄老汉用手指敲击着桌面：“为何看上去，他们两个都有些匪气？”
<br/>　　“的确都是江湖上的豪客，他们的来历么……”黄为善皱起眉头：“那个姓刘的看上去没多少本事，那个姓程的……”
<br/>　　黄为善顿了顿，一字一顿道：“来历绝不简单。”
<br/>　　如今帝国，没听过程大雷名字的人怕是不多，即便住在荒山僻野的人，也知道有个杀人如麻的程大雷。而兖州距离幽州并州都谈不上远，在此地程大雷的名声，早已经糟糕到极点。就算谈不上小儿止啼，其实也差不了太多。
<br/>　　到现在，大家称呼程大雷，一般不敢提他的名字，往往用『姓程的』『那个人』代替。今天黄为善父子之间的谈话也是一样。
<br/>　　虽然黄为善没有提那三个字，但黄老汉还是知道了儿子口中说得是谁。意识到这一点后，黄老汉脸上表情明显不一样了。
<br/>　　毕竟，明知道家里住着一个杀人魔头，谁也不可能还保持心平气和。他在房间中来回踱步，脸上由白转青，由青转红，几次变幻。
<br/>　　“呀，我的儿，这是我们黄家天大的造化啊！”黄老汉突然将手压在桌子上。
<br/>　　黄为善一头雾水：“什么造化？”
<br/>　　“孩儿你有所不知，现今兖州王在城里建立一座招贤馆，招募天下英雄。为父一直想要替你谋一个出路，可无论使多少银子，都攀不上兖州王的高枝。而今，那个人是帝国通缉的要犯，将他献给兖州王，定能为你换一份泼天前程。”
<br/>　　“这……”黄为善一惊：“可他们是孩儿的救命恩人。”
<br/>　　“我的儿你现在还是太过年轻，不知道为父的难处。我们黄家虽然算这方圆百里的首富，可有钱无权又能做什么。当兵的来了我们要与他们粮，山贼强人来了也要与他们粮。说来说去，我这黄善人也不过人人可欺。你若能攀上兖州王，以后便只有我们欺负人，而没人再敢欺负我们。”
<br/>　　黄为善脸色变幻，一时间拿不定注意。
<br/>　　“为善，我的儿，要知道量小非君子，无毒不丈夫。你的余生难道想为父一样，看各种人的脸色干活。”
<br/>　　“孩儿做了。”黄为善将牙一咬，道：“他们现在睡在厢房，孩儿带着家丁过去，将他们杀了，人头献给兖州王。”
<br/>　　“不，不妥。”黄老汉摇摇头：“如果真是那个人，黄家的家丁怕充不得什么用。这样，你连夜骑快马去兖州城，请得兖州王大军，我在紧挨着安排好宴席，先稳住他们。等你将大军引到了，再将他们一举擒获。”
<br/>　　“好，孩儿这就去办。”
<br/>　　“你这一路辛苦，回到家中还没有好好休息，又要连夜赶到兖州城。孩儿，委屈你了，但这件事关乎我黄家日后的前程，你要拼一拼，撑一撑，只要撑过这一遭，日后我黄家就谁也不用怕了。”
<br/>　　天还没亮的时候，黄为善拉着一批快马离了黄宅，等出了镇子，立刻骑上马，向兖州城狂奔。
<br/>　　黄老汉说得不错，他这一路风餐露宿，早已经是疲惫不堪，又要连夜赶路，奔袭上百里去请大兵，当然，也正如父亲说的，自己要拼一拼，撑一撑。
<br/>　　等过了今天，黄家便是一片光明。
<br/>　　东方，天快亮了，光明，就在眼前。
<br/>　　……
<br/>　　程大雷和刘发财一直睡到当天下午，当然，程大雷一直没有睡踏实，一方面是经年养成的警惕，另外一方面……是刘发财打呼噜的声音太大了。
<br/>　　等二人醒来，各自打着哈欠看着彼此。程大雷丝毫不怀疑，自己的目光能将刘发财杀死。
<br/>　　“我说兄弟呀，你太能打呼噜了，我这一天没睡好啊！”刘发财。
<br/>　　“呃……”程大雷下意识默默鼻子：我睡觉也打呼噜么。
<br/>　　这时候门开了，两个水灵的丫头推门进来，手中端着清水毛巾。
<br/>　　“二位爷，你们醒了，我们老爷吩咐，让我们伺候二位爷洗漱后去大厅，老爷已经备好了宴席。”
<br/>　　程大雷在蛤蟆城就有人服侍起居，对这些早已经习惯如常，其他大概就是嫌丫鬟伺候的不够周到。
<br/>　　当然，黄家不过是兖州此地的一户土财主，也不能对他们要求太多。程大雷觉得，自己在外面多受些委屈也是应该的。
<br/>　　而相比程大雷，刘发财就处处透着不自在，双手都不该往哪里放。
<br/>　　等二人被丫鬟引着来到黄家大厅，见大厅上已摆满酒菜。黄老汉并黄家几个长辈一起在厅外迎着二人。
<br/>　　“二位好汉，不知休息得怎样，手下有没有招待不周的地方。”黄老汉呵呵笑着：“二位好汉救了小儿的性命，老汉聊备薄酒，感谢二位好汉的救命之恩。”
<br/>　　刘发财顿时抖了起来，他早已习惯这样的场面，只要报出『程大雷』的名字，总是能换几顿好饭的。
<br/>　　一桌人杯来盏去，酒宴进行到半酣。黄老汉一直陪着笑，陪着二人饮酒。但渐渐，黄老汉觉得有些事情不对了。
<br/>　　据自己儿子说，二人中，姓程的为主，可观他气度，却没什么耐人咀嚼的城府。反倒是那个闷头吃饭的……就是那个无足轻重的『刘发财』，看着有些不简单。
<br/>(本章完)
