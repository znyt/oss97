　　拜占庭人虽然并不觉得唐军有什么可怕，但也没有大意到毫不理会的程度。
<br/>　　所以拜占庭人的先头部队驻扎下来之后，第一件事就是通知明尼城主他们到来的消而，而第二件便是撒开人马去探查唐军的动静。
<br/>　　便是这样，方圆三十余里的明尼城周围成了双方探子的杀戮场，每天都会有人死去，同时每天也都会有小规模的战斗发生。
<br/>　　伊力克作为拜占庭人先头部队的最高长官，眼看着自己的部下每天都要死上数十上百人，心情自然不会太好。毕竟在他的手上只有两万人，而对面的唐军则有二十万，不管是总人数还是探子的人数，比较起来他都显的十分吃亏。
<br/>　　可就算这样，他却依旧不敢入城，两万人进城总是要时间的，如果这个时候唐军绕过明尼城发起进攻，城池会不会丢失他不敢确定，但他的手下死上一半却是一定的。
<br/>　　而就在他忧心忡忡盘算着是不是要向后面的援兵求援时，一份情报从明尼城中送了出来：今夜，本城主欲亲率城中大军偷大唐粮草，若事有不皆，望将军能助一臂之力。
<br/>　　简简单单几十个字，表现出了明尼城主的决心与魄力，看的伊力克愧疚不已。
<br/>　　但是上阵打仗这种事情终究是死道友不死贫道，拜占庭的国家体制决定，每个城主都有自己的私兵，而每个将军也同样有自己的私兵。这样的情况下，如果不是国王的命令，自然没人喜欢拿自己的家底去探敌人的虚实。
<br/>　　所以明尼城主想去就让他去好了，正好让他去探探路，反正战争一打起来明尼城也是首当其冲，就算是拜占庭最终取得胜利，他的城池基本上也废了，不如让他物尽其用。
<br/>　　想到这里，伊力克命令他的手下全部打点行装，作好行动准备，如果到时候明尼城主不敌战败自己也可以跑的快一些，而如果明尼城主成功了……抢功劳同样也可以快一些。
<br/>　　夜色下，明尼城南面城门在悄无声息中缓缓打开，数十骑蹄子上包着厚厚麻布，口中勒着嚼头的战马在骑士的牵引下从里面缓缓出来，慢慢消失于夜色之中。
<br/>　　片刻之后，顺着打开的城门，又开始陆陆续续的有战马出来，无一例外的全都是在马蹄上包着厚厚的麻布，口中勒着嚼头，力求不让战马发出一丝的声音。
<br/>　　不过这一次出来的人数显然有些多，整整一刻钟过去，竟然还不断有人从里面出来，直到近小半个时辰之后，那阴森恐怖的城门里面才没有了一丝动静，城外的吊桥也开始缓缓收了上去。
<br/>　　五千骑兵已经全部出城，不过为了不引起唐军的警觉，这些从城里出来的骑士并没有马上上马，而是就那么牵着马走着，一点点向唐军主营的方向靠过去。
<br/>　　时间缓缓流逝，不知不觉中已经过了午夜，原本有些喧哗的唐军营地开始陷入沉睡，远远望去，只有哨位上还有唐军士兵在那里梭巡。
<br/>　　看着已经安静下去的唐军营地，‘明尼城主’对身边的亲兵吩咐道：“通知下面的人准备，时间差不多了，让他们检查好身上的火种，一会儿千万不要出任何差错。记住，最要紧的就是不能恋战，我们的目的是要烧掉那些粮草，而不是与唐军决战。”
<br/>　　此时他脸上正带着一副青面獠牙的面具，他身边的那些骑兵也是一样的打扮，在漆黑的夜里乍一看就好像无数来自幽冥的恶鬼，显然是为了这次‘偷袭’做了充分的准备。
<br/>　　那些亲卫似早就已经在等着‘明尼城主’的命令，闻言立刻转身而去，片刻之后又转了回来，沉声说道：“城主，已经安排好了，勇士们已经做好了准备，随时可以进攻。”
<br/>　　“好！”‘明尼城主’重重一点头，回头身了一眼身后，把手一挥：“出发！”
<br/>　　‘哗啦……’一连串衣甲碰撞的声音，五千骑兵全部翻身上了马背，接着在一声唿哨声中悍然向数百步之外的唐军营地冲了上去。
<br/>　　战马的马蹄虽然已经被包起来，但两万只马蹄敲击地面引起的阵动还是让唐军哨位上的哨兵警惕了起来，将目光投向远处的黑暗。
<br/>　　‘嗖’的一声，一枝不知从哪里射来的箭矢突然间‘穿’过那士兵的咽喉，随后便见他无声无息的倒了下去。
<br/>　　接着又是数支不知从何处射出的长箭，依次‘射穿’了其他几个哨兵的咽喉，而随着几个哨兵消无声息的倒下，疾驰而来的战马亦冲到了营门口。
<br/>　　‘嗖嗖嗖……’长箭如雨，骑在马上的那些带着面具的骑兵纷纷拉开手中长弓，将手中带着火的长箭射向四周的营帐，随后马不停蹄的入唐军营地的深处冲去。
<br/>　　足足有二十余个呼吸的时间，一声凄厉的示警声从唐军大营的一个角落传出，接着在漫开大火中，整个唐军大营乱了起来。
<br/>　　五千骑兵对阵二十万唐军，看似数量相差悬殊，但是不要忘了，他们并不是想要与唐军决一死战，这些人的目的其实就是放火，只要他们能够冲到唐军堆放粮草的位置，就可以算是大功告成。
<br/>　　所以这些‘杀’进唐军大营的‘明尼城守军’根本不做任何停留，制造出大量混乱之后，便向着唐军营地的后方冲去，不到一刻钟时间已经将整个营地凿穿，那堆集如山的‘粮草’已经出现在他们面前。
<br/>　　“烧光，快快快……”带着面具的‘明尼城主’在数十护卫的保护下，大声的吼着，催促着手下。
<br/>　　而那些好不容易穿过唐军大营的‘明尼城守军’自然也不敢怠慢，赶在身后那些唐军追上来之前，不顾一切的将身上携带的一切引火之物全都丢了出去。
<br/>　　很快，火便烧了起来，远远看着好像天空都被烧红了一般。
<br/>　　而‘明尼城守军’则借着这个机会逃向远方，留下身后无数唐军大呼小叫的围着冲天大火骂个不休。
