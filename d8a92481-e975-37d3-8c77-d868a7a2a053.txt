　　但，虽然实力比普通的SS级喰种要强一些，也不过只是SS级罢了，对于刘哔来说，这样的实力对他造不成什么威胁。
<br/>　　“修罗，危险程度SSS，疑似击杀有马贵将，不知修罗先生，来我们这里，真的只是为了来找人吗？”
<br/>　　轻轻抿了一口咖啡，诗摘下面具，露出了一张有些妖异的脸庞。
<br/>　　这是喰种之间解除敌意的方式，在私下交谈的时候，没有喰种会戴着面具，这是最起码的尊重。
<br/>　　一直戴着面具的话，是一种防备的表现吧，虽然诗对修罗也不怎么信任，但表面功夫还是要做足的。
<br/>　　看到诗的动作，刘哔也摘下了自己从月山家顺出来的面具，却伸手制止了什造的动作。
<br/>　　人类和喰种不同，两者之间，也没有什么规矩存在。
<br/>　　更重要的是，刘哔不想暴露什造，这太过危险，反正只要他表明姿态就足够了。
<br/>　　“啊，来找人，顺便也来看看，你们小丑是怎么处理危机的。”刘哔翘起二郎腿，漫不经心的开口说道。
<br/>　　先是月山家被灭，又是和修宗太死亡，作为现残存最大的自由喰种组织，他可不相信小丑一点危险的气息都嗅不出来。
<br/>　　如果反应没那么迟钝的话，估计现在早就已经做好准备了吧。
<br/>　　“这样啊……那你恐怕要失望了，我们并没做什么应对方案。”顿了一下，诗无奈的说道：“敌人太强大，如果一心想要对我们动手，那我们不会有任何活下去的希望。”
<br/>　　脸上的笑意更深，刘哔眼中透出一抹玩味：“那么，你们应该怎么办呢？”
<br/>　　“看着办呗，除此之外，还有更好的办法吗？”耸了耸肩，诗的语气略有些颓废。
<br/>　　皱了皱眉头，刘哔大脑飞速运转起来，他可不相信，小丑就会这么坐以待毙。
<br/>　　蝼蚁尚且苟活，更何况是这么一个庞大的喰种机构呢。
<br/>　　绝对是有什么算计，但是信息太少，他一时之间，也猜想不到。
<br/>　　“说起来，修罗先生，听闻你在美林大厦的时候，杀掉了和修宗太大人？”喝了口咖啡，呗的话锋一转，突然提起了宗太的事情。
<br/>　　那是他们的首领，于情于理，作为核心成员的他也应该过问一下。
<br/>　　“恩，杀了，怎么，那家伙和你们小丑有关系？”装出一副惊奇的样子，刘哔并没有否认击杀和修宗太的事实。
<br/>　　这件事情，随便打听就能打听的到，泄露出去也纯属正常。
<br/>　　“哦，他是我们的首领，也是和修一族派来控制我们的监管人，说起来，和修一族对我们动手，也和修罗先生有很大的关系呢。”眼中闪过一丝惊讶，诗开口解释道。
<br/>　　虽然已经提前知道了这则消息，但真的得到了确认，依旧还是有些不可思议。
<br/>　　作为核心成员，他自然知道和修宗太的强大，或者说，和修一族出来的喰种，没有一个简单的。
<br/>　　“哦？那你们的意思是……”态度一转，刘哔的语气有些危险，虽然不太可能，但他还是做好了战斗的准备。
<br/>　　反正来这里的另一个目的，也是为了参与战争的，虽说时间现在早了点，但也没有太大影响。
<br/>　　“修罗先生果然还是很危险呢，我们可没打算和你起什么冲突，既然你能杀掉首领，那自然也有能力灭掉我们。”听到刘哔的话语，诗的表情一凝，摊了摊手：“这样的危险人物，我们可得罪不起。”
<br/>　　看了眼时间，刘哔微微有些烦躁，这个家伙，一直和他绕来绕去，他感觉自己的耐心已经快要被磨没了。
<br/>　　毕竟，改变了这么多剧情，他也不清楚CCG什么时候到。
<br/>　　他可不太喜欢计划之外的任何变故。
<br/>　　“直接说吧，有什么事情，如果只是对我不放心的话，那你可以离开了。”刘哔放下手腕，语气有些不耐：“我来只是为了寻找一个人，仅此而已。”
<br/>　　相对于刘哔的不耐，诗反倒是一脸轻松的样子，即使知道对方已经很不耐烦了，可他心中依旧没有丝毫在意。
<br/>　　“BigMadam是吧，我们已经派人去请了，很快就会将他带到包厢里来。”顿了一下，诗的语气充满自信：“放心，一个面前算是SS级的喰种而已，我们拍卖行还是能处理好的。”
<br/>　　“啪！”
<br/>　　一巴掌拍碎面前的大理石茶几，刘哔脸上带着怒意，咬牙切齿的质问道：“我什么时候，让你们去抓人了！”
<br/>　　似乎是太过于愤怒，刘哔的声音充满暴戾，让人遍体发寒。
<br/>　　虽然，他也不喜欢那个喰种，甚至于讨厌，但就算如此，BigMadam也应该交给什造来审判。
<br/>　　哪怕是BigMadam曾经将什造当做赚钱的工具，甚至还给了什造无尽的折磨，但说起来，没有BigMadam，什造都还不一定是什么样子。
<br/>　　养育之恩是真的啊，什造对他的感情也是真啊。
<br/>　　几乎每天晚上，他都能听见什造口中低喃着这个名字，以及那份深深的依恋。
<br/>　　“妈妈，很疼……”
<br/>　　“妈妈，今天，什造是个乖孩子吗……”
<br/>　　“妈妈，什造不痛，真的不痛，但就是不知道，为什么会哭……”
<br/>　　每每听到这些话语，刘哔的心情都会十分压抑，早就没有任何感觉的心脏，都有些异样的难受。
<br/>　　他能理解，，因为他们，是同一种人。
<br/>　　但他不能接受，他不想让什造经历自己所经历的老路。
<br/>　　所以，他才决定带着什造出来，就是为了寻找到BigMadam打开她的心结。
<br/>　　甚至为了避免什造胡思乱想，他都没敢说一些过分，或者不敬的话。
<br/>　　现在这种情况，哪怕一点带有侮辱性的词汇，都有可能让什造的心结无法顺利解开。
<br/>　　而这些不知所谓的家伙，居然，就这么大大咧咧的去绑人了。
<br/>　　真是该死。
<br/>　　如果一会BigMadam进来的时候状态有什么不对，那他一定会将这些人的头给锤烂。
<br/>　　“咚咚咚。”
<br/>　　不等两人继续有什么动作，门外传来了一阵敲门声，刘哔强压住自己的怒气，将视线换到了房门处。
<br/>　　紧接着，包厢门“吱呀”一声缓缓打开了。
