　　苏莫离开了宫殿之后，便走下了普阳山。
<br/>　　山脚下，阴蒙执事正在等着他。
<br/>　　“这么快！”阴蒙执事见苏莫下山，不由的有些意外，因为苏莫进入普阳神长老的宫殿，前前后后也就半柱香的功夫。
<br/>　　这时间也太短了，指点一次居然这么快！
<br/>　　“阴蒙执事！”苏莫走到对方的身前，微微抱拳。
<br/>　　“苏莫，我送你离开吧！”阴蒙执事点了点头说道。
<br/>　　苏莫闻言，略一沉吟，问道：“阴蒙执事，我能否在内族中逛逛？”
<br/>　　他是想去寻找青璇，既然来了内族，当然不能就这么离开！”
<br/>　　“不能，你必须离开！”阴蒙长老摇头，不是内族之人，若是没有族中长老的允许，是严禁在内族中滞留的。
<br/>　　“好吧！”苏莫无奈，既然对方不让他多留，他也不能强求。
<br/>　　随后，阴蒙执事便带着苏莫原路返回，没过多久便离开了内族，回到了太阴城。
<br/>　　回到太阴城之后，苏莫也没有久留，告别阴蒙执事，便离开了太阴城。
<br/>　　天阴山脉之外，苏莫身形凌空而立，回头望了一眼浩大的太阴城，心中暗叹一声。
<br/>　　这一次来太阴族，是见到了青璇，但却没有单独接触对方的机会。
<br/>　　不过，以后会有机会的，他还会再来。
<br/>　　少倾，苏莫身形划破长空，向道宗的方向而去。
<br/>　　……
<br/>　　数日之后，苏莫回到了问道山，身形落在了山门之前。
<br/>　　随着苏莫回到道宗，道宗几乎沸腾了起来。
<br/>　　“看，苏莫回来了！”
<br/>　　“他就是苏莫？”
<br/>　　“真是厉害啊，这次会武横扫其它六大势力的妖孽，正是出尽了风头！”
<br/>　　“是啊，算是让我们道宗颜面大涨！”
<br/>　　“这个家伙也没有什么不凡啊？真的这么厉害！”
<br/>　　苏莫大步的走在山道上，路过的弟子大多对他指指点点，各种议论之声不绝于耳。
<br/>　　这一段时间，苏莫之名在道宗可谓是震天响，参加会武的道宗弟子回来之后，自然便将会武的情况传开了。
<br/>　　苏莫之名也几乎是无人不知，无人不晓，夺得会武的第一名，横扫六大势力的妖孽天才，这份战绩可谓是轰动了整个道宗。
<br/>　　最主要的是，苏莫只有武帝境八重的修为，这更是让所有人震惊。
<br/>　　苏莫没有理会任何人，独自的回到了自己的院落。
<br/>　　坐在院落中，苏莫低头沉思，他现在又是一穷二白了，灵晶消耗完了，连身上的宝器等战利品，也在九重浮屠塔中消耗的差不多了。
<br/>　　想要突破到武帝境九重的境界，还需要寻找资源。
<br/>　　不过，真枯半圣说是等他回来之后，宗内会给予他奖励，不知道奖励是什么？
<br/>　　就在苏莫思索之色，院门被人敲响了，冯光长老来了。
<br/>　　苏莫打开了院门，冯光便一阵噼里啪啦的开口了。
<br/>　　“苏莫，你终于回来了，我跟你说，宗门为了你的奖励，还专门召开了一次高层探讨，现在奖励都已经定下来了，我这就带你去！”冯光显得很是兴奋，仿佛得到奖励的人是他自己一般。
<br/>　　“哦？都有什么奖励？”苏莫笑着问道，他现在最缺资源，若是能有资源奖励，那就再好不过了。
<br/>　　“嘿嘿，等会你就知晓了，我带你去见真枯长老！”冯光嘿嘿一笑，却是卖了一个关子。
<br/>　　“那我们走吧！”苏莫说道，他也有些迫不及待了，想要看看宗门会给予什么奖励！
<br/>　　随后，冯光便带着苏莫，向真枯半圣的居所而去。
<br/>　　真枯长老的居所，在问道山的上方部位，独立的宫殿占据了一片很大的地域，周围还种满了各种灵药、灵草。
<br/>　　各种灵草灵药围拢着宫殿，形成了一个优雅的院落。
<br/>　　真枯长老作为半圣，地位在道宗之内仅次于六大核心圣长老，所以居所在山上占据了很大一片地域。
<br/>　　当冯光和苏莫到来之时，真枯半圣正盘坐在院落中，闭目打坐。
<br/>　　众多的灵草灵药，散发着强烈的灵气，纷纷向真枯半圣的身上汇聚。
<br/>　　苏莫暗暗惊奇，别人都是直接吞服灵药，这真枯半圣居然只吸收灵草灵药散发的灵气。
<br/>　　“苏莫，你终于回来了！”
<br/>　　苏莫刚刚来到院落之前，真枯半圣便睁开了眼眸，面上露出淡淡的笑容。
<br/>　　“见过真枯长老！”苏莫向对方抱拳，冯光亦是躬身行礼。
<br/>　　“来，进来坐吧！”
<br/>　　真枯半圣招了招手，随即袖袍一挥，身前便出现了两张蒲团。
<br/>　　苏莫和冯光走进了院落中，在真枯半圣的身前盘坐了下来。
<br/>　　“苏莫，这次会武，你为道宗争光，功不可没！”真枯半圣微笑着说道。
<br/>　　“真枯长老过奖了，身为道宗弟子，这是理所应当的！”苏莫沉声说道。
<br/>　　“嗯，你能有如此心态，实在是再好不过了！”
<br/>　　真枯半圣微微颌首，道：“宗门要给予你的奖励，也已经安排了下来，希望你会满意！”
<br/>　　“真枯长老，不知是何种奖励？”苏莫好奇的问道。
<br/>　　“第一种奖励，是灵晶，宗门特赐予你一千中品灵晶！”真枯半圣微笑着说道。
<br/>　　“一千中品灵晶！”苏莫闻言心中一震，顿时面露喜色，居然是中品灵晶，他还从来没有见过中品灵晶。
<br/>　　中品灵晶的价值，自然不是下品灵晶可比，一块中品灵晶，便相当于一千下品灵晶，一千中品灵晶就是百万下品灵晶，这个奖励不可谓不大。
<br/>　　“第二个奖励，是让你直接晋升为真传弟子！”真枯半圣继续说道。
<br/>　　苏莫闻言眼眸一亮，这真是好事不断啊！
<br/>　　虽然他不在乎真传弟子的地位，但是真传弟子，就有资格进出太阴族，这是他之前的目标，现在如此轻易的就达到了！
<br/>　　要知道，在道宗之内，就算是武尊境的弟子，都不一定能成为真传弟子。
<br/>　　因为真传弟子不仅名额有限，还有年龄的限制，庞大的道宗武尊境的弟子也有很多，但真传弟子只设区区四十八个名额。
<br/>　　而现在，他只是武帝境八重的修为，居然直接破例成为了真传弟子。
<br/>　　成了真传弟子之后，他以后去太阴族就方便了。
<br/>　　一般的弟子，若不是在特定的情况下，有宗门高层带领，是没有资格进入太阴族的。
<br/>　　就比如此次会武，一般的弟子才能有进入太阴族的机会。
<br/>　　但是成为了道宗真传弟子，就能随意出入太阴城了。
<br/>　　苏莫心中暗喜，能出入太阴族，虽然只是外族的太阴城，对于他的计划也是有很大的帮助。
