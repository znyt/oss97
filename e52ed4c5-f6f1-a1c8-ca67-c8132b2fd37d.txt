　　噗通！
<br/>　　叶尘这掌没有留手，直接将叶家的这位少爷当场击毙，甚至连惨叫都没有来得及发出，就重重栽倒在了地上。
<br/>　　“啊！！”
<br/>　　另外那三名随从看到这一幕，顿时吓得尿都快要出来了，一个个连忙拔腿就想要往外面逃窜。
<br/>　　“死吧！”
<br/>　　叶尘自然不会让这些人前去通风报信，直接身体一晃，便拦住了三人的去路，再次施展出杀招。
<br/>　　三人的修为虽然要超过叶尘不少，可是本来战斗水平就太差，此时又是惊恐之下，根本不堪一击，被叶尘轻松秒杀。
<br/>　　叶尘一连杀了四人，知道此时一旦被叶家得知，肯定在劫难逃，当即又身体一晃，冲到小兰的跟前。
<br/>　　小兰此时早已经吓傻了，结结巴巴，道：
<br/>　　“少爷，你、你怎么可以……”
<br/>　　叶尘一把抓住小兰的手腕，直接就朝外面走去，
<br/>　　“无妨，我们这就离开叶家！”
<br/>　　带着小兰离开房间的时候，叶尘心念一动，屈指将房间正在燃烧的烛台打翻，
<br/>　　呼！
<br/>　　整个房间顿时便开始燃烧起来，火势越烧越大。
<br/>　　等到火焰蔓延到整个屋子，演变变成汹汹大火，引起了叶家的注意，叶尘早已经带着小兰，翻墙离开了叶家。
<br/>　　……

<br/>　　一处幽静偏僻的山谷之中，
<br/>　　叶尘盘腿坐在一块岩石之上，两手交错，双眼闭合，似乎正处于修炼状态。

<br/>　　好在，叶家一开始，并没有将他们两人放在眼里，所以派的都是一些实力不高的下人，再加上叶尘的战斗经验何其丰富，虽然受了几次伤，但最终还是带着小兰，都挺了过来。
<br/>　　而且，经过三个月的磨炼，叶尘的修为也突飞猛进，如今已经是入窍巅峰境界。
<br/>　　就连小兰，在叶尘的指点之下，修为也达到了胎动后期。
<br/>　　“少爷，你真的还是以前的少爷吗？”
<br/>　　不远处，小兰正在为叶尘护法，感受到叶尘身上磅礴的气息，不由得低声喃喃道。
<br/>　　这三个月以来的逃亡生涯，她的这位叶枫少爷，一次次惊艳的表现，给了她太多的冲击！
<br/>　　原本，每一次叶家来人，她都以为他们必死无疑，结果叶家派来的那些高手，一个个全都死在了‘叶枫’的手上。
<br/>　　要知道，她以前的那位叶枫少爷，不但修为弱的可怜，而且生性懦弱胆小，和眼前的‘叶枫’相比，简直完全就是两个人！
<br/>　　轰隆隆！
<br/>　　就在小兰正在思绪纷飞之时，空间忽然一阵巨震！
<br/>　　呼！
<br/>　　伴随着空间一阵晃动之后，叶尘的修为突然暴涨，气息也变得越发凝实。
<br/>　　霍！
<br/>　　叶尘猛然张开双眼，眼眸中闪过一抹喜色，
<br/>　　“总算达到丹成境界了！”
<br/>　　在这个世界，完全抽离了其他法则的存在，唯独只有造化法则，而叶尘在造化法则的领悟上，本就一窍不通，若非这三个月来，在生死之间磨炼，让他隐隐领悟到了一丝造化法则的皮毛，以他这具身体的资质，想要修炼到丹成境界，恐怕至少也得数年时间。
<br/>　　“看来想要彻底参悟造化法则，我需要多经历一些生死之战，只有在生死之间徘徊之时，我对造化法则的领悟，才会更加的明了……”
<br/>　　叶尘低声喃喃了一阵之后，忽然嘴角微微一扬，自言自语道：
<br/>　　“还真是想什么就来什么呀！”
<br/>　　他这话音才刚刚落下，
<br/>　　唰唰唰！
<br/>　　远处九天之上，忽然几道人影闪过，转眼之间，便已经来到山谷上方。

<br/>　　“叶枫小畜生！你给老子纳命来吧！！”
<br/>　　这道声音，赫然是出自一名，气势极强的老者之口。
<br/>　　而一旁的小兰看到此人，顿时俏脸大变，继而满脸焦急之色，道：
<br/>　　“少爷，没想到连叶家的家主，竟然都亲自来了，据说他可是破神境的强者！还有三大长老也来了！这万万不是我们可以抗拒的，你不要管我了，赶紧逃命去吧！！”
<br/>　　小兰急得不行，在她看来，这位叶枫少爷虽然已经今非昔比，可是和叶家家主这样的大人物相比，还是要差上十万八千里，再加上叶家其他高手，根本不是他们两个可以抗拒的！

<br/>　　“还想跑？你觉得你们还跑得了吗？”

<br/>　　“小畜生，不得不说，之前老夫还真是看走了眼！”
<br/>　　“短短三个月，从一个几乎没有修为的废物，修炼到丹成境界，而且还一连斩杀了我叶家几十名高手！”
<br/>　　“如果老夫没有猜错的话，你的手上，肯定掌握着叶家嫡系的某种秘术功法吧？否则的话，你的实力不可能提升这么快！”

<br/>　　“好！我可以答应你！”

<br/>　　小兰眼眶通红，还想拒绝，
<br/>　　叶尘故意脸色一板，
<br/>　　“如果你还当我是少爷，就听我的命令，赶紧走！走的越远越好！”
