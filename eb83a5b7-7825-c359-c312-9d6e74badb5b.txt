　　“楚风眠，司空家的人，这不好对付，他们跟沧澜世家，风家的人，都关系极为密切，你如今镇压了这司空景胜，也等于得罪了他们。”
<br/>　　冰漓长老犹豫了一下开口道。
<br/>　　“现在放了他们，还可以留个善终。”
<br/>　　“善终？”

<br/>　　楚风眠冷笑连连的开口道。
<br/>　　之前在颜古龙帝洞府的时候，楚风眠就跟这两大世家的人交手过来，他们要是敢来找麻烦，楚风眠一样会让他们死。
<br/>　　这司空景胜，司空弘二人，如今念在冰漓长老的面子上，楚风眠可以不杀了他们，不过死罪难免，活罪难逃，就让他们跪在这里，跪上七天七夜。
<br/>　　对司空景胜，司空弘这样的武道世家出身的人来说，这种下跪，简直就是一种侮辱，他们出身极高，一个个极为高傲。
<br/>　　如今楚风眠就这样的粉碎了他们的骄傲，也就意味着在他们的心中，种下了一个心魔，如今这二人，除非是破开了自己的心魔，不然的话便是就会永远困在其中，实力一生都无法再进一步。
<br/>　　听到楚风眠狂妄的话，冰漓长老本来还想说些什么。
<br/>　　可是当她看到地上跪着的司空景胜跟司空弘二人时，却是一句话都说不出来了。
<br/>　　楚风眠的话，并非狂妄，而是楚风眠真正有着说到做到的能力。
<br/>　　沧澜世家，风家，跟司空家相比，实力都差不多，如今司空家的老祖都被楚风眠击败镇压，这沧澜世家跟风家出手对付楚风眠，只怕是会沦落到一样的结果。
<br/>　　冰漓长老，虽然跟司空景胜有些交情，但是如今也不好在说些什么。
<br/>　　楚风眠现在饶他们二人一命，实际上已经是在给冰漓长老几分颜面了，不然的话，楚风眠直接便是足以轰杀了他们。
<br/>　　以楚风眠的性格，杀了他们，也并非没有可能。
<br/>　　“这两个废物，就让他们跪在这里，好好反省反省吧。”
<br/>　　楚风眠平静的开口道，突然楚风眠话锋一转，向着冰漓长老开口道。
<br/>　　“冰漓长老，可否知道归墟天，武斗大会的事？”
<br/>　　楚风眠突然开口询问道。
<br/>　　在玄北圣者哪里，楚风眠也是大概听闻了一下，便是离开了，因为玄北圣者要来帮楚风眠争取学院方面的机会，所以也没有多聊。
<br/>　　这武斗大会，楚风眠还是极为看重的，他现在想要突破，必须要在去寻常新的奇遇才行。
<br/>　　这归墟天，正是远古宗门遗址，其中拥有这无数奇遇，楚风眠必须要进入其中，寻找奇遇，就可以就此一举突破神力境极限，度过生死大劫踏上生死境。
<br/>　　等到楚风眠踏上生死境，像是什么司空家，沧澜世家，风家这样的武道家族，楚风眠随意就足以将他们捏死。

<br/>　　楚风眠还是打算要了解其中一二。

<br/>　　冰漓长老平静的开口道。
<br/>　　对于楚风眠要去参加，他倒是没有多少稀奇的，毕竟这归墟天，只要是实力足够争取的武者，谁不想进入其中，寻找奇遇。
<br/>　　以现在楚风眠的实力，倒是的确有着争夺的资格，楚风眠想要去参加，也是一件极为正常的事情。
<br/>　　“这一次的武斗大会，依然是在武王城举行，不过只有在武斗大会上，取得前十名次的武者，才有资格去归墟天。”
<br/>　　冰漓长老开口解释起来。
<br/>　　“这武斗大会，只要是北域之中各大势力的弟子，都可以前来参加，并不局限于神力境武者，许多生死境武者，也都可以参加。”

<br/>　　“北域之中的各大势力，不乏天才，几乎所有北域百绝榜上的弟子，都会前去参加。”
<br/>　　“以你现在的实力，争取进去前十，并不算困难，不过却是想要争夺前三的话，就有些难度了，至于夺取头名，以你现在的实力，应该还无法做到。”
