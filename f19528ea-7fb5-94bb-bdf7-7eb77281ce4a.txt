　　吞灵冰鲸皇将萧逸三人轰飞后。
<br/>　　并未再次发起攻击，而是立刻继续下潜逃离。
<br/>　　它的灵智虽然高于普通吞灵冰鱼，乃至吞灵冰鲸。
<br/>　　但它们有着共同的特点，那就是敏感。
<br/>　　它只是被海面精纯而浓郁的寒冰属性气息吸引上去的。
<br/>　　遭受攻击，第一反应还是逃回冰海深处。
<br/>　　而萧逸三人，自然不肯罢手。
<br/>　　三人稳下身子后，再次速度飙升到极致，瞬间拦下打算逃跑的吞灵冰鲸皇。
<br/>　　“寒冰覆海斩。”
<br/>　　“暗冰武破掌。”
<br/>　　“冰炎蚀心。”
<br/>　　三道狂猛攻击，再次将它击伤。
<br/>　　吞灵冰鲸皇，确实强，它发出的攻击，恐怕连段云那个层次的武者都吃不了多少下。
<br/>　　但，它却有着一个致命的弱点。
<br/>　　那便是身躯极其脆弱。
<br/>　　它并没有能与七级巅峰妖兽媲美的强悍妖兽肉体。
<br/>　　它的身躯，只是一层精纯灵气，根本没有多少防御性可言。
<br/>　　它的攻击强大，只不过是来源于它拥有极其庞大的身躯，和以之施展攻击的浑身精纯灵气。
<br/>　　此时，它的庞大身躯，被三道攻击所伤。
<br/>　　大片精纯灵气，如汩汩鲜血，从它身躯中流出。
<br/>　　萧逸三人，已不打算再冰封这些灵气。
<br/>　　而是边打伤这头吞灵冰鲸皇，边降低它的实力。
<br/>　　虽说承受它的攻击有些恐怖，也会受伤。
<br/>　　但萧逸三人，几乎有着绝对的把握，在彻底重伤前，彻底耗死这头吞灵冰鲸皇。
<br/>　　将它体内的精纯灵气消耗殆尽。
<br/>　　这才是三人敢找它麻烦的原因。
<br/>　　“呼…好精纯的力量。”
<br/>　　吞灵冰鲸皇刚才外泄的一丝灵气，飘散于周遭，被萧逸三人所吸收。
<br/>　　三人，顿时双眼发亮。
<br/>　　那种精纯度，绝非吞灵冰鲸王能比拟的。
<br/>　　这绝对是更高层次的力量。
<br/>　　“吼。”吞灵冰鲸皇，怒吼一声。
<br/>　　又是三道恐怖的巨大水流打向萧逸三人。
<br/>　　随后，继续下潜逃离。
<br/>　　嘭…嘭…嘭…
<br/>　　三人再次被吐血击飞。
<br/>　　随后，又再次追击，再次拦下吞灵冰鲸皇。
<br/>　　又是三道强悍攻击，打在吞灵冰鲸皇身上，让它外泄出大量精纯灵气。
<br/>　　相应的，萧逸三人，再次被击飞。
<br/>　　情况，在这毫无变化地循环中进行着。
<br/>　　追上、拦下、打伤、被轰飞；再追上，再拦下，再打伤，再被轰飞。
<br/>　　三人，早已拿出了自己的武器。
<br/>　　萧逸的武器，是寒霜剑，自不必多说。
<br/>　　而沐冰的武器，是一根颜色锃白的冰属性长棍。
<br/>　　韩苍的武器，是一双手套。
<br/>　　三人的武器，皆是中品元器。
<br/>　　沐冰与韩苍，本只是天元一重修为。
<br/>　　但在元器和自身秘法的增幅下，以及一身手段。
<br/>　　现今爆发出来的战力，达到了巅峰层次。
<br/>　　大概在木青云那个水准。
<br/>　　而其中最大的增幅，来源于他们的秘法。
<br/>　　不过，萧逸并未看出来是何种秘法。
<br/>　　只知，这种秘法，有些类似于‘吸元掌’。
<br/>　　他们乃是修习寒冰一道的武者。
<br/>　　在这极寒冰海深处，无比的冰冷之下，反倒成了他们最好的战斗地方。
<br/>　　这也是他们如今战力大增的原因。
<br/>　　言归正传。
<br/>　　三人不断追击，承受着的，是受伤的代价。
<br/>　　半晌。
<br/>　　当吞灵冰鲸皇下潜的深度，达到近万米之时。
<br/>　　三人已经彻底重伤。
<br/>　　萧逸的伤势是最轻的，毕竟他的实力本就比两人强。
<br/>　　且有寒冰铠甲护身。
<br/>　　周遭的环境，早已是漆黑一片，冰冷至极。
<br/>　　这里，早已是极光，包括阳光所无法照耀的地方。
<br/>　　普通的深海，在这个深度，已会因常年无法受阳光照射，而导致海水冰冷无比。
<br/>　　更别说这里是极寒冰海。
<br/>　　周遭的冰冷，已经无法用言语去形容。
<br/>　　若非三人一身修为深厚，怕是早已被冻死。
<br/>　　而哪怕如此，周遭奇黑无比，如同置身冰冷深渊的感觉，还是让人有些发怵。
<br/>　　没人知道极寒冰海到底有多深。
<br/>　　更没人知道极寒冰海最深处，到底存在着什么。
<br/>　　这本就辽阔无边的极寒冰海，乃是险地之一。
<br/>　　一股未知的危险意味，忽然涌上三人心头。
<br/>　　不过，萧逸向来胆大包天，这种危险意味，并未能让他退缩。
<br/>　　咬了咬牙，继续追击。
<br/>　　而韩苍和沐冰，在这更加恐怖的冰海深处寒冷下，似乎使得他们的秘法，威力更甚。
<br/>　　战力，也因此上升了不少。
<br/>　　勉强达到了血成河那个层次。
<br/>　　战力增长下，这二人更加不会退缩。
<br/>　　三人再次追击。
<br/>　　虽然周遭奇黑无比，但武者拥有着过人的目力，还是能勉强看到周遭情况。
<br/>　　另外，武者拥有着感知，只需外放，即可知晓方圆百米内的情况。
<br/>　　嘭…嘭…嘭…
<br/>　　三人一次次将吞灵冰鲸皇打伤，自身也在承受着一次次攻击。
<br/>　　不知过去了多久。
<br/>　　三人甚至无法确定已经下潜多少米了。
<br/>　　只知，绝对远远超过了万米。
<br/>　　而此时，当三人再一次被恐怖海流击飞后。
<br/>　　韩苍和沐冰，已猛地吐出一大口腥血。
<br/>　　腥血在这漆黑海水中，凭空增添了一股莫名的氛围。
<br/>　　“该死，撑不住了。”韩苍咬了咬牙，身上的气息，已在不由自主地飞速下降。
<br/>　　沐冰也同样如此。
<br/>　　显然，他们的秘法不能维持太久，只能散去。
<br/>　　同时，在冰海的这个深度，虽然愈发冰冷，增幅着他们的战力。
<br/>　　却也水压的压迫力愈发恐怖，让他们的身体难以承受。
<br/>　　最重要的是，身旁，已不时出现一些奇怪的寒冰妖兽。
<br/>　　虽实力不太强，大概只在地元九重左右。
<br/>　　但天知道待会儿再继续下去，伤势会达到何种程度，自身又还能坚持多久。
<br/>　　若继续贸然追击，一旦伤势过重，怕是没命回到海面。
<br/>　　同时，逃跑中的吞灵冰鲸皇，其庞大的身躯，虽因为不断被击伤，而蕴含的力量外泄。
<br/>　　导致身躯微微变小。
<br/>　　但这种变小，对比他那庞大身躯，几乎微不足道，怕是不足百分之一。
<br/>　　原本打算不停让它体积缩小，实力减弱，最终一举擒获。
<br/>　　现在看来，这种念头，不得不打消了。
<br/>　　再追下去，很可能它还未变得足够弱，自身就率先承受不住了。
<br/>　　韩苍与沐冰二人，已经停下了追击的步伐。
<br/>　　只有萧逸，一直未曾放弃，还在不断追击。
<br/>　　“那个疯子。”两人低喝一声。
<br/>　　“算了，他自己找死便随他去吧。”韩苍冷笑一声。
<br/>　　“吞灵冰鲸皇虽珍稀无比，我等得到，将修为剧增。”
<br/>　　“可性命没了，一切都是空谈。”
<br/>　　沐冰不语，但点了点头。
<br/>　　萧逸一直在追击着。
<br/>　　嗖的一声，速度忽然飙升到极致，再次拦下了吞灵冰鲸皇。
<br/>　　他一直不放弃，自然有他的原因。
<br/>　　其实，早在吞灵冰鲸皇出现之时，体内冰鸾剑，就已经躁动不已。
<br/>　　剑身轻吟，透漏着欢呼的意味。
<br/>　　能让冰鸾剑如此大反应之物，已经能证明此物蕴含的能量层次，绝对在极高的范畴。
<br/>　　萧逸怎可能放弃。
<br/>　　但，若是继续下潜，先不说这未知的冰海深渊存在着什么。
<br/>　　单单海水的压力，就会逐渐让他承受不住。
<br/>　　不能再拖下去了，萧逸咬了咬牙，心头发狠。
<br/>　　身影一闪，跳到了吞灵冰鲸皇巨大的身躯上。
<br/>　　手掌狠狠按下，打算强行吸收。
<br/>　　如今吞灵冰鲸皇实力未降到可以强行擒获的地步。
<br/>　　萧逸这般强行吸收，后果难料，能否成功也未可知。
<br/>　　但他只能试试。
<br/>　　然，几乎在他双手触碰到吞灵冰鲸皇的瞬间。
<br/>　　嘭的一声，仿佛气球破散的声音。
<br/>　　整头巨大的吞灵冰鲸皇，凭空消失。
<br/>　　“这…”萧逸一愣。
<br/>　　随即，感受到体内一阵躁动。
<br/>　　内视一番。
<br/>　　小世界处，竟已紫芒涌动。
<br/>　　一股澎湃而精纯无比的力量，已存在于小世界中。
<br/>　　毫无疑问，整头吞灵冰鲸皇蕴含的精纯力量，是被冰鸾剑瞬间吸收了。
<br/>　　对于冰鸾剑的恐怖吸收速度，萧逸不是第一次见识了。
<br/>　　不过，让他惊喜的是。
<br/>　　此刻，气泉中的气态元力，已化作了液态。
<br/>　　这代表着，现在的他，只需一个念头，即可突破天元境。
<br/>　　萧逸满意一笑，脚步一动，打算离开。
<br/>　　但，恰在此时，一股莫名的阴寒，笼罩了全身。
<br/>　　萧逸瞬间打了个冷颤。
<br/>　　循着心头感觉，微微转过头。
<br/>　　一瞬间，萧逸瞳孔一缩，头皮发麻。
<br/>　　在这极寒冰海远超万米的深度之下，冰冷至极、漆黑如深渊的环境下。
<br/>　　在他不远处，出现了一只巨大的眼睛。
<br/>　　眼睛，散发着幽光；散发着强大而邪恶冰冷的气息。
<br/>　　在它的注视下，自己竟动弹不得，浑身血液结冰。
<br/>　　一种无尽的恐慌感，涌上心头，让萧逸不知所措。
<br/>　　那是一种仿佛比死亡更能令人畏惧的感觉。
<br/>　　“动。”萧逸心头低喝一声，他发现，自己的拳头，早已颤抖着紧握。
<br/>　　但，萧逸终究是萧逸，一下子摆脱了心头恐慌。
<br/>　　下一秒，脚步一动，瞬间上游。
<br/>　　待得上升了近千米高度时。
<br/>　　心头的那种恐慌，才算消散。
<br/>　　萧逸微微松了口气，边上游，边心有余悸地思索着。
<br/>　　“怎么回事？那是错觉吗？”
<br/>　　他的目光，看向下方，那里，漆黑一片，没有任何东西。
<br/>　　但，这种漆黑而冰冷的感觉，始终让他心头有些不舒服。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的速度，飙升到极致。
<br/>　　不多时，嘭的一声，身影破冰而去，回到了海面之上。
<br/>　　现在的他，处于地元巅峰与天元一重的临界点。
<br/>　　一个念头，即可突破。
<br/>　　只不过，突破的气息，暂时被他压下。
<br/>　　现在，显然不是适合突破的时候。
<br/>　　周遭，九大冰宫长老，韩苍、沐冰等人，已经围了上来。
