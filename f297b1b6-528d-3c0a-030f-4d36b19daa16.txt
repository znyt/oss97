　　在楚风眠把身躯之中的战龙血脉全部催动起来，浑厚的龙威从楚风眠的身上笼罩而下。
<br/>　　在楚风眠的背后，天涯剑匣也显现出来，剑意冲天而起，他的身躯，十二万九千六百枚组成的血肉晶体，也紧紧凝聚在了一起。
<br/>　　楚风眠还是楚风眠。
<br/>　　样貌没有任何的变化，但是如今楚风眠身上的气势已经是跟之前，截然不同，这才是楚风眠的真正实力。
<br/>　　“战龙血脉？不，这其中还蕴含有着始祖天龙血脉，你有着先天神兽的血脉？”
<br/>　　看到楚风眠身上的气息，陈少主愣了一下，随即脸上大喜。
<br/>　　“很好，没想到在九域之中还有人有着如此大机缘！”
<br/>　　“始祖天龙血脉，我吞噬了，就可以成就双先天神兽血脉！一起上，诛杀此人！”
<br/>　　陈少主一声命令之下，陈家在场的古帝，一同出手了，狂暴的灵力爆发而来，数十道杀招向着楚风眠纷纷轰击而去。
<br/>　　陈家在场古帝之中，最先出手的，却是两名八劫古帝，他们二人一出手，都是凝聚雄厚的天地之力，化为了两道神芒，从天笼罩下来。
<br/>　　“我们来对付他们二人！”
<br/>　　嗖！
<br/>　　在这两名八劫古帝出手的一刻，站在楚风眠身后的寒月漓跟洛汐二人，也是突然一步上前。
<br/>　　同时他们二人身上的力量也当即运转起来，雄厚的天地之力散发而出。
<br/>　　“八劫古帝！”
<br/>　　“这二人也是八劫古帝？”
<br/>　　陈家一众古帝的脸色都有些阴沉。
<br/>　　八劫古帝，可不是大白菜，整个陈家的八劫古帝，都是一只手数的过来。
<br/>　　现在不止是楚风眠一人有着八劫古帝的时候，跟随在她身后的寒月漓跟洛汐，居然也是八劫古帝。
<br/>　　尤其是寒月漓，他们可是曾经跟寒月漓交手过的，将寒月漓击伤，甚至是陈少主更是将一枚吞天之力，打入到了寒月漓的身躯之中。
<br/>　　按照他们的想法，就算是寒月漓化解了吞天之力，也要浪费很多的经历才对，实力不退就不错了。
<br/>　　可现在寒月漓的实力，明显提升了无数倍，境界更是踏入到了八劫古帝，这远超了他们的意料之中。
<br/>　　轰隆。
<br/>　　没等他们二人反应过来，寒月漓已经是率先出手了，冰蓝长剑在她的手心之中凝聚。
<br/>　　剑光一闪，漫天剑影斩杀而去，那两名陈家的八劫古帝，也无暇在估计楚风眠，顿时迎面跟寒月漓，洛汐，战在了一起。
<br/>　　“化解了吞天之力？”
<br/>　　陈少主的脸上都露出了几分惊讶，打量了楚风眠一眼。
<br/>　　“哦？你的身上，居然也有着吞天之力？”
<br/>　　吞天之力，是属于吞天兽的力量，就算是其他先天神兽的力量，都难以化解吞天之力。
<br/>　　能够无声无息之间，化解了吞天之力的，唯有同样是吞天之力本身。
<br/>　　这陈少主看了楚风眠一眼，也察觉到了这一点。
<br/>　　“天助我也，我在阎罗星域，寻找百年，都未曾找到其他传承人，没想到居然在这九域之中被碰到了！”
<br/>　　陈少主的眼神之中露出了几分炙热之色。
<br/>　　他也曾经得到过吞天兽的传承，掌控吞天秘术，拥有吞天兽血脉。
<br/>　　如果杀了楚风眠，得到了楚风眠的血脉之力，那他身躯之中的吞天兽血脉，就会晋升，到时候足以得到更加强大的吞天之力。
<br/>　　吞天兽，可是先天神兽，像是楚风眠掌控的这部分力量，不足千万分之一，唯有得到更多的血脉，才可以掌控更强的力量。
<br/>　　“吞天秘术！吞噬天地！”
<br/>　　刹那之间这陈少主也动了，他身上的气息猛然爆发而出，天地之力在这陈少主的身上涌现而出。
<br/>　　就在楚风眠的身边，化为了一道巨大无比的漩涡，这漩涡深邃无比，看不见尽头，庞大的力量将朝着楚风眠拉扯而去。
<br/>　　“八劫古帝巅峰！”
<br/>　　这陈少主看起来无比年轻，但是一身境界，已经是达到了八劫古帝巅峰。
<br/>　　在他的身躯之上，足足是有着两千道天地法则！
<br/>　　比起楚风眠身上的天地法则，多上两倍有余。
<br/>　　在催动这吞天秘术的一刻，陈少主身躯之中的天地法则都给融入其中，这一道漩涡的出现，真的有着吞噬天地之能。
<br/>　　“好玄妙的手段！”
<br/>　　楚风眠目光闪烁，看到这陈少主催动吞天秘术的一刻。
<br/>　　楚风眠就看的出来，这陈少主对于吞天之力的掌控，要远超楚风眠。
<br/>　　楚风眠也有着吞天之力，甚至是得到了吞天兽心脏，但是这并非是楚风眠力量的根本，楚风眠最强的，乃是他的剑术。
<br/>　　至于吞天之力，多半是作为辅佐的手段。
<br/>　　而这陈少主，却是一生苦修吞天秘术！施展出来的，自然要比起楚风眠施展的玄妙的多。
<br/>　　对于吞天秘术，楚风眠可不敢掉以轻心，这可是先天神兽的力量！
<br/>　　以这陈少主的手段，只要是楚风眠被拉扯到了吞天秘术之中，便是再也没有挣扎离开的可能。

<br/>　　嗖！
<br/>　　楚风眠的背后，天涯剑匣当即打开，两把灵剑飞了出来，于此同时，还有着数道剑气，同时飞出。

<br/>　　陈少主不屑的说道。

<br/>　　“这吞天兽血脉，落入到你的手中，也算是蒙尘了，还是交给我吧，未来我定然会成为跟吞天兽一样伟大的存在，位于诸天之上！”
<br/>　　“吞噬！”
<br/>　　陈少主长啸一声，两千道天地法则的力量，融入到了吞天秘术之中，顿时令吞天秘术的吞噬之力，增强数倍。

<br/>　　这每一道铁索，都是以吞天之力所化，交织在一起，咔嚓咔嚓的作响，化为了一道封锁大阵，封锁这整个宫殿之中。
