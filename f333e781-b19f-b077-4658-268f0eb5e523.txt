<br/>李振邦在街上随意的游走着，在一个个的摊位前驻足，好奇的看着商贩们卖的都是什么东西。<br/>　　<br/>　　四处溜达的李振邦突然发现一个摊位竟然卖的是饺子，而且还是现吃现包的。摊位上摆放着几个大盆，盆里面都是各种馅料，有纯肉的，有纯菜的，还有肉菜混合的，李振邦不由得食指大动。<br/>　　<br/>　　李振邦点了一盘纯肉的，一盘肉菜混合的。老板娘娴熟的抱着饺子，老板将水烧开，把包好的饺子下锅。不一会儿的功夫，两盘热气腾腾的饺子就端了上来，同时端上来的还有一大碗饺子汤。<br/>　　<br/>　　李振邦哪里还顾得上饺子烫不烫，直接狼吞虎咽的吃了起来。<br/>　　<br/>　　“娃子，慢点吃，小心烫。”看起来有四十多岁的老板娘笑眯眯的提醒道。<br/>　　<br/>　　“没关系，我已经好久没有吃过饺子了。”李振邦嘴里塞着几个饺子，呜呜噜噜的说道。<br/>　　<br/>　　老板娘看到李振邦的吃相，不由得笑了起来。<br/>　　<br/>　　“你和俺家狗娃小时候的吃相太像了，每次都是嘴里面塞着几个饺子，还总是叫着让俺再多煮一些。”<br/>　　<br/>　　“大娘，狗娃是您儿子吗？”李振邦咽下了嘴里的饺子，嘿嘿笑着问道。<br/>　　<br/>　　“是啊！狗娃是俺儿子，可孝顺了，就是现在很少回家了，总是在外面忙着挣钱。好不容易回来一次，待不上两天，就又走了。”老板娘叹了一口气，眼神中充满了思念。<br/>　　<br/>　　“老婆子，又想狗娃了？俺天天算着日子呢！最多不超过一个星期，狗娃就该回来了。”头发有些发白的老板笑呵呵的说道。<br/>　　<br/>　　“老头子，你说这回狗娃回来，咱们给他整点儿啥吃啊！”老板娘有些急迫的问道。<br/>　　<br/>　　“还能整啥？他从小到大就爱吃饺子。”<br/>　　<br/>　　“那也不能每次都整饺子啊！”<br/>　　<br/>　　“那咋了？咱们家的饺子，就算他跑遍整个大陆也找不到第二家味道这么好的。”老板撇了撇嘴，一脸傲气的说着。<br/>　　<br/>　　不过紧接着老板兴奋的说道：“不过你说的也对，饺子得整，但是也得整点儿别的，也不知道那小兔崽子现在喜欢吃啥，是不是还喜欢吃那一口儿。”<br/>　　<br/>　　虽然还有一个星期左右老板和老板娘的儿子才能回来，但是他们却讨论的热火朝天，仿佛儿子已经回来了一般。看着他们争执的样子，李振邦的眼角不由得有些湿润。<br/>　　<br/>　　“娃子，你咋哭了？做的不好吃啊？”<br/>　　<br/>　　“不是，是太好吃了。”李振邦笑着看着两人，声音有些哽咽。<br/>　　<br/>　　“看你这娃子，好吃哭啥，吓我一跳，还以为你不喜欢吃吃呢！喜欢吃的话以后常来。”老板娘笑着说道。<br/>　　<br/>　　李振邦点了点头，将盘子里最后一个饺子放进了嘴里。<br/>　　<br/>　　“大娘，多少钱？”<br/>　　<br/>　　“两盘饺子，一共七个铜币。”老板娘用围裙擦了擦手，笑着说道。<br/>　　<br/>　　“啥？七个铜币？”李振邦瞪大了眼睛看着老板娘，他身上似乎已经很久没有见过铜币了，每次出去吃饭最差也是银币计算的。<br/>　　<br/>　　“娃子你觉得贵了？你是第一次来吃吧？俺们家的饺子一直都是这个价，绝对没有要谎啊！”<br/>　　<br/>　　“大娘，你误会了。不是贵，是太便宜了，给你钱。”李振邦递过去了一枚银币。<br/>　　<br/>　　“娃子，俺们这是刚开张，没有那么多零钱找你，要不你明天吃饭的时候再给俺们送来吧！”老板娘将银币又递还给了李振邦。<br/>　　<br/>　　李振邦没有接，笑着说道：“大娘，要不这样吧！我还有几个朋友都没有吃饭，你把剩下的钱都煮成饺子，我给他们带回去。”<br/>　　<br/>　　“你几个朋友啊？”老板娘疑惑的问道。<br/>　　<br/>　　“三个人。”<br/>　　<br/>　　“你这娃子，三个人哪能吃的了这么多的饺子啊！钱不能浪费，粮食也不能浪费。”老板娘摇了摇头，劝着李振邦。<br/>　　<br/>　　“没事，大娘，你是不知道，他们都特别能吃，一个人能吃三四个人的饭量，没准这些饺子还不够呢！”李振邦笑着说道。<br/>　　<br/>　　“你这娃子，净胡说，哪有一个人能吃三四个人饭量的，那不得撑死了。”<br/>　　<br/>　　“大娘，你放心，我不会浪费粮食和钱的，你放心的做吧！”<br/>　　<br/>　　老板娘还是疑惑的看着李振邦，不过看他的样子似乎不像说假话，这才又动手包起了饺子……<br/>　　<br/>　　“少爷？你怎么从外面回来的？”宇辰疑惑的看着刚刚从外面走进院子里的李振邦问道。他刚刚起来，准备收拾一下为大家准备早饭，没想到竟然碰到李振邦从外面拎着几个牛皮纸包走了过来。<br/>　　<br/>　　“我起的早了点，出去吃了早饭，给你们带的饺子，趁热吃吧。”李振邦笑着将牛皮纸包交给了宇辰。<br/>　　<br/>　　“对了，淼淼还没有起来吗？”<br/>　　<br/>　　“应该没有吧！我还没有见到她。”<br/>　　<br/>　　李振邦点了点头，走进了别墅。<br/>　　<br/>　　龙淼淼正躺在床上懊恼着，昨天怎么就迷迷糊糊睡着了，还没有让那个讨厌的家伙赔礼道歉呢！竟然嫌弃自己等级低，说的好像他很强一样，找机会一定要暴揍他一顿，让他知道知道到底谁最弱。<br/>　　<br/>　　“咚咚咚。”<br/>　　<br/>　　“谁啊？大早上都不让人清净清净。”龙淼淼没有好气的说道。<br/>　　<br/>　　“淼淼，你醒了啊！我是来叫你吃饭的。”李振邦听到龙淼淼带着怒气的声音讪笑着说道。<br/>　　<br/>　　“不吃，我不修炼成圣级强者，我就不出这个屋，不吃饭了！”<br/>　　<br/>　　龙淼淼听到李振邦的声音，从床上一下子坐了起来，不过很快又躺了下去，嘟着小嘴赌气道。<br/>　　<br/>　　“好了，淼淼。昨天是我错了，你原谅我吧！”李振邦苦笑道。<br/>　　<br/>　　“你错哪了？”<br/>　　<br/>　　“呃……”李振邦满脑门的黑线，这个问题对任何男人来说都是送命题。<br/>　　<br/>　　李振邦还在考虑如何回答的时候，龙淼淼看到了李振邦头上伤口，所有的问题也就都迎刃而解了……<br/>　　<br/>　　吃完早饭，李振邦告别了众人，独自回到了学院。站在宿舍门前，李振邦诧异的发现，宿舍竟然已经开门了，看样子已经有人回来了。<br/>　　<br/>　　李振邦很好奇，到底是谁这么早回来了，急忙推门进去……<br/><br/><br/>
