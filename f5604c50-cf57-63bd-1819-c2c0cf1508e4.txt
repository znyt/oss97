        这一坐，就是一天一夜。
<br/>
<br/>        暮霭沉沉的静夜里，前院里的金合欢花谢了又开，悄悄地绽放在月光溶溶的夜色中。
<br/>
<br/>        一盏橘黄色的小夜灯微染了静谧的卧室，童筱颖侧卧在大床上睡得并不深沉。
<br/>
<br/>        与其说她睡得不深沉，倒不如说她是失眠了而睡不着。
<br/>
<br/>        哪怕她的双眼是闭着的，但她的大脑，却在不断地回放着季思妍的那些话。
<br/>
<br/>        忽然，耳畔传来轻轻地开门声，随后是不急不缓地脚步声。
<br/>
<br/>        童筱颖能感觉到到身边有一种沉下去的压重感。
<br/>
<br/>        她身上的薄被忽然慢慢地往下滑落。
<br/>
<br/>        这隔日，原定要三个月后才会回来的丈夫，却突然在半夜三更的时候回来了，意味着什么？
<br/>
<br/>        童筱颖睁开了双眼，下一秒，令她感到陌生的重量却是非常熟悉的气息就这么猝不及防地覆上了来。
<br/>
<br/>        空气里弥漫着暧昧的荷尔蒙气息……
<br/>
<br/>        “筱颍，我回来了……”
<br/>
<br/>        这声透着磁性又迷人的嗓音，让童筱颖一阵身心紧绷。
<br/>
<br/>        她的纤纤玉手有气无力地抵着傅昼景那精壮的胸膛。
<br/>
<br/>        “小卫，你不是说你要等项目完工后，才会回来吗？”
<br/>
<br/>        童筱颖努力压抑着自己那悲痛的情绪，装作若无其事地询问。
<br/>
<br/>        “嗯——本来是要三个月后才回来的，但我想你了，所以……”傅昼景微微偏头，温柔地吻了吻童筱颖的耳垂，在她耳畔呢喃，“我就坐了我老板的私人直升飞机提前赶回来了。”
<br/>
<br/>        童筱颖觉得耳朵上有点儿痒，脑袋下意识地躲了躲，心痛却不敢明说，还要继续装作毫不知情的样子，陪着笑脸，撒娇：“嗯，小卫最疼我了！”
<br/>
<br/>        “筱颍，今晚我们就生个孩子吧！”傅昼景忽然意味深长地呢喃。
<br/>
<br/>        这个时刻，他提出要跟她生个孩子，是为了什么？
<br/>
<br/>        事实已经显而易见了。
<br/>
<br/>        季思妍带着那个孩子联系上了他，他一定也见过那孩子了，得知了那孩子所有的情况。
<br/>
<br/>        所以，他才在这半夜三更的时候，赶回来，跟她怀孩子。
<br/>
<br/>        血淋淋的现实，就这样残酷地摆在了她童筱颖的面前。
<br/>
<br/>        以前的小卫，不是这样的。
<br/>
<br/>        他每次出差回来的第一件事情，就是给她送上他特意为她带回来的礼物。
<br/>
<br/>        无论礼物是否贵重，但都是他精心为她挑选的，她每一次都心满意足。
<br/>
<br/>        因为双方身体的缘故，需要禁欲，所以，在这之前的每一个两人独处的夜晚，他都只是安安分分，非常有绅士风度地搂着她睡。
<br/>
<br/>        甚至，有个时候，他对她有生理反应了，也只是独自一个人去浴室里冲凉水澡解决。
<br/>
<br/>        而今夜，小卫变了。
<br/>
<br/>        他迫不及待地想跟她要个孩子……
<br/>
<br/>        想到这里，童筱颖猛地把傅昼景的手给推开。
<br/>
<br/>        傅昼景身心一滞，但并未做过多的顾虑，然后整个人直接压到了童筱颖的身上。
<br/>
<br/>        “小卫，不要！”童筱颖忍不住再次推开傅昼景，支支吾吾地接着说道，“我、我还没有做好心理准备！”
<br/>
<br/>        傅昼景抬起头来，神情难测地看了童筱颖一眼。
<br/>
<br/>        随后，他选择了就此作罢，利索地从她身上下来，单手将她揽入怀中，抚摸着她的长发，轻轻地吻了吻她的头顶，温柔地说道：“对不起，是我太心急了。”
<br/>
<br/>        童筱颖故作娇羞地转过身去背对着傅昼景，而实则她差点哽咽出声，过了许久，她才弱弱地问道：“小卫，你爱我吗？”
<br/>
<br/>        这是她，第一次问他这个问题。
<br/>
<br/>        “爱。”傅昼景将童筱颖搂紧，在她耳畔轻轻地呢喃。
<br/>
<br/>        而童筱颖整个人都僵硬地绷直了身子，任由他抱着。
<br/>
<br/>        “有多爱？”
<br/>
<br/>        “死生契阔，与子成说。执子之手，与子偕老。”傅昼景说完后，伸手关了卧室的灯，“睡吧！明早，我就得回项目上去了。”
<br/>
<br/>        童筱颖一动也不动，直至身后那人传来规律的呼吸声。
<br/>
<br/>        她再三在心底确定傅昼景已经睡过去后，她才慢慢转过身来。
<br/>
<br/>        借着从阳台那儿透过来的微弱的光线，她死死地看着睡着的傅昼景的睡颜。
