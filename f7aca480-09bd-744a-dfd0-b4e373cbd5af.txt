　　所有道宗参加会武的弟子，均是咬紧牙关，面色坚定。
<br/>　　所谓佛争一炷香，人争一口气，就是这个道理，既然所有人都不看好他们，那他们就更要拿出实际的战绩，来打所有人的脸。
<br/>　　让所有人都看看，他们道宗的弟子，不比任何势力的弟子差。
<br/>　　“吕罡、林皇、唐古，你们三人切不可大意！”真枯半圣继续道，他目光看向吕罡三人，眸中露出郑重之色，这一次会武，道宗能有什么成绩，就看这三人的表现了。
<br/>　　“真枯长老放心！”
<br/>　　三人郑重的点了点头，随即纷纷深吸一口气，调整自身的状态，他们三人也知道，若是这次会武，他们三人没有得到好成绩的话，道宗会再次一败涂地。
<br/>　　苏莫瞥了真枯半圣一眼，心中暗暗的摇了摇头，他估计吕罡三人想进入前十，希望很小。
<br/>　　那些异族的天才，实力都非常的强啊，而且到目前为止，还有三大势力没来呢！
<br/>　　“苏莫，你看那位，就是黑炎族的炎澜天！”冯光目光望着黑炎族的方向，示意苏莫观看。
<br/>　　苏莫抬头看去，便见在黑炎族的阵营中，炎火的身边，坐着一名消瘦的黑炎族青年，其眼眸紧闭，对周围的声音莫不关心。
<br/>　　黑炎族之人，由于太黑了，面如焦炭，粗略看起来容貌都没有多大的差别。
<br/>　　“还有那两位，分别是啸天狼族的幽狼和闪灵族的灵枫！”冯光又看向啸天狼族和闪灵族的阵营，为苏莫介绍。
<br/>　　他之所以如此殷切，也就是为了和苏莫打好关系，因为他认定后者与真贤道圣关系不一般。
<br/>　　苏莫又转头看向啸天狼族的阵营和闪灵族的阵营，幽狼和墨狼坐在一其，身材同样高大，面容狂野，充满厉色，给人一种狠辣之感。
<br/>　　其身上浓密的毛发居然和其他啸天狼族之人不同，不是黑色，而是银色。
<br/>　　其一身银色的毛发，如同披上了一层银装，极为吸引眼球。
<br/>　　至于另一位闪灵族的灵枫，苏莫则是看不出对方的深浅，此人看起来像是一个十岁左右的孩童，身形消瘦单薄，面容较为青秀。
<br/>　　“冯长老，太阴族内族之人不来吗？”
<br/>　　苏莫不关心这些天才，他只关心青璇是否来此，但现在北方看台空无一人，不见任何太阴族内族之人。
<br/>　　“当然会来，七大势力会武，就是由太阴族操持举办，怎么可能不来？”冯光说道。
<br/>　　“那……你说神女会来吗？”苏莫轻声问道。
<br/>　　“神女？”
<br/>　　冯光闻言一怔，随即打量了苏莫一眼，轻笑了起来，道：“嘿嘿，苏莫，原来你也心仪神女啊！”
<br/>　　“我只是随口一问而已！”苏莫轻声道。
<br/>　　“神女是否会来，这不好说，若是她无事的话，应该会来，毕竟她也是武帝境修为，对于这种武帝境修为顶尖天才会武，肯定不会错过！”
<br/>　　冯光分析道，言罢，他又继续道：“当然，她若是有事，或者在闭关静修，那就不一定了！”
<br/>　　苏莫闻言点了点头，对方言之有理，的确是这样。
<br/>　　他不由得暗叹一声，青璇会不会来此，就要看天意了。
<br/>　　冯光见苏莫面色有些黯然，心中有些惊讶，难道这苏莫很痴迷神女，来此不是为了观看会武，而是想要一睹神女风采。
<br/>　　“这小子，看起来很沉稳，没想到还有这种心思！”
<br/>　　冯光心中暗自腹诽，随即，他略一沉吟，语重心长的道：“苏莫啊，神女那是太阴族道统的继承人，身份崇高无比，你还是不要抱有不切实际的幻想了，努力修炼武道才是正途！”
<br/>　　苏莫闻听此言，脸皮不由的抽了抽，感情对方以为他痴迷神女啊！
<br/>　　“我知道！”苏莫淡淡的点了点头。
<br/>　　“苏莫，道宗有不少美貌的女弟子，你若是觉得孤单了，本长老给你介绍一位容颜姿色都上佳的女弟子，作为双.修道侣，岂不更好！”冯光眸中噙着笑意，轻声说道。
<br/>　　“冯长老，你想多了，我不需要！”苏莫摇了摇头，直接拒绝了对方。
<br/>　　他若是想要找双.修道侣，哪里需要对方帮忙？
<br/>　　再说了，就算要找的话，也要找自己爱的人，不可能随意滥.情。
<br/>　　“嘿嘿，你什么时候需要，随时可以和我说！”冯长老嘿嘿一笑，面上露出猥琐的表情。
<br/>　　苏莫沉默不语，这个冯长老他也看出来了，估计满脑子坏水。
<br/>　　就在两人谈话之间，又有一群人来到了武斗场，这群人约莫一百六、七十人左右，很多人背上都背负着古剑，个个眸光凌厉，身上剑势弥漫。
<br/>　　“是御剑谷的人！”
<br/>　　“御剑谷人好少啊，只有一百多人！”
<br/>　　“哈哈！人少又怎么样？本命剑客的战斗力，可不是盖的，他们还能极大的克制闪灵族！”
<br/>　　见到这群到来之人，整个武斗场再次议论声四起，震耳欲聋。
<br/>　　而御剑谷的人前脚刚到，天虚宗和雷云禽族的人也来了。
<br/>　　这两个势力之人较多，均是不下三百人，浩浩荡荡。
<br/>　　片刻之后，三大势力的人，亦是全部飞到南方看台落座。
<br/>　　七大势力之人已经全部到齐，不足两千人，占据整个南方看台。
<br/>　　整个武斗场一片闹哄哄的场景，气氛火爆不已，各种讨论之声、大笑之声，冲荡九霄，浩荡八方。
<br/>　　苏莫闭上了双眼，屏弃双耳中的声音，静坐不动。
<br/>　　太阴族内族之人都还没有来，会武暂时不会开始，他也懒得关注七大势力的天才。
<br/>　　可以说，他来到此地，完全不是为了会武，如果可以的话，他不会上场参加比斗。
<br/>　　他完全没有和别人比斗的兴趣，此来太阴族，只为一见青璇。
<br/>　　时间在无数人的等待之中，快速流逝，约莫小半个时辰之后，太阴族内族之人，终于来了。
<br/>　　只见武斗场所在的空间入口之处，大量的武者鱼贯而入来，飞向武斗场。
<br/>　　这些人数量不少，但并不是一起到来，而是三三两两，不断的来到此地。
<br/>　　苏莫睁开了眼眸，目光紧紧的望着入口之处，期盼着能看到心中的倩影。
